1. <span data-ttu-id="2e3ca-115">No painel de navegação, selecione o botão de reticências (**...**) ao lado de **Rodapé** e depois **Adicionar módulo**.</span><span class="sxs-lookup"><span data-stu-id="2e3ca-115">In the navigation pane, select the ellipsis button (**...**) next to **Footer**, and then select **Add Module**.</span></span>

1. <span data-ttu-id="2e3ca-117">No painel de navegação, selecione o botão de reticências ao lado de **Categoria de rodapé** e depois **Adicionar módulo**.</span><span class="sxs-lookup"><span data-stu-id="2e3ca-117">In the navigation pane, select the ellipsis button next to **Footer category**, and then select **Add Module**.</span></span>
1. <span data-ttu-id="2e3ca-118">Na caixa de diálogo, selecione **Bloco de texto** e depois **OK**.</span><span class="sxs-lookup"><span data-stu-id="2e3ca-118">In the dialog box, select **Text block**, and then select **OK**.</span></span>
1. <span data-ttu-id="2e3ca-119">No painel de navegação, selecione **Bloco de texto**.</span><span class="sxs-lookup"><span data-stu-id="2e3ca-119">In the navigation pane, select **Text block**.</span></span>
1. <span data-ttu-id="2e3ca-120">No painel de propriedades à direita, no campo **Parágrafo**, adicione sua mensagem de direitos autorais.</span><span class="sxs-lookup"><span data-stu-id="2e3ca-120">In the properties pane on the right, in the **Paragraph** field, add your copyright message.</span></span> <span data-ttu-id="2e3ca-121">Por exemplo, insira **(C) Fabrikam 2019**.</span><span class="sxs-lookup"><span data-stu-id="2e3ca-121">For example, enter **(C) Fabrikam 2019**.</span></span>
1. <span data-ttu-id="2e3ca-122">Selecione **Salvar**, **Concluir a edição** e depois **Publicar**.</span><span class="sxs-lookup"><span data-stu-id="2e3ca-122">Select **Save**, select **Finish editing**, and then select **Publish**.</span></span>
1. <span data-ttu-id="2e3ca-123">Vá para **Modelos**, selecione o modelo e depois **Editar**.</span><span class="sxs-lookup"><span data-stu-id="2e3ca-123">Go to **Templates**, select the template, and then select **Edit**.</span></span>
1. <span data-ttu-id="2e3ca-124">Em **Estrutura de Tópicos da Página**, expanda **Corpo** e depois **Página Padrão**.</span><span class="sxs-lookup"><span data-stu-id="2e3ca-124">Under **Page Outline**, expand **Body**, and then expand **Default Page**.</span></span>
1. <span data-ttu-id="2e3ca-125">Selecione o botão de reticências ao lado de **Slot de rodapé** e depois escolha **Adicionar fragmento**.</span><span class="sxs-lookup"><span data-stu-id="2e3ca-125">Select the ellipsis button next to **Footer Slot**, and then select **Add Fragment**.</span></span>
1. <span data-ttu-id="2e3ca-126">Selecione o fragmento que você criou antes e escolha **Selecionar**.</span><span class="sxs-lookup"><span data-stu-id="2e3ca-126">Select the fragment that you created earlier, and then select **Select**.</span></span>
1. <span data-ttu-id="2e3ca-127">Selecione **Concluir edição** para fazer check-in do modelo e depois selecione **Publicar** para publicá-lo.</span><span class="sxs-lookup"><span data-stu-id="2e3ca-127">Select **Finish editing** to check in the template, and then select **Publish** to publish it.</span></span>

<span data-ttu-id="2e3ca-128">O rodapé que contém o aviso de direitos autorais aparece automaticamente na parte inferior de todas as páginas que usam o modelo selecionado.</span><span class="sxs-lookup"><span data-stu-id="2e3ca-128">The footer that contains the copyright notice automatically appears at the bottom of all pages that use the selected template.</span></span>
