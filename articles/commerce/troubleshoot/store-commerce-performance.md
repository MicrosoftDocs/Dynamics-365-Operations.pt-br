# <a name="troubleshoot-store-commerce-performance-issues"></a>Solucionar problemas de desempenho do Store Commerce

[!include [banner](../includes/banner.md)]
[!include [banner](../includes/preview-banner.md)]

Este artigo explica como solucionar problemas de desempenho no aplicativo do Store Commerce do Microsoft Dynamics 365 Commerce.

- Ao solucionar problemas de desempenho do Store Commerce, evite a entrada remota. Em vez disso, entre diretamente ao dispositivo do Store Commerce.
- Se o aplicativo do Store Commerce for lento, verifique a CPU, a memória, a rede e o uso do disco usando o Gerenciador de Tarefas ou o Monitor de Recursos. Verifique se não há nenhum consumo significativo dos recursos de dispositivo.
- Verifique o status da conectividade de rede e confirme se as solicitações/respostas HTTPS não estão demorando mais do que alguns segundos.
