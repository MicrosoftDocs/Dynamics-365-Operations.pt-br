# <a name="online-store-overview"></a>Visão geral de loja online
[!include [banner](includes/banner.md)]


Este tópico apresenta o conceito de uma loja online e explica como as lojas online são usadas no Microsoft Dynamics 365 Commerce. Ele também fornece um link para obter mais informações sobre armazenamentos online e informações sobre como configurar uma loja online.

Antes de criar seu site no Dynamics 365 Commerce, pelo menos uma loja online deve ser configurada. No Dynamics 365 Commerce, você usa uma loja online para estabelecer produtos, preços, idiomas, métodos de pagamento, modos de entrega, centros de preenchimento e outros aspectos da experiência online que devem estar disponíveis para seus clientes.

Somente uma loja online deve ser configurada para que você possa iniciar o Dynamics 365 Commerce. Porém, um site único do Dynamics 365 Commerce pode fornecer a experiência online para várias lojas online. Por exemplo, se várias lojas online forem configuradas para suportar diferentes regiões geográficas, um único conjunto de páginas poderá ser usado para fornecer as experiências exclusivas definidas por cada loja. Para obter mais informações sobre como configurar um site para oferecer suporte a várias lojas online, consulte [Associar um site online a um canal](associate-site-online-store.md).

Depois que uma loja online for configurada, ela pode ser associada ao site do Dynamics 365 Commerce que servirá como uma vitrine online. Para obter mais informações sobre lojas online e sobre como configurá-las, consulte [Configurar lojas onlines](https://docs.microsoft.com/dynamics365/unified-operations/retail/online-stores).
