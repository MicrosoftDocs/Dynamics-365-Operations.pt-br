## <a name="overview"></a><span data-ttu-id="a50d9-105">Visão Geral</span><span class="sxs-lookup"><span data-stu-id="a50d9-105">Overview</span></span>

<span data-ttu-id="a50d9-106">Os módulos de trilha de navegação são usados para fornecer navegação secundária em páginas do site.</span><span class="sxs-lookup"><span data-stu-id="a50d9-106">Breadcrumb modules are used to provide secondary navigation on site pages.</span></span> <span data-ttu-id="a50d9-107">Em geral, são mostradas na parte superior de uma página, abaixo do cabeçalho.</span><span class="sxs-lookup"><span data-stu-id="a50d9-107">They are typically shown at the top of a page, below the header.</span></span> <span data-ttu-id="a50d9-108">Embora os módulos de trilha de navegação possam ser adicionados a qualquer página, eles costumam ser usados em páginas de detalhes do produto (PDPs), para mostrar a hierarquia de categoria de produtos e fornecer uma maneira rápida de se movimentar em um site.</span><span class="sxs-lookup"><span data-stu-id="a50d9-108">Although breadcrumb modules can be added to any page, they are most often used on product details pages (PDPs), to show the product category hierarchy and provide a quick way to move around a site.</span></span> <span data-ttu-id="a50d9-109">Um módulo de trilha de navegação também pode ser usado para mostrar um link "Voltar para resultados" quando os usuários abrem um PDP de uma página de pesquisa ou de listagem.</span><span class="sxs-lookup"><span data-stu-id="a50d9-109">A breadcrumb module can also be used to show a "Back to results" link when users open a PDP from a search or list page.</span></span> <span data-ttu-id="a50d9-110">Dessa forma, os usuários podem retornar rapidamente à página de listagem filtrada para continuar comprando.</span><span class="sxs-lookup"><span data-stu-id="a50d9-110">In this way, users can quickly return to their filtered list page to continue shopping.</span></span>

<span data-ttu-id="a50d9-111">Nas páginas que têm contexto de categoria de produto, como as páginas PDPs e de categoria, os módulos de trilha de navegação mostram a hierarquia de categoria.</span><span class="sxs-lookup"><span data-stu-id="a50d9-111">On pages that have product category context, such as PDPs and category pages, breadcrumb modules show the category hierarchy.</span></span> <span data-ttu-id="a50d9-112">Em páginas sem contexto de categoria, os módulos de trilha de navegação mostram **&lt;Site raiz&gt; / &lt;Página atual&gt;** por padrão.</span><span class="sxs-lookup"><span data-stu-id="a50d9-112">On pages that don't have category context, breadcrumb modules show **&lt;Site root&gt; / &lt;Current page&gt;** by default.</span></span> <span data-ttu-id="a50d9-113">Os módulos de trilha de navegação também podem ser configurados manualmente em outros tipos de páginas do site para mostrar links para páginas específicas no site.</span><span class="sxs-lookup"><span data-stu-id="a50d9-113">Breadcrumb modules can also be manually configured on other types of site pages to show links to specific pages on the site.</span></span>

## <a name="breadcrumb-module-settings"></a><span data-ttu-id="a50d9-117">Configurações do módulo de trilha de navegação</span><span class="sxs-lookup"><span data-stu-id="a50d9-117">Breadcrumb module settings</span></span>

- <span data-ttu-id="a50d9-120">**Mostrar hierarquia de categoria** – quando esse valor for selecionado, o módulo de trilha de navegação mostrará a hierarquia de categoria completa do produto exibido no PDP.</span><span class="sxs-lookup"><span data-stu-id="a50d9-120">**Show category hierarchy** – When this value is selected, the breadcrumb module will show the full category hierarchy of the product that is viewed on the PDP.</span></span>
- <span data-ttu-id="a50d9-121">**Mostrar voltar para resultados** – quando esse valor for selecionado, o módulo de trilha de navegação mostrará um link "Voltar para resultados" em um PDP se o usuário funcionalidade está disponível quando os usuários navegam em páginas de listagem de categorias, pesquisa, listas e recomendação.</span><span class="sxs-lookup"><span data-stu-id="a50d9-122">This functionality is available when users navigate from category, search, list, and recommendation lists pages.</span></span> <span data-ttu-id="a50d9-123">Para dar suporte a essa funcionalidade, a support this functionality, product collection and search results modules have a property that is named **Allow back to results on PDP**.</span></span> <span data-ttu-id="a50d9-124">Essa propriedade para resultados** for selecionado para a configuração **Tipo de exibição de trilha de navegação no PDP** do módulo de trilha de navegação e **Permitir voltar para resultados no PDP** for selecionado para o módulo de resultados de pesquisa da página de pesquisa, um link "Voltar para resultados" será mostrado quando os usuários navegarem da página de pesquisa para um PDP.</span><span class="sxs-lookup"><span data-stu-id="a50d9-125">For example, when **Show back to results** is selected for the **Breadcrumb display type on PDP** setting of the breadcrumb module, and **Allow back to results on PDP** is selected for the search page search results module, a "Back to results" link will be shown when users navigate from the search page to a PDP.</span></span>
- <span data-ttu-id="a50d9-126">**Mostrar hierarquia de categoria e voltar para resultados** – esse valor é uma combinação dos dois anteriores.</span><span class="sxs-lookup"><span data-stu-id="a50d9-126">**Show category hierarchy and back to results** – This value is a combination of the previous two.</span></span> <span data-ttu-id="a50d9-127">Quando esse valor for selecionado, o módulo de trilha de navegação mostrará a hierarquia de categoria completa e um link "Voltar para resultados" (se estiver configurado) em um PDP.</span><span class="sxs-lookup"><span data-stu-id="a50d9-127">When this value is selected, the breadcrumb module will show both the full category hierarchy and a "Back to results" link (if it's configured) on a PDP.</span></span>
