<span data-ttu-id="f2802-111">Nas páginas que têm contexto de categoria de produto, como as páginas PDPs e de categoria, os módulos de trilha de navegação mostram a hierarquia de categoria.</span><span class="sxs-lookup"><span data-stu-id="f2802-111">On pages that have product category context, such as PDPs and category pages, breadcrumb modules show the category hierarchy.</span></span> <span data-ttu-id="f2802-112">Em páginas sem contexto de categoria, os módulos de trilha de navegação mostram **&lt;Site raiz&gt; / &lt;Página atual&gt;** por padrão.</span><span class="sxs-lookup"><span data-stu-id="f2802-112">On pages that don't have category context, breadcrumb modules show **&lt;Site root&gt; / &lt;Current page&gt;** by default.</span></span> <span data-ttu-id="f2802-113">Os módulos de trilha de navegação também podem ser configurados manualmente em outros tipos de páginas do site para mostrar links para páginas específicas no site.</span><span class="sxs-lookup"><span data-stu-id="f2802-113">Breadcrumb modules can also be manually configured on other types of site pages to show links to specific pages on the site.</span></span>

## <a name="breadcrumb-module-settings"></a><span data-ttu-id="f2802-116">Configurações do módulo de trilha de navegação</span><span class="sxs-lookup"><span data-stu-id="f2802-116">Breadcrumb module settings</span></span>

- <span data-ttu-id="f2802-119">**Mostrar hierarquia de categoria** – quando esse valor for selecionado, o módulo de trilha de navegação mostrará a hierarquia de categoria completa do produto exibido no PDP.</span><span class="sxs-lookup"><span data-stu-id="f2802-119">**Show category hierarchy** – When this value is selected, the breadcrumb module will show the full category hierarchy of the product that is viewed on the PDP.</span></span>
- <span data-ttu-id="f2802-120">**Mostrar voltar para resultados** – quando esse valor for selecionado, o módulo de trilha de navegação mostrará um link "Voltar para resultados" em um PDP se o usuário funcionalidade está disponível quando os usuários navegam em páginas de listagem de categorias, pesquisa, listas e recomendação.</span><span class="sxs-lookup"><span data-stu-id="f2802-121">This functionality is available when users navigate from category, search, list, and recommendation lists pages.</span></span> <span data-ttu-id="f2802-122">Para dar suporte a essa funcionalidade, a coleção de produtos e os módulos de resultados de pesquisa têm uma propriedade denominada **Permitir voltar para resultados em PDP**.</span><span class="sxs-lookup"><span data-stu-id="f2802-122">To support this functionality, product collection and search results modules have a property that is named **Allow back to results on PDP**.</span></span> <span data-ttu-id="f2802-123">Essa propriedade oferece a flexibilidade de definir quais módulos devem dar suporte à funcionalidade de link "Voltar para resultados" no PDP.</span><span class="sxs-lookup"><span data-stu-id="f2802-123">This property gives you the flexibility to define which modules should support the "Back to results" link functionality on the PDP.</span></span> <span data-ttu-id="f2802-124">Por exemplo, quando **Mostrar voltar para resultados** for selecionado para a configuração **Tipo de exibição de trilha de navegação no PDP** do módulo de trilha de navegação e **Permitir voltar para resultados no PDP** for selecionado para o módulo de resultados de pesquisa da página de pesquisa, um link "Voltar para resultados" será mostrado quando os usuários navegarem da página de pesquisa para um PDP.</span><span class="sxs-lookup"><span data-stu-id="f2802-124">For example, when **Show back to results** is selected for the **Breadcrumb display type on PDP** setting of the breadcrumb module, and **Allow back to results on PDP** is selected for the search page search results module, a "Back to results" link will be shown when users navigate from the search page to a PDP.</span></span>
- <span data-ttu-id="f2802-125">**Mostrar hierarquia de categoria e voltar para resultados** – esse valor é uma combinação dos dois anteriores.</span><span class="sxs-lookup"><span data-stu-id="f2802-125">**Show category hierarchy and back to results** – This value is a combination of the previous two.</span></span> <span data-ttu-id="f2802-126">Quando esse valor for selecionado, o módulo de trilha de navegação mostrará a hierarquia de categoria completa e um link "Voltar para resultados" (se estiver configurado) em um PDP.</span><span class="sxs-lookup"><span data-stu-id="f2802-126">When this value is selected, the breadcrumb module will show both the full category hierarchy and a "Back to results" link (if it's configured) on a PDP.</span></span>

1. <span data-ttu-id="f2802-149">Na caixa de diálogo **Adicionar Módulo**, selecione o módulo **Trilha de navegação** e, depois, **OK**.</span><span class="sxs-lookup"><span data-stu-id="f2802-149">In the **Add Module** dialog box, select the **Breadcrumb** module, and then select **OK**.</span></span>
