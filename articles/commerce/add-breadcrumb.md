# <a name="breadcrumb-module"></a><span data-ttu-id="10e84-103">Módulo de trilha de navegação</span><span class="sxs-lookup"><span data-stu-id="10e84-103">Breadcrumb module</span></span>

[!include [banner](includes/banner.md)]

<span data-ttu-id="10e84-104">Este tópico abrange os módulos de trilha de navegação e descreve como adicioná-los às páginas de site no Microsoft Dynamics 365 Commerce.</span><span class="sxs-lookup"><span data-stu-id="10e84-104">This topic covers breadcrumb modules and describes how to add them to site pages in Microsoft Dynamics 365 Commerce.</span></span>

<span data-ttu-id="10e84-105">Os módulos de trilha de navegação são usados para fornecer navegação secundária em páginas do site.</span><span class="sxs-lookup"><span data-stu-id="10e84-105">Breadcrumb modules are used to provide secondary navigation on site pages.</span></span> <span data-ttu-id="10e84-106">Em geral, são mostradas na parte superior de uma página, abaixo do cabeçalho.</span><span class="sxs-lookup"><span data-stu-id="10e84-106">They are typically shown at the top of a page, below the header.</span></span> <span data-ttu-id="10e84-107">Embora os módulos de trilha de navegação possam ser adicionados a qualquer página, eles costumam ser usados em páginas de detalhes do produto (PDPs), para mostrar a hierarquia de categoria de produtos e fornecer uma maneira rápida de se movimentar em um site.</span><span class="sxs-lookup"><span data-stu-id="10e84-107">Although breadcrumb modules can be added to any page, they are most often used on product details pages (PDPs), to show the product category hierarchy and provide a quick way to move around a site.</span></span> <span data-ttu-id="10e84-108">Um módulo de trilha de navegação também pode ser usado para mostrar um link "Voltar para resultados" quando os usuários abrem um PDP de uma página de pesquisa ou de listagem.</span><span class="sxs-lookup"><span data-stu-id="10e84-108">A breadcrumb module can also be used to show a "Back to results" link when users open a PDP from a search or list page.</span></span> <span data-ttu-id="10e84-109">Dessa forma, os usuários podem retornar rapidamente à página de listagem filtrada para continuar comprando.</span><span class="sxs-lookup"><span data-stu-id="10e84-109">In this way, users can quickly return to their filtered list page to continue shopping.</span></span>

<span data-ttu-id="10e84-110">Nas páginas que têm contexto de categoria de produto, como as páginas PDPs e de categoria, os módulos de trilha de navegação mostram a hierarquia de categoria.</span><span class="sxs-lookup"><span data-stu-id="10e84-110">On pages that have product category context, such as PDPs and category pages, breadcrumb modules show the category hierarchy.</span></span> <span data-ttu-id="10e84-111">Em páginas sem contexto de categoria, os módulos de trilha de navegação mostram **&lt;Site raiz&gt; / &lt;Página atual&gt;** por padrão.</span><span class="sxs-lookup"><span data-stu-id="10e84-111">On pages that don't have category context, breadcrumb modules show **&lt;Site root&gt; / &lt;Current page&gt;** by default.</span></span> <span data-ttu-id="10e84-112">Os módulos de trilha de navegação também podem ser configurados manualmente em outros tipos de páginas do site para mostrar links para páginas específicas no site.</span><span class="sxs-lookup"><span data-stu-id="10e84-112">Breadcrumb modules can also be manually configured on other types of site pages to show links to specific pages on the site.</span></span>

## <a name="breadcrumb-module-settings"></a><span data-ttu-id="10e84-116">Configurações do módulo de trilha de navegação</span><span class="sxs-lookup"><span data-stu-id="10e84-116">Breadcrumb module settings</span></span>

<span data-ttu-id="10e84-117">O módulo de trilha de navegação depende da configuração de **Tipo de exibição de trilha de navegação em PDP**, que é definido em **Configurações de site \> Extensões** no assistente para criação de sites.</span><span class="sxs-lookup"><span data-stu-id="10e84-117">The breadcrumb module relies on the **Breadcrumb display type on PDP** setting, which is defined at **Site Settings \> Extensions** in site builder.</span></span> <span data-ttu-id="10e84-118">Esta configuração tem três valores possíveis:</span><span class="sxs-lookup"><span data-stu-id="10e84-118">This setting has three possible values:</span></span>

- <span data-ttu-id="10e84-119">**Mostrar hierarquia de categoria** – quando esse valor for selecionado, o módulo de trilha de navegação mostrará a hierarquia de categoria completa do produto exibido no PDP.</span><span class="sxs-lookup"><span data-stu-id="10e84-119">**Show category hierarchy** – When this value is selected, the breadcrumb module will show the full category hierarchy of the product that is viewed on the PDP.</span></span>
- <span data-ttu-id="10e84-120">**Mostrar voltar para resultados** – quando esse valor for selecionado, o módulo de trilha de navegação mostrará um link "Voltar para resultados" em um PDP se o usuário abrir o PDP a partir de um módulo que permita um link "Voltar para resultados".</span><span class="sxs-lookup"><span data-stu-id="10e84-120">**Show back to results** – When this value is selected, the funcionalidade está disponível quando os usuários navegam em páginas de listagem de categorias, pesquisa, listas e recomendação.</span><span class="sxs-lookup"><span data-stu-id="10e84-121">This functionality is available when users navigate from category, search, list, and recommendation lists pages.</span></span> <span data-ttu-id="10e84-122">Para dar suporte a essa funcionalidade, a coleção de produtos e os módulos de resultados de pesquisa têm uma propriedade denominada **Permitir voltar para resultados em PDP**.</span><span class="sxs-lookup"><span data-stu-id="10e84-122">To support this functionality, product collection and search results modules have a property that is named **Allow back to results on PDP**.</span></span> <span data-ttu-id="10e84-123">Essa propriedade oferece a flexibilidade de definir quais módulos devem dar suporte à funcionalidade de link "Voltar para resultados" no PDP.</span><span class="sxs-lookup"><span data-stu-id="10e84-123">This property gives you the flexibility to define which modules should support the "Back to results" link functionality on the PDP.</span></span> <span data-ttu-id="10e84-124">Por exemplo, quando **Mostrar voltar para resultados** for selecionado para a configuração **Tipo de exibição de trilha de navegação no PDP** do módulo de trilha de navegação e **Permitir voltar para resultados no PDP** for selecionado para o módulo de resultados de pesquisa da página de pesquisa, um link "Voltar para resultados" será mostrado quando os usuários navegarem da página de pesquisa para um PDP.</span><span class="sxs-lookup"><span data-stu-id="10e84-124">For example, when **Show back to results** is selected for the **Breadcrumb display type on PDP** setting of the breadcrumb module, and **Allow back to results on PDP** is selected for the search page search results module, a "Back to results" link will be shown when users navigate from the search page to a PDP.</span></span>
- <span data-ttu-id="10e84-125">**Mostrar hierarquia de categoria e voltar para resultados** – esse valor é uma combinação dos dois anteriores.</span><span class="sxs-lookup"><span data-stu-id="10e84-125">**Show category hierarchy and back to results** – This value is a combination of the previous two.</span></span> <span data-ttu-id="10e84-126">Quando esse valor for selecionado, o módulo de trilha de navegação mostrará a hierarquia de categoria completa e um link "Voltar para resultados" (se estiver configurado) em um PDP.</span><span class="sxs-lookup"><span data-stu-id="10e84-126">When this value is selected, the breadcrumb module will show both the full category hierarchy and a "Back to results" link (if it's configured) on a PDP.</span></span>

> <span data-ttu-id="10e84-127">Essas configurações estão disponíveis na versão 10.0.12 do Dynamics 365 Commerce.</span><span class="sxs-lookup"><span data-stu-id="10e84-127">These settings are available in the Dynamics 365 Commerce 10.0.12 release.</span></span> <span data-ttu-id="10e84-128">Se estiver atualizando de uma versão mais antiga do Dynamics 365 Commerce, você deverá atualizar manualmente o arquivo appsettings.json.</span><span class="sxs-lookup"><span data-stu-id="10e84-128">If you are updating from an older version of Dynamics 365 Commerce, you must manually update the appsettings.json file.</span></span> <span data-ttu-id="10e84-129">Para obter instruções sobre como atualizar o arquivo appsettings.json, consulte [SDK e atualizações da biblioteca de módulos](e-commerce-extensibility/sdk-updates.md#update-the-appsettingsjson-file).</span><span class="sxs-lookup"><span data-stu-id="10e84-129">For instructions on updating the appsettings.json file, see [SDK and module library updates](e-commerce-extensibility/sdk-updates.md#update-the-appsettingsjson-file).</span></span>

## <a name="add-a-breadcrumb-module-to-a-new-page"></a><span data-ttu-id="10e84-142">Adicionar um módulo de trilha de navegação a uma nova página</span><span class="sxs-lookup"><span data-stu-id="10e84-142">Add a breadcrumb module to a new page</span></span>

<span data-ttu-id="10e84-143">Para adicionar um módulo de trilha de navegação a um PDP e definir as propriedades necessárias, siga estas etapas.</span><span class="sxs-lookup"><span data-stu-id="10e84-143">To add a breadcrumb module to a PDP and set the required properties, follow these steps.</span></span>

1. <span data-ttu-id="10e84-144">Vá para **Configurações do Site \> Extensões**. 1. <span data-ttu-id="10e84-146">No slot **Contêiner** que contém o módulo de caixa de compra, selecione as reticências (**...**) e, depois, **Adicionar Módulo**.</span><span class="sxs-lookup"><span data-stu-id="10e84-146">In the **Container** slot that contains the buy box module, select the ellipsis (**...**), and then select **Add Module**.</span></span>

1. <span data-ttu-id="10e84-149">Vá para **Páginas** e abra um PDP que utilize o modelo PDP.</span><span class="sxs-lookup"><span data-stu-id="10e84-149">Go to **Pages**, and open a PDP that uses the PDP template.</span></span> <span data-ttu-id="10e84-150">Se ainda não existir um PDP, crie um.</span><span class="sxs-lookup"><span data-stu-id="10e84-150">If a PDP doesn't yet exist, create one.</span></span>
1. <span data-ttu-id="10e84-151">No slot **Contêiner** que contém o módulo de caixa de compra, selecione as reticências (**...**) e, depois, **Adicionar Módulo**.</span><span class="sxs-lookup"><span data-stu-id="10e84-151">In the **Container** slot that contains the buy box module, select the ellipsis (**...**), and then select **Add Module**.</span></span>
1. <span data-ttu-id="10e84-152">Na caixa de diálogo **Adicionar Módulo**, selecione o módulo **Trilha de navegação** e, depois, **OK**.</span><span class="sxs-lookup"><span data-stu-id="10e84-152">In the **Add Module** dialog box, select the **Breadcrumb** module, and then select **OK**.</span></span>
1. <span data-ttu-id="10e84-153">No painel de propriedades do slot **Trilha de navegação**, em **Raiz**, selecione **Texto do link**.</span><span class="sxs-lookup"><span data-stu-id="10e84-153">In the properties pane of the **Breadcrumb** slot, under **Root**, select **Link text**.</span></span>
1. <span data-ttu-id="10e84-154">Na caixa de diálogo **Vincular texto**, digite **Início** e, em **Destino do link**, selecione **Adicionar um link**.</span><span class="sxs-lookup"><span data-stu-id="10e84-154">In the **Link text** dialog box, enter **Home**, and then, under **Link target**, select **Add a link**.</span></span>
