[!include [banner](includes/banner.md)]

<span data-ttu-id="1a6b2-104">Quando uma [ordem do cliente](customer-orders-overview.md) é criada para retirada na loja, um usuário da loja pode usar o aplicativo de POS (ponto de venda) para iniciar a retirada do estoque.</span><span class="sxs-lookup"><span data-stu-id="1a6b2-104">When a [customer order](customer-orders-overview.md) is created for store pickup, a store user can use the point of sale (POS) application to start the pickup of inventory.</span></span> <span data-ttu-id="1a6b2-105">O PDV obterá o pagamento final, conforme necessário.</span><span class="sxs-lookup"><span data-stu-id="1a6b2-105">POS will run the final payment capture as required.</span></span> <span data-ttu-id="1a6b2-106">Ele também concluirá o lançamento de estoque e financeiro para as quantidades retiradas.</span><span class="sxs-lookup"><span data-stu-id="1a6b2-106">It will also complete the inventory and financial posting for the quantities that are picked up.</span></span>

<span data-ttu-id="1a6b2-107">Se você for um usuário de armazenamento, poderá realizar a retirada usando a operação de **Ordem de cancelamento** ou a operação de **Atendimento de ordem** no PDV.</span><span class="sxs-lookup"><span data-stu-id="1a6b2-107">If you're a store user, you can perform the pickup by using either the **Recall order** operation or the **Order fulfillment** operation in POS.</span></span> <span data-ttu-id="1a6b2-108">Para poder acessar a operação de **Retirada**, primeiro você deve seguir uma das seguintes etapas:</span><span class="sxs-lookup"><span data-stu-id="1a6b2-108">To make the **Pick up** operation available, you must first follow one of these steps:</span></span>

<span data-ttu-id="1a6b2-111">Se a ordem ou linhas da ordem selecionadas não estiverem configuradas para retirada nessa loja específica ou se a ordem já tiver sido totalmente retirada, a operação de **Retirada** não estará disponível.</span><span class="sxs-lookup"><span data-stu-id="1a6b2-111">If the selected order or order lines aren't configured for pickup at that specific store, or if the order has already been fully picked up, the **Pick up** operation will be unavailable.</span></span>

<span data-ttu-id="1a6b2-113">No Microsoft Dynamics 365 Commerce versão 10.0.17 e posterior, o recurso **Experiência de usuário aprimorada para o processamento de ordens de retirada no Ponto de venda** pode ser ativada com o gerenciamento de recursos na matriz do Commerce.</span><span class="sxs-lookup"><span data-stu-id="1a6b2-113">In Microsoft Dynamics 365 Commerce version 10.0.17 and later, the **Improved user experience for pick up order processing in Point of Sale** feature can be turned on through Feature management in Commerce headquarters.</span></span> <span data-ttu-id="1a6b2-114">Se esse recurso estiver desativado, os usuários não poderão selecionar as quantidades de retirada.</span><span class="sxs-lookup"><span data-stu-id="1a6b2-114">If this feature is turned off, users can't select pickup quantities.</span></span> <span data-ttu-id="1a6b2-115">Por padrão, a quantidade total pedida da linha é a quantidade que será retirada.</span><span class="sxs-lookup"><span data-stu-id="1a6b2-115">By default, the full quantity that was ordered for the line is the quantity that will be picked up.</span></span> <span data-ttu-id="1a6b2-116">Essa experiência pode causar problemas, já que os usuários podem esquecer de selecionar alguns itens para retirada ao executarem a retirada usando o atendimento da ordem.</span><span class="sxs-lookup"><span data-stu-id="1a6b2-116">This experience can be problematic, because users might forget to select some items for pickup when they perform the pickup through order fulfillment.</span></span>

<span data-ttu-id="1a6b2-117">O recurso **Experiência de usuário aprimorada para o processamento de ordens de retirada no Ponto de venda** oferece aos usuários mais controle sobre a seleção de produtos que serão retirados e a quantidade de produtos que serão retirados.</span><span class="sxs-lookup"><span data-stu-id="1a6b2-117">The **Improved user experience for pick up order processing in Point of Sale** feature gives users more control over the selection of products that will be picked up and the quantity of those products that will be picked up.</span></span> <span data-ttu-id="1a6b2-118">Os usuários não precisam selecionar cada linha da ordem de venda na página de atendimento da ordem para selecionar **Retirada**.</span><span class="sxs-lookup"><span data-stu-id="1a6b2-118">Users don't have to select every line of the sales order on the order fulfillment page before they select **Pick up**.</span></span> <span data-ttu-id="1a6b2-119">Todos os itens que podem ser retirados serão mostrados.</span><span class="sxs-lookup"><span data-stu-id="1a6b2-119">All the items that can be picked up will be shown.</span></span> <span data-ttu-id="1a6b2-120">Os usuários podem especificar várias linhas para retirada, mesmo que apenas uma linha de produto tenha sido selecionada.</span><span class="sxs-lookup"><span data-stu-id="1a6b2-120">Users can specify multiple lines for pickup even if only one product line is selected.</span></span>

<span data-ttu-id="1a6b2-121">Quando o recurso **Experiência de usuário aprimorada para o processamento de ordens de retirada no Ponto de venda** está ativado e você seleciona a operação de **Retirada**, a caixa de diálogo **Retirada** é exibida.</span><span class="sxs-lookup"><span data-stu-id="1a6b2-121">When the **Improve user experience for pick up order processing in Point of Sale** feature is turned on, and you select the **Pick up** operation, the **Pick up** dialog box appears.</span></span> <span data-ttu-id="1a6b2-122">Nela, você pode selecionar os itens e as quantidades que serão retirados.</span><span class="sxs-lookup"><span data-stu-id="1a6b2-122">There, you can select the items and quantities that will be picked up.</span></span> <span data-ttu-id="1a6b2-123">Por padrão, qualquer quantidade pedida que tenha estoque em um estado separado ou embalado é considerada qualificada para retirada.</span><span class="sxs-lookup"><span data-stu-id="1a6b2-123">By default, any ordered quantity that has inventory in a picked or packed state is considered eligible for pickup.</span></span> <span data-ttu-id="1a6b2-124">Por padrão, essa quantidade é definida como a quantidade de retirada.</span><span class="sxs-lookup"><span data-stu-id="1a6b2-124">By default, that quantity is set as the pickup quantity.</span></span> <span data-ttu-id="1a6b2-125">Você pode alterar a quantidade que foi inserida, desde que a quantidade não seja 0 (zero) e não exceda a quantidade total aberta (ou seja, não faturada) da linha selecionada.</span><span class="sxs-lookup"><span data-stu-id="1a6b2-125">You can change the quantity that was entered, provided that the quantity isn't 0 (zero) and doesn't exceed the total open (that is, non-invoiced) quantity for the selected line.</span></span>
