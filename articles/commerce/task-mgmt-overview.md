# <a name="task-management-overview"></a><span data-ttu-id="e2afb-103">Visão geral do gerenciamento de tarefas</span><span class="sxs-lookup"><span data-stu-id="e2afb-103">Task management overview</span></span>

[!include [banner](includes/banner.md)]

<span data-ttu-id="e2afb-104">Este tópico fornece uma visão geral do gerenciamento de tarefas para gerentes e trabalhadores no Microsoft Dynamics 365 Commerce.</span><span class="sxs-lookup"><span data-stu-id="e2afb-104">This topic provides an overview of task management for managers and workers in Microsoft Dynamics 365 Commerce.</span></span>

<span data-ttu-id="e2afb-105">Em um ambiente de varejo, é sempre difícil verificar se as tarefas são realizadas pela pessoa certa no momento certo.</span><span class="sxs-lookup"><span data-stu-id="e2afb-105">In a retail environment, it's always difficult to make sure that tasks are performed by the right person at the right time.</span></span> <span data-ttu-id="e2afb-106">Os varejistas devem ser capazes de notificar os trabalhadores sobre tarefas futuras e fornecer contexto comercial relacionado, de forma que as tarefas possam ser concluídas corretamente e a tempo.</span><span class="sxs-lookup"><span data-stu-id="e2afb-106">Retailers must be able to notify workers about upcoming tasks and provide related business context, so that the tasks can be completed correctly and on time.</span></span>

<span data-ttu-id="e2afb-107">O gerenciamento de tarefas é um recurso de produtividade no Dynamics 365 Commerce, que permite que os gerentes e trabalhadores criem listas de tarefas, gerenciem critérios de atribuição, rastreiem o status da tarefa e integrem essas operações entre o back-office do Commerce e as aplicações de ponto de venda (PDV).</span><span class="sxs-lookup"><span data-stu-id="e2afb-107">Task management is a productivity feature in Dynamics 365 Commerce that lets managers and workers create task lists, manage assignment criteria, track task status, and integrate these operations between Commerce back office and point of sale (POS) applications.</span></span>

<span data-ttu-id="e2afb-108">As pessoas da sede podem usar o gerenciamento de tarefas para criar listas de tarefas para lojas de varejo e rastrear o status por loja ou trabalhador.</span><span class="sxs-lookup"><span data-stu-id="e2afb-108">Headquarters personas can use task management to create task lists for retail stores, and to track status by store or worker.</span></span> <span data-ttu-id="e2afb-109">Eles também podem criar tarefas recorrentes (por exemplo, "lista de verificação de fechamento noturno de quinta-feira").</span><span class="sxs-lookup"><span data-stu-id="e2afb-109">They can also create recurrent tasks (for example, "Thursday night closing checklist").</span></span>

<span data-ttu-id="e2afb-110">Os gerentes de armazenamento podem usar o gerenciamento de tarefas para atribuir tarefas a trabalhadores individuais, enviar notificações sobre tarefas ou tarefas futuras que estão vencidas, atualizar status da tarefa e criar tarefas de finalidade única no aplicativo de PDV.</span><span class="sxs-lookup"><span data-stu-id="e2afb-110">Store managers can use task management to assign tasks to individual workers, send notifications about upcoming tasks or tasks that are past due, update task status, and create single-purpose tasks in the POS application.</span></span> <span data-ttu-id="e2afb-111">Os trabalhadores poderão ver notificações, exibir detalhes da tarefa e atualizar o status da tarefa no PDV.</span><span class="sxs-lookup"><span data-stu-id="e2afb-111">Workers can then see notifications, view task details, and update task status at the POS.</span></span>
