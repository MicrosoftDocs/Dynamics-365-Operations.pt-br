## <a name="overview"></a><span data-ttu-id="f3612-105">Visão Geral</span><span class="sxs-lookup"><span data-stu-id="f3612-105">Overview</span></span>

<span data-ttu-id="f3612-106">A Biblioteca de Mídia do assistente para criação de sites do Commerce permite cortar imagens para otimizá-las para diferentes tipos de módulo e portas de exibição.</span><span class="sxs-lookup"><span data-stu-id="f3612-106">The Commerce site builder Media Library allows you to crop images to optimize them for different module types and viewports.</span></span>

## <a name="crop-an-image"></a><span data-ttu-id="f3612-107">Cortar uma imagem</span><span class="sxs-lookup"><span data-stu-id="f3612-107">Crop an image</span></span>

<span data-ttu-id="f3612-108">Para cortar uma imagem no assistente para criação de sites, siga as etapas a seguir.</span><span class="sxs-lookup"><span data-stu-id="f3612-108">To crop an image in site builder, follow these steps.</span></span>

1. <span data-ttu-id="f3612-109">No painel de navegação esquerdo do assistente para criação de sites do Commerce, selecione **Biblioteca de Mídia**.</span><span class="sxs-lookup"><span data-stu-id="f3612-109">In the left navigation pane of Commerce site builder, select **Media Library**.</span></span>
1. <span data-ttu-id="f3612-110">Na janela principal, selecione a imagem que deseja modificar.</span><span class="sxs-lookup"><span data-stu-id="f3612-110">In the main window, select the image you want to modify.</span></span>
1. <span data-ttu-id="f3612-111">Na barra de comandos, selecione **Editar**.</span><span class="sxs-lookup"><span data-stu-id="f3612-111">On the command bar, select **Edit**.</span></span>
1. <span data-ttu-id="f3612-112">Selecione a imagem para entrar no **Modo de Edição**.</span><span class="sxs-lookup"><span data-stu-id="f3612-112">Select the image to enter **Edit Mode**.</span></span>
1. <span data-ttu-id="f3612-113">Em **Modo de Edição**, selecione **Editar Exibição por Módulo**.</span><span class="sxs-lookup"><span data-stu-id="f3612-113">Under **Edit Mode**, select **Edit View by Module**.</span></span>
1. <span data-ttu-id="f3612-114">No menu suspenso **Módulo**, selecione o tipo de módulo.</span><span class="sxs-lookup"><span data-stu-id="f3612-114">From the **Module** drop-down menu, select the module type.</span></span>
1. <span data-ttu-id="f3612-115">No menu suspenso **Tipo de Exibição**, selecione o tipo de exibição.</span><span class="sxs-lookup"><span data-stu-id="f3612-115">From the **View type** drop-down menu, select the view type.</span></span>
1. <span data-ttu-id="f3612-116">No menu suspenso **Posicionamento**, selecione o posicionamento da imagem.</span><span class="sxs-lookup"><span data-stu-id="f3612-116">From the **Placement** drop-down menu, select the image placement.</span></span>
1. <span data-ttu-id="f3612-117">No menu suspenso **Porta de Exibição**, selecione o tamanho da porta de exibição.</span><span class="sxs-lookup"><span data-stu-id="f3612-117">From the **Viewport** drop-down menu, select the viewport size.</span></span>
1. <span data-ttu-id="f3612-118">A imagem é sobreposta à área que representa a região de corte.</span><span class="sxs-lookup"><span data-stu-id="f3612-118">The image is overlaid with the area representing the crop region.</span></span> <span data-ttu-id="f3612-119">Mova e redimensione a região de corte, conforme necessário.</span><span class="sxs-lookup"><span data-stu-id="f3612-119">Move and resize the crop region as needed.</span></span> <span data-ttu-id="f3612-120">A taxa de proporção será mantida automaticamente.</span><span class="sxs-lookup"><span data-stu-id="f3612-120">The aspect ratio will be maintained automatically.</span></span>
1. <span data-ttu-id="f3612-121">Quando terminar, na barra de comandos, selecione **Salvar** e depois selecione **Concluir edição**.</span><span class="sxs-lookup"><span data-stu-id="f3612-121">When you're done, on the command bar, select **Save**, and then select **Finish editing**.</span></span> 

<span data-ttu-id="f3612-122">Após a conclusão do corte personalizado, as modificações na imagem têm efeito quase imediato.</span><span class="sxs-lookup"><span data-stu-id="f3612-122">After custom cropping is completed, image modifications will take effect almost immediately.</span></span>
