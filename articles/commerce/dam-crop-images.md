# <a name="crop-images"></a><span data-ttu-id="04a0c-103">Cortar imagens</span><span class="sxs-lookup"><span data-stu-id="04a0c-103">Crop images</span></span>

[!include [banner](includes/banner.md)]

<span data-ttu-id="04a0c-104">Este tópico descreve como cortar imagens no construtor de sites do Microsoft Dynamics 365 Commerce.</span><span class="sxs-lookup"><span data-stu-id="04a0c-104">This topic describes how to crop images in Microsoft Dynamics 365 Commerce site builder.</span></span>

<span data-ttu-id="04a0c-105">A Biblioteca de Mídia do assistente para criação de sites do Commerce permite cortar imagens para otimizá-las para diferentes tipos de módulo e portas de exibição.</span><span class="sxs-lookup"><span data-stu-id="04a0c-105">The Commerce site builder Media Library allows you to crop images to optimize them for different module types and viewports.</span></span>

## <a name="crop-an-image"></a><span data-ttu-id="04a0c-106">Cortar uma imagem</span><span class="sxs-lookup"><span data-stu-id="04a0c-106">Crop an image</span></span>

<span data-ttu-id="04a0c-107">Para cortar uma imagem no assistente para criação de sites, siga as etapas a seguir.</span><span class="sxs-lookup"><span data-stu-id="04a0c-107">To crop an image in site builder, follow these steps.</span></span>

1. <span data-ttu-id="04a0c-108">No painel de navegação esquerdo do assistente para criação de sites do Commerce, selecione **Biblioteca de Mídia**.</span><span class="sxs-lookup"><span data-stu-id="04a0c-108">In the left navigation pane of Commerce site builder, select **Media Library**.</span></span>
1. <span data-ttu-id="04a0c-109">Na janela principal, selecione a imagem que deseja modificar.</span><span class="sxs-lookup"><span data-stu-id="04a0c-109">In the main window, select the image you want to modify.</span></span>
1. <span data-ttu-id="04a0c-110">Na barra de comandos, selecione **Editar**.</span><span class="sxs-lookup"><span data-stu-id="04a0c-110">On the command bar, select **Edit**.</span></span>
1. <span data-ttu-id="04a0c-111">Selecione a imagem para entrar no **Modo de Edição**.</span><span class="sxs-lookup"><span data-stu-id="04a0c-111">Select the image to enter **Edit Mode**.</span></span>
1. <span data-ttu-id="04a0c-112">Em **Modo de Edição**, selecione **Editar Exibição por Módulo**.</span><span class="sxs-lookup"><span data-stu-id="04a0c-112">Under **Edit Mode**, select **Edit View by Module**.</span></span>
1. <span data-ttu-id="04a0c-113">No menu suspenso **Módulo**, selecione o tipo de módulo.</span><span class="sxs-lookup"><span data-stu-id="04a0c-113">From the **Module** drop-down menu, select the module type.</span></span>
1. <span data-ttu-id="04a0c-114">No menu suspenso **Tipo de Exibição**, selecione o tipo de exibição.</span><span class="sxs-lookup"><span data-stu-id="04a0c-114">From the **View type** drop-down menu, select the view type.</span></span>
1. <span data-ttu-id="04a0c-115">No menu suspenso **Posicionamento**, selecione o posicionamento da imagem.</span><span class="sxs-lookup"><span data-stu-id="04a0c-115">From the **Placement** drop-down menu, select the image placement.</span></span>
1. <span data-ttu-id="04a0c-116">No menu suspenso **Porta de Exibição**, selecione o tamanho da porta de exibição.</span><span class="sxs-lookup"><span data-stu-id="04a0c-116">From the **Viewport** drop-down menu, select the viewport size.</span></span>
1. <span data-ttu-id="04a0c-117">A imagem é sobreposta à área que representa a região de corte.</span><span class="sxs-lookup"><span data-stu-id="04a0c-117">The image is overlaid with the area representing the crop region.</span></span> <span data-ttu-id="04a0c-118">Mova e redimensione a região de corte, conforme necessário.</span><span class="sxs-lookup"><span data-stu-id="04a0c-118">Move and resize the crop region as needed.</span></span> <span data-ttu-id="04a0c-119">A taxa de proporção será mantida automaticamente.</span><span class="sxs-lookup"><span data-stu-id="04a0c-119">The aspect ratio will be maintained automatically.</span></span>
1. <span data-ttu-id="04a0c-120">Quando terminar, na barra de comandos, selecione **Salvar** e depois selecione **Concluir edição**.</span><span class="sxs-lookup"><span data-stu-id="04a0c-120">When you're done, on the command bar, select **Save**, and then select **Finish editing**.</span></span> 

<span data-ttu-id="04a0c-121">Após a conclusão do corte personalizado, as modificações na imagem têm efeito quase imediato.</span><span class="sxs-lookup"><span data-stu-id="04a0c-121">After custom cropping is completed, image modifications will take effect almost immediately.</span></span>
