# <a name="customize-image-focal-points"></a><span data-ttu-id="7dc01-103">Personalizar pontos focais da imagem</span><span class="sxs-lookup"><span data-stu-id="7dc01-103">Customize image focal points</span></span>

[!include [banner](includes/banner.md)]

<span data-ttu-id="7dc01-104">Este tópico descreve como personalizar pontos focais da imagem no construtor de sites do Microsoft Dynamics 365 Commerce.</span><span class="sxs-lookup"><span data-stu-id="7dc01-104">This topic describes how to customize image focal points in Microsoft Dynamics 365 Commerce site builder.</span></span>

<span data-ttu-id="7dc01-105">Quando uma imagem é carregada na Biblioteca de Mídia do assistente para criação de sites do Commerce, o sistema tenta determinar o ponto focal da imagem.</span><span class="sxs-lookup"><span data-stu-id="7dc01-105">When an image is uploaded to the Commerce site builder Media Library, the system attempts to determine the focal point of the image.</span></span> <span data-ttu-id="7dc01-106">Por exemplo, se a imagem tiver uma pessoa, o sistema definirá o ponto focal para o rosto da pessoa por padrão.</span><span class="sxs-lookup"><span data-stu-id="7dc01-106">For example, if the image has a person on it, the system will set the focal point to the face of the person by default.</span></span> <span data-ttu-id="7dc01-107">Na maioria dos casos, o ponto focal definido automaticamente funciona bem para todas as portas de exibição, mas às vezes você pode ajustar o ponto focal para garantir que uma parte específica da imagem esteja sempre visível.</span><span class="sxs-lookup"><span data-stu-id="7dc01-107">In most cases the automatically set focal point works well for all viewports, but sometimes you may want to adjust the focal point to ensure that a specific part of the image is always visible.</span></span>

### <a name="define-a-custom-focal-point-for-an-image"></a><span data-ttu-id="7dc01-108">Definir um ponto focal personalizado de uma imagem</span><span class="sxs-lookup"><span data-stu-id="7dc01-108">Define a custom focal point for an image</span></span>

<span data-ttu-id="7dc01-109">Para definir um ponto focal personalizado de uma imagem, siga as etapas a seguir.</span><span class="sxs-lookup"><span data-stu-id="7dc01-109">To define a custom focal point for an image, follow these steps.</span></span>

1. <span data-ttu-id="7dc01-110">No painel de navegação esquerdo do assistente para criação de sites do Commerce, selecione **Biblioteca de Mídia**.</span><span class="sxs-lookup"><span data-stu-id="7dc01-110">In the left navigation pane of Commerce site builder, select **Media Library**.</span></span>
1. <span data-ttu-id="7dc01-111">Na janela principal, selecione a imagem que deseja modificar.</span><span class="sxs-lookup"><span data-stu-id="7dc01-111">In the main window, select the image you want to modify.</span></span>
1. <span data-ttu-id="7dc01-112">Na barra de comandos, selecione **Editar**.</span><span class="sxs-lookup"><span data-stu-id="7dc01-112">On the command bar, select **Edit**.</span></span>
1. <span data-ttu-id="7dc01-113">Selecione a imagem para entrar no **Modo de Edição**.</span><span class="sxs-lookup"><span data-stu-id="7dc01-113">Select the image to enter **Edit Mode**.</span></span>
1. <span data-ttu-id="7dc01-114">Em **Modo de Edição**, selecione **Alterar Ponto Focal**.</span><span class="sxs-lookup"><span data-stu-id="7dc01-114">Under **Edit Mode**, select **Change Focal Point**.</span></span> <span data-ttu-id="7dc01-115">Um controle circular do ponto focal aparece sobre a imagem.</span><span class="sxs-lookup"><span data-stu-id="7dc01-115">A circular focal point control appears over the image.</span></span>
1. <span data-ttu-id="7dc01-116">Selecione o controle do ponto focal para movê-lo sobre o ponto focal desejado.</span><span class="sxs-lookup"><span data-stu-id="7dc01-116">Select the focal point control to move it over the desired focal point.</span></span>
1. <span data-ttu-id="7dc01-117">Quando terminar, na barra de comandos, selecione **Salvar** e depois selecione **Concluir edição**.</span><span class="sxs-lookup"><span data-stu-id="7dc01-117">When you're done, on the command bar select **Save**, and then select **Finish editing**.</span></span>
