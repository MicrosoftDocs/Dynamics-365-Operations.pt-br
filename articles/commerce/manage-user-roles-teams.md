# <a name="manage-user-roles-in-microsoft-teams"></a>Gerenciar funções do usuário no Microsoft Teams

[!include [banner](includes/banner.md)]

Este tópico descreve como gerenciar os arquivos do usuário do Microsoft Dynamics 365 Commerce no Microsoft Teams.

Ao criar uma equipe para cada armazenamento ou canal no Teams, uma associação de grupo que corresponde à equipe é criada (por exemplo, `HOUSTON_D365@<YourTenantAzureADDomain>.com`). Todos os trabalhadores do armazenamento em uma associação de grupo de equipe recebem uma das duas funções de usuário: **Proprietário** ou **Membro**. Os funcionários do armazenamento que têm a função de usuário **Proprietário** podem executar operações, como adicionar um canal privado e adicionar ou excluir membros. Normalmente, os gerentes de armazenamento têm a função de usuário **Proprietário**.

A ilustração a seguir mostra um exemplo de uma lista de membros da equipe e suas funções de usuário no centro de administração do Microsoft Teams.

![Membros da equipe e funções de usuário no centro de administração do Microsoft Teams.](media/d365-commerce-teams-integration-user-roles.png)

Para obter mais informações, consulte [Atribuir proprietários e membros de equipe no Microsoft Teams](/microsoftteams/assign-roles-permissions).
