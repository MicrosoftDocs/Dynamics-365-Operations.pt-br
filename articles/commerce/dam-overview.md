# <a name="digital-asset-management-overview"></a>Visão geral do gerenciamento de ativos digitais

[!include [banner](includes/banner.md)]

Este tópico fornece uma visão geral do gerenciamento de ativos digitais no assistente para criação de sites do Microsoft Dynamics 365 Commerce.

## <a name="overview"></a>Visão Geral

A Biblioteca de Mídia do assistente para criação de sites do Dynamics 365 Commerce apresenta uma sofisticada funcionalidade de gerenciamento de ativos digitais, incluindo suporte para o seguinte:
- Ativos de imagem
- Ativos de vídeo
- Outros tipos de ativos binários, como documentos do Office e arquivos PDF
- Imagens localizadas
- Uploads de pastas
- Recorte de imagens
- Personalização de pontos focais de imagem
- Recursos de imagem de omnicanal (produto, grades de produtos, catálogo, categoria, trabalhador e imagens de clientes)

Os ativos digitais são específicos de canal, com exceção dos ativos de omnicanal, que são padrão para todo o ambiente. 

Recursos digitais e recursos de gerenciamento estão localizados no assistente para criação de sites em **Biblioteca de Mídia** no painel de navegação esquerdo de um site.
