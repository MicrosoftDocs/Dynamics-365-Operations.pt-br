# <a name="commerce-chat-features-overview"></a>Visão geral dos recursos do chat comercial

[!include [banner](includes/banner.md)]

Ao integrar os aplicativos Omnicanal para Customer Service e Power Virtual Agents do Microsoft Dynamics 365 aos sites de comércio eletrônico do Dynamics 365 Commerce, as empresas podem habilitar seus compradores online por meio de recursos de chat. Este artigo fornece uma visão geral dos recursos de chat do Commerce que estão disponíveis para compradores online quando o Omnicanal para Customer Service e o Power Virtual Agents são integrados aos sites de comércio eletrônico do Dynamics 365 Commerce.

## <a name="omnichannel-for-customer-service"></a>Omnicanal para Customer Service

O Omnicanal para Customer Service é um aplicativo robusto que amplia a capacidade do Dynamics 365 Customer Service. Ele permite que as empresas se conectem e participem instantaneamente de seus clientes por meio de canais como chat ao vivo, voz e SMS (Serviço de Mensagens Curtas). Para obter mais informações sobre o Omnicanal para Customer Service, consulte [introdução ao Omnicanal para Customer Service](/dynamics365/customer-service/introduction-omnichannel). Para obter informações sobre como integrar o Omnicanal para Customer Service com um site de comércio eletrônico do Dynamics 365 Commerce, consulte [Módulo Chat do Commerce com Omnicanal para Customer Service](commerce-chat-module.md).

## <a name="power-virtual-agents"></a>Power Virtual Agents

O Power Virtual Agents permite criar chatbots avançados da plataforma AI para um intervalo de solicitações. Para obter mais informações sobre o Power Virtual Agents, consulte [Visão geral do Power Virtual Agents](/power-virtual-agents/fundamentals-what-is-power-virtual-agents). Para obter informações sobre como integrar o Power Virtual Agents a um site de comércio eletrônico do Dynamics 365 Commerce, consulte [Chat do Commerce com Omnicanal para Microsoft Power Virtual Agents](chat-module-pva.md).

> [!NOTE]
> O Power Virtual Agents já está incluído no Omnicanal para Customer Service. Se você precisar de recursos do agente virtual (chatbot) e do agente humano, use Omnicanal para Customer Service.
