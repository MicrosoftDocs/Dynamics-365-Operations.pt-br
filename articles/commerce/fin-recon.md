[!include [banner](includes/banner.md)]

<span data-ttu-id="ae19e-104">No Microsoft Dynamics 365 Commerce versão 10.0.10 e anteriores, a funcionalidade que o cliente de PDV (ponto de venda) oferece para os processos de fechamento do dia nas <span data-ttu-id="ae19e-106">No entanto, não existe um recurso no PDV para finalizar as informações financeiras de turnos de forma que elas possam ser usadas para lançar dados financeiros no Commerce
