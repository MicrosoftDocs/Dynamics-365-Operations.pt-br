# <a name="learning-catalog-for-dynamics-365-commerce-administrators"></a>Catálogo de aprendizagem para administradores do Dynamics 365 Commerce

Você precisa manter os sistemas e os dados fluindo, provisionados e protegidos 24 horas?

O catálogo a seguir é organizado com base no conhecimento fundamental para domínios específicos, desde o mais básico até o mais avançado. Se o conteúdo existir em vários formatos, o usuário será informado, permitindo escolher o formato de treinamento que melhor atender às suas necessidades. Este catálogo inclui apenas recursos específicos do Dynamics 365 Commerce. Para obter treinamento sobre recursos compartilhados com outros aplicativos de finanças e operações, consulte o [Catálogo de aprendizagem do Dynamics 365 Finance](../../finance/get-started/learning-catalog-administrator.md).
