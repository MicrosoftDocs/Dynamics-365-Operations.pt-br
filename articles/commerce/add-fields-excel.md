## <a name="add-fields-to-a-worksheet-in-an-excel-workbook"></a><span data-ttu-id="500af-108">Adicionar campos a uma planilha em uma pasta de trabalho do Excel</span><span class="sxs-lookup"><span data-stu-id="500af-108">Add fields to a worksheet in an Excel workbook</span></span>

<span data-ttu-id="500af-109">Para adicionar campos a uma pasta de trabalho do Excel para que você possa editar transações de varejo, siga estas etapas:</span><span class="sxs-lookup"><span data-stu-id="500af-109">To add fields to an Excel workbook so that you can edit retail transactions, follow these steps.</span></span>

1. <span data-ttu-id="500af-110">Baixe e abra o arquivo Excel na página **Demonstrativos**, na página **Transações de varejo** ou no bloco **Falhas na validação de transações** no espaço de trabalho **Finanças da loja**.</span><span class="sxs-lookup"><span data-stu-id="500af-110">Download and open the Excel file from the **Statements** page, the **Retail transactions** page, or the **Transaction validation failures** tile in the **Store financials** workspace.</span></span>
1. <span data-ttu-id="500af-111">Selecione **Design**.</span><span class="sxs-lookup"><span data-stu-id="500af-111">Select **Design**.</span></span>
1. <span data-ttu-id="500af-112">Selecione o símbolo do lápis para a tabela desejada e, na lista de campos disponíveis, encontre e selecione o campo que deseja adicionar.</span><span class="sxs-lookup"><span data-stu-id="500af-112">Select the pencil symbol for the desired table, and then, in the list of available fields, find and select the field that you want to add.</span></span>

1. <span data-ttu-id="500af-115">Após a conclusão da atualização, selecione **Atualizar** para obter os dados da nova coluna.</span><span class="sxs-lookup"><span data-stu-id="500af-115">After the update is completed, select **Refresh** to fetch the data for the new column.</span></span>

<span data-ttu-id="500af-116">O novo campo e seus dados devem agora estar disponíveis para edição no Excel.</span><span class="sxs-lookup"><span data-stu-id="500af-116">The new field and data for it should now be available for editing in Excel.</span></span>
