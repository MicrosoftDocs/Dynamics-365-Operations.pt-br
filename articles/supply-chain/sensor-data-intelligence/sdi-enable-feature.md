# <a name="turn-on-sensor-data-intelligence-for-your-system"></a>Ativar Sensor Data Intelligence para o seu sistema

[!include [banner](../includes/banner.md)]
[!INCLUDE [preview-banner](../includes/preview-banner.md)]

Para que você possa usar a Sensor Data Intelligence, ela deve estar ativado no sistema.

1. Acesse **Administrador do sistema \> Espaços de trabalho \> Gerenciamento de recursos**. (Para obter mais informações, consulte [Visão geral do gerenciamento de recursos](../../fin-ops-core/fin-ops/get-started/feature-management/feature-management-overview.md).)
1. Na guia **Tudo**, use o campo **Filtro** para procurar o recurso chamado *Sensor Data Intelligence*.
1. Se o recurso *Sensor Data Intelligence* estiver habilitado no seu sistema, selecione-o na lista e, em seguida, selecione **Desabilitar** para desabilitá-lo. Não é possível usar essa versão mais antiga do recurso em conjunto com a nova versão de visualização que será habilitada na próxima etapa.
1. Use o campo **Filtro** para procurar o recurso chamado *(Preview) Sensor Data Intelligence*.
1. Selecione o recurso na lista e, depois, **Habilitar agora** para habilitá-lo.
