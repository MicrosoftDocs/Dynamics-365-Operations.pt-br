# <a name="replenishment"></a><span data-ttu-id="03279-103">Reabastecimento</span><span class="sxs-lookup"><span data-stu-id="03279-103">Replenishment</span></span>

[!include[banner](../includes/banner.md)]

<span data-ttu-id="03279-104">Este tópico descreve as estratégias de reabastecimento que estão disponíveis para os depósitos que usam a funcionalidade que está disponível no Gerenciamento de depósito.</span><span class="sxs-lookup"><span data-stu-id="03279-104">This topic describes the replenishment strategies that are available for warehouses that use the functionality that is available in Warehouse management.</span></span> <span data-ttu-id="03279-105">As informações deste tópico não se aplicam à solução de armazenamento disponível no Gerenciamento de estoque.</span><span class="sxs-lookup"><span data-stu-id="03279-105">The information in this topic doesn't apply to the warehousing solution that is available in Inventory management.</span></span>

<span data-ttu-id="03279-106">As seguintes estratégias de reabastecimento estão disponíveis:</span><span class="sxs-lookup"><span data-stu-id="03279-106">The following replenishment strategies are available:</span></span>

- <span data-ttu-id="03279-107">**Reabastecimento baseado em demanda** – essa estratégia cria o trabalho de reabastecimento para ordens de saída ou cargas, caso o estoque não esteja disponível quando o trabalho for criado pela onda.</span><span class="sxs-lookup"><span data-stu-id="03279-107">**Wave demand replenishment** – This strategy creates replenishment work for outbound orders or loads if inventory isn't available when the wave creates work.</span></span> <span data-ttu-id="03279-108">Por exemplo, se a quantidade necessária para uma ordem de venda não estiver disponível quando uma onda for processada, será possível criar o trabalho de reabastecimento.</span><span class="sxs-lookup"><span data-stu-id="03279-108">For example, replenishment work can be created if the quantity that is required for a sales order isn't available when a wave is processed.</span></span>
- <span data-ttu-id="03279-109">**Reabastecimento mín./máx.** – essa estratégia usa limites mínimo e máximo de estoque para determinar quando as localizações deverão ser reabastecidas.</span><span class="sxs-lookup"><span data-stu-id="03279-109">**Min/Max replenishment** – This strategy uses minimum and maximum stocking limits to determine when locations should be replenished.</span></span> <span data-ttu-id="03279-110">Os critérios de item e localização definem o estoque avaliado para reabastecimento.</span><span class="sxs-lookup"><span data-stu-id="03279-110">The item and location criteria define the inventory that is evaluated for replenishment.</span></span> <span data-ttu-id="03279-111">Modelos de reabastecimento mín./máx. são o principal mecanismo para manter os níveis ideais em locais de separação.</span><span class="sxs-lookup"><span data-stu-id="03279-111">Min/Max replenishment templates are the primary mechanism for maintaining optimal levels in picking locations.</span></span> <span data-ttu-id="03279-112">Para ajudar a garantir que estoque nominal escolhido suficiente esteja disponível para ser compatível com a demanda da onda, você pode usar a demanda de reabastecimento como um suplemento entre ciclos de reabastecimento mín./máx.</span><span class="sxs-lookup"><span data-stu-id="03279-112">To help guarantee that enough pick face inventory is available to meet wave demand, you can use demand replenishment as a supplement between Min/Max replenishment cycles.</span></span>
- <span data-ttu-id="03279-113">**Carregar o reabastecimento por demanda** – essa estratégia soma a demanda por várias cargas e cria o trabalho de reabastecimento necessário para os locais de separação relevante de ações.</span><span class="sxs-lookup"><span data-stu-id="03279-113">**Load demand replenishment** – This strategy sums the demand for several loads and creates the replenishment work that is required in order to stock the relevant picking locations.</span></span> <span data-ttu-id="03279-114">Essa estratégia ajuda a garantir que as cargas criadas possam ser separadas no depósito depois que elas são lançadas.</span><span class="sxs-lookup"><span data-stu-id="03279-114">This strategy helps guarantee that the loads that are created can be picked in the warehouse after they are released.</span></span>
- <span data-ttu-id="03279-115">**Reabastecimento imediato** – Esta estratégia reabastece o estoque antes de uma onde ser executada, se a alocação falhar para uma linha de diretiva de localização que tem o modelo de reabastecimento.</span><span class="sxs-lookup"><span data-stu-id="03279-115">**Immediate replenishment** – This strategy replenishes inventory before a wave is run if allocation fails for a location directive line that has a replenishment template.</span></span> 

<span data-ttu-id="03279-116">Todas as quatro estratégias criam trabalho de reabastecimento, com base em um modelo de reabastecimento.</span><span class="sxs-lookup"><span data-stu-id="03279-116">All four strategies create replenishment work, based on a replenishment template.</span></span>

<span data-ttu-id="03279-118">O reabastecimento de demanda de onda cria um trabalho de reabastecimento, baseado na demanda, se a quantidade que é necessária para ordens de produção, kanbans, ordens de saída ou carregamentos não está disponível quando uma onda cria trabalho.</span><span class="sxs-lookup"><span data-stu-id="03279-118">Wave demand replenishment creates replenishment work, based on demand, if the quantity that is required for production orders, kanbans, outbound orders, or loads isn't available when a wave creates work.</span></span> <span data-ttu-id="03279-119">O modelo de reabastecimento contém informações sobre os critérios de item, a unidade de medida, o incremento de demanda e o local.</span><span class="sxs-lookup"><span data-stu-id="03279-119">The replenishment template contains information about the item criteria, the unit of measure, the demand increment, and the location.</span></span> 

<span data-ttu-id="03279-120">Diretivas locais são usadas para determinar qual local deve ser reabastecido.</span><span class="sxs-lookup"><span data-stu-id="03279-120">Location directives are used to determine which location should be replenished.</span></span> <span data-ttu-id="03279-121">Você vincula essas diretivas de local ao modelo de reabastecimento usando o campo **Código de diretiva**.</span><span class="sxs-lookup"><span data-stu-id="03279-121">You link these location directives to the replenishment template by using the **Directive code** field.</span></span> <span data-ttu-id="03279-122">Se o campo **Código de diretiva** não for definido, as consultas são usadas para determinar qual diretiva local deve ser usada.</span><span class="sxs-lookup"><span data-stu-id="03279-122">If the **Directive code** field isn't set, queries are used to determine which location directive should be used.</span></span> <span data-ttu-id="03279-123">Observe que se não for especificado um código de diretiva no modelo de reabastecimento e a diretiva local tiver um código de diretiva, a diretiva local será ignorada, mesmo se a consulta na diretiva local estiver correta.</span><span class="sxs-lookup"><span data-stu-id="03279-123">Note that if a directive code isn't specified in the replenishment template, and the location directive has a directive code, the location directive will be ignored, even if the query on the location directive is correct.</span></span> <span data-ttu-id="03279-124">Escolha diretivas locais que são usadas para determinar de onde obter o estoque para o reabastecimento.</span><span class="sxs-lookup"><span data-stu-id="03279-124">Pick location directives are used to determine where to get inventory for the replenishment.</span></span> 

<span data-ttu-id="03279-125">Além de criar um modelo, você deve especificar algumas configurações de reabastecimento no modelo de onda.</span><span class="sxs-lookup"><span data-stu-id="03279-125">In addition to creating a template, you must specify some replenishment settings in the wave template.</span></span> <span data-ttu-id="03279-126">O modelo de onda deve conter uma etapa de onda para reabastecimento executada somente se um item não for alocado com sucesso.</span><span class="sxs-lookup"><span data-stu-id="03279-126">The wave template should contain a wave step for replenishment that is run only if an item isn't successfully allocated.</span></span> <span data-ttu-id="03279-127">Esta etapa de onda de reabastecimento usa um código de etapa de onda para determinar qual modelo de reabastecimento deve ser usado.</span><span class="sxs-lookup"><span data-stu-id="03279-127">This replenishment wave step uses a wave step code to determine which replenishment template should be used.</span></span> <span data-ttu-id="03279-128">Além de ter uma etapa de onda para reabastecimento, você deve garantir que o **Reabastecimento** seja selecionado na seção **Métodos** do modelo de onda.</span><span class="sxs-lookup"><span data-stu-id="03279-128">In addition to having a wave step for replenishment, you must make sure that **Replenish** is selected in the **Methods** section of the wave template.</span></span>

<span data-ttu-id="03279-133">O reabastecimento de demanda é suportado para pedidos de vendas, ordens de transferência, ordens de produção e kanbans.</span><span class="sxs-lookup"><span data-stu-id="03279-133">Demand replenishment is supported for sales orders, transfer orders, production orders, and kanbans.</span></span>
