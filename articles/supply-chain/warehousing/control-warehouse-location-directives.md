## <a name="location-directives"></a><span data-ttu-id="f20e9-134">Diretivas de localização</span><span class="sxs-lookup"><span data-stu-id="f20e9-134">Location directives</span></span>

<span data-ttu-id="f20e9-135">As diretivas de localização são regras que ajudam a identificar locais de separação e armazenamento para a movimentação do estoque.</span><span class="sxs-lookup"><span data-stu-id="f20e9-135">Location directives are rules that help identify pick and put locations for inventory movement.</span></span> <span data-ttu-id="f20e9-136">Por exemplo, em uma transação de ordem de venda, uma diretiva de localização determina onde os itens serão separados e onde os itens separados serão armazenados.</span><span class="sxs-lookup"><span data-stu-id="f20e9-136">For example, in a sales order transaction, a location directive determines where the items will be picked, and where the picked items will be put.</span></span> <span data-ttu-id="f20e9-137">As diretivas de localização consistem em um cabeçalho e linhas associadas, e você pode criá-los na página **Diretivas de localização**.</span><span class="sxs-lookup"><span data-stu-id="f20e9-137">Location directives consist of a header and associated lines, and you create them on the **Location directives** page.</span></span> 

<span data-ttu-id="f20e9-138">No cabeçalho, cada diretiva de localização deve estar associada a um *tipo de ordem de trabalho* que especifica para qual tipo de transação de estoque essa diretiva será usada, como ordens de venda, reabastecimento ou separação de matéria-prima.</span><span class="sxs-lookup"><span data-stu-id="f20e9-138">On the header, each location directive must be associated with a *work order type* that specifies the type of inventory transaction that the directive will be used for, such as sales orders, replenishment, or raw material picking.</span></span> <span data-ttu-id="f20e9-139">O *tipo de trabalho* especifica se a diretiva de localização será usada para trabalho de separação ou armazenamento ou para qualquer outro processo de depósito, como contagem ou alterações de status do estoque.</span><span class="sxs-lookup"><span data-stu-id="f20e9-139">The *work type* specifies whether the location directive will be used for picking or putting work, or for some other warehouse process, such as counting or inventory status changes.</span></span> <span data-ttu-id="f20e9-140">Você também deve especificar um *local* e um *depósito*.</span><span class="sxs-lookup"><span data-stu-id="f20e9-140">You must also specify a *site* and a *warehouse*.</span></span> <span data-ttu-id="f20e9-141">Um *código de diretiva* que você especifica no cabeçalho pode ser usado para vincular a diretiva de localização a um ou mais modelos de trabalho.</span><span class="sxs-lookup"><span data-stu-id="f20e9-141">A *directive code* that you specify on the header can be used to link the location directive to one or more work templates.</span></span>

<span data-ttu-id="f20e9-145">Linhas das diretivas definem local restrições adicionais no aplicativo do local que encontra regras.</span><span class="sxs-lookup"><span data-stu-id="f20e9-145">The location directive lines set additional restrictions on the application of the location finding rules.</span></span> <span data-ttu-id="f20e9-146">Você pode especificar uma quantidade mínima e máxima ao qual a diretiva deve ser aplicada; você pode especificar ainda que a diretiva se refere a uma unidade específica do estoque.</span><span class="sxs-lookup"><span data-stu-id="f20e9-146">You can specify a minimum quantity and a maximum quantity that the directive should apply to, and you can specify that the directive should be for a specific inventory unit.</span></span> <span data-ttu-id="f20e9-147">Por exemplo, se a unidade de medida for paletes, os itens em paletes poderão ser colocados em um local específico.</span><span class="sxs-lookup"><span data-stu-id="f20e9-147">For example, if the unit of measure is pallets, the items in pallets can be put in a specific location.</span></span> <span data-ttu-id="f20e9-148">Você também pode especificar se a quantidade pode ser dividida entre vários locais.</span><span class="sxs-lookup"><span data-stu-id="f20e9-148">You can also specify whether the quantity can be split across multiple locations.</span></span> <span data-ttu-id="f20e9-149">Como o cabeçalho diretivo localização, de cada linha da localização diretiva tiver um número de sequência que determina a ordem em que as linhas sejam avaliadas.</span><span class="sxs-lookup"><span data-stu-id="f20e9-149">Like the location directive header, each location directive line has a sequence number that determines the order that the lines are assessed in.</span></span>
