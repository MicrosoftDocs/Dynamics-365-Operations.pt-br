# <a name="scale-unit-capability-is-currently-paused-for-new-customers"></a>No momento, a capacidade da unidade de escala está pausada para novos clientes

[!include [banner](../includes/banner.md)]

> [!IMPORTANT]
> A capacidade de unidade de escala para o Microsoft Dynamics 365 Supply Chain Management está pausada para novos clientes, pois a arquitetura está sendo aperfeiçoada. No momento, não estamos aceitando novos clientes para essa solução, mas estamos trabalhando ativamente em uma arquitetura combinável, flexível e fácil de usar para nossa linha de negócios de Cadeia de Fornecedores, que aumentará ainda mais nossa capacidade de fornecer uma solução de unidade de escala de nível internacional para nossos clientes.
>
> Estamos trabalhando para aprimorar o Dynamics 365 Supply Chain Management de modo a permitir maior capacidade de composição por módulos específicos, como o Warehouse Management. Continuaremos planejando e desenvolvendo novas ofertas de unidades de escala de nuvem e de borda para atender às necessidades de nossos clientes. Para obter mais informações, entre em contato pelo email [d365scmcneext@microsoft.com](mailto:d365scmcneext@microsoft.com).
