# <a name="purchase-orders-dont-show-translated-product-text"></a>As ordens de compra não mostram tradução do texto do produto

Número de KB: 4612098

## <a name="symptoms"></a>Sintomas

Você definiu traduções de texto para nomes de produtos. No entanto, os nomes de produtos sempre são mostrados no idioma original, independentemente da configuração de idioma do aplicativo.

## <a name="resolution"></a>Resolução

As traduções de produtos são mostradas somente em documentos externos. Elas não são mostradas para usuários internos. Para obter mais informações, consulte [Perguntas frequentes sobre traduções relacionadas aos produtos](../../pim/translations-product-related-information.md#where-can-i-view-the-translated-information).
