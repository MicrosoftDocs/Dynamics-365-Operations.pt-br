# <a name="cant-move-inventory-to-a-location-that-is-license-plate-controlled"></a>Não é possível mover o estoque para um local controlado por placa de licença

## <a name="symptoms"></a>Sintomas

Você não consegue reduzir as quantidades selecionadas em uma carga.

## <a name="resolution"></a>Resolução

Em versões anteriores, não é possível reduzir as quantidades selecionadas em uma carga. Contudo, agora você pode desfazer a separação em um local controlado por placa de licença. Você deve especificar um valor **Local** e um valor **Placa de licença** para a linha de carga na página **Reduzir quantidade separada**.
