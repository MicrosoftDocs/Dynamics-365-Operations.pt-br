# <a name="quantity-is-not-valid-when-performing-a-split-pick-across-multiple-batches"></a>A quantidade não é válida ao executar uma separação de divisão entre vários lotes

## <a name="symptoms"></a>Sintomas

Ao tentar executar uma *separação de divisão* em vários lotes, você recebe a seguinte mensagem de erro:

> A quantidade não é válida para a unidade %1.

## <a name="resolution"></a>Resolução

O funcionário do depósito deve usar o processo *separação insuficiente* no aplicativo móvel Warehouse Management. Se você estiver tentando selecionar vários lotes do mesmo local, também pode usar a opção **Completo** no aplicativo.
