# <a name="can-only-use-administrator-role-to-access-the-mobile-device-app-emulator"></a>Só é possível usar a função Administrador para acessar o emulador de aplicativo de dispositivo móvel

## <a name="symptoms"></a>Sintomas

Ao configurar depósitos no Supply Chain Management, você não pode usar nenhuma função, exceto a função de administrador para acessar o emulador de aplicativo de dispositivo móvel.

## <a name="resolution"></a>Resolução

O emulador de aplicativo de dispositivo móvel está configurado para funcionar somente com a conta do administrador. Para todos os fins de teste e processo ativo, recomendamos que você use o próprio aplicativo móvel Gerenciamento de Depósito.
