# <a name="cant-trace-serial-numbers-of-finished-goods-against-those-of-consumed-goods"></a>Não é possível rastrear números de série de mercadorias concluídas contra aqueles de mercadorias consumidas

## <a name="symptoms"></a>Sintomas

Não é possível rastrear os números de série dos produtos concluídos em relação aos números de série dos materiais que uma ordem de produção consome para fazer esses produtos concluídos.

## <a name="workaround"></a>Solução alternativa

Para uma solução alternativa, crie ordens de produção para uma quantidade 1.
