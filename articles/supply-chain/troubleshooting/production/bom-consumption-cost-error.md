# <a name="cant-end-a-production-order-because-of-a-bom-consumptioncost-value-error"></a>Não é possível encerrar uma ordem de produção devido a um erro no valor de consumptionCost da BOM

## <a name="symptoms"></a>Sintomas

Ao tentar encerrar uma ordem de produção, a seguinte mensagem de erro é exibida:

> O cálculo do valor de consumptionCost da BOM precisa ser negativo na saída do estoque.

## <a name="resolution"></a>Resolução

Esse problema foi corrigido na versão 10.0.15.
