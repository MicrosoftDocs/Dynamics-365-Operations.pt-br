# <a name="users-can-unpick-material-lines-for-products-that-have-been-reported-as-finished"></a>Os usuários podem desselecionar linhas de material para produtos que foram relatados como finalizados

Número da base de dados de conhecimento: 4614721

## <a name="symptoms"></a>Sintomas

O sistema permite aos usuários desselecionar linhas de material para produtos que foram relatados como finalizados.

## <a name="resolution"></a>Resolução

O sistema está agindo como esperado. O consumo de matéria prima pode ser ajustado até que o pedido de produção chegue ao status de *Encerrado*.
