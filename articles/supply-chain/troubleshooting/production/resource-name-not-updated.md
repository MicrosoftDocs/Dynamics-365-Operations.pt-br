# <a name="resource-name-isnt-updated-after-changing-a-worker-name-in-the-address-book"></a>O nome do recurso não é atualizado após alterar o nome de um trabalhador no catálogo de endereços

## <a name="symptoms"></a>Sintomas

Se você alterar o nome de um trabalhador no catálogo de endereços global, o nome do recurso correspondente não será atualizado no mestre do grupo de recursos.

## <a name="resolution"></a>Resolução

No momento, o nome do recurso correspondente não é atualizado automaticamente. Para corrigir o problema, você deve atualizar manualmente o nome do recurso.
