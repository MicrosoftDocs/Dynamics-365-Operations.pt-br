# <a name="vendor-isnt-specified-when-planned-orders-are-firmed"></a>O fornecedor não é especificado quando as ordens planejadas são firmadas

Código de erro: SYS23532

## <a name="symptoms"></a>Sintomas

Ao tentar firmar ordens planejadas, você recebe a seguinte mensagem de erro:

> O fornecedor não foi especificado

## <a name="resolution"></a>Resolução

Para especificar um fornecedor, siga estas etapas.

1. Abra a ordem planejada que está faltando um vendedor.
1. Na FastTab **Fornecimento planejado**, no campo **Fornecedor** selecione um fornecedor.
