-   <span data-ttu-id="cd240-110">Atribua um tipo de custo de **Custo padrão**.</span><span class="sxs-lookup"><span data-stu-id="cd240-110">Assign a costing type of **Standard cost**.</span></span>

-   <span data-ttu-id="cd240-112">No grupo de campos **Permitir tipos de preço**, verifique se **Preço de custo** está definido como **Sim**.</span><span class="sxs-lookup"><span data-stu-id="cd240-112">In the **Allow price types** field group, make sure that **Cost price** is set to **Yes**.</span></span>

manufactured items by using the **Calculation** page.</span></span> <span data-ttu-id="cd240-124">Abra a página **Cálculo** da página **Manutenção de versão de avaliação de custo** e selecione a class="sxs-lookup"><span data-stu-id="cd240-128">The from-date should be in the effective dates for the item’s BOM version and route version.</span></span> <span data-ttu-id="cd240-129">**Observação:** um registro de custo ausente pode indicar um novo item fabricado.</span><span class="sxs-lookup"><span data-stu-id="cd240-129">**Note:** A missing cost record can indicate a new manufactured item.</span></span> <span data-ttu-id="cd240-130">Os cálculos de BOM podem ser limitados a itens que têm custos ausentes.</span><span class="sxs-lookup"><span
