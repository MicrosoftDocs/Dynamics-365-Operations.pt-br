# <a name="asset-attribute-overview"></a>Visão geral de atributos de ativos

[!include [banner](../../includes/banner.md)]

 

Este tópico oferece uma visão geral sobre atributos de ativo no Asset Management. Os atributos do ativo são as propriedades relacionadas a um tipo de ativo ou a um ativo. Se tiver configurado [tipos de atributo](../setup-for-functional-locations/specification-types.md) e os usado em ativos, você poderá obter uma visão geral dos valores de atributo que estão configurados nos ativos. Esta visão geral é mostrada em duas páginas no Asset Management: **Visão geral de atributo de ativo** e **Atributo de ativo**.

A página **Visão geral de atributo de ativo** mostra uma linha separada para cada ativo. A lista mostra todos os tipos de atributo relacionados a esse ativo. Siga estas etapas para abrir e usar a página **Visão geral de atributo de ativo**.

1. Selecione **Asset Management** \> **Consultas** \> **Ativos** \> **Visão geral de atributo de ativo**.
2. No campo **Tipo de ativo**, selecione um tipo de ativo.
3. Selecione **OK**. A lista mostra todos os ativos que usam o tipo de ativo selecionado. Para cada ativo, todos os tipos de atributo de ativo relacionados são mostrados na mesma linha.

A página **Atributo de ativo** mostra uma linha separada para cada tipo de atributo usado em um ativo. Para abrir esta página, selecione **Asset Management** \> **Consultas** \> **Ativos** \> **Atributo de ativo**.
