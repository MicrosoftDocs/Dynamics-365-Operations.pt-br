# <a name="introduction-to-functional-locations"></a><span data-ttu-id="7f989-103">Introdução a locais funcionais</span><span class="sxs-lookup"><span data-stu-id="7f989-103">Introduction to functional locations</span></span>

[!include [banner](../../includes/banner.md)]

 

<span data-ttu-id="7f989-104">Este tópico oferece uma visão geral de locais funcionais em Gerenciamento de Ativos.</span><span class="sxs-lookup"><span data-stu-id="7f989-104">This topic provides an overview of functional locations in Asset Management.</span></span> <span data-ttu-id="7f989-105">Os locais funcionais são elementos de uma estrutura técnica, como unidades funcionais em um sistema.</span><span class="sxs-lookup"><span data-stu-id="7f989-105">Functional locations are elements of a technical structure, such as the functional units in a system.</span></span> <span data-ttu-id="7f989-106">Os locais funcionais são criados hierarquicamente e você instala ativos neles.</span><span class="sxs-lookup"><span data-stu-id="7f989-106">Functional locations are created hierarchically, and you install assets on them.</span></span> <span data-ttu-id="7f989-107">A configuração de locais funcionais na sua empresa depende dos requisitos da empresa.</span><span class="sxs-lookup"><span data-stu-id="7f989-107">The setup of functional locations in your company depends on the company's requirements.</span></span>

<span data-ttu-id="7f989-108">Estes são alguns exemplos de como você pode usar locais funcionais:</span><span class="sxs-lookup"><span data-stu-id="7f989-108">Here are some examples of how you can use functional locations:</span></span>

- <span data-ttu-id="7f989-109">**Funcional** – Os locais funcionais podem ser orientados ao usuário e usados para gerenciar ativos que têm comportamento semelhante.</span><span class="sxs-lookup"><span data-stu-id="7f989-109">**Functional** – The functional locations can be user-oriented and used to manage assets that have similar behavior.</span></span>

<span data-ttu-id="7f989-112">Cada local funcional é gerenciado independentemente em Gerenciamento de Ativos.</span><span class="sxs-lookup"><span data-stu-id="7f989-112">Each functional location is managed independently in Asset Management.</span></span> <span data-ttu-id="7f989-113">Estes são alguns dos recursos úteis de locais funcionais:</span><span class="sxs-lookup"><span data-stu-id="7f989-113">Here are some of the useful features of functional locations:</span></span>

- <span data-ttu-id="7f989-114">Configure especificações de local funcional.</span><span class="sxs-lookup"><span data-stu-id="7f989-114">Set up functional location specifications.</span></span>
- <span data-ttu-id="7f989-115">Configure requisitos de especificação de ativos.</span><span class="sxs-lookup"><span data-stu-id="7f989-115">Set up asset specification requirements.</span></span>
- <span data-ttu-id="7f989-116">Configure sequências de manutenção para manutenção preventiva e reativa.</span><span class="sxs-lookup"><span data-stu-id="7f989-116">Set up maintenance sequences for preventive and reactive maintenance.</span></span>
- <span data-ttu-id="7f989-117">Gerenciar ativos instalados.</span><span class="sxs-lookup"><span data-stu-id="7f989-117">Manage installed assets.</span></span>
- <span data-ttu-id="7f989-118">Rastreie solicitações de ativos e ordens de serviço relacionadas a ativos instalados.</span><span class="sxs-lookup"><span data-stu-id="7f989-118">Track active requests and work orders that are related to installed assets.</span></span>
- <span data-ttu-id="7f989-119">Rastreie falhas registrados em ativos.</span><span class="sxs-lookup"><span data-stu-id="7f989-119">Track faults that are registered on assets.</span></span>
- <span data-ttu-id="7f989-120">Rastreie custos de manutenção nos ativos que estão relacionados a um local funcional em determinado momento.</span><span class="sxs-lookup"><span data-stu-id="7f989-120">Track maintenance costs on the assets that are related to a functional location at any given time.</span></span>

<span data-ttu-id="7f989-121">Os locais funcionais fornecem o rastreamento de ativos em relação a solicitações, ordens de serviço, registros de falhas, avaliações de condição, registros de interrupção de produção e registros de contador de ativos.</span><span class="sxs-lookup"><span data-stu-id="7f989-121">Functional locations provide traceability of assets in relation to requests, work orders, fault registrations, condition assessments, production stop registrations, and asset counter registrations.</span></span>

> [!NOTE]
> <span data-ttu-id="7f989-122">Se um ativo for instalado em vários locais funcionais no seu tempo de vida, os custo poderão ser relativos a cada local.</span><span class="sxs-lookup"><span data-stu-id="7f989-122">Even if an asset is installed on various functional locations during its lifetime, the costs can be related to each location.</span></span> <span data-ttu-id="7f989-123">Ou seja, os custos de ativos são sempre relacionadas ao local funcional no qual o ativo foi instalado em algum momento.</span><span class="sxs-lookup"><span data-stu-id="7f989-123">In other words, asset costs are always related to the functional location that the asset was installed on at a given time.</span></span>

<span data-ttu-id="7f989-124">Os locais funcionais **não** são flexíveis.</span><span class="sxs-lookup"><span data-stu-id="7f989-124">Functional locations are **not** flexible.</span></span> <span data-ttu-id="7f989-125">Portanto, após configurar uma hierarquia de locais funcionais, você não poderá mover locais ao redor dela.</span><span class="sxs-lookup"><span data-stu-id="7f989-125">Therefore, after you set up a functional location hierarchy, you can't move locations around in it.</span></span> 

<span data-ttu-id="7f989-126">Depois de criar uma hierarquia de locais funcionais, a próxima etapa será instalar ativos nela.</span><span class="sxs-lookup"><span data-stu-id="7f989-126">After you create a functional location hierarchy, the next step is to install assets on it.</span></span> <span data-ttu-id="7f989-127">Para obter mais informações, consulte [Instalar ativos em locais funcionais](../functional-locations/install-objects-on-functional-locations.md).</span><span class="sxs-lookup"><span data-stu-id="7f989-127">For more information, see [Install assets on functional locations](../functional-locations/install-objects-on-functional-locations.md).</span></span>
