10. <span data-ttu-id="2ac4f-133">No campo **Tipo**, selecione o tipo de mensagem (informações, aviso, erro) a ser mostrado se ocorrerem desvios fora do intervalo definido quando você fizer o registro manual de contadores.</span><span class="sxs-lookup"><span data-stu-id="2ac4f-133">In the **Type** field, select the type of message (information, warning, error) to be shown if deviations outside the defined range occur when you make manual counter registrations.</span></span>
