<span data-ttu-id="8926f-104">O tópico explica como criar a medida de ativos no Gerenciamento de Ativos.</span><span class="sxs-lookup"><span data-stu-id="8926f-104">The topic explains how to create asset measure types in Asset Management.</span></span> <span data-ttu-id="8926f-105">Os tipos de medidas de ativos são usados para fazer registros de medidas em ativos, por exemplo, em relação ao número de horas de produção ou à quantidade produzida no ativo.</span><span class="sxs-lookup"><span data-stu-id="8926f-105">Asset measure types are used to make measurement registrations on assets, for example, regarding number of production hours, or quantity produced on the asset.</span></span> <span data-ttu-id="8926f-106">Os tipos de ativos são relacionados aos tipos de medida de ativos.</span><span class="sxs-lookup"><span data-stu-id="8926f-106">Asset types are related to the asset measure types.</span></span> <span data-ttu-id="8926f-107">Isso significa que uma medida de ativo só poderá ser usada em um ativo se a medida de ativo estiver configurada no tipo de ativo usado no ativo.</span><span class="sxs-lookup"><span data-stu-id="8926f-107">This means that an asset measure can only be used on an asset if the asset measure is set up on the asset type used on the asset.</span></span>

<span data-ttu-id="8926f-110">As medidas de ativos podem ser usadas em planos de manutenção.</span><span class="sxs-lookup"><span data-stu-id="8926f-110">Asset measures can be used on maintenance plans.</span></span> <span data-ttu-id="8926f-111">Uma linha do plano de manutenção pode ser do tipo "Contador", por exemplo, em relação ao número de horas de produção ou à quantidade produzida.</span><span class="sxs-lookup"><span data-stu-id="8926f-111">A maintenance plan line can be of type "Counter", for example, relating to number of production hours or quantity produced.</span></span> 

<span data-ttu-id="8926f-112">Um registro de medida de ativos pode ser atualizado de forma manual ou automática com base nas horas de produção ou na quantidade produzida.</span><span class="sxs-lookup"><span data-stu-id="8926f-112">An asset measurement registration can be updated manually or automatically based on production hours or quantity produced.</span></span> <span data-ttu-id="8926f-113">Uma medida de ativos pode ser configurada para usar um entre três métodos de atualização (selecionado no campo **Atualizar** em **Contadores**):</span><span class="sxs-lookup"><span data-stu-id="8926f-113">An asset measure can be set up to use one of three update methods (selected in the **Update** field in **Counters**):</span></span>

>[!NOTE]
><span data-ttu-id="8926f-117">Se a quantidade produzida for utilizada, *todos* os itens registrados serão incluídos no registro de medida, com a quantidade de acertos e a quantidade de erros.</span><span class="sxs-lookup"><span data-stu-id="8926f-117">If quantity produced is used, *all* registered items are included in the measurement registration, good quantity as well as error quantity.</span></span> <span data-ttu-id="8926f-118">É sempre possível fazer registros manuais de medida de ativos, se necessário.</span><span class="sxs-lookup"><span data-stu-id="8926f-118">It is always possible to make manual asset measurement registrations, if required.</span></span>

## <a name="create-counter-types-for-asset-counter-registrations"></a><span data-ttu-id="8926f-119">Crie tipos de contador para registros de contador de ativos</span><span class="sxs-lookup"><span data-stu-id="8926f-119">Create counter types for asset counter registrations</span></span>

2. <span data-ttu-id="8926f-121">Selecione **Novo** para criar um novo tipo de medida de ativo.</span><span class="sxs-lookup"><span data-stu-id="8926f-121">Select **New** to create a new asset measure type.</span></span>
3. <span data-ttu-id="8926f-122">Insira uma ID no campo **Contador** e um nome de contador no campo **Nome**.</span><span class="sxs-lookup"><span data-stu-id="8926f-122">Insert an ID in the **Counter** field, and a counter name in the **Name** field.</span></span>
4. <span data-ttu-id="8926f-123">Na Guia Rápida **Geral**, selecione uma unidade de medida no campo **Unidade**.</span><span class="sxs-lookup"><span data-stu-id="8926f-123">On the **General** FastTab, select a measurement unit in the **Unit** field.</span></span>
5. <span data-ttu-id="8926f-124">No campo **Atualizar**, selecione o método de atualização a ser usado para a medida do ativo.</span><span class="sxs-lookup"><span data-stu-id="8926f-124">In the **Update** field, select the update method to be used for the asset measure.</span></span>
6. <span data-ttu-id="8926f-125">Selecione "Sim" no botão de alternância **Herdar valores do contador** se os ativos filho em uma estrutura de ativos herdarem automaticamente registros de medida de ativos criados em ativos pai.</span><span class="sxs-lookup"><span data-stu-id="8926f-125">Select "Yes" on the **Inherit counter values** toggle button if child assets in an asset structure should automatically inherit asset measure registrations made on the parent asset.</span></span>

9. <span data-ttu-id="8926f-131">No campo **Desvio abaixo de**, insira o nível inferior em porcentagem para validar se o registro manual de medida de ativo está em um intervalo esperado.</span><span class="sxs-lookup"><span data-stu-id="8926f-131">In the **Deviation under** field, insert the lower level in percent for validating if manual asset measure registrations are within an expected range.</span></span> <span data-ttu-id="8926f-132">A validação é baseada em uma redução linear em registros existentes de medida de ativos.</span><span class="sxs-lookup"><span data-stu-id="8926f-132">The validation is based on a linear decrease in existing asset measure registrations.</span></span>
10. <span data-ttu-id="8926f-133">No campo **Tipo**, selecione o tipo de mensagem (informações, aviso, erro) a ser mostrado se ocorrerem desvios fora do intervalo definido quando você fizer o registro manual de medida de ativos.</span><span class="sxs-lookup"><span data-stu-id="8926f-133">In the **Type** field, select the type of message (information, warning, error) to be shown if deviations outside the defined range occur when you make manual asset measure registrations.</span></span>

12. <span data-ttu-id="8926f-135">Na Guia Rápida **Medidas de ativos relacionadas**, adicione as medidas de ativos a serem atualizadas automaticamente quando essa medida de ativos for atualizada.</span><span class="sxs-lookup"><span data-stu-id="8926f-135">On the **Related asset measures** FastTab, add the asset measures that you want to be automatically updated when this asset measure is updated.</span></span>
