# <a name="scheduled-work-order-maintenance-jobs"></a><span data-ttu-id="c0043-103">Trabalhos de manutenção da ordem de serviço agendados</span><span class="sxs-lookup"><span data-stu-id="c0043-103">Scheduled work order maintenance jobs</span></span>

[!include [banner](../../includes/banner.md)]

 

<span data-ttu-id="c0043-104">A página **Trabalhos de manutenção de ordens de serviço agendadas** mostra uma visão geral das ordens de serviço alocadas a um recurso.</span><span class="sxs-lookup"><span data-stu-id="c0043-104">The **Scheduled work order maintenance jobs** page shows an overview of the work orders allocated to a resource.</span></span> <span data-ttu-id="c0043-105">As ordens de serviço usando tipos de recurso "Recursos humanos", "Ferramenta" e "Máquina" são exibidas.</span><span class="sxs-lookup"><span data-stu-id="c0043-105">Work orders using resource types "Human resources", "Tool", and "Machine" are shown.</span></span> <span data-ttu-id="c0043-106">Por exemplo, se um funcionário de manutenção falte ao trabalho por motivo de doença, você pode usar esta página para localizar rapidamente as ordens de serviço alocadas ao funcionário e, em seguida, alocar outro funcionário de manutenção ao trabalho.</span><span class="sxs-lookup"><span data-stu-id="c0043-106">For example, if a maintenance worker calls in sick, you can use this page to quickly find work orders allocated to the worker, and then allocate another maintenance worker to the job.</span></span>

## <a name="view-scheduled-work-order-maintenance-jobs"></a><span data-ttu-id="c0043-107">Exibir trabalhos de manutenção de ordens de serviço agendadas</span><span class="sxs-lookup"><span data-stu-id="c0043-107">View scheduled work order maintenance jobs</span></span>

1. <span data-ttu-id="c0043-108">Clique em **Gerenciamento de ativos** > **Comum** > **Ordens de serviço** > **Trabalhos de manutenção de ordens de serviço agendadas**.</span><span class="sxs-lookup"><span data-stu-id="c0043-108">Click **Asset management** > **Common** > **Work orders** > **Scheduled work order maintenance jobs**.</span></span> <span data-ttu-id="c0043-109">Você verá uma lista de todas as ordens de serviço definidas para o estado de ciclo de vida da ordem de serviço "Agendado" ou "Em progresso".</span><span class="sxs-lookup"><span data-stu-id="c0043-109">You see a list of all work orders set to work order lifecycle state "Scheduled" or "In progress".</span></span>

2. <span data-ttu-id="c0043-110">Você pode classificar a lista, por exemplo, pelo funcionário de manutenção.</span><span class="sxs-lookup"><span data-stu-id="c0043-110">You can sort the list, for example, by maintenance worker.</span></span> <span data-ttu-id="c0043-111">Você também pode usar o filtro para limitar a lista a exibir ordens de serviço alocadas a um recurso específico ou funcionário de manutenção.</span><span class="sxs-lookup"><span data-stu-id="c0043-111">You can also use the filter to limit the list to display work orders allocated to a specific resource or maintenance worker.</span></span>

3. <span data-ttu-id="c0043-112">Você pode ver as notas na ordem de serviço e, se necessário, adicionar novas notas selecionando o trabalho da ordem de serviço e clicando em **Notas**.</span><span class="sxs-lookup"><span data-stu-id="c0043-112">You can see notes on the work order and, if required, add new notes by selecting the work order job, and then click **Notes**.</span></span>

4. <span data-ttu-id="c0043-113">Se desejar alocar um funcionário de manutenção a uma ordem de serviço, selecione a ordem de serviço na lista e, em seguida, clique em **Ordem de serviço**.</span><span class="sxs-lookup"><span data-stu-id="c0043-113">If you want to allocate one maintenance worker to a work order, select the work order, and then click **Work order**.</span></span>

5. <span data-ttu-id="c0043-114">A página **Ordem de serviço** é aberta.</span><span class="sxs-lookup"><span data-stu-id="c0043-114">The **Work order** page opens.</span></span> <span data-ttu-id="c0043-115">Clique em **Expedir** para agendar a ordem de serviço a um funcionário de manutenção específico.</span><span class="sxs-lookup"><span data-stu-id="c0043-115">Click **Dispatch** to schedule the work order to a specific maintenance worker.</span></span>
