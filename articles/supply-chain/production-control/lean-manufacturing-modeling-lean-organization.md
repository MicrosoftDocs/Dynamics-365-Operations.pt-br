## <a name="production-flows-in-lean-manufacturing"></a><span data-ttu-id="bbc9c-108">Fluxos de Produção em Lean Manufacturing</span><span class="sxs-lookup"><span data-stu-id="bbc9c-108">Production flows in lean manufacturing</span></span>
<span data-ttu-id="bbc9c-109">Os cenários de produção baseados em ordens de produção, o material é emitido para uma ordem de produção específica.</span><span class="sxs-lookup"><span data-stu-id="bbc9c-109">In production scenarios that are based on production orders, material is issued to a specific production order.</span></span> <span data-ttu-id="bbc9c-110">Durante uma sequência de operações, com base em uma lista de materiais (BOM) e nos roteiros, os produtos são criados e finalmente recebidos na localização fornecida.</span><span class="sxs-lookup"><span data-stu-id="bbc9c-110">During a sequence of operations that is based on a bill of materials (BOM) and routes, products are created and are finally received at the supplied location.</span></span> <span data-ttu-id="bbc9c-111">As horas de produtividade das ordens de produção variam em intervalos de minutos a semanas.</span><span class="sxs-lookup"><span data-stu-id="bbc9c-111">The throughput time of production orders varies from minutes to weeks.</span></span> <span data-ttu-id="bbc9c-112">Todo o custo, material e trabalho relacionados são acumulados na ordem de produção.</span><span class="sxs-lookup"><span data-stu-id="bbc9c-112">All related cost, material, and labor are accumulated on the production order.</span></span> 

<span data-ttu-id="bbc9c-113">Para reduzir prazos de entrega e o estoque excedente entre os centros de trabalho causados pela produção de lote, o lean manufacturing apresenta o reabastecimento do kanban e supermercados na fabricação e no reabastecimento do depósito.</span><span class="sxs-lookup"><span data-stu-id="bbc9c-113">To reduce the delivery lead times and excess inventory between work centers that batch production causes, lean manufacturing introduces kanban replenishment and supermarkets in manufacturing and warehouse replenishment.</span></span> <span data-ttu-id="bbc9c-114">Isso geralmente interrompe a produção de ciclos parcialmente independentes do kanban.</span><span class="sxs-lookup"><span data-stu-id="bbc9c-114">Typically, these features disrupt the production of partially independent kanban cycles.</span></span> <span data-ttu-id="bbc9c-115">O reabastecimento de um kanban para um produto semiacabado não é mais disparado por uma ordem para um produto acabado.</span><span class="sxs-lookup"><span data-stu-id="bbc9c-115">The replenishment of a kanban for a semi-finished product is no longer triggered by an order for a finished product.</span></span> 

<span data-ttu-id="bbc9c-116">Para restabelecer um contexto de produção e custo para os vários cenários do kanban propostos no Microsoft Dynamics 365 for Finance and Operations, Enterprise edition, os fluxos de produção baseados em atividade são apresentados como a base do lean manufacturing.</span><span class="sxs-lookup"><span data-stu-id="bbc9c-116">To re-establish a production and cost context for the various kanban scenarios that are proposed in Microsoft Dynamics 365 for Finance and Operations, Enterprise edition, activity-based production flows have been introduced as the backbone of lean manufacturing.</span></span> <span data-ttu-id="bbc9c-117">Todas as regras kanban se referem a essa estrutura predefinida.</span><span class="sxs-lookup"><span data-stu-id="bbc9c-117">All kanban rules refer to this predefined structure.</span></span> <span data-ttu-id="bbc9c-118">O módulo com base em atividade suporta a configuração de uma variação maior de cenários que as versões anteriores de lean manufacturing para Dynamics AX suportavam.</span><span class="sxs-lookup"><span data-stu-id="bbc9c-118">The activity-based model supports the setup of a wider range of scenarios than previous versions of Lean manufacturing for Dynamics AX supported.</span></span> <span data-ttu-id="bbc9c-119">No entanto, este módulo não adiciona a complexidade para trabalhadores de chão de fábrica, como todos os cenários usam a mesma interface do usuário com base na atividade.</span><span class="sxs-lookup"><span data-stu-id="bbc9c-119">However, this model doesn't add complexity for the shop floor workers, because all scenarios use the same activity-based user interface.</span></span>
