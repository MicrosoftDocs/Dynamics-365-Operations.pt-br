# <a name="turn-on-the-landed-cost-module-for-your-system"></a>Ativar o módulo Custo de entrega para seu sistema

[!include [banner](../includes/banner.md)]

Para que você possa usar o módulo Custo de entrega, ele deve estar ativado no sistema. Os administradores podem usar as configurações do [gerenciamento de recursos](../../fin-ops-core/fin-ops/get-started/feature-management/feature-management-overview.md) para verificar o status do recurso e ativá-lo conforme o necessário. No espaço de trabalho **Gerenciamento de recursos**, o recurso está listado da seguinte forma:

- **Módulo:** *Gerenciamento de transporte*
- **Nome do recurso:** *Custo de entrega*
