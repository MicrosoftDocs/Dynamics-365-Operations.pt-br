<span data-ttu-id="847de-117">No custo de entrega, uma viagem é um movimento distinto de um local de saída, por meio de um conjunto específico de destinos em um período específico, para um local de depósito de entrada especificado.</span><span class="sxs-lookup"><span data-stu-id="847de-117">In Landed cost, a voyage is a distinct movement from an outbound location, through a specific set of destinations over a specified period, to a specified inbound warehouse location.</span></span> <span data-ttu-id="847de-118">As ordens de compra e linhas de ordem podem ser adicionadas a um único contêiner ou a vários contêineres para uma viagem, e os custos serão corretamente alocados novamente para a linha do item.</span><span class="sxs-lookup"><span data-stu-id="847de-118">Purchase orders and order lines can be added to either a single container or multiple containers for a voyage, and the costs will be correctly allocated back to the item line.</span></span> <span data-ttu-id="847de-119">Também é possível adicionar ordens e linhas de ordem em entidades legais para uma única viagem.</span><span class="sxs-lookup"><span data-stu-id="847de-119">Orders and order lines can also be added across legal entities for a single voyage.</span></span>

<span data-ttu-id="847de-127">Quando você cria uma viagem no custo de entrega, os custos podem ser adicionados automaticamente a ela.</span><span class="sxs-lookup"><span data-stu-id="847de-127">When you create a voyage in Landed cost, costs can automatically be added to it.</span></span> <span data-ttu-id="847de-128">Esses custos são configurados no custo de entrega; várias opções de custo e bases de custo estão disponíveis para eles.</span><span class="sxs-lookup"><span data-stu-id="847de-128">These costs are set up in Landed cost, and various cost options and cost bases are available for them.</span></span> <span data-ttu-id="847de-129">Cada custo pode ser configurado para diferentes níveis de uma viagem e distribuído para o nível de item por meio de regras de divisão que dão suporte a quantidade, volume, peso, valor e divisores volumétricos definidos.</span><span class="sxs-lookup"><span data-stu-id="847de-129">Each cost can be set up for different levels of a voyage and apportioned to the item level through apportionment rules that support quantity, volume, weight, amount, and defined volumetric divisors.</span></span> <span data-ttu-id="847de-130">Esses custos estimados fornecem uma estimativa precisa de todos os custos de uma viagem.</span><span class="sxs-lookup"><span data-stu-id="847de-130">These estimated costs provide an accurate estimate of all costs for a voyage.</span></span>
