# <a name="landed-cost-entities-overview"></a>Visão geral de entidades de custo de entrega

[!include [banner](../includes/banner.md)]
[!INCLUDE [preview-banner](../includes/preview-banner.md)]
<!-- KFM: Preview until GA with 10.0.28 -->

O módulo **Custo de entrega** ajuda as empresas a simplificar operações de remessa de entrada, fornecendo o total controle financeiro e logístico do frete importado, do fabricante ao depósito. Entidades de dados de transação para Custo de entrega permitem que as fontes de dados externas (como serviços do controlador de frete) criem viagens e custos, e atualizem os registros de rastreamento de contêiner.

Estas são algumas das operações compatíveis:

- Crie e atualize registros de viagem, contêiner de remessa e fólio para as linhas de ordem de entrada existentes.
- Crie e atualize os custos previstos para cada área de custo.
- Crie e atualize registros de atividade de contêiner de remessa para capturar a data de início, a data final prevista e a data de término real de cada trecho da jornada do contêiner.
- Crie alocações de faturas de custo do fornecedor.
