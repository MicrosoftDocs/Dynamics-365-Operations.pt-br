# <a name="set-up-an-appointment-for-a-load"></a>Configurar um compromisso para uma carga

[!include[task guide banner](../../includes/task-guide-banner.md)]

Este procedimento mostra como definir e planejar um compromisso para uma doca de carga. Normalmente isso é feito por um coordenador de transporte. Você pode usar este procedimento na empresa USMF de dados de demonstração.


## <a name="select-the-load"></a>Selecione a carga
1. Vá para Gerenciamento de transporte > Planejamento > Bancada de planejamento de carga.
2. Desmarque a caixa de seleção Esconder envio e recebido.
3. Na lista, selecione custo indireto que tem um status Enviado.
4. Clique em Transporte.
5. Clique em Agendamento de compromisso.

## <a name="create-an-appointment"></a>Criar um compromisso
1. Clique em Novo.
2. No campo Regra de compromisso, insira ou selecione um valor.
3. Clique em Salvar.
4. Clique em Atualizar status.
5. Clique em Confirmar.
6. Clique em Salvar.
7. Feche a página.
