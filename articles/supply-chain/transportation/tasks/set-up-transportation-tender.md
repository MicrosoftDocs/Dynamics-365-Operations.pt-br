# <a name="set-up-a-transportation-tender"></a>Configurar uma proposta de transporte

[!include [banner](../../includes/banner.md)]

Este procedimento mostra como configurar um meio de pagamento do transporte. Normalmente isso é feito por um coordenador de transporte. Você pode usar este procedimento na empresa USMF de dados de demonstração.


## <a name="select-a-route"></a>Selecione uma rota
1. Vá para Gerenciamento de transporte > Planejamento > Bancada de planejamento de carga.
2. Desmarque a caixa de seleção Esconder envio e recebido.
3. Selecione a linha com ID de carga 00006.
4. Clique em Classificação e roteiro.
5. Clique em Rotas.

## <a name="create-the-transportation-tender"></a>Criar o meio de pagamento de transporte
1. Clique em Meios de pagamento do transporte.
2. Clique em Novo.
3. Expanda a seção Geral.
4. No campo Taxas solicitadas, insira um número.
5. Clique em Salvar.
6. Clique em Atualizar status.
7. Clique em Enviar.
8. Selecione uma rota.
