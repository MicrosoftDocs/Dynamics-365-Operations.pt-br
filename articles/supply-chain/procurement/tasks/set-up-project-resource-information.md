# <a name="set-up-project-resource-information"></a>Configurar informações de recursos do projeto

[!include [banner](../../includes/banner.md)]

Este procedimento mostra como configurar um recurso do projeto. Essa tarefa usa o conjunto de dados de USMF.

1. Vá para Gerenciamento e contabilidade de projeto > Recursos do projeto > Lista de recursos.
2. Clique em Novo recurso.
3. No campo Trabalhador, insira ou selecione um valor.
4. No campo Calendário, insira ou selecione um valor.
5. No campo Código de período, insira ou selecione um valor.
6. Clique em Criar.
7. Clique em Preços de custo.
8. Clique em Preço de custo - hora.
9. Clique em Novo.
10. No campo Data efetiva, insira uma data.
11. No campo Preço de custo, insira um número.
12. Clique em Salvar.
13. Feche a página.
14. Clique em Preços de venda.
15. Clique em Preço de vendas - hora.
16. Clique em Novo.
17. No campo Data efetiva, insira uma data.
18. No campo Definição de preços, insira um número.
19. No campo Válido para, selecione uma opção.
20. Clique em Salvar.
21. Feche a página.
