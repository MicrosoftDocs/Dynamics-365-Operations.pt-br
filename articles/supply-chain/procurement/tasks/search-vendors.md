# <a name="search-for-vendors"></a>Procurar fornecedores

[!include [task guide banner](../../includes/task-guide-banner.md)]

Aprenda como pesquisar fornecedores com base em critérios específicos. Este exemplo mostra como procurar fornecedores que estão aprovados para uma categoria de aquisição específica e têm seu endereço principal em um país específico. Você pode executar esse procedimento na empresa de dados demonstrativos USMF, ou usando seus próprios dados. Essa tarefa é tipicamente realizada por um profissional de aquisição.

1. Vá para Aquisição e fornecimento > Fornecedores > Pesquisa de fornecedores.
2. Clique no ícone Mais para abrir a página de seleção da categoria de aquisição.  
3. Na árvore, selecione 'CATEGORIAS DE AQUISIÇÃO CORPORATIVA\MÁQUINAS DE ESCRITÓRIO'.
    * Se você estiver executando este procedimento como um guia de tarefa, você talvez precise clicar no botão Desbloquear antes de poder selecionar o nó correto. Se você não estiver usando USMF, selecione uma das categorias que você tenha.  
4. Clique em Adicionar.
    * É possível selecionar mais de uma categoria aqui. Se você fizer isso, a pesquisa irá encontrar todos os fornecedores que estão aprovados para em pelo menos uma das categorias.  
5. Clique em OK.
6. No campo País/Região, digite um valor.
7. Clique em OK.
