<span data-ttu-id="3169d-104">Este procedimento mostra como criar uma ordem de compra do projeto.</span><span class="sxs-lookup"><span data-stu-id="3169d-104">This procedure shows you how to create a project purchase order.</span></span> <span data-ttu-id="3169d-105">Essa tarefa usa o conjunto de dados de USSI.</span><span class="sxs-lookup"><span data-stu-id="3169d-105">This task uses the USSI data set.</span></span>

1. <span data-ttu-id="3169d-106">Vá para Gerenciamento e contabilidade de projeto > Projetos > Todos os projetos.</span><span class="sxs-lookup"><span data-stu-id="3169d-106">Go to Project management and accounting > Projects > All projects.</span></span>
2. <span data-ttu-id="3169d-107">Na lista, clique no link na linha selecionada.</span><span class="sxs-lookup"><span data-stu-id="3169d-107">In the list, click the link in the selected row.</span></span>
3. <span data-ttu-id="3169d-108">No Painel de Ação, clique em Gerenciar.</span><span class="sxs-lookup"><span data-stu-id="3169d-108">On the Action Pane, click Manage.</span></span>
4. <span data-ttu-id="3169d-109">Clique em Tarefa de item.</span><span class="sxs-lookup"><span data-stu-id="3169d-109">Click Item task.</span></span>
5. <span data-ttu-id="3169d-110">Clique em Ordem de compra.</span><span class="sxs-lookup"><span data-stu-id="3169d-110">Click Purchase order.</span></span>
6. <span data-ttu-id="3169d-111">No campo Conta de fornecedor, insira ou selecione um valor.</span><span class="sxs-lookup"><span data-stu-id="3169d-111">In the Vendor account field, enter or select a value.</span></span>

    * <span data-ttu-id="3169d-113">Essas etapas não são obrigatórias, mas simplificam a ordem de compra, configurando um local padrão e um depósito para as linhas de ordem de compra.</span><span class="sxs-lookup"><span data-stu-id="3169d-113">These steps aren't required, but they do simplify the purchase order by setting up a default site and warehouse for the purchase order lines.</span></span>  
8. <span data-ttu-id="3169d-114">No campo Depósito, insira ou selecione um valor.</span><span class="sxs-lookup"><span data-stu-id="3169d-114">In the Warehouse field, enter or select a value.</span></span>
9. <span data-ttu-id="3169d-115">Clique em OK.</span><span class="sxs-lookup"><span data-stu-id="3169d-115">Click OK.</span></span>
10. <span data-ttu-id="3169d-116">Na lista, marque a linha selecionada.</span><span class="sxs-lookup"><span data-stu-id="3169d-116">In the list, mark the selected row.</span></span>
11. <span data-ttu-id="3169d-117">No campo Número do item, insira ou selecione um valor.</span><span class="sxs-lookup"><span data-stu-id="3169d-117">In the Item number field, enter or select a value.</span></span>
    * <span data-ttu-id="3169d-118">Pode ser o número de item ou uma categoria de compras.</span><span class="sxs-lookup"><span data-stu-id="3169d-118">This can be the item number or a procurement category.</span></span>

13. <span data-ttu-id="3169d-120">Clique na guia Projeto.</span><span class="sxs-lookup"><span data-stu-id="3169d-120">Click the Project tab.</span></span>
    * <span data-ttu-id="3169d-121">Verifique se as vendas e os preços de custo estão disponíveis.</span><span class="sxs-lookup"><span data-stu-id="3169d-121">Verify that the sales and cost prices are available.</span></span> <span data-ttu-id="3169d-122">Se não estiverem disponíveis, mas forem necessárias, insira as informações.</span><span class="sxs-lookup"><span data-stu-id="3169d-122">If they are not available but needed, enter the information.</span></span>
