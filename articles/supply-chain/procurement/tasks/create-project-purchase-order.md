# <a name="create-project-purchase-order"></a>Criar ordem de compra de projeto

[!include [banner](../../includes/banner.md)]

Este procedimento mostra como criar uma ordem de compra do projeto. Essa tarefa usa o conjunto de dados de USSI.

1. Vá para Gerenciamento e contabilidade de projeto > Projetos > Todos os projetos.
2. Na lista, clique no link na linha selecionada.
3. No Painel de Ação, clique em Gerenciar.
4. Clique em Tarefa de item.
5. Clique em Ordem de compra.
6. No campo Conta de fornecedor, insira ou selecione um valor.
7. No campo Local, insira ou selecione um valor.
    * Essas etapas não são obrigatórias, mas simplificam a ordem de compra, configurando um local padrão e um depósito para as linhas de ordem de compra.  
8. No campo Depósito, insira ou selecione um valor.
9. Clique em OK.
10. Na lista, marque a linha selecionada.
11. No campo Número do item, insira ou selecione um valor.
    * Pode ser o número de item ou uma categoria de compras.  
12. Expanda a seção Detalhes da linha.
13. Clique na guia Projeto.
    * Verifique se as vendas e os preços de custo estão disponíveis. Se não estiverem disponíveis, mas forem necessárias, insira as informações.  
14. Clique em Salvar.
