# <a name="create-dimension-based-configurations"></a>Criar configurações baseadas em dimensão

[!include [task guide banner](../../includes/task-guide-banner.md)]

Este procedimento mostra como definir uma configuração para um produto baseado na dimensão. Este é o último procedimento da série que explica como criar combinações para configuração baseada em dimensão. A execução desse procedimento depende dos dados criados nos sete registros anteriores. A empresa de dados demo usada para criar este procedimento é USMF.


## <a name="find-the-dimension-based-product-master"></a>Encontre o produto mestre baseado na dimensão
1. Clique em Manutenção de produto liberado.
2. Clique em Produtos liberados.
3. Na lista, marque a linha selecionada.
    * Selecione o produto mestre baseado na dimensão que você criou no primeiro registro nesta sequência de 8 registros.  

## <a name="create-configurations"></a>Criar configurações
1. No Painel de Ação Engenharia, clique em Manter configurações.
2. Clique em Configurar.
3. Na lista, marque a linha selecionada.
4. No campo Número do item, insira ou selecione um valor.
    * Selecione um dos itens no grupo de configuração.  
5. Na lista, localize e selecione o PDV desejado.
6. No campo Número do item, insira ou selecione um valor.
    * Selecione qualquer item do segundo grupo de configuração.  
7. Clique em OK.
8. Na lista, marque a linha selecionada.
9. No campo Configuração, digite um valor.
    * Insira um nome de configuração que seja de fácil identificação da configuração.  
10. No campo Descrição, digite um valor.
    * Insira uma descrição da configuração para explicar o que ela contém.  
11. Clique em OK.
