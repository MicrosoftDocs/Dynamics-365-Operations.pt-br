# <a name="create-a-bar-code-for-a-product"></a>Criar um código de barra para um produto

[!include [banner](../../includes/banner.md)]

Este tópico mostra como criar manualmente um código de barras usando o número de item M0001 como exemplo. A empresa de dados demo usada para criar este procedimento é USMF.

1. Selecione **Manutenção de produto liberado** na home page.
2. Vá para **Produtos > Produtos liberados** na seção **Links**.
3. Na lista, localize e selecione o registro desejado. Para este exemplo, selecione o número de item **M0001**.
4. No Painel de Ação, selecione **Gerenciar estoque**.
5. Selecione **Códigos de barras**.
6. Selecione **Novo**.
7. Marque a linha selecionada que é criada na lista a seguir.
8. No campo **Configuração de código de barras**, insira ou selecione um valor.
9. No campo **Código de barras**, insira ou selecione um valor.
10. No campo **Código de barras**, digite um valor  
11. No campo **Quantidade.**, insira um número
12. Selecione **Salvar**.
13. Feche a página.
