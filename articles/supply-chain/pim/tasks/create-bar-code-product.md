# <a name="create-a-bar-code-for-a-product"></a>Criar um código de barra para um produto

[!include [task guide banner](../../includes/task-guide-banner.md)]

Este procedimento mostra como criar manualmente um código de barras usando o número de item M0001 como exemplo. A empresa de dados demo usada para criar este procedimento é USMF.

1. Clique em Manutenção de produto liberado.
2. Clique em Produtos liberados.
3. Na lista, localize e selecione o registro desejado.
4. No Painel de Ação, clique em Gerenciar estoque.
5. Clique em Código de barras.
6. Clique em Novo.
7. Na lista, marque a linha selecionada.
8. No campo Configuração de código de barras, insira ou selecione um valor.
9. No campo Código de barras, insira ou selecione um valor.
10. No campo Código de barras, digite um valor.
    * Pressione a tecla Tab.  
11. Feche a página.
12. No campo Quantidade, insira um número.
13. Clique em Salvar.
    * Ao clicar em salvar, a verificação do código de barras é executada e, nesse caso, exibirá um erro informando que o dígito de verificação esperado é 8, mas que 3 foi encontrado. Atualize manualmente o número do código de barras para que 8 esteja no fim.  
14. No campo Código de barras, insira ou selecione um valor.
15. No campo Código de barras, digite um valor.
    * Pressione a tecla Tab.  
16. Feche a página.
17. Clique em Salvar.
18. Feche a página.
