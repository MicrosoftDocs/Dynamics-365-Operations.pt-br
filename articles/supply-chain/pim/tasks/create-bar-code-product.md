1. <span data-ttu-id="8af79-106">Clique em Manutenção de produto liberado.</span><span class="sxs-lookup"><span data-stu-id="8af79-106">Click Released product maintenance.</span></span>
2. <span data-ttu-id="8af79-107">Clique em Produtos liberados.</span><span class="sxs-lookup"><span data-stu-id="8af79-107">Click Released products.</span></span>
3. <span data-ttu-id="8af79-108">Na lista, localize e selecione o registro desejado.</span><span class="sxs-lookup"><span data-stu-id="8af79-108">In the list, find and select the desired record.</span></span>

11. <span data-ttu-id="8af79-117">Feche a página.</span><span class="sxs-lookup"><span data-stu-id="8af79-117">Close the page.</span></span>
12. <span data-ttu-id="8af79-118">No campo Quantidade, insira um número.</span><span class="sxs-lookup"><span data-stu-id="8af79-118">In the Quantity field, enter a number.</span></span>
