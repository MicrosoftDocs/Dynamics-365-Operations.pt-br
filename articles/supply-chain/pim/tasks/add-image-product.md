# <a name="add-an-image-to-a-product"></a>Adicionar uma imagem a um produto

[!include [banner](../../includes/banner.md)]

Este procedimento mostra como adicionar uma imagem a um produto. A empresa de dados de demonstração usada para criar esta tarefa é a USMF.

1. Vá para Gerenciamento de informações do produto > Produtos > Produtos liberados.
2. Expanda a imagem do produto FactBox.
3. Na lista, localize e selecione o PDV desejado.
    * Por exemplo, selecione o número de item C0002.  
4. Clique em Alterar imagem.
5. Clique em Novo.
    * Clique em Procurar. Encontre a imagem que deseja usar, selecione-a e depois feche a caixa de diálogo.
