# <a name="subscription-groups"></a>Grupos de assinaturas    

[!include [banner](../includes/banner.md)]


As subscrições compartilham as seguintes informações, que são definidas para o grupo de subscrições:

  - Código de sequência numérica da subscrição, se houver.

  - O período de faturamento para o qual as transações de subscrição são criadas. Por exemplo, se o código de período **Faturamento** no formulário **Grupos de subscrições** estiver definido como **Trimestral**, as transações de todas as subscrições anexadas ao grupo de subscrições serão criadas com base trimestral.

  - Com que frequência, se houver, a subscrição deverá ser acumulada.

Também é possível concluir as seguintes tarefas para todas as subscrições anexadas ao grupo de subscrições:

  - Criar transações de subscrição.

  - Atualizar o preço base ou o índice das subscrições.

  - Lançar a receita acumulada para transações de subscrição.
