# <a name="copy-service-templates-lines"></a><span data-ttu-id="42b73-103">Copiar linhas de modelos de serviço</span><span class="sxs-lookup"><span data-stu-id="42b73-103">Copy service templates lines</span></span> 

[!INCLUDE [banner](../includes/banner.md)]

<span data-ttu-id="42b73-104">As informações de um modelo podem ser copiadas para um contrato de serviço ou uma ordem de serviço.</span><span class="sxs-lookup"><span data-stu-id="42b73-104">Template information can be copied to either a service agreement or a service order.</span></span>

## <a name="copy-service-template-lines-into-a-service-agreement"></a><span data-ttu-id="42b73-105">Copiar as linhas do modelo de serviço para um contrato de serviço</span><span class="sxs-lookup"><span data-stu-id="42b73-105">Copy service template lines into a service agreement</span></span>

1. <span data-ttu-id="42b73-106">Clique em **Contratos de serviço**.</span><span class="sxs-lookup"><span data-stu-id="42b73-106">Click **Service agreements**.</span></span>
2. <span data-ttu-id="42b73-107">Clique duas vezes em um contrato de serviço para abrir a página **Contratos de serviço**.</span><span class="sxs-lookup"><span data-stu-id="42b73-107">Double-click a service agreement to open the **Service agreements** page.</span></span>
3. <span data-ttu-id="42b73-108">Clique em **Linhas de modelo** para abrir a página **Copiar linhas de modelo de serviço**.</span><span class="sxs-lookup"><span data-stu-id="42b73-108">Click **Template lines** to open the **Copy service template lines** page.</span></span>
4. <span data-ttu-id="42b73-109">Na guia **Controle em árvore**, expanda a árvore, localize o modelo que contém as linhas que você deseja copiar para o contrato de serviço e selecione o modelo de serviço.</span><span class="sxs-lookup"><span data-stu-id="42b73-109">On the **Tree control** tab, expand the tree, find the template that contains the lines that you want to copy to the service agreement, and then select the service template.</span></span>

6. <span data-ttu-id="42b73-111">Clique em **OK**.</span><span class="sxs-lookup"><span data-stu-id="42b73-111">Click **OK**.</span></span>
7. <span data-ttu-id="42b73-112">Feche as páginas para salvar suas alterações.</span><span class="sxs-lookup"><span data-stu-id="42b73-112">Close the pages to save your changes.</span></span>

<span data-ttu-id="42b73-113">Agora as linhas do modelo de serviço serão copiadas para o contrato de serviço, junto com as relações associadas de tarefas e objetos de serviço.</span><span class="sxs-lookup"><span data-stu-id="42b73-113">The service template lines are now copied to the service agreement, together with the attached service object and service task relations.</span></span>
