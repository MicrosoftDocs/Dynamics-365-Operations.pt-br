# <a name="copy-service-templates-lines"></a>Copiar linhas de modelos de serviço 

[!include [banner](../includes/banner.md)]

As informações de um modelo podem ser copiadas para um contrato de serviço ou uma ordem de serviço.

## <a name="copy-service-template-lines-into-a-service-agreement"></a>Copiar as linhas do modelo de serviço para um contrato de serviço

1. Clique em **Contratos de serviço**.
2. Clique duas vezes em um contrato de serviço para abrir a página **Contratos de serviço**.
3. Clique em **Linhas de modelo** para abrir a página **Copiar linhas de modelo de serviço**.
4. Na guia **Controle em árvore**, expanda a árvore, localize o modelo que contém as linhas que você deseja copiar para o contrato de serviço e selecione o modelo de serviço.
5. Na guia **Linhas**, marque a caixa de seleção **Selecionar** correspondente às linhas do modelo de serviço que você deseja copiar.
6. Clique em **OK**.
7. Feche as páginas para salvar suas alterações.

Agora as linhas do modelo de serviço serão copiadas para o contrato de serviço, junto com as relações associadas de tarefas e objetos de serviço.

## <a name="copy-service-template-lines-into-a-service-order"></a>Copiar as linhas do modelo de serviço para uma ordem de serviço

1. Clique em **Ordens de serviço**.
2. Clique duas vezes em uma ordem de serviço para abrir a página **Ordens de serviço**.
3. Clique em Copiar \> Copiar linhas do modelo para abrir a página **Copiar linhas de modelo de serviço**.
4. Na guia **Controle em árvore**, expanda a árvore, localize o modelo que contém as linhas que você deseja copiar para a ordem de serviço e selecione o modelo de serviço.
5. Na guia **Linhas**, marque a caixa de seleção **Selecionar** correspondente às linhas do modelo de serviço que você deseja copiar.
6. Clique em **OK**.
7. Feche as páginas para salvar suas alterações.

Agora as linhas do modelo de serviço serão copiadas para a nova ordem de serviço, junto com as relações associadas de tarefas e objetos de serviço.
