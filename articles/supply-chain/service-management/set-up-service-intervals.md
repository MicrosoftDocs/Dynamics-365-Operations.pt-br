# <a name="set-up-service-intervals"></a><span data-ttu-id="7e585-103">Configurar intervalos de serviço</span><span class="sxs-lookup"><span data-stu-id="7e585-103">Set up service intervals</span></span>  

[!include [banner](../includes/banner.md)]

<span data-ttu-id="7e585-104">O intervalo de serviço indica a frequência com a qual as linhas de ordem de serviço são criadas para linhas de contrato de serviço quando você cria ordens de serviço.</span><span class="sxs-lookup"><span data-stu-id="7e585-104">Service interval indicates the frequency with which service order lines are created for service agreement lines when you create service orders.</span></span>

1. <span data-ttu-id="7e585-105">Clique em **Gerenciamento de serviços** \> **Configuração** \> **Contratos de serviço** \> **Intervalos de serviço**.</span><span class="sxs-lookup"><span data-stu-id="7e585-105">Click **Service management** \> **Setup** \> **Service agreements** \> **Service intervals**.</span></span>
2. <span data-ttu-id="7e585-106">Crie um novo intervalo de serviço.</span><span class="sxs-lookup"><span data-stu-id="7e585-106">Create a new service interval.</span></span>
3. <span data-ttu-id="7e585-107">Insira a ID e a descrição do intervalo de serviço.</span><span class="sxs-lookup"><span data-stu-id="7e585-107">Enter the ID and description of the service interval.</span></span>
4. <span data-ttu-id="7e585-108">No campo **Intervalo**, selecione o intervalo.</span><span class="sxs-lookup"><span data-stu-id="7e585-108">In the **Range** field, select the range.</span></span>
5. <span data-ttu-id="7e585-109">No campo **Frequência**, digite a frequência.</span><span class="sxs-lookup"><span data-stu-id="7e585-109">In the **Frequency** field, type the frequency.</span></span> <span data-ttu-id="7e585-110">A frequência é o fator pelo qual você deve multiplicar o intervalo para obter o intervalo de um contrato de serviço.</span><span class="sxs-lookup"><span data-stu-id="7e585-110">The frequency is the factor by which you must multiply the range to obtain the interval for a service agreement.</span></span>
6. <span data-ttu-id="7e585-111">Pressione **Alt+S** para salvar o intervalo de serviço.</span><span class="sxs-lookup"><span data-stu-id="7e585-111">Press **Alt+S** to save the service interval.</span></span>
