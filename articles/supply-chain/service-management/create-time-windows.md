# <a name="create-time-windows"></a>Criar janelas de horas   

[!include [banner](../includes/banner.md)]

Você pode usar janelas de tempo para otimizar o agendamento das linhas de ordem de serviço.

1. Clique em **Gerenciamento de serviços** \> **Configuração** \> **Contratos de serviço** \> **Janela de tempo**.
2. Pressione **Alt+N** para criar uma nova janela de tempo.
3. No campo **Janela de tempo**, insira um nome para a janela de tempo.
4. No campo **Descrição**, insira uma descrição para a janela de tempo
5. No campo **Método**, selecione o método a ser criado automaticamente. Se o método for do tipo **Manual**, especifique o número de dias antes ou depois da data calculada que a linha da ordem de serviço pode ser movida.
