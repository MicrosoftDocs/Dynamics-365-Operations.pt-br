# <a name="set-up-dispatch-teams"></a>Configurar equipes de expedição 

[!include [banner](../includes/banner.md)]


Este tópico descreve como criar uma equipe de expedição. Depois de criar uma equipe de expedição, você pode atribuir trabalhadores à equipe. Quando você cria uma ordem de serviço, você pode relacionar um membro de uma equipe de expedição a uma atividade na ordem de serviço.

1.  Clique **Gerenciamento de serviço** \> **Configuração** \> **Equipes de expedição**.

2.  No formulário **Equipes de expedição**, clique em **Novo** para criar uma nova equipe.

3.  No campo **Equipe de expedição**, digite um nome exclusivo para a equipe de expedição.

4.  No campo **Proprietário**, selecione o funcionário responsável por esta equipe.
    

    > [!NOTE]
    > <P>É possível selecionar somente os funcionários nesse campo. Você não pode selecionar fornecedores.</P>



5.  Opcional: No campo **Descrição**, digite uma breve descrição desta equipe de expedição.


> [!NOTE]
> <P>Você pode atribuir um trabalhador à esta equipe de expedição no formulário do <STRONG>Trabalhador</STRONG>. Para obter mais informações sobre como atribuir um trabalhador a uma equipe de expedição, consulte <A href="/dynamicsax-2012//dispatch-workers-form">Trabalhadores de expedição (formulário)</A>.</P>
