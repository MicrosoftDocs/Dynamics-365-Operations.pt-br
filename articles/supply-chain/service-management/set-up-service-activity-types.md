# <a name="set-up-service-activity-types"></a>Configurar tipos de atividade de serviço 

[!include [banner](../includes/banner.md)]


1.  Clique em **Vendas e marketing** \> **Configuração** \> **Atividades** \> **Tipos de atividade**.
    
    –ou–
    
    Clique em **Administração da organização** \> **Configuração** \> **Atividades** \> **Tipos de atividade**.

2.  No formulário **Tipos de atividade**, pressione CTRL+N para adicionar uma nova linha de tipo de atividade.

3.  No campo **Tipo**, digite um nome para o tipo de atividade.

4.  No campo **Descrição**, digite o texto que descreve o tipo de atividade.

5.  Feche o formulário para salvar suas alterações.
