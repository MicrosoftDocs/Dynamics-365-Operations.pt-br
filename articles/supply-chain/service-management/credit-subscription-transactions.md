## <a name="credit-subscription-transactions"></a><span data-ttu-id="72688-103">Creditar transações de subscrição</span><span class="sxs-lookup"><span data-stu-id="72688-103">Credit subscription transactions</span></span>

1.  <span data-ttu-id="72688-104">Clique em **Gerenciamento de serviço** \> **Comum** \> **Subscrições de serviço** \> **Todas as subscrições de serviço**.</span><span class="sxs-lookup"><span data-stu-id="72688-104">Click **Service management** \> **Common** \> **Service subscriptions** \> **All service subscriptions**.</span></span>

2.  <span data-ttu-id="72688-105">Selecione a subscrição anexada à transação de subscrição para a qual deseja criar uma nota de crédito.</span><span class="sxs-lookup"><span data-stu-id="72688-105">Select the subscription attached to the subscription transaction for which you want to create a credit note.</span></span>

3.  <span data-ttu-id="72688-106">Selecione a guia **Analisar** e, em seguida, clique no botão **Transações de taxa** no Painel de Ações.</span><span class="sxs-lookup"><span data-stu-id="72688-106">Select the **Analyze** tab, and then click the **Fee transactions** button on the Action Pane.</span></span>

4.  <span data-ttu-id="72688-107">No formulário **Transações de taxa**, selecione a transação para a qual deseja criar uma nota de crédito.</span><span class="sxs-lookup"><span data-stu-id="72688-107">From the **Fee transactions** form, select the transaction for which you want to create a credit note.</span></span>

5.  <span data-ttu-id="72688-108">Clique em **Funções** \> **Selecionar para a nota de crédito**.</span><span class="sxs-lookup"><span data-stu-id="72688-108">Click **Functions** \> **Select for credit note**.</span></span>

6.  <span data-ttu-id="72688-109">No formulário **Selecionar para nota de crédito**, selecione a transação que deseja creditar e, em seguida, clique em **OK**.</span><span class="sxs-lookup"><span data-stu-id="72688-109">From the **Select for credit note** form, select the transaction that you want to credit and then click **OK**.</span></span>


> [!NOTE]
> <P><span data-ttu-id="72688-110">Ao criar a nota de crédito, verifique se as <STRONG>Notas de crédito</STRONG> estão selecionadas.</span><span class="sxs-lookup"><span data-stu-id="72688-110">When you create the credit note, make sure that you select <STRONG>Credit notes</STRONG>.</span></span> <span data-ttu-id="72688-111">Isso é encontrado na lista <STRONG>Método de faturamento</STRONG> na caixa de diálogo <STRONG>Criar fatura</STRONG>.</span><span class="sxs-lookup"><span data-stu-id="72688-111">This is found in the <STRONG>Invoicing method</STRONG> list in the <STRONG>Create invoice</STRONG> dialog box.</span></span></P>

<span data-ttu-id="72688-112">Se o campo **Reverter acúmulos em créditos** do formulário **Parâmetros de gerenciamento de serviço** estiver definido como **Manual**, será necessário reverter cada transação de receita acumulada individualmente para poder criar uma proposta de nota de crédito para a transação.</span><span class="sxs-lookup"><span data-stu-id="72688-112">If the **Reverse accruals on crediting** field in the **Service management parameters** form is set to **Manual**, you have to reverse each accrued revenue transaction individually before you create a credit note proposal for the transaction.</span></span>
