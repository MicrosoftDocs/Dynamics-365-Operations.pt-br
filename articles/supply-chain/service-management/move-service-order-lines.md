# <a name="move-service-order-lines"></a><span data-ttu-id="3ab28-103">Mover linhas da ordem de serviço</span><span class="sxs-lookup"><span data-stu-id="3ab28-103">Move service order lines</span></span> 

[!include [banner](../includes/banner.md)]


1.  <span data-ttu-id="3ab28-104">Clique em **Gerenciamento de serviços** \> **Comum** \> **Ordens de serviço** \> **Ordens de serviço**.</span><span class="sxs-lookup"><span data-stu-id="3ab28-104">Click **Service management** \> **Common** \> **Service orders** \> **Service orders**.</span></span>

2.  <span data-ttu-id="3ab28-105">Clique duas vezes na ordem de serviço que contém as linhas que você deseja mover para outra ordem de serviço.</span><span class="sxs-lookup"><span data-stu-id="3ab28-105">Double-click the service order that contains the lines that you want to move to another service order.</span></span>

3.  <span data-ttu-id="3ab28-106">No formulário **Ordens de serviço**, no Painel de ação, clique no botão **Mover linhas**.</span><span class="sxs-lookup"><span data-stu-id="3ab28-106">In the **Service orders** form, on the Action Pane, click the **Move lines** button.</span></span>

4.  <span data-ttu-id="3ab28-107">No formulário **Mover linhas de ordem de serviço**, no campo **Ordem de serviço**, selecione a ordem de serviço para onde deseja mover as linhas.</span><span class="sxs-lookup"><span data-stu-id="3ab28-107">In the **Move service order lines** form, in the **Service order** field, select the service order that you want to move the lines to.</span></span>

5.  <span data-ttu-id="3ab28-108">Marque a caixa de seleção **Excluir ordem de serviço vazia** para excluir a ordem de serviço da qual você moveu as linhas de ordem de serviço.</span><span class="sxs-lookup"><span data-stu-id="3ab28-108">Select the **Delete empty service order** check box to delete the service order that you moved the service order lines from.</span></span>

6.  <span data-ttu-id="3ab28-109">Clique em **OK**.</span><span class="sxs-lookup"><span data-stu-id="3ab28-109">Click **OK**.</span></span>


> [!NOTE]
> <P><span data-ttu-id="3ab28-110">Você só poderá mover as linhas entre as ordens de serviço que estão anexadas ao mesmo contrato de serviço.</span><span class="sxs-lookup"><span data-stu-id="3ab28-110">You can move service order lines only between service orders that are attached to the same service agreement.</span></span></P>
