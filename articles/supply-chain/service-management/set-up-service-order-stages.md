# <a name="set-up-service-order-stages"></a><span data-ttu-id="b8327-103">Configurar estágios da ordem de serviço</span><span class="sxs-lookup"><span data-stu-id="b8327-103">Set up service order stages</span></span> 

[!include [banner](../includes/banner.md)]


1.  <span data-ttu-id="b8327-104">Clique em **Gerenciamento de serviços** \> **Configuração** \> **Ordens de serviço** \> **Fases de serviço**.</span><span class="sxs-lookup"><span data-stu-id="b8327-104">Click **Service management** \> **Setup** \> **Service orders** \> **Service stages**.</span></span>

2.  <span data-ttu-id="b8327-105">Pressione CTRL+N para criar um novo registro.</span><span class="sxs-lookup"><span data-stu-id="b8327-105">Press CTRL+N to create a new record.</span></span>

3.  <span data-ttu-id="b8327-106">Nos campos **Etapa de serviço** e **Descrição**, especifique uma ID de etapa de serviço e uma descrição.</span><span class="sxs-lookup"><span data-stu-id="b8327-106">In the **Service stage** and **Description** fields, specify a service stage ID and description.</span></span>

4.  <span data-ttu-id="b8327-107">Selecione os parâmetros apropriados para o estágio.</span><span class="sxs-lookup"><span data-stu-id="b8327-107">Select the appropriate parameters for the stage.</span></span>

5.  <span data-ttu-id="b8327-108">Selecione a fase pai da fase atual ou deixe o campo **Pai** em branco, se a fase for a fase inicial na configuração de fase.</span><span class="sxs-lookup"><span data-stu-id="b8327-108">Select the parent stage for the current stage or leave the **Parent** field blank if the stage is the initial stage in the stage setup.</span></span>
