# <a name="create-service-subscriptions-from-a-project"></a>Criar subscrições de serviço a partir de um projeto    

[!include [banner](../includes/banner.md)]


Você pode criar uma subscrição de serviço de um projeto em **Gerenciamento e contabilidade do projeto**.

1.  Clique em **Gerenciamento e contabilidade de projeto** \> **Comum** \> **Projetos** \> **Todos os projetos**. Abra o projeto para o qual você deseja criar uma subscrição.

2.  No formulário **Projetos**, no **Painel de Ação**, na guia **Gerenciar** clique em **Serviço** \> **Subscrição de serviço**.

3.  No formulário **Subscrição**, no campo **Subscrições de serviço**, insira um número de ID exclusivo. O número do projeto é inserido automaticamente na subscrição do projeto selecionado.

4.  Insira as informações restantes no formulário.

Agora, você pode criar transações de subscrição para a subscrição.


> [!NOTE]
> <P>Se você tiver configurado uma sequência numérica para o grupo de subscrições, um ID de subscrição será gerado automaticamente para a subscrição quando você especificar o grupo de subscrições.</P>
