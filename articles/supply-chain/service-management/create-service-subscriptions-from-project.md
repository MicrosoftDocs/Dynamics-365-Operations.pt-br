[!include [banner](../includes/banner.md)]


<span data-ttu-id="387f2-104">Você pode criar uma subscrição de serviço de um projeto em **Gerenciamento e contabilidade do projeto**.</span><span class="sxs-lookup"><span data-stu-id="387f2-104">You can create a service subscription from a project in **Project management and accounting**.</span></span>

1.  <span data-ttu-id="387f2-105">Clique em **Gerenciamento e contabilidade de projeto** \> **Comum** \> **Projetos** \> **Todos os projetos**.</span><span class="sxs-lookup"><span data-stu-id="387f2-105">Click **Project management and accounting** \> **Common** \> **Projects** \> **All projects**.</span></span> <span data-ttu-id="387f2-106">Abra o projeto para o qual você deseja criar uma subscrição.</span><span class="sxs-lookup"><span data-stu-id="387f2-106">Open the project that you want to create a subscription for.</span></span>

2.  <span data-ttu-id="387f2-107">No formulário **Projetos**, no **Painel de Ação**, na guia **Gerenciar** clique em **Serviço** \> **Subscrição de serviço**.</span><span class="sxs-lookup"><span data-stu-id="387f2-107">On the **Projects** form, on the **Action Pane**, on the **Manage** tab, click **Service** \> **Service subscription**.</span></span>

<span data-ttu-id="387f2-111">Agora, você pode criar transações de subscrição para a subscrição.</span><span class="sxs-lookup"><span data-stu-id="387f2-111">You can now create subscription transactions for the subscription.</span></span>


> [!NOTE]
> <P><span data-ttu-id="387f2-112">Se você tiver configurado uma sequência numérica para o grupo de subscrições, um ID de subscrição será gerado automaticamente para a subscrição quando você especificar o grupo de subscrições.</span><span class="sxs-lookup"><span data-stu-id="387f2-112">If you have set up a number sequence for the subscription group, a subscription ID is generated automatically for the subscription when you specify the subscription group.</span></span></P>
