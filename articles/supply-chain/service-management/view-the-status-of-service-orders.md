# <a name="view-the-status-of-service-orders"></a>Exibir o status de ordens de serviço 

[!include [banner](../includes/banner.md)]


Você pode usar o formulário de **Quadro de expedição** para visualizar o status das atividades de serviço em ordens de serviço.

## <a name="more-information-about-service-orders"></a>Mais informações sobre as ordens de serviço

O formulário também fornece ferramentas para rastrear o fluxo de ordens de serviço e inclui detalhes adicionais para ajudá-lo a avaliar a urgência de uma ordem de serviço, como o contrato de serviço e o contrato de nível de serviço ao qual está associado.

Para ajudá-lo a aferir rapidamente a urgência de uma ordem de serviço, o gráfico de Gantt oferece suporte ao uso de cores. As cores são usadas para diferenciar a prioridade de uma chamada de serviço. Por padrão, essas cores são vermelho para alta prioridade, amarelo para prioridade normal e verde para baixa prioridade.


> [!TIP]
> <P>Você poderá alterar as cores na guia <STRONG>Expedição</STRONG> no formulário <STRONG>Parâmetros de gerenciamento de serviços</STRONG>.</P>
