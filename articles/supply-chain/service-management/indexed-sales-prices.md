# <a name="indexed-sales-prices"></a><span data-ttu-id="f3336-103">Preços de venda indexados</span><span class="sxs-lookup"><span data-stu-id="f3336-103">Indexed sales prices</span></span>  

[!include [banner](../includes/banner.md)]


<span data-ttu-id="f3336-104">Você configura o índice para o preço de venda de uma subscrição quando cria uma taxa de subscrição.</span><span class="sxs-lookup"><span data-stu-id="f3336-104">You set up the index for a subscription sales price when you create a subscription fee.</span></span>

<span data-ttu-id="f3336-105">No formulário **Criar taxa de subscrição**, defina o campo **Obter preço de** como **Preço base indexado**, e depois multiplique o preço base pela porcentagem no campo **Porcentagem de alteração de preço** para obter o preço de vendas da transação de subscrição.</span><span class="sxs-lookup"><span data-stu-id="f3336-105">In the **Create subscription fee** form, set the **Get pricing from** field to **Indexed base price**, and then multiply the base price by the percentage in the **Percent price change** field to get the sales price of the subscription transaction.</span></span>
