# <a name="reason-codes-for-service-orders"></a>Códigos de motivo para ordens de serviço   

[!include [banner](../includes/banner.md)]


Você pode usar códigos de motivo para ajudar a explicar o status de uma ordem de serviço quando a fase de uma ordem de serviço é atualizada. Por exemplo, se cancelar uma ordem de serviço, você pode selecionar um código de motivo para o cancelamento.

Para exibir informações sobre os códigos de motivo usados para acompanhar o andamento da ordem de serviço, execute o relatório de andamento da ordem de serviço. Esse relatório lista todas as ordens de serviço, independentemente de sua fase, e os códigos de motivo que são especificados quando uma fase da ordem de serviço é atualizada.

## <a name="turn-reason-codes-on-or-off"></a>Ativar ou desativar códigos de motivo

Os códigos de motivo são opcionais. Você pode decidir se deseja exigir um código de motivo quando você atualiza uma ordem de serviço para uma fase específica do serviço.

1.  Clique em **Gerenciamento de serviços** \> **Configuração** \> **Ordens de serviço** \> **Fases de serviço**.

2.  No formulário **Fases de serviço**, selecione uma fase de serviço e marque a caixa de seleção **Motivo** para a fase do serviço.

3.  Feche o formulário para salvar suas alterações.
