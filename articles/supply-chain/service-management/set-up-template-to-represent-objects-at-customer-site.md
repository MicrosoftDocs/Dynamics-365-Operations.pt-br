# <a name="set-up-template-to-represent-objects-at-customer-site"></a><span data-ttu-id="229a9-103">Configurar modelo para representar objetos no local do cliente</span><span class="sxs-lookup"><span data-stu-id="229a9-103">Set up template to represent objects at customer site</span></span> 

[!include [banner](../includes/banner.md)]


<span data-ttu-id="229a9-104">Use este procedimento para configurar um modelo que representa objetos no local do cliente.</span><span class="sxs-lookup"><span data-stu-id="229a9-104">Use this procedure to set up a template that represents objects at a customer site.</span></span>

1.  <span data-ttu-id="229a9-105">Clique em **Gerenciamento de serviços** \> **Comum** \> **Contratos de serviço** \> **Contratos de serviço**.</span><span class="sxs-lookup"><span data-stu-id="229a9-105">Click **Service management** \> **Common** \> **Service agreements** \> **Service agreements**.</span></span>

2.  <span data-ttu-id="229a9-106">Clique duas vezes em um contrato de serviço ou clique em **Contrato de serviço** para criar um novo contrato de serviço.</span><span class="sxs-lookup"><span data-stu-id="229a9-106">Double-click a service agreement, or click **Service agreement** to create a new service agreement.</span></span>

3.  <span data-ttu-id="229a9-107">Clique na guia **Configuração**.</span><span class="sxs-lookup"><span data-stu-id="229a9-107">Click the **Setup** tab.</span></span>

4.  <span data-ttu-id="229a9-108">Clique em **Objetos de serviço** para anexar uma lista de materiais (BOM) de modelo ao contrato de serviço.</span><span class="sxs-lookup"><span data-stu-id="229a9-108">Click **Service objects** to attach a template bill of materials (BOM) to the service agreement.</span></span>

5.  <span data-ttu-id="229a9-109">No formulário **Objetos de serviço**, selecione um objeto de serviço, e depois clique em **Funções** \> **Anexar BOM de modelo**.</span><span class="sxs-lookup"><span data-stu-id="229a9-109">In the **Service objects** form, select a service object, and then click **Functions** \> **Attach template BOM**.</span></span>

6.  <span data-ttu-id="229a9-110">No formulário **Selecionar BOM de modelo**, selecione uma BOM de modelo no campo **BOM de modelo** e clique em **OK**.</span><span class="sxs-lookup"><span data-stu-id="229a9-110">In the **Select template BOM** form, select a template BOM in the **Template BOM** field, and then click **OK**.</span></span>

7.  <span data-ttu-id="229a9-111">No formulário **Objetos de serviço**, clique em **Designer** para modificar as linhas da BOM de modelo de forma que a BOM represente o objeto desejado.</span><span class="sxs-lookup"><span data-stu-id="229a9-111">In the **Service objects** form, click **Designer** to modify the template BOM so that the BOM represents the objects that you want.</span></span>

8.  <span data-ttu-id="229a9-112">Feche os formulários para salvar suas alterações.</span><span class="sxs-lookup"><span data-stu-id="229a9-112">Close the forms to save your changes.</span></span>
