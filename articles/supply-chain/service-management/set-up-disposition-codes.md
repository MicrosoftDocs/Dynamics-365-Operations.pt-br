[!include [banner](../includes/banner.md)]


<span data-ttu-id="1d43f-104">Você pode configurar códigos de disposição para especificar como processar um item que é devolvido por um cliente.</span><span class="sxs-lookup"><span data-stu-id="1d43f-104">You can set up disposition codes to specify how to process an item that is returned by a customer.</span></span> <span data-ttu-id="1d43f-105">Por exemplo, crie um código de disposição chamado **Reparo e devolução** para indicar que o item devolvido será reparado e devolvido para o cliente.</span><span class="sxs-lookup"><span data-stu-id="1d43f-105">For example, create a disposition code named **Repair and return** to indicate that the returned item will be repaired and then returned to the customer.</span></span> <span data-ttu-id="1d43f-106">Para obter mais exemplos de códigos de disposição que são usados para itens que são devolvidos pelos clientes, consulte [Especificar o destino dos itens devolvidos](specify-how-to-dispose-of-returned-items.md).</span><span class="sxs-lookup"><span data-stu-id="1d43f-106">For more examples of disposition codes that are typically used for items that are returned by customers, see [Specify how to dispose of returned items](specify-how-to-dispose-of-returned-items.md).</span></span>

<span data-ttu-id="1d43f-107">Você também pode configurar um código de motivo para ajudar a explicar por que um item foi devolvido.</span><span class="sxs-lookup"><span data-stu-id="1d43f-107">You can also set up a reason code to help explain why an item was returned.</span></span> <span data-ttu-id="1d43f-108">Para obter mais informações sobre como configurar os códigos de motivo, consulte [Configurar códigos de motivo de devolução](set-up-return-reason-code.md).</span><span class="sxs-lookup"><span data-stu-id="1d43f-108">For more information about reason codes, see [Set up return reason codes](set-up-return-reason-code.md).</span></span>

1.  <span data-ttu-id="1d43f-109">Clique em **Vendas e marketing** \> **Configuração** \> **Ordens de venda** \> **Devoluções** \> **Códigos de disposição**.</span><span class="sxs-lookup"><span data-stu-id="1d43f-109">Click **Sales and marketing** \> **Setup** \> **Sales orders** \> **Returns** \> **Disposition codes**.</span></span>

2.  <span data-ttu-id="1d43f-110">Clique em **Novo** ou pressione CTRL+N para criar um novo código de disposição.</span><span class="sxs-lookup"><span data-stu-id="1d43f-110">Click **New** or press CTRL+N to create a new disposition code.</span></span>

3.  <span data-ttu-id="1d43f-111">Insira um nome exclusivo e descritivo, selecione uma ação e insira uma descrição para o código de disposição.</span><span class="sxs-lookup"><span data-stu-id="1d43f-111">Enter a unique, descriptive name, select an action, and enter a description for the disposition code.</span></span>

4.  <span data-ttu-id="1d43f-112">Se você deseja associar quaisquer encargos de cliente com esse código de disposição, clique no botão **Encargos** para abrir o formulário **Configurar encargos**.</span><span class="sxs-lookup"><span data-stu-id="1d43f-112">If you want to associate any customer charges with this disposition code, click the **Charges** button to open the **Set up charges** form.</span></span>

5.  <span data-ttu-id="1d43f-113">Se você deseja definir quaisquer códigos externos para corresponder aos códigos de disposição da própria empresa, clique no botão **Códigos externos** para abrir o formulário **Códigos externos**.</span><span class="sxs-lookup"><span data-stu-id="1d43f-113">If you want to define any external codes to match with the company's own disposition codes, click the **External codes** button to open the **External codes** form.</span></span>
