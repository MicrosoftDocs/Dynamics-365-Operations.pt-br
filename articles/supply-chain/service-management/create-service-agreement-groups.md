# <a name="create-service-agreement-groups"></a>Criar grupos de contrato de serviço

[!include [banner](../includes/banner.md)]

Os grupos de contratos de serviço são úteis para classificar e filtrar contratos de serviço.

1. Acesse **Gerenciamento de serviços** \> **Configuração** \> **Contratos de serviço** \> **Grupos de contratos de serviço**.
1. No Painel de Ações, selecione **Novo** para criar um grupo de contratos de serviço.
1. No campo **Grupo de contratos de serviço**, insira uma ID do grupo de contratos de serviço.
1. No campo **Descrição**, insira uma descrição para o grupo de contratos de serviço.
1. No campo **Contrato de nível de serviço**, selecione o contrato que se aplica ao grupo.
1. Feche a página para salvar suas alterações.

> [!NOTE]
> Caso um grupo de contratos de serviço seja definido no campo **Grupo de contratos de serviço** da página **Parâmetros de gerenciamento de serviços**, o programa irá sugerir esse grupo para todos os novos contratos de serviços que você criar.
