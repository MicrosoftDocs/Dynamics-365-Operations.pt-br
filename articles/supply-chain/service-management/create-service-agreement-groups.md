# <a name="create-service-agreement-groups"></a><span data-ttu-id="a88ab-103">Criar grupos de contrato de serviço</span><span class="sxs-lookup"><span data-stu-id="a88ab-103">Create service agreement groups</span></span> 

[!include [banner](../includes/banner.md)]

<span data-ttu-id="a88ab-104">Os grupos de contratos de serviço são úteis para classificar e filtrar contratos de serviço.</span><span class="sxs-lookup"><span data-stu-id="a88ab-104">Service agreement groups are useful for sorting and filtering service agreements.</span></span>

1. <span data-ttu-id="a88ab-105">Clique em **Gerenciamento de serviços** \> **Configuração** \> **Contratos de serviço** \> **Grupos de contratos de serviço**.</span><span class="sxs-lookup"><span data-stu-id="a88ab-105">Click **Service management** \> **Setup** \> **Service agreements** \> **Service agreement groups**.</span></span>
2. <span data-ttu-id="a88ab-106">Pressione **Alt+N** para criar um novo grupo de contratos de serviço.</span><span class="sxs-lookup"><span data-stu-id="a88ab-106">Press **Alt+N** to create a new service agreement group.</span></span>
3. <span data-ttu-id="a88ab-107">No campo **Grupo de contratos de serviço**, insira uma ID do grupo de contratos de serviço.</span><span class="sxs-lookup"><span data-stu-id="a88ab-107">In the **Service agreement group** field, enter an ID for the service agreement group.</span></span>
4. <span data-ttu-id="a88ab-108">No campo **Descrição**, insira uma descrição para o grupo de contratos de serviço.</span><span class="sxs-lookup"><span data-stu-id="a88ab-108">In the **Description** field, enter a description for the service agreement group.</span></span>
5. <span data-ttu-id="a88ab-109">Feche a página para salvar suas alterações.</span><span class="sxs-lookup"><span data-stu-id="a88ab-109">Close the page to save your changes.</span></span>

> [!NOTE]
> <span data-ttu-id="a88ab-110">Caso um grupo de contratos de serviço seja definido no campo **Grupo de contratos de serviço** da página **Parâmetros de gerenciamento de serviços**, o programa irá sugerir esse grupo para todos os novos contratos de serviços que você criar.</span><span class="sxs-lookup"><span data-stu-id="a88ab-110">If a service agreement group is defined in the **Service agreement group** field of the **Service management parameters** page, the program suggests this group for all new service agreements that you create.</span></span>
