# <a name="view-the-start-time-and-duration-of-a-service-order"></a>Exibir a hora de início e a duração de uma ordem de serviço 

[!include [banner](../includes/banner.md)]


Você pode exibir quando o trabalho na ordem de serviço foi iniciado e quando a ordem será concluída.

Também é possível exibir quando o registro de tempo de uma ordem de serviço foi iniciado ou parado. Quando uma ordem de serviço é parada, o tempo no qual a ordem deve ser concluída é adiado.

## <a name="view-the-start-time-for-a-service-order"></a>Exibir a hora inicial de uma ordem de serviço

1.  Clique em **Gerenciamento de serviços** \> **Comum** \> **Ordens de serviço** \> **Ordens de serviço**. Selecione e clique duas vezes em uma ordem para abrir o formulário de detalhes.

2.  Na guia **Geral**, exiba a hora em que o trabalho começou para uma ordem de serviço no campo **Iniciar tempo**.

## <a name="view-the-time-remaining-to-complete-a-service-order"></a>Exibir o tempo restante para concluir uma ordem de serviço

1.  Clique em **Gerenciamento de serviços** \> **Comum** \> **Ordens de serviço** \> **Ordens de serviço**. Selecione e clique duas vezes em uma ordem para abrir o formulário de detalhes.

2.  Na guia **Geral**, exiba o tempo restante para concluir uma ordem de serviço no campo **Tempo de conclusão mais recente**.

## <a name="view-the-start-time-and-stop-time-recording-entries-for-a-service-order"></a>Exibir as entradas de registro de hora inicial e final de uma ordem de serviço

1.  Clique em **Gerenciamento de serviços** \> **Comum** \> **Ordens de serviço** \> **Ordens de serviço**. Selecione e clique duas vezes em uma ordem para abrir o formulário de detalhes.

2.  No **Painel de ação**, clique na guia **Despachar** \> **Tempo de gravação** para abrir o formulário **Tempo de gravação do SLA** e exibir as entradas de registro de tempo da ordem de serviço.
