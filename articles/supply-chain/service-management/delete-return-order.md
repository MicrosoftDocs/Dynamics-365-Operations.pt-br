# <a name="delete-a-return-order"></a>Excluir uma ordem de devolução 

[!include [banner](../includes/banner.md)]


Use o formulário **Excluir ordem de devolução** para excluir várias ordens de retorno.

1.  Clique em **Vendas e marketing** \> **Periódico** \> **Limpar** \> **Excluir ordens de devolução**.

2.  Clique em **Selecionar** para abrir o formulário **Atualização de vendas**.

3.  Especifique os critérios e então clique em **OK** para voltar para o formulário **Excluir ordem de devolução**. As ordens de devolução criadas na entidade legal são exibidas com base nos critérios especificados.

4.  Selecione a linha da ordem de devolução que deseja excluir e clique em **OK**.

5.  Clique em **Sim** para excluir as ordens de devolução.
