# <a name="set-up-subscription-groups"></a><span data-ttu-id="e74a3-103">Configurar grupos de subscrições</span><span class="sxs-lookup"><span data-stu-id="e74a3-103">Set up subscription groups</span></span> 

[!include [banner](../includes/banner.md)]


1.  <span data-ttu-id="e74a3-104">Vá para **Gerenciamento de serviços** \> **Configuração** \> **Subscrições de serviços** \> **Grupos de subscrições**.</span><span class="sxs-lookup"><span data-stu-id="e74a3-104">Go to **Service management** \> **Setup** \> **Service subscriptions** \> **Subscription groups**.</span></span>

2.  <span data-ttu-id="e74a3-105">Selecione **Novo** para criar um novo grupo de subscrições.</span><span class="sxs-lookup"><span data-stu-id="e74a3-105">Select **New** to create a new subscription group.</span></span>

3.  <span data-ttu-id="e74a3-106">No campo **Grupo de subscrição**, insira um nome para o grupo da nova subscrição.</span><span class="sxs-lookup"><span data-stu-id="e74a3-106">In the **Subscription group** field, enter a name for the new subscription group.</span></span>

4.  <span data-ttu-id="e74a3-107">No campo **Fatura**, selecione uma frequência de faturamento para a subscrição.</span><span class="sxs-lookup"><span data-stu-id="e74a3-107">In the **Invoicing** field, select an invoicing frequency for the subscription.</span></span> <span data-ttu-id="e74a3-108">A frequência de faturamento é indicada por um código de período configurado no formulário **Tipos de período**.</span><span class="sxs-lookup"><span data-stu-id="e74a3-108">The invoicing frequency is indicated by a period code that is set up in the **Period types** form.</span></span>

5.  <span data-ttu-id="e74a3-109">Para acumular receita para transações de subscrição, marque a caixa de seleção **Acumular receita** e, no campo **Acumular**, selecione uma frequência.</span><span class="sxs-lookup"><span data-stu-id="e74a3-109">To accrue revenue for subscription transactions, select the **Accrue revenue** check box, and then in the **Accrual** field, select a frequency.</span></span> <span data-ttu-id="e74a3-110">A frequência de acumulação se baseia em um código de período configurado no formulário **Períodos**.</span><span class="sxs-lookup"><span data-stu-id="e74a3-110">The accrual frequency is based on a period code that is set up in the **Periods** form.</span></span>
