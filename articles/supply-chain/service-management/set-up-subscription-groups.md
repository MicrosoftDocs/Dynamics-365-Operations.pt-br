# <a name="set-up-subscription-groups"></a>Configurar grupos de subscrições 

[!include [banner](../includes/banner.md)]


1.  Acesse **Gerenciamento de serviços** \> **Configuração** \> **Subscrições de serviços** \> **Grupos de subscrições**.

2.  Selecione **Novo** para criar um novo grupo de subscrições.

3.  No campo **Grupo de subscrição**, insira um nome para o grupo da nova subscrição.

4.  No campo **Fatura**, selecione uma frequência de faturamento para a subscrição. A frequência de faturamento é indicada por um código de período configurado no formulário **Tipos de período**.

5.  Para acumular receita para transações de subscrição, marque a caixa de seleção **Acumular receita** e, no campo **Acumular**, selecione uma frequência. A frequência de acumulação se baseia em um código de período configurado no formulário **Períodos**.
