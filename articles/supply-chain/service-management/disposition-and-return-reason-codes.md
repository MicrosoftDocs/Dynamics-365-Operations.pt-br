# <a name="disposition-codes-and-return-reason-codes"></a>Códigos de disposição e códigos de motivo de devolução 

[!include [banner](../includes/banner.md)]


Crie e use códigos de motivo de devolução e códigos de disposição para dar suporte ao processo de devolução de produtos.

Use um código de motivo para descrever o motivo pelo qual o cliente quer devolver um item. Você pode atribuir um código de motivo no formulário **Criar ordens de devolução**.

Atribuir um código de disposição quando um item for recebido ou durante a inspeção física de um item devolvido. Você pode usar códigos de disposição para descrever a condição do item. Você também pode usar códigos de disposição para indicar se a ação adicional é necessária para a transação. Por exemplo, crie códigos de disposição para as seguintes ações:

  - Descartar o item devolvido e fornecer um item de substituição para o cliente.

  - Devolver o item ao estoque e creditar o cliente pelo custo do item.

  - Reparar o item e devolvê-lo ao cliente.
