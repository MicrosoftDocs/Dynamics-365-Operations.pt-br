# <a name="modify-a-demand-forecast-manually"></a>Modificar uma previsão de demanda manualmente

[!include [task guide banner](../../includes/task-guide-banner.md)]

Este procedimento mostra como alterar a previsão para um item. A empresa de dados demo usada para criar este procedimento é USMF. Esse registro é criado para o planejador de produção. 


## <a name="modify-the-forecast-for-an-item"></a>Modifique a previsão para um item
1. Vá para Gerenciamento de informações do produto > Produtos > Produtos liberados.
2. Na lista, localize e selecione o PDV desejado.
    * Selecione o item cujo tipo deseja alterar a previsão. Por exemplo, você pode selecionar o item D0001.  
3. No Painel de Ação, clique em Plano.
4. Clique em Previsão de demanda.
5. Na lista, marque a linha selecionada.
    * Se não houver nenhuma linha de previsão, crie uma nova linha clicando em Novo na barra do aplicativo.  
6. No campo Quantidade de vendas, insira um número.
    * Este número representa a quantidade prevista para o item.  
7. Clique em Salvar.

## <a name="modify-the-forecast-in-excel"></a>Modifique a previsão do Excel
1. Clique em Abrir no Microsoft Office.
2. Clique em Editar Previsão de demanda no Excel.
    * No Excel, você pode adicionar, editar e excluir linhas de previsão de demanda. Se não conseguir ver os dados do Excel, você precisa entrar no Microsoft Dynamics 365 for Finance and Operations, Enterprise edition com a opção “Mantenha-me conectado” habilitada e você precisa confiar na conexão do aplicativo de dados.
