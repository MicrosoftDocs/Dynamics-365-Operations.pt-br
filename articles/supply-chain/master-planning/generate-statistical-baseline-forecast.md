# <a name="generate-a-statistical-baseline-forecast"></a><span data-ttu-id="646c1-103">​Gerar uma previsão estatística​</span><span class="sxs-lookup"><span data-stu-id="646c1-103">Generate a statistical baseline forecast</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="646c1-104">Este tópico fornece informações sobre os parâmetros e filtros usados no cálculo da previsão de demanda.</span><span class="sxs-lookup"><span data-stu-id="646c1-104">This topic provides information about the parameters and filters that are used in the calculation of demand forecasting.</span></span> 

<span data-ttu-id="646c1-105">Ao criar uma previsão de baseline, primeiro você deve especificar os parâmetros e os filtros usados no cálculo.</span><span class="sxs-lookup"><span data-stu-id="646c1-105">When you create a baseline forecast, you must first specify the parameters and filters that are used in the calculation.</span></span> <span data-ttu-id="646c1-106">Por exemplo, você pode criar uma previsão de baseline que estima a demanda baseada nos dados de transação do ano passado para uma empresa específica, para o mês de entrada e para um grupo de itens selecionado.</span><span class="sxs-lookup"><span data-stu-id="646c1-106">For example, you can create a baseline forecast that estimates demand based on transaction data from the past year for a specific company, for the coming month, and for a selected group of items.</span></span> 

<span data-ttu-id="646c1-107">Para gerar uma previsão de demanda, vá para **Planejamento mestre &gt; Previsão &gt; Previsão de demanda &gt; Gerar previsão estatística**.</span><span class="sxs-lookup"><span data-stu-id="646c1-107">To generate a demand forecast, go to **Master planning &gt; Forecasting &gt; Demand forecasting &gt; Generate statistical baseline forecast**.</span></span> 

<span data-ttu-id="646c1-108">A classificação de previsão pode ser selecionada no momento da geração da previsão.</span><span class="sxs-lookup"><span data-stu-id="646c1-108">The forecast bucket can be selected at forecast generation time.</span></span> <span data-ttu-id="646c1-109">Os valores disponíveis são: Dia, Semana e Mês.</span><span class="sxs-lookup"><span data-stu-id="646c1-109">The available values are: Day, Week, and Month.</span></span> 

<span data-ttu-id="646c1-110">O número de classificações para as quais serão geradas uma previsão é definido no campo **Horizonte de previsão**.</span><span class="sxs-lookup"><span data-stu-id="646c1-110">The number of buckets to generate a forecast for is set in the **Forecast horizon** field.</span></span> 

<span data-ttu-id="646c1-111">Quando a estratégia de previsão é definida para **Copiar sobre demanda histórica**, o fim do horizonte histórico é ignorado.</span><span class="sxs-lookup"><span data-stu-id="646c1-111">When the forecast strategy is set to **Copy over historical demand**, the end of the historical horizon is ignored.</span></span> <span data-ttu-id="646c1-112">O sistema copia o número de classificações especificado no campo **Horizonte de previsão** para a demanda de previsão, começando pela data definida no campo **Data inicial** em **Horizonte histórico**.</span><span class="sxs-lookup"><span data-stu-id="646c1-112">The system copies the number of buckets specified in the **Forecast horizon** field to the forecast demand, starting from the date set in the **From date** field under **Historical horizon**.</span></span> <span data-ttu-id="646c1-113">Copiando a demanda histórica de uma determinada data em diante, os planejadores de produção podem planejar o próximo trimestre de duas maneiras:</span><span class="sxs-lookup"><span data-stu-id="646c1-113">By copying historical demand from a certain date forward, production planners can make the plan for the next quarter in two ways:</span></span>

-   <span data-ttu-id="646c1-114">Copiando a demanda do mesmo trimestre do ano passado.</span><span class="sxs-lookup"><span data-stu-id="646c1-114">By copying the demand from the same quarter last year.</span></span>
-   <span data-ttu-id="646c1-115">Copiando a demanda do trimestre anterior.</span><span class="sxs-lookup"><span data-stu-id="646c1-115">By copying the demand from the previous quarter.</span></span>

<span data-ttu-id="646c1-116">Para evitar confusão nos planos de produção, um determinado número de classificações de previsão pode ser congelado.</span><span class="sxs-lookup"><span data-stu-id="646c1-116">To prevent confusion in the production plans, a certain number of forecast buckets can be frozen.</span></span> <span data-ttu-id="646c1-117">Esse número é definido no campo **Congelar limite de tempo**.</span><span class="sxs-lookup"><span data-stu-id="646c1-117">This number is set in the **Freeze time fence** field.</span></span> <span data-ttu-id="646c1-118">Na página **Previsão de demanda ajustada**, as células das classificações congeladas são desabilitadas para fornecer uma indicação visual de que esses valores não devem ser alterados.</span><span class="sxs-lookup"><span data-stu-id="646c1-118">On the **Adjusted demand forecast** page, the cells for the frozen buckets are disabled, to give a visual indication that those values should not be changed.</span></span> 

<span data-ttu-id="646c1-119">A data inicial da previsão de demanda de linha de base não precisa ser a data atual ou uma data futura.</span><span class="sxs-lookup"><span data-stu-id="646c1-119">The start date for the baseline demand forecast doesn’t have to be the current date or a date in the future.</span></span> <span data-ttu-id="646c1-120">Para definir uma data inicial diferente, use **Data de início de previsão de linha de base - Data inicial**.</span><span class="sxs-lookup"><span data-stu-id="646c1-120">To set a different start date, use the **Baseline forecast start date - From date** field.</span></span> <span data-ttu-id="646c1-121">Por exemplo, em junho, os usuários poderão gerar uma previsão para o próximo ano.</span><span class="sxs-lookup"><span data-stu-id="646c1-121">For example, in June, users can generate a forecast for the next year.</span></span> <span data-ttu-id="646c1-122">Como as classificações de previsão entre o fim da demanda histórica e o início da linha de base estão ausentes, as previsões podem não ser precisas.</span><span class="sxs-lookup"><span data-stu-id="646c1-122">Because the forecast buckets between the end of historical demand and the start of the baseline are missing, the predictions might not be accurate.</span></span> <span data-ttu-id="646c1-123">Se você estiver usando o serviço de previsão de demanda, há quatro maneiras de preencher as lacunas ausentes.</span><span class="sxs-lookup"><span data-stu-id="646c1-123">If you are using the Demand forecasting service, there are four ways in which you can fill in the missing gaps.</span></span> <span data-ttu-id="646c1-124">Você pode escolher o método que deseja definindo o parâmetro MISSING\_VALUE\_SUBSTITUTION na página **Parâmetros de previsão de demanda**.</span><span class="sxs-lookup"><span data-stu-id="646c1-124">You can choose the method that you want by setting the MISSING\_VALUE\_SUBSTITUTION parameter on the **Demand forecasting parameters** page.</span></span> 

> [!NOTE]
> <span data-ttu-id="646c1-125">A substituição de valor ausente funciona somente para os intervalos em dados entre o início e o fim de dados históricos.</span><span class="sxs-lookup"><span data-stu-id="646c1-125">Missing value substitution only works for the gaps in data in between the start and end dates for historical data.</span></span> <span data-ttu-id="646c1-126">Ela não preencherá dados antes ou depois do último ponto de dados físico; ela só funciona como uma extrapolação entre pontos de dados existentes reais.</span><span class="sxs-lookup"><span data-stu-id="646c1-126">It will not fill in data before or after the last physical data point, it only acts as extrapolation between actual existing data points.</span></span> 

<span data-ttu-id="646c1-127">O campo **Data de início de previsão de linha de base** - **Data inicial** deve ser definido como o início de uma classificação de previsão, por exemplo, nos Estados Unidos, um domingo, se a previsão for semanal.</span><span class="sxs-lookup"><span data-stu-id="646c1-127">The **Baseline forecast start date** - **From date** field has to be set to the beginning of a forecast bucket, for example, in the United States, a Sunday if the forecasting bucket is the week.</span></span> <span data-ttu-id="646c1-128">O sistema ajusta automaticamente o campo **Data de início de previsão de linha de base** - **Data inicial** de modo que corresponda ao início de uma classificação de previsão.</span><span class="sxs-lookup"><span data-stu-id="646c1-128">The system automatically adjusts the **Baseline forecast start date** - **From date** field to match the beginning of a forecast bucket.</span></span> 

<span data-ttu-id="646c1-129">O campo **Data de início de previsão de linha de base** - **Data inicial** pode ser definido para uma data no passado.</span><span class="sxs-lookup"><span data-stu-id="646c1-129">The **Baseline forecast start date** - **From date** field can be set to a date in in the past.</span></span> <span data-ttu-id="646c1-130">Em outras palavras, é possível gerar uma previsão de demanda no passado.</span><span class="sxs-lookup"><span data-stu-id="646c1-130">In other words, it is possible to generate a demand forecast in the past.</span></span> <span data-ttu-id="646c1-131">Isso será útil porque permite que os usuários ajustem os parâmetros do serviço de previsão para que a previsão estatística gerada no passado corresponda à demanda histórica real.</span><span class="sxs-lookup"><span data-stu-id="646c1-131">This is useful, because it lets users adjust the forecast service parameters so that the statistical forecast generated in the past matches the actual historical demand.</span></span> <span data-ttu-id="646c1-132">Os usuários poderão, então, continuar usando essas configurações de parâmetro para gerar uma previsão estatística para o futuro.</span><span class="sxs-lookup"><span data-stu-id="646c1-132">Users can then continue using these parameter settings to generate a statistical baseline forecast for the future.</span></span> 

<span data-ttu-id="646c1-133">Os ajustes manuais feitos nas iterações de previsão de demanda anteriores poderão ser aplicados automaticamente à nova previsão estatística se a caixa de seleção **Transferir ajustes manuais para a previsão de demanda**.</span><span class="sxs-lookup"><span data-stu-id="646c1-133">Manual adjustments made in previous demand forecasting iterations can be automatically applied to the new baseline forecast if the **Transfer manual adjustments to the demand forecas** t check box is selected.</span></span> <span data-ttu-id="646c1-134">Se a caixa de seleção estiver desmarcada, os ajustes manuais não serão adicionadas à previsão estatísticas, mas não serão excluídos.</span><span class="sxs-lookup"><span data-stu-id="646c1-134">If the check box is cleared, the manual adjustments are not added to the baseline forecast – but they are not deleted.</span></span> <span data-ttu-id="646c1-135">Os ajustes manuais feitos em uma previsão podem ser class="sxs-lookup"><span data-stu-id="646c1-135">Manual adjustments made to a forecast can be deleted only at forecast import time, by clearing the **Save the manual adjustments made to the baseline demand forecast** check box.</span></span> <span data-ttu-id="646c1-136">Os ajustes manuais são salvos no momento da autorização.</span><span class="sxs-lookup"><span data-stu-id="646c1-136">Manual adjustments are saved at authorization time.</span></span> <span data-ttu-id="646c1-137">Portanto, se um usuário fizer ajustes manuais na previsão, mas não autorizar a previsão no Supply Chain Management, as alterações serão perdidas.</span><span class="sxs-lookup"><span data-stu-id="646c1-137">Therefore, if a user makes manual adjustments to the forecast, but doesn’t authorize the forecast back to Supply Chain Management, the changes are lost.</span></span> <span data-ttu-id="646c1-138">Para obter mais informações sobre os ajustes manuais e como eles funcionam, consulte [Autorizar uma adjusted forecast](authorize-adjusted-forecast.md).</span></span> 

<span data-ttu-id="646c1-139">Uma geração de previsão de demanda pode ter um nome e comentários para facilitar a identificação da previsão gerada.</span><span class="sxs-lookup"><span data-stu-id="646c1-139">A demand forecast generation can have a name and comments to help users identify the forecast that has been generated.</span></span> <span data-ttu-id="646c1-140">Esses valores podem ser vistos no histórico de geração de previsão na página **Histórico de geração de previsão estatística**.</span><span class="sxs-lookup"><span data-stu-id="646c1-140">These values are visible in forecast generation history on the **Statistical baseline forecast generation history** page.</span></span> 

<span data-ttu-id="646c1-141">O grupo de planejamento intercompanhia, as chaves de alocação de itens e outros filtros podem ser aplicados no momento da geração da previsão.</span><span class="sxs-lookup"><span data-stu-id="646c1-141">The intercompany planning group, item allocation keys, and other filters can be applied at forecast generation time.</span></span> <span data-ttu-id="646c1-142">Eles podem ser usados para melhorar o desempenho ou para dividir os dados em partes gerenciáveis.</span><span class="sxs-lookup"><span data-stu-id="646c1-142">These can be used to improve performance or to split the data into manageable chunks.</span></span> <span data-ttu-id="646c1-143">No entanto, observe que uma previsão de demanda não será gerada para os membros de qualquer chave de alocação de item que não esteja associada a um grupo de planejamento intercompanhia, mesmo se a chave de alocação de item for selecionada na consulta.</span><span class="sxs-lookup"><span data-stu-id="646c1-143">However, note that a demand forecast is not generated for the members of any item allocation key that is not associated with an intercompany planning group, even if the item allocation key is selected in the query.</span></span> 

> [!TIP]
> <span data-ttu-id="646c1-144">Os usuários talvez recebam erros ao gerar uma previsão de demanda ou a geração de previsão será concluída sem log de sessão.</span><span class="sxs-lookup"><span data-stu-id="646c1-144">Sometimes users might receive errors while generating a demand forecast, or forecast generation is completed with no session log.</span></span> <span data-ttu-id="646c1-145">Isso pode ocorrer devido aos dados que restaram na consulta usada anteriormente para a geração da demanda.</span><span class="sxs-lookup"><span data-stu-id="646c1-145">This can happen because of leftover data in the query that was previously used for forecast generation.</span></span> <span data-ttu-id="646c1-146">Para resolver esse problema, clique em **Selecionar** para abrir a página **Consulta**, selecione **Redefinir** e gere novamente a previsão de linha de base.</span><span class="sxs-lookup"><span data-stu-id="646c1-146">To resolve this issue, click **Select** to open the **Query** page, select **Reset**, and then regenerate the baseline forecast.</span></span>

> [!NOTE]
> <span data-ttu-id="646c1-148">Uma previsão aparentemente tranquila talvez deva-se aos dados históricos que precisam ser de um período histórico maior (no mínimo 3 períodos para selecionar padrões, como 3 anos com previsão mensal).</span><span class="sxs-lookup"><span data-stu-id="646c1-148">A potentially flat looking forecast can be due to the historical data that has to be of a longer historical timeframe (a minimum 3 of time periods in order to pick out patterns, such as 3 years with monthly forecast).</span></span> <span data-ttu-id="646c1-149">Para obter um resultado melhor, tente alterar a granularidade do intervalo de tempo ou aumentar o intervalo de tempo.</span><span class="sxs-lookup"><span data-stu-id="646c1-149">To get a better result, you can try changing the granularity of the time range or increase the time range.</span></span>
