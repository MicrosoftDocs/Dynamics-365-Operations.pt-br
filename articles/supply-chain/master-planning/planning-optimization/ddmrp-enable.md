# <a name="turn-on-the-ddmrp-feature-for-your-system"></a>Ativar o recurso de DDMRP para seu sistema

[!include [banner](../../includes/banner.md)]
[!INCLUDE [preview-banner](../../includes/preview-banner.md)]

Para poder usar a funcionalidade DDMRP (Planejamento de Requisitos de Material Orientado por Demanda), ela e seu recurso de pré-requisito devem ser ativados para o sistema. Os administradores podem usar as configurações de [gerenciamento de recursos](../../../fin-ops-core/fin-ops/get-started/feature-management/feature-management-overview.md) para verificar o status desses recursos e ativá-los. Ative os seguintes recursos nesta ordem:

1. **Nome do recurso:** *Suporte ao MRP baseado em prioridade para a Otimização de Planejamento*
1. **Nome do recurso:** *(Versão preliminar) DDMRP para Otimização de Planejamento*

Para obter mais informações, consulte [Planejamento baseado em prioridade](priority-based-planning.md).
