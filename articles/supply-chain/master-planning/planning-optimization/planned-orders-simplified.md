# <a name="planned-orders-simplified"></a>Ordens planejadas simplificadas

[!include [banner](../../includes/banner.md)]

Este tópico descreve como você pode exibir, gerenciar e confirmar ordens planejadas usando a página **Ordens planejadas simplificadas**.

A página **Ordens planejadas simplificadas** é uma versão simplificada da página **Ordens planejadas**. Ela oferece menos opções, mas é mais rápido trabalhar, quando tudo o que você tem a fazer é revisar, aprovar e confirmar suas ordens planejadas.

## <a name="turn-on-this-feature-for-your-system"></a>Ative este recurso para o seu sistema

Se o sistema ainda não incluir os recursos descritos neste tópico, acesse [Gerenciamento de recursos](../../../fin-ops-core/fin-ops/get-started/feature-management/feature-management-overview.md) e ative o recurso *Ordens planejadas simplificadas*.

## <a name="view-manage-and-firm-planned-orders"></a>Exibir, gerenciar e confirmar ordens planejadas

A página **Ordens planejadas simplificada** destina-se principalmente à funcionalidade de confirmação. Ela mostra todas as ordens planejadas que foram geradas para o plano selecionado, da mesma forma que a página **Ordens planejadas** padrão. Portanto, ela pode ser usada como uma alternativa otimizada para exibir os resultados de uma execução de planejamento mestre.

Para abrir a página **Ordens planejadas simplificadas**, vá para **Planejamento mestre \> Planejamento mestre \> Ordens planejadas simplificadas**. Você pode usar esta página para concluir as seguintes tarefas:

- Exibir todas as ordens planejadas para um plano específico.
- Criar novas ordens planejadas.
- Excluir uma ou várias ordens planejadas.
- Confirmar uma ou várias ordens planejadas.
- Alterar o status da ordem planejada para uma ou várias ordens planejadas.
- Abrir com perfeição detalhes da ordem planejada para uma ordem planejada individual para exibir detalhes completos da ordem.
- Criar exibições personalizadas usando exibições salvas.

Cenários adicionais, como Divisão, são suportados pela página **Ordens planejadas** padrão.
