# <a name="demand-forecasting-setup"></a><span data-ttu-id="5c368-103">Configuração da previsão de demanda</span><span class="sxs-lookup"><span data-stu-id="5c368-103">Demand forecasting setup</span></span>

## <a name="item-allocation-key"></a><span data-ttu-id="5c368-106">Chave de alocação de itens</span><span class="sxs-lookup"><span data-stu-id="5c368-106">Item allocation key</span></span>
<span data-ttu-id="5c368-107">Uma previsão de demanda será calculada para um item e suas dimensões somente se o item fizer parte de uma chave de alocação de item.</span><span class="sxs-lookup"><span data-stu-id="5c368-107">A demand forecast is calculated for an item and its dimensions only if the item is part of an item allocation key.</span></span> <span data-ttu-id="5c368-108">A regra é imposta agrupar grandes números de itens, para que as previsões de demanda possam ser criadas mais rapidamente.</span><span class="sxs-lookup"><span data-stu-id="5c368-108">This rule is enforced to group large numbers of items, so that demand forecasts can be created more quickly.</span></span> <span data-ttu-id="5c368-109">O percentual da chave de alocação de item é ignorado quando as previsões de demanda são geradas.</span><span class="sxs-lookup"><span data-stu-id="5c368-109">The item allocation key percentage is ignored when demand forecasts are generated.</span></span> <span data-ttu-id="5c368-110">As previsões são criadas com base em dados históricos apenas.</span><span class="sxs-lookup"><span data-stu-id="5c368-110">Forecasts are created based on historical data only.</span></span> 

<span data-ttu-id="5c368-111">Um item e suas dimensões devem ser parte de apenas uma chave de alocação de item, caso a chave de alocação de item seja usada durante a criação da previsão.</span><span class="sxs-lookup"><span data-stu-id="5c368-111">An item and its dimensions must be part of only one item allocation key if the item allocation key is used during forecast creation.</span></span> 

<span data-ttu-id="5c368-112">Para adicionar uma SKU (unidade de manutenção de estoque) a uma chave de alocação de item, vá para **Planejamento mestre** &gt; **Configuração** &gt; **Previsão de demanda** &gt; **Chaves de alocação de itens**.</span><span class="sxs-lookup"><span data-stu-id="5c368-112">To add a stock keeping unit (SKU) to an item allocation key, go to **Master planning** &gt; **Setup** &gt; **Demand forecasting** &gt; **Item allocation keys**.</span></span> <span data-ttu-id="5c368-113">Use a página **Atribuir itens** para atribuir um item a uma chave de alocação.</span><span class="sxs-lookup"><span data-stu-id="5c368-113">Use the **Assign items** page to assign an item to an allocation key.</span></span>

## <a name="intercompany-planning-groups"></a><span data-ttu-id="5c368-114">Grupos de planejamento intercompanhia</span><span class="sxs-lookup"><span data-stu-id="5c368-114">Intercompany planning groups</span></span>
<span data-ttu-id="5c368-115">A previsão de demanda gera previsões interempresariais.</span><span class="sxs-lookup"><span data-stu-id="5c368-115">Demand forecasting generates cross-company forecasts.</span></span> <span data-ttu-id="5c368-116">No Microsoft Dynamics 365 for Finance and Operations, as empresas que são planejadas juntas são mantidas em um grupo de planejamento intercompanhia.</span><span class="sxs-lookup"><span data-stu-id="5c368-116">In Microsoft Dynamics 365 for Finance and Operations, companies that are planned together are grouped into one intercompany planning group.</span></span> <span data-ttu-id="5c368-117">Para especificar, por empresa, quais chaves de alocação de item devem ser consideradas para previsão de demanda, associe uma chave de alocação de item ao membro do grupo de planejamento intercompanhia, acessando **Planejamento mestre** &gt; **Configuração** &gt; **Grupos de planejamento intercompanhia**.</span><span class="sxs-lookup"><span data-stu-id="5c368-117">To specify, per company, which item allocation keys should be considered for demand forecasting, associate an item allocation key with the intercompany planning group member by going to **Master planning** &gt; **Setup** &gt; **Intercompany planning groups**.</span></span> 

<span data-ttu-id="5c368-118">Por padrão, se nenhuma chave de alocação de item for atribuída aos membros do grupo de planejamento intercompanhia, uma previsão de demanda será calculada para todos os itens atribuídos a todas as chaves de alocação de itens de todas as empresas do Finance and Operations.</span><span class="sxs-lookup"><span data-stu-id="5c368-118">By default, if no item allocation keys are assigned to intercompany planning group members, a demand forecast is calculated for all items that are assigned to all item allocation keys from all Finance and Operations companies.</span></span> <span data-ttu-id="5c368-119">Outras opções de filtragem para empresas e chaves de alocação de itens estão disponíveis na página **Gerar previsão estatística**.</span><span class="sxs-lookup"><span data-stu-id="5c368-119">Additional filtering options for companies and item allocation keys are available on the **Generate statistical baseline forecast** page.</span></span> 

<span data-ttu-id="5c368-120">Analise o número de itens previstos.</span><span class="sxs-lookup"><span data-stu-id="5c368-120">Review the number of items that are forecasted.</span></span> <span data-ttu-id="5c368-121">Itens desnecessários podem ocasionar o aumento dos custos quando você usar o Aprendizado de Máquina do Microsoft Azure.</span><span class="sxs-lookup"><span data-stu-id="5c368-121">Unnecessary items might cause increased costs when you use Microsoft Azure Machine Learning.</span></span>

## <a name="demand-forecasting-parameters"></a><span data-ttu-id="5c368-122">Parâmetros de previsão de demanda</span><span class="sxs-lookup"><span data-stu-id="5c368-122">Demand forecasting parameters</span></span>
<span data-ttu-id="5c368-123">Para configurar os parâmetros de previsão de demanda, vá para **Planejamento mestre** &gt; **Configuração** &gt; **Parâmetros de previsão de demanda**.</span><span class="sxs-lookup"><span data-stu-id="5c368-123">To set up demand forecasting parameters, go to **Master planning** &gt; **Setup** &gt; **Demand forecasting parameters**.</span></span> <span data-ttu-id="5c368-124">Como a previsão de demanda é realizada entre empresas, a configuração é global.</span><span class="sxs-lookup"><span data-stu-id="5c368-124">Because demand forecasting runs cross-company, the setup is global.</span></span> <span data-ttu-id="5c368-125">Em outras palavras, a configuração se aplica a todas as empresas.</span><span class="sxs-lookup"><span data-stu-id="5c368-125">In other words, the setup applies to all companies.</span></span> 

<span data-ttu-id="5c368-126">A previsão de demanda gera a previsão de quantidades.</span><span class="sxs-lookup"><span data-stu-id="5c368-126">Demand forecasting generates the forecast in
