# <a name="master-plans"></a><span data-ttu-id="a391e-103">Planos mestres</span><span class="sxs-lookup"><span data-stu-id="a391e-103">Master plans</span></span>

[!include[banner](../includes/banner.md)]


<span data-ttu-id="a391e-104">Use diversos planos mestre para oferecer suporte a operações de produção diárias da empresa, simular diferentes estratégias de planejamento que deseja monitorar e implementar uma política corporativa, por exemplo, com relação ao desempenho interno ou à satisfação do cliente.</span><span class="sxs-lookup"><span data-stu-id="a391e-104">Use various master plans to support your company's daily working operations, simulate different planning strategies that you want to monitor, and implement a company policy, such as a policy about internal performance or customer satisfaction.</span></span> 

<span data-ttu-id="a391e-105">Você pode configurar planos mestres na página **Planos mestres**.</span><span class="sxs-lookup"><span data-stu-id="a391e-105">You can configure master plans on the **Master plans** page.</span></span>

<span data-ttu-id="a391e-106">Há dois tipos de planos:</span><span class="sxs-lookup"><span data-stu-id="a391e-106">There are two types of plans:</span></span>
-   <span data-ttu-id="a391e-107">**Plano estático** - o cálculo do planejamento mestre usa os dados atuais para gerar um plano de requisições líquidas.</span><span class="sxs-lookup"><span data-stu-id="a391e-107">**Static plan** – The master planning calculation uses the current data to generate a net requirements plan.</span></span> <span data-ttu-id="a391e-108">Esse plano permanece inalterado até a próxima vez que você executar o planejamento mestre.</span><span class="sxs-lookup"><span data-stu-id="a391e-108">This plan remains unchanged until the next time that you run master planning.</span></span> <span data-ttu-id="a391e-109">É um plano de operações que várias pessoas da empresa, como um comprador ou planejador de produção, podem usar como base de suas decisões e para a realização de tarefas e atividades diárias.</span><span class="sxs-lookup"><span data-stu-id="a391e-109">It is an operating plan that various company personnel, such as a purchaser or production planner, can use to base their decisions on and perform their daily tasks and activities.</span></span>
-   <span data-ttu-id="a391e-110">**Plano dinâmico** - este plano é iniciado com o mesmo plano de requisições líquidas gerado pelo planejamento mestre.</span><span class="sxs-lookup"><span data-stu-id="a391e-110">**Dynamic plan** – This plan starts with the same net requirements plan that was generated by master planning.</span></span> <span data-ttu-id="a391e-111">Contudo, é possível atualizar o plano dinâmico a cada vez que os dados mestres são alterados.</span><span class="sxs-lookup"><span data-stu-id="a391e-111">However, you can update the dynamic plan every time that the master data changes.</span></span> <span data-ttu-id="a391e-112">Isso pode ocorrer na criação de uma nova ordem de venda, por exemplo.</span><span class="sxs-lookup"><span data-stu-id="a391e-112">This could be when you create a new sales order, for example.</span></span> <span data-ttu-id="a391e-113">Isso permite que você monitore a rede de ordens em alteração e a disponibilidade do item sem atrapalhar o plano estático que outros usam para seus processos de trabalho.</span><span class="sxs-lookup"><span data-stu-id="a391e-113">This enables you to monitor the changing order network and item availability without disturbing the static plan that others are using for their work processes.</span></span>

<span data-ttu-id="a391e-114">Uma empresa pode optar por trabalhar apenas com um plano dinâmico ou usar tanto planos estáticos quanto dinâmicos.</span><span class="sxs-lookup"><span data-stu-id="a391e-114">A company may choose to work with just a dynamic plan or it may use both static and dynamic plans.</span></span> <span data-ttu-id="a391e-115">Além disso, é possível configurar qualquer plano mestre para refletir uma determinada estratégia ou abordar um problema.</span><span class="sxs-lookup"><span data-stu-id="a391e-115">In addition, you can configure any master plan to reflect a particular strategy or address an issue.</span></span> <span data-ttu-id="a391e-116">Os exemplos são os seguintes:</span><span class="sxs-lookup"><span data-stu-id="a391e-116">Examples are as follows:</span></span>
-   <span data-ttu-id="a391e-117">Configurar níveis de estoque maiores de modo a evitar o esgotamento do estoque.</span><span class="sxs-lookup"><span data-stu-id="a391e-117">Set higher inventory levels to guarantee against stockouts.</span></span>
-   <span data-ttu-id="a391e-118">Configurar margens de segurança maiores para proteção contra fornecedores não confiáveis.</span><span class="sxs-lookup"><span data-stu-id="a391e-118">Set longer safety margins to protect against unreliable vendors.</span></span>

<span data-ttu-id="a391e-119">Também é possível configurar o início do plano dinâmico para ser atualizado com o novo plano de requisições, cada vez que você executar o planejamento mestre.</span><span class="sxs-lookup"><span data-stu-id="a391e-119">You can also set up the starting dynamic plan so that it is updated with the new requirements plan every time that you run master planning.</span></span> <span data-ttu-id="a391e-120">Você pode especificar essas configurações na página **Parâmetros de planejamento mestre**.</span><span class="sxs-lookup"><span data-stu-id="a391e-120">You can specify these settings on the **Master planning parameters** page.</span></span>
