# <a name="rebate-statuses"></a>Status de reembolsos

[!include [banner](../includes/banner.md)]

O status de reembolsos ajuda a rastrear o status de cada acordo de gerenciamento de reembolso. Você também pode usá-los para fins de relatório e consulta. Você pode configurar e nomear os valores de status de reembolso necessários para dar suporte ao processo comercial. 

Para configurar o status de reembolsos, acesse **Gerenciamento de reembolsos \> Configuração \> Status \> Status**. Use os botões no Painel de Ações para adicionar e remover valores de status, conforme necessário. Em cada status, defina os seguintes campos:

- **Status do reembolso** – Insira um nome para o status (como *Negociação*, *Acordado* ou *Concluído*).
- **Descrição** – Insira uma descrição do status para fornecer mais informações sobre como ela deve ser usada.

Na [página **Parâmetros de gerenciamento de reembolso**](rebate-management-parameters.md), você pode selecionar um status padrão. Esse status será atribuído a todos os novos Acordos de gerenciamento de reembolso criados.
