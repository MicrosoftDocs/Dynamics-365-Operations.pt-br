# <a name="turn-on-the-rebate-management-module-for-your-system"></a>Ativar o módulo de Gerenciamento de reembolsos para o seu sistema

[!include [banner](../includes/banner.md)]

Para que você possa usar o módulo Gerenciamento de reembolsos, ele deve estar ativado no sistema. Os administradores podem usar as configurações do [gerenciamento de recursos](../../fin-ops-core/fin-ops/get-started/feature-management/feature-management-overview.md) para verificar o status do recurso e ativá-lo conforme o necessário. No espaço de trabalho **Gerenciamento de recursos**, o recurso está listado da seguinte forma:

- **Módulo:** *AP e RA compartilhados*
- **Nome do recurso:** *Gerenciamento de reembolsos*
