# <a name="set-up-a-ledger-account-alias"></a>Configurar um alias de conta contábil

[!include [banner](../../includes/banner.md)]

Este procedimento mostra como criar um apelido de conta que fornece um atalho para inserir um número de conta. Este procedimento usa a empresa de dados de demonstração USMF.

1. Acesse **Contabilidade > Plano de contas > Contas > Alias da conta contábil**, clique em **Novo**.
2. No campo **Alias da conta contábil**, digite um valor.
3. No campo **Estrutura de conta**, selecione a estrutura à qual a conta e as dimensões pertencem.
4. No campo **Empresa** na lista, selecione a empresa à qual o alias se aplica.
5. No campo **Definição do alias da conta contábil**, especifique a conta e as dimensões.
    * A conta e as dimensões serão preenchidas quando o atalho for usado.  
6. No campo **Foco inicial**, selecione a dimensão que terá foco quando o alias for usado.
    * Após digitar o atalho e a conta e as dimensões serem preenchidas, o campo **Foco inicial** é para onde o cursor ou foco será movido.
