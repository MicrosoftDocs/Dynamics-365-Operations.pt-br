# <a name="main-account-types"></a>Tipos de conta principal

[!include [banner](../includes/banner.md)]

Os tipos de conta principais determinam como uma conta principal é usada no gráfico de contas.

Você deve selecionar um tipo de conta principal quando criar uma conta principal na página **Contas principais - plano de contas**.
-   **Lucros e perdas**, **Receita** e **Despesa** – essas contas principais são usadas como contas de lançamento. As contas de **Lucros e perdas** são usadas para registrar receitas e despesas. A soma de todas as contas de lucros e perdas, de receita e de despesa determina os resultados de fim de ano. As contas principais de **Receita** e **Despesa** têm a mesma função das contas principais de **Lucros e perdas**.
-   **Balanço**, **Ativo**, **Passivo** e **Capital Próprio** – essas contas principais são usadas como contas de transação para registrar os valores que uma entidade tem ou deve.
-   **Total** – essas contas principais são usadas para adicionar intervalos de contas. Configure o tipo de conta principal **Total** usando a página **Intervalo de contas**, que exibe o total de saldos do grupo para todos os intervalos de contas. Acesse a página **Intervalo de contas** selecionando **Totais** na guia **Geral** após a seleção de um tipo de conta **Total**.
-   **Relatórios** – essas contas principais são usadas para o relatório de demonstrativo financeiro somente para o Brasil.

Para saber mais, consulte [Configurar categorias da conta principal](tasks/set-up-main-account-categories.md).
