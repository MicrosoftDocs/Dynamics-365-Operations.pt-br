# <a name="tax-calculation-data-model"></a>Modelo de dados de cálculo de imposto

Este artigo fornece informações sobre o modelo de dados de imposto e sobre como os valores de campo de cada modelo de dados de imposto são determinados nas transações de cálculo de imposto.

O *modelo de dados de imposto* é composto de campos que são necessários para cálculos de imposto. A Microsoft fornece o *modelo de dados de cálculo de impostos*, que inclui os campos de cabeçalho e os campos de linha dos documentos de transação nos aplicativos de finanças e operações. Os campos definidos no modelo de dados de cálculo de impostos são as colunas disponíveis das tabelas de regras de aplicabilidade na configuração da configuração do recurso de cálculo de imposto.

> [!NOTE] 
> Alguns nós definidos no modelo de dados, como **ID do Registro** e **ID da Tabela**, são para fins técnicos. Elas não são colunas disponíveis na configuração do recurso de cálculo de imposto.

Siga estas etapas para exibir o modelo de dados de cálculo de imposto.

1. Entre no RCS (Regulatory Configuration Service) e acesse **Relatório eletrônico** \> **Configurações de imposto**.
2. Selecione **Modelo de Dados de Imposto** \> **Modelo de Dados de Cálculo de Imposto**.
3. Na Guia Rápida **Versões**, selecione uma versão.
4. Selecione **Designer**.

    [![Botão Designer.](./media/tax-calculation-model-mapping-1.png)](./media/tax-calculation-model-mapping-1.png)

5. Expanda o nó **Serviço de imposto**.
6. Expanda o nó **Cabeçalho** para exibir os modelos de dados de cabeçalho.
7. Expanda o nó **Linhas** para exibir os modelos de dados de linha.

    [![Modelos de dados.](./media/tax-calculation-model-mapping-2.png)](./media/tax-calculation-model-mapping-2.png)

Você também pode exibir o modelo de dados de cálculo de impostos e as colunas disponíveis na tabela de regras de aplicabilidade.

1. No RCS, acesse **Recursos de globalização** \> **Cálculo de imposto**.
2. Localize um recurso de imposto que tenha um status **Rascunho** e, em seguida, selecione **Editar**.
3. Na guia **Geral**, selecione a versão de configuração com base na versão do modelo de dados de cálculo de imposto. Por exemplo, selecione **40.46.212**.
4. Na guia **Aplicabilidade do grupo de impostos**, selecione **Gerenciar colunas**.

    [![Gerenciar colunas.](./media/tax-calculation-model-mapping-3.png)](./media/tax-calculation-model-mapping-3.png)

O modelo de dados de cálculo de imposto é integrado aos aplicativos de finanças e operações. Durante cada transação relacionada ao cálculo de imposto, os valores dos campos definidos nas tabelas de regras de aplicabilidade são coletados e enviados ao serviço de cálculo de imposto para cálculo. Na versão 40.46, 57 modelos de dados predefinidos estão disponíveis para os seguintes tipos de transação: ordens de compra, ordens de venda, ordens de transferência, requisições de compra, solicitações de cotação e cotações de venda.

## <a name="version-updates-and-restrictions"></a>Atualizações e restrições de versão

- **Fatura de texto livre:** compatível com a versão 40.50.221 e o Finance versão 10.0.23.
- **Diários Gerais:** compatível com a versão 40.54.234 e o Finance versão 10.0.26.

    *Restrição:* não há suporte para [um comprovante](../general-ledger/one-voucher.md#issues-with-one-voucher) e uma taxa de pagamento.

- **Diário de faturas de fornecedores:** compatível com a versão 40.54.234 e o Finance versão 10.0.26.
- **Diários periódicos:** compatível com a versão 40.55.236 e o Finance versão 10.0.29.
