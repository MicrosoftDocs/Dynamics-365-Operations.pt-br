# <a name="set-up-tds-ledger-accounts"></a>Configurar contas contábeis do TDS

[!include [banner](../includes/banner.md)]

Este tópico explica como configurar contas contábeis para o recurso de Imposto Deduzido na Origem (TDS). Use a página **Plano de contas** para configurar contas contábeis para TDS.

1. Vá para **Contabilidade \> Plano de contas \> Contas \> Contas principais**.
2. Na guia **Visão geral**, selecione **Alt+N** para criar uma conta contábil de TDS e insira os detalhes necessários.
3. Na guia **Configuração**, no campo **Tipo de lançamento**, selecione **Imposto retido na fonte**.     

    > [!NOTE]
    > As contas contábeis que têm um tipo de lançamento de **Imposto retido na fonte** podem ser definidas somente como contas a receber usadas para lançar o valor de TDS a receber para um código de imposto de TDS.

4. Feche a página.
