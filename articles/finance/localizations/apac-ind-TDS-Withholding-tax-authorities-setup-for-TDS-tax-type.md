# <a name="set-up-withholding-tax-authorities-for-the-tds-tax-type"></a><span data-ttu-id="699e4-103">Configurar autoridades de imposto retido na fonte para os tipos de imposto TDS</span><span class="sxs-lookup"><span data-stu-id="699e4-103">Set up withholding tax authorities for the TDS tax type</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="699e4-104">Este tópico explica como configurar autoridades de Imposto Deduzido na Origem (TDS).</span><span class="sxs-lookup"><span data-stu-id="699e4-104">This topic explains how to set up Tax Deducted at Source (TDS) authorities.</span></span>

1. <span data-ttu-id="699e4-105">Vá para **Imposto \> Impostos indiretos \> Autoridades de imposto retido na fonte**.</span><span class="sxs-lookup"><span data-stu-id="699e4-105">Go to **Tax \> Indirect taxes \> Withholding tax authorities**.</span></span>

    <span data-ttu-id="699e4-106">[![Página Autoridades de imposto retido na fonte](./media/apac-ind-TDS-12.png)](./media/apac-ind-TDS-12.png)</span><span class="sxs-lookup"><span data-stu-id="699e4-106">[![Withholding tax authorities page](./media/apac-ind-TDS-12.png)](./media/apac-ind-TDS-12.png)</span></span>

2. <span data-ttu-id="699e4-107">No campo **Tipo de imposto**, selecione **TDS** para configurar autoridades de imposto retido na fonte para o tipo de imposto TDS.</span><span class="sxs-lookup"><span data-stu-id="699e4-107">In the **Tax type** field, select **TDS** to set up withholding tax authorities for the TDS tax type.</span></span>
3. <span data-ttu-id="699e4-108">No Painel de Ações, selecione **Novo** para criar uma linha.</span><span class="sxs-lookup"><span data-stu-id="699e4-108">On the Action Pane, select **New** to create a line.</span></span>
4. <span data-ttu-id="699e4-109">No campo **Autoridade do imposto retido na fonte**, informe o nome da autoridade TDS.</span><span class="sxs-lookup"><span data-stu-id="699e4-109">In the **Withholding tax authority** field, enter a name for the TDS authority.</span></span>
5. <span data-ttu-id="699e4-110">No campo **Descrição**, insira uma descrição.</span><span class="sxs-lookup"><span data-stu-id="699e4-110">In the **Description** field, enter a description.</span></span>
6. <span data-ttu-id="699e4-111">Na FastTab **Geral**, no campo **Conta do fornecedor**, selecione a conta do fornecedor para a autoridade de TDS.</span><span class="sxs-lookup"><span data-stu-id="699e4-111">On the **General** FastTab, in the **Vendor account** field, select the vendor account for the TDS authority.</span></span>

    > [!NOTE]
    > <span data-ttu-id="699e4-112">Você deve definir o nome do banco em que os fundos devidos ao fornecedor da autoridade de TDS serão depositados.</span><span class="sxs-lookup"><span data-stu-id="699e4-112">You must define the name of the bank where funds that are owed to the TDS authority vendor will be deposited.</span></span> <span data-ttu-id="699e4-113">O nome do banco é definido na página **Contas bancárias** (**Contas a pagar \> Todos os fornecedores \> Fornecedor \> Configurar \> Contas bancárias**).</span><span class="sxs-lookup"><span data-stu-id="699e4-113">The bank's name is defined on the **Bank accounts** page (**Accounts payable \> All vendors \> Vendor \> Set up \> Bank accounts**).</span></span>

7. <span data-ttu-id="699e4-114">Feche a página.</span><span class="sxs-lookup"><span data-stu-id="699e4-114">Close the page.</span></span>
