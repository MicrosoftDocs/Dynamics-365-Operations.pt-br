# <a name="mandatory-default-descriptions-for-posting-of-bank-transactions"></a>Descrições padrão obrigatórias para lançamento de transações bancárias

[!include [banner](../includes/banner.md)]

O recurso de descrição da **transação bancária obrigatória (Brasil)** permite que você exija que as descrições sejam inseridas em transações de reconciliação de conta. Dessa forma, você pode impedir que os diários de transações bancárias sejam lançados, caso contenham linhas de transação que não tenham descrição.

1. No Microsoft Dynamics 365 Finance ou Dynamics 365 Supply Chain Management abrir o espaço de trabalho **Gerenciamento de recursos**.
2. Na lista de recursos, procurar por **(Brasil) Descrição da transação bancária obrigatória**. Se esse recurso não aparecer na página, selecione **Verificar se há atualizações**.
3. Selecione o recurso e **Habilitar agora**.

Você pode desativar o recurso após ele ter sido ativado.
