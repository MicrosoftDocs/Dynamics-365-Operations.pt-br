# <a name="process-electronic-payment-remittances"></a><span data-ttu-id="06478-103">Processar remessas de pagamento eletrônico</span><span class="sxs-lookup"><span data-stu-id="06478-103">Process electronic payment remittances</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="06478-104">Utilize esta funcionalidade para criar o arquivo que deve ser enviado ao banco com as instruções para o pagamento eletrônico de faturas do fornecedor.</span><span class="sxs-lookup"><span data-stu-id="06478-104">Use this functionality to create the file that must be sent to the bank with the instructions for electronic payment of vendor invoices.</span></span>

1.  <span data-ttu-id="06478-105">Na página **Diário de pagamentos**, clique em **Novo**.</span><span class="sxs-lookup"><span data-stu-id="06478-105">On the **Payment journal** page, click **New**.</span></span>
3.  <span data-ttu-id="06478-106">No campo Nome, insira um nome para o diário de pagamentos.</span><span class="sxs-lookup"><span data-stu-id="06478-106">In the Name field, enter a name of the payment journal.</span></span>
4.  <span data-ttu-id="06478-107">Clique em **Linhas**.</span><span class="sxs-lookup"><span data-stu-id="06478-107">Click **Lines**.</span></span>
5.  <span data-ttu-id="06478-108">No campo **Conta**, selecione a conta.</span><span class="sxs-lookup"><span data-stu-id="06478-108">In the **Account** field, select the account.</span></span>
6.  <span data-ttu-id="06478-109">No campo **Descrição**, insira ou selecione um valor.</span><span class="sxs-lookup"><span data-stu-id="06478-109">In the **Description** field, enter or select a value.</span></span>
7.  <span data-ttu-id="06478-110">Clique em **Liquidar transações** e, na lista, encontre e selecione as transações a liquidar.</span><span class="sxs-lookup"><span data-stu-id="06478-110">Click **Settle transactions** and in the list, find and select the transactions to settle.</span></span>
8.  <span data-ttu-id="06478-111">Clique na guia **Pagamento** e no campo **Método de pagamento** selecione um método de pagamento.</span><span class="sxs-lookup"><span data-stu-id="06478-111">Click the **Payment** tab and in the **Method of payment** field, select a method of payment.</span></span>
9.  <span data-ttu-id="06478-112">No campo **Especificação de Pagamento**, insira ou selecione um valor.</span><span class="sxs-lookup"><span data-stu-id="06478-112">In the **Payment specification** field, enter or select a value.</span></span>
10. <span data-ttu-id="06478-113">No campo **Código de barras**, digite um valor</span><span class="sxs-lookup"><span data-stu-id="06478-113">In the **Bar code** field, type a value.</span></span>
11. <span data-ttu-id="06478-114">Clique em **OK**.</span><span class="sxs-lookup"><span data-stu-id="06478-114">Click **OK**.</span></span>
12. <span data-ttu-id="06478-115">Clique em **Gerar pagamentos** e, no campo **Método de pagamento**, selecione um método de pagamento.</span><span class="sxs-lookup"><span data-stu-id="06478-115">Click **Generate payments** and in the **Method of payment** field, select a method of payment.</span></span>
13. <span data-ttu-id="06478-116">No campo **Conta bancária**, insira ou selecione uma conta bancária.</span><span class="sxs-lookup"><span data-stu-id="06478-116">In the **Bank account** field, enter or select a bank account.</span></span>
14. <span data-ttu-id="06478-117">Clique em **OK**.</span><span class="sxs-lookup"><span data-stu-id="06478-117">Click **OK**.</span></span>
15. <span data-ttu-id="06478-118">Clique em **Salvar**.</span><span class="sxs-lookup"><span data-stu-id="06478-118">Click **Save**.</span></span>
