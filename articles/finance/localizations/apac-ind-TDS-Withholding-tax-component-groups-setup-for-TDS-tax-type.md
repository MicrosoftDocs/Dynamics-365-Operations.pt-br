# <a name="set-up-withholding-tax-component-groups-for-the-tds-tax-type"></a><span data-ttu-id="34c34-103">Configurar grupos de componentes de imposto retido na fonte para tipos de imposto TDS</span><span class="sxs-lookup"><span data-stu-id="34c34-103">Set up withholding tax component groups for the TDS tax type</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="34c34-104">Este tópico explica como configurar grupos de componentes de imposto retido na fonte, como **Aluguel** e **Prestador de serviço**, para o tipo de Imposto Deduzido na Origem (TDS).</span><span class="sxs-lookup"><span data-stu-id="34c34-104">This topic explains how to set up withholding tax component groups, such as **Rent** and **Contractor**, for the Tax Deducted at Source (TDS) tax type.</span></span>

1. <span data-ttu-id="34c34-105">Vá para **Imposto \> Configuração \> Imposto retido na fonte \> Grupos de componentes de imposto retido na fonte**.</span><span class="sxs-lookup"><span data-stu-id="34c34-105">Go to **Tax \> Setup \> Withholding tax \> Withholding tax component groups**.</span></span>

    <span data-ttu-id="34c34-106">[![Página grupos de componentes de imposto retido na fonte](./media/apac-ind-TDS-8.png)](./media/apac-ind-TDS-8.png)</span><span class="sxs-lookup"><span data-stu-id="34c34-106">[![Withholding tax component groups page](./media/apac-ind-TDS-8.png)](./media/apac-ind-TDS-8.png)</span></span>

2. <span data-ttu-id="34c34-107">No campo **Tipo de imposto**, selecione **TDS** para configurar grupos de componentes do imposto retido na fonte para o tipo de imposto TDS.</span><span class="sxs-lookup"><span data-stu-id="34c34-107">In the **Tax type** field, select **TDS** to set up withholding tax component groups for the TDS tax type.</span></span>
3. <span data-ttu-id="34c34-108">No Painel de Ações, selecione **Novo** para criar uma linha.</span><span class="sxs-lookup"><span data-stu-id="34c34-108">On the Action Pane, select **New** to create a line.</span></span>
4. <span data-ttu-id="34c34-109">No campo **Grupo de componentes do imposto retido na fonte**, informe o grupo de componentes de TDS.</span><span class="sxs-lookup"><span data-stu-id="34c34-109">In the **Withholding tax component group** field, enter a name for the TDS component group.</span></span>
5. <span data-ttu-id="34c34-110">No campo **Descrição**, insira uma descrição.</span><span class="sxs-lookup"><span data-stu-id="34c34-110">In the **Description** field, enter a description.</span></span>
6. <span data-ttu-id="34c34-111">Na FastTab **Geral**, no campo **Status**, selecione o status residencial do grupo de componentes de TDS.</span><span class="sxs-lookup"><span data-stu-id="34c34-111">On the **General** FastTab, in the **Status** field, select the residential status of the TDS component group.</span></span> <span data-ttu-id="34c34-112">As opções são **Residente** e **Não residente**.</span><span class="sxs-lookup"><span data-stu-id="34c34-112">The options are **Resident** and **Non-resident**.</span></span>
7. <span data-ttu-id="34c34-113">No campo **Código da seção**, insira o código de seção que é aplicado ao grupo de componentes de TDS.</span><span class="sxs-lookup"><span data-stu-id="34c34-113">In the **Section code** field, enter the section code that is applied to the TDS component group.</span></span>
8. <span data-ttu-id="34c34-114">Feche a página.</span><span class="sxs-lookup"><span data-stu-id="34c34-114">Close the page.</span></span>
