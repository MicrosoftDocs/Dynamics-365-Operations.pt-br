# <a name="set-up-withholding-tax-component-groups-for-the-tds-tax-type"></a>Configurar grupos de componentes de imposto retido na fonte para tipos de imposto TDS

[!include [banner](../includes/banner.md)]

Este tópico explica como configurar grupos de componentes de imposto retido na fonte, como **Aluguel** e **Prestador de serviço**, para o tipo de Imposto Deduzido na Origem (TDS).

1. Vá para **Imposto \> Configuração \> Imposto retido na fonte \> Grupos de componentes de imposto retido na fonte**.

    [![Página grupos de componentes de imposto retido na fonte](./media/apac-ind-TDS-8.png)](./media/apac-ind-TDS-8.png)

2. No campo **Tipo de imposto**, selecione **TDS** para configurar grupos de componentes do imposto retido na fonte para o tipo de imposto TDS.
3. No Painel de Ações, selecione **Novo** para criar uma linha.
4. No campo **Grupo de componentes do imposto retido na fonte**, informe o grupo de componentes de TDS.
5. No campo **Descrição**, insira uma descrição.
6. Na FastTab **Geral**, no campo **Status**, selecione o status residencial do grupo de componentes de TDS. As opções são **Residente** e **Não residente**.
7. No campo **Código da seção**, insira o código de seção que é aplicado ao grupo de componentes de TDS.
8. Feche a página.
