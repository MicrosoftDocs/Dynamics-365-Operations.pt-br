# <a name="copy-a-billing-schedule"></a>Copiar uma agenda de cobrança

[!include [banner](../includes/banner.md)]

Você pode usar a página **Copiar agenda** para criar uma nova agenda de cobrança de uma agenda de cobrança existente. A nova agenda de cobrança terá as mesmas configurações que a agenda de cobrança original. Você também pode usar essa página para copiar itens de uma agenda de cobrança para outra.

## <a name="copy-a-billing-schedule"></a>Copiar uma agenda de cobrança

Para copiar uma agenda de cobrança, siga estas etapas.

1. Na página de listagem **Todas as agendas de cobrança** ou **Agendas de cobrança ativas**, selecione uma agenda de cobrança.
2. Na guia **Agenda de cobrança**, selecione **Copiar agenda**.
3. No campo **Conta de cliente**, selecione a conta do cliente.
4. Atualize as datas de início e de término da cobrança conforme necessário.
5. Atualize os valores dos seguintes campos opcionais, conforme necessário: **Conta de usuário final**, **Conta de parceiro**, **Referência de cliente**, **Requisição de cliente** e **Data de início do adiamento**.
6. Selecione **Exibir pré-visualização** para exibir e selecionar as linhas da agenda de cobrança que você deseja copiar.
7. Selecione **Copiar**.

Depois que o processo de cópia for concluído, a nova agenda de cobrança será exibida na parte inferior da página de listagem **Todas as agendas de cobrança** ou **Agendas de cobrança ativas** e na página **Agendas de cobrança**.

## <a name="copy-items-from-one-billing-schedule-to-another"></a>Copiar itens de uma agenda de cobrança para outra

1. Na página de listagem **Todas as agendas de cobrança** ou **Agendas de cobrança ativas**, selecione uma agenda de cobrança.
2. Na guia **Agenda de cobrança**, selecione **Copiar agenda**.
3. No campo **Conta de cliente**, selecione a conta do cliente.
4. No campo **Número da agenda de cobrança**, selecione um número de agenda de cobrança.
4. Atualize as datas de início e de término da cobrança.
5. Atualize os valores dos seguintes campos, conforme necessário: **Conta de usuário final**, **Conta de parceiro**, **Referência de cliente**, **Requisição de cliente** e **Data de início do adiamento**.
6. Selecione **Exibir pré-visualização** para exibir e selecionar as linhas da agenda de cobrança que você deseja copiar para uma agenda de cobrança existente.
7. Selecione **Copiar**.

Ao concluir o processo de cópia, revise a agenda de cobrança para a qual você copiou os itens e verifique se os itens foram adicionados conforme o esperado.
