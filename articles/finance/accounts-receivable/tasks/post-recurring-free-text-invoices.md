# <a name="generate-and-post-recurring-free-text-invoices"></a>Gerar e lançar faturas de texto livre recorrentes

[!include [banner](../../includes/banner.md)]

Faturas recorrentes são usadas para faturar clientes regularmente com o mesmo valor. Este registro usa a empresa de dados de demonstração USMF. O registro é direcionado para a pessoa responsável por gerenciar e processar faturas A/R.


## <a name="generate-recurring-invoices"></a>Gerar faturas recorrentes

## <a name="post-recurring-invoices"></a>Lançar faturas recorrentes
1. Acesse **Contas a receber > Faturas > Faturas recorrentes > Lançar faturas recorrentes**.
    * Use esta página para exibir e imprimir as notas fiscais recorrentes que foram geradas anteriormente.  
2. Na lista, clique no link na linha selecionada.
    * Selecione o grupo de fatura recorrente.  
3. Clique em **Totais**.
    * Verificar totais do grupo da nota fiscal recorrente.  
4. Clique em **Fechar**.
    * Cada linha abaixo é uma fatura de texto livre recorrente. Você pode selecionar uma linha e clicar no botão de **Detalhes** para exibir detalhes da fatura em texto livre.  
5. Clique em **Validar**.
    * Verifique se as notas fiscais selecionadas não têm erros, mas não lance as notas fiscais.  
6. Clique em **Enviar**.
    * Lançar as faturas selecionadas.
