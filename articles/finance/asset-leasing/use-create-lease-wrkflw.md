# <a name="use-lease-approval-workflows"></a><span data-ttu-id="29783-103">Usar fluxos de trabalho de aprovação de arrendamento</span><span class="sxs-lookup"><span data-stu-id="29783-103">Use lease approval workflows</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="29783-104">Este tópico explica como usar fluxos de trabalho para aprovar arrendamentos de ativos e como rastrear o status e o histórico dos fluxos de trabalho.</span><span class="sxs-lookup"><span data-stu-id="29783-104">This topic explains how to use workflows to approve asset leases, and how to track the status and history of the workflows.</span></span> <span data-ttu-id="29783-105">Os fluxos de trabalho ajudam a proporcionar consistência ao gerenciamento de aprovações de arrendamento, fornecendo um conjunto padrão de etapas de aprovação e atribuindo usuários específicos que aprovem cada etapa do processo.</span><span class="sxs-lookup"><span data-stu-id="29783-105">Workflows help bring consistency to the management of lease approvals by providing a standard set of approval steps and assigning specific users who approve each step of the process.</span></span> <span data-ttu-id="29783-106">Um aprovador pode aprovar um arrendamento, rejeitá-lo, solicitar uma alteração nele ou atribuí-lo a outro usuário para aprovação.</span><span class="sxs-lookup"><span data-stu-id="29783-106">An approver can approve a lease, reject it, request a change to it, or assign it to another user for approval.</span></span> <span data-ttu-id="29783-107">Os fluxos de trabalho também podem trazer mais visibilidade para o processo de aprovação, permitindo que você controle o status e o histórico.</span><span class="sxs-lookup"><span data-stu-id="29783-107">Workflows can also bring more visibility into the approval process by letting you track their status and history.</span></span> <span data-ttu-id="29783-108">Além disso, é possível exibir uma lista de trabalho centralizada que liste as tarefas e aprovações atribuídas a aprovadores específicos.</span><span class="sxs-lookup"><span data-stu-id="29783-108">Additionally, you can view a centralized worklist that lists the tasks and approvals that are assigned to specific approvers.</span></span>

<span data-ttu-id="29783-109">Antes de usar esse procedimento, certifique-se de que pelo menos o fluxo de trabalho de aprovação de arrendamento tenha sido criado.</span><span class="sxs-lookup"><span data-stu-id="29783-109">Before you use this procedure, make sure that at least on lease approval workflow has been created.</span></span> <span data-ttu-id="29783-110">Se não houver nenhum fluxo de trabalho, crie um.</span><span class="sxs-lookup"><span data-stu-id="29783-110">If no workflow exists, create one.</span></span> <span data-ttu-id="29783-111">Para obter informações sobre como configurar um fluxo de trabalho, consulte [Configurar fluxos de trabalho de aprovação de arrendamento](set-up-lease-wrkflw.md).</span><span class="sxs-lookup"><span data-stu-id="29783-111">For information about how to set up a workflow, see [Set up lease approval workflows](set-up-lease-wrkflw.md).</span></span>

1. <span data-ttu-id="29783-112">Envie um arrendamento para aprovação.</span><span class="sxs-lookup"><span data-stu-id="29783-112">Submit a lease for approval.</span></span> <span data-ttu-id="29783-113">Na página **Detalhes do registro** do arrendamento, selecione **Fluxo de trabalho** e, em seguida, selecione **Enviar**.</span><span class="sxs-lookup"><span data-stu-id="29783-113">On the **Book details** page for the lease, select **Workflow**, and then select **Submit**.</span></span>
2. <span data-ttu-id="29783-114">Na caixa de diálogo exibida, você pode adicionar um comentário.</span><span class="sxs-lookup"><span data-stu-id="29783-114">In the dialog box that appears, you can add a comment.</span></span> <span data-ttu-id="29783-115">O aprovador designado verá esse comentário juntamente com o arrendamento.</span><span class="sxs-lookup"><span data-stu-id="29783-115">The designated approver will see this comment together with the lease.</span></span> <span data-ttu-id="29783-116">Ao terminar de inserir o comentário, selecione **Enviar**.</span><span class="sxs-lookup"><span data-stu-id="29783-116">When you've finished entering the comment, select **Submit**.</span></span> <span data-ttu-id="29783-117">O arrendamento é enviado para o sistema de fluxo de trabalho e o aprovador o recebe para aprovação.</span><span class="sxs-lookup"><span data-stu-id="29783-117">The lease is submitted to the workflow system, and the approver receives it for approval.</span></span>
3. <span data-ttu-id="29783-118">Para exibir os arrendamentos designados para aprovação, vá para **Módulos \> Comum \> Itens de trabalho \> Itens de trabalho atribuídos a mim**.</span><span class="sxs-lookup"><span data-stu-id="29783-118">To view the leases that they are designated for approval, go to **Modules \> Common \> Work items \> Work items assigned to me**.</span></span>
4. <span data-ttu-id="29783-119">Na página **Itens de trabalho atribuídos a mim**, selecione o link **ID do Arrendamento** para o arrendamento que você deseja exibir.</span><span class="sxs-lookup"><span data-stu-id="29783-119">On the **Work items assigned to me** page, select the **Lease ID** link for the lease that you want to view.</span></span> <span data-ttu-id="29783-120">A página exibida depende dos registros de arrendamento e dos detalhes do arrendamento aos quais você tem acesso.</span><span class="sxs-lookup"><span data-stu-id="29783-120">The page that appears depends on the lease books and lease details that you have access to.</span></span>
5. <span data-ttu-id="29783-121">Quando terminar de exibir o arrendamento, selecione **Fluxo de trabalho** e decida qual ação deve ser executada.</span><span class="sxs-lookup"><span data-stu-id="29783-121">When you've finish viewing the lease, select **Workflow**, and decide what action should be taken.</span></span> <span data-ttu-id="29783-122">As opções incluem **Aprovar**, **Rejeitar**, **Solicitar alteração**, **Delegar** e **Cancelado**.</span><span class="sxs-lookup"><span data-stu-id="29783-122">The options include **Approve**, **Rejected**, **Request change**, **Delegate**, and **Canceled**.</span></span> <span data-ttu-id="29783-123">Você também pode selecionar **Exibir histórico** para exibir o histórico de aprovação para o arrendamento selecionado.</span><span class="sxs-lookup"><span data-stu-id="29783-123">You can also select **View history** to view the approval history for the selected lease.</span></span>
6. <span data-ttu-id="29783-124">Depois de selecionar uma ação, insira um comentário para descrever a ação.</span><span class="sxs-lookup"><span data-stu-id="29783-124">After you've selected an action, enter a comment to describe the action.</span></span> <span data-ttu-id="29783-125">Quando tiver terminado de inserir o comentário, selecione a ação **Aprovado** na lista.</span><span class="sxs-lookup"><span data-stu-id="29783-125">When you've finished entering the comment, select the **Approved** action in the list.</span></span>
7. <span data-ttu-id="29783-126">Para exibir as ações de aprovação, volte para a página **Detalhes do arrendamento** na página **Resumo do arrendamento** e selecione **Fluxo de trabalho \> Exibir histórico**.</span><span class="sxs-lookup"><span data-stu-id="29783-126">To view the approval actions, go back to the **Lease details** page from the **Lease summary** page, and then select **Workflow \> View history**.</span></span>

    <span data-ttu-id="29783-127">Você pode exibir as atividades de fluxo de trabalho na página **Histórico do fluxo de trabalho**.</span><span class="sxs-lookup"><span data-stu-id="29783-127">You can view the workflow activities on the **Workflow history** page.</span></span> <span data-ttu-id="29783-128">Esta página mostra as etapas do fluxo de trabalho que foram executadas no arrendamento específico.</span><span class="sxs-lookup"><span data-stu-id="29783-128">This page shows the workflow steps that have been taken on the specific lease.</span></span> <span data-ttu-id="29783-129">Você também pode usar o campo **Itens de trabalho** para exibir o status dos itens de trabalho atribuídos.</span><span class="sxs-lookup"><span data-stu-id="29783-129">You can also use the **Work items** field to view the status of the assigned work items.</span></span>

8. <span data-ttu-id="29783-130">Para interromper um fluxo de trabalho, na página **Histórico do fluxo de trabalho**, selecione **Recuperar**.</span><span class="sxs-lookup"><span data-stu-id="29783-130">To stop a workflow, on the **Workflow history** page, select **Recall**.</span></span> <span data-ttu-id="29783-131">Na caixa de diálogo mostrada, insira um comentário e selecione **OK**.</span><span class="sxs-lookup"><span data-stu-id="29783-131">In the dialog box that appears, enter a comment, and then select **OK**.</span></span>
9. <span data-ttu-id="29783-132">Para desativar um fluxo de trabalho ou ativar um fluxo de trabalho criado anteriormente, acesse **Arrendamento de ativo \> Configuração \> Fluxo de trabalho de arrendamento**.</span><span class="sxs-lookup"><span data-stu-id="29783-132">To inactivate a workflow, or to activate a workflow that was previously created, go to **Asset leasing \> Setup \> Lease workflow**.</span></span> <span data-ttu-id="29783-133">Em seguida, na página **Fluxo de trabalho de arrendamento**, selecione **Fluxo de trabalho \> Versões**.</span><span class="sxs-lookup"><span data-stu-id="29783-133">Then, on the **Lease workflow** page, select **Workflow \> Versions**.</span></span> <span data-ttu-id="29783-134">Para tornar um fluxo de trabalho atual inativo, selecione o arrendamento ativo na caixa de diálogo da versão do arrendamento e, em seguida, selecione **Tornar inativo**.</span><span class="sxs-lookup"><span data-stu-id="29783-134">To make a current workflow inactive, select the active lease in the lease version dialog box, and then select **Make inactive**.</span></span> <span data-ttu-id="29783-135">Para ativar um fluxo de trabalho existente, selecione o fluxo de trabalho e selecione **Tornar ativo**.</span><span class="sxs-lookup"><span data-stu-id="29783-135">To make an existing workflow active, select the workflow, and then select **Make active**.</span></span>
