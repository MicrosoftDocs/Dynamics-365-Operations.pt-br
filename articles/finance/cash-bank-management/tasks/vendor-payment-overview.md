# <a name="vendor-payment-overview"></a><span data-ttu-id="b4a41-103">Visão geral de pagamentos de fornecedor</span><span class="sxs-lookup"><span data-stu-id="b4a41-103">Vendor payment overview</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="b4a41-104">Este guia de tarefas irá apresentar vários métodos utilizados para criar pagamentos de fornecedor, incluindo como usar uma proposta de pagamento ou inserir manualmente um pagamento único.</span><span class="sxs-lookup"><span data-stu-id="b4a41-104">This task guide will walk you through various methods used to create vendor payments, including how to use a payment proposal or manually entering a one-off payment.</span></span> <span data-ttu-id="b4a41-105">Este procedimento usa a empresa de dados de demonstração USMF.</span><span class="sxs-lookup"><span data-stu-id="b4a41-105">This procedure uses the USMF demo company.</span></span>

1. <span data-ttu-id="b4a41-106">Vá para **Painel de Navegação > Módulos > Contas a pagar > Pagamentos > Diário de pagamentos**.</span><span class="sxs-lookup"><span data-stu-id="b4a41-106">Go to **Navigation pane > Modules > Accounts payable > Payments > Payment journal**.</span></span>
2. <span data-ttu-id="b4a41-107">Clique em **Novo**.</span><span class="sxs-lookup"><span data-stu-id="b4a41-107">Click **New**.</span></span>
3. <span data-ttu-id="b4a41-108">Selecione o diário de pagamentos para salvar os pagamentos do fornecedor.</span><span class="sxs-lookup"><span data-stu-id="b4a41-108">Select the payment journal in which to save the vendor payments.</span></span> 
4. <span data-ttu-id="b4a41-109">Selecione o diário ou insira-o manualmente.</span><span class="sxs-lookup"><span data-stu-id="b4a41-109">Select the journal or manually enter it.</span></span>
5. <span data-ttu-id="b4a41-110">Clique em **Linhas**.</span><span class="sxs-lookup"><span data-stu-id="b4a41-110">Click **Lines**.</span></span>
6. <span data-ttu-id="b4a41-111">No **Painel de ação**, clique em **Proposta de pagamento**.</span><span class="sxs-lookup"><span data-stu-id="b4a41-111">On **Ation pane**, click **Payment proposal**.</span></span>
7. <span data-ttu-id="b4a41-112">Clique em **Criar proposta de pagamento**.</span><span class="sxs-lookup"><span data-stu-id="b4a41-112">Click **Create payment proposal**.</span></span> <span data-ttu-id="b4a41-113">A proposta de pagamento é uma consulta utilizada para selecionar faturas para o pagamento.</span><span class="sxs-lookup"><span data-stu-id="b4a41-113">The payment proposal is a query used to select invoices for payment.</span></span> <span data-ttu-id="b4a41-114">Você pode editar a lista de faturas a serem pagas antes de criar ou de gerar os pagamentos do fornecedor.</span><span class="sxs-lookup"><span data-stu-id="b4a41-114">You can edit the list of invoices to pay before creating or generating the vendor payments.</span></span>
8. <span data-ttu-id="b4a41-115">Selecionar faturas para o pagamento por data de vencimento, desconto à vista, ou ambos.</span><span class="sxs-lookup"><span data-stu-id="b4a41-115">Select invoices for payment by due date, cash discount, or both.</span></span> 
9. <span data-ttu-id="b4a41-116">Insira a data para comparação com a data de vencimento ou de desconto à vista.</span><span class="sxs-lookup"><span data-stu-id="b4a41-116">Enter the date for comparing to the due date or cash discount.</span></span> 
10. <span data-ttu-id="b4a41-117">Opcional: Insira uma data de pagamento usada para todos os pagamentos.</span><span class="sxs-lookup"><span data-stu-id="b4a41-117">Optional: Enter a payment date used for all payments.</span></span> <span data-ttu-id="b4a41-118">A data inserida aqui será a data de pagamento para todos os pagamentos criados, independentemente da data de vencimento ou da data de desconto à vista.</span><span class="sxs-lookup"><span data-stu-id="b4a41-118">The date entered here will be the payment date for all payments created, regardless of the due date or cash discount date.</span></span>  
11. <span data-ttu-id="b4a41-119">Opcional: Insira uma data de pagamento mínimo que pode ser usada como a data de pagamento.</span><span class="sxs-lookup"><span data-stu-id="b4a41-119">Optional: Enter a minimum payment date which may be used as the payment date.</span></span> <span data-ttu-id="b4a41-120">A data de pagamento mínimo será a primeira data usada durante a criação dos pagamentos.</span><span class="sxs-lookup"><span data-stu-id="b4a41-120">The minimum payment date will be the earliest date used when creating payments.</span></span> <span data-ttu-id="b4a41-121">Por exemplo, se uma fatura tem sua data de vencimento após a data de pagamento mínimo, a data de vencimento passará a ser a data de pagamento em vez da data de pagamento mínima, visando pagar a fatura na data mais distante possível.</span><span class="sxs-lookup"><span data-stu-id="b4a41-121">For example, if an invoice has a due date after the minimum payment date, the due date will become the payment date instead of the minimum payment date in order to pay the invoice on the latest possible date.</span></span>
12. <span data-ttu-id="b4a41-122">Insira restrições de consulta adicionais na seção **Registros a serem incluídos**.</span><span class="sxs-lookup"><span data-stu-id="b4a41-122">Enter additional query restrictions under **Records to include** section.</span></span> <span data-ttu-id="b4a41-123">O filtro é usado geralmente para restringir as faturas selecionadas para pagamento por grupo de fornecedores ou método de pagamento.</span><span class="sxs-lookup"><span data-stu-id="b4a41-123">The filter is often used to restrict the invoices selected for payment by vendor group or method of payment.</span></span> <span data-ttu-id="b4a41-124">Por exemplo, você pode adicionar um filtro para pagar somente faturas com cheque neste ciclo de pagamentos.</span><span class="sxs-lookup"><span data-stu-id="b4a41-124">For example, you may add a filter to only pay invoices by check in this pay run.</span></span>
13. <span data-ttu-id="b4a41-125">Insira restrições de consulta adicionais ou padrões de pagamento.</span><span class="sxs-lookup"><span data-stu-id="b4a41-125">Enter additional query restriction or payment defaults.</span></span> <span data-ttu-id="b4a41-126">Os parâmetros adicionais podem ser usados para definir a moeda de pagamento ou para permitir pagamentos centralizados para esse ciclo de pagamentos.</span><span class="sxs-lookup"><span data-stu-id="b4a41-126">The additional parameters can be used to define the payment currency or to enable centralized payments for this pay run.</span></span>  
14. <span data-ttu-id="b4a41-127">Clique em **OK**.</span><span class="sxs-lookup"><span data-stu-id="b4a41-127">Click **OK**.</span></span> <span data-ttu-id="b4a41-128">Após clicar em **OK**, os resultados da consulta serão exibidos.</span><span class="sxs-lookup"><span data-stu-id="b4a41-128">After clicking **OK**, the results of the query will appear.</span></span> <span data-ttu-id="b4a41-129">Se você não deseja visualizar a lista de faturas selecionadas para pagamento, você pode voltar para a guia rápida **Parâmetros** e alterar a opção **Criar pagamentos sem visualização da fatura** como "Sim".</span><span class="sxs-lookup"><span data-stu-id="b4a41-129">If you don't want to preview the list of invoices selected to pay, you can go back to the **Parameters** fast tab and change the setting **Create payments without invoice preview** to "Yes".</span></span>  
15. <span data-ttu-id="b4a41-130">Escolha o botão **Exibir visão geral do pagamento** para visualizar os pagamentos que serão criados para o fornecedor na fatura selecionada.</span><span class="sxs-lookup"><span data-stu-id="b4a41-130">Choose the **Show payment overview** button to view the payments that will be created for the vendor on the invoice selected.</span></span>
16. <span data-ttu-id="b4a41-131">Escolha o botão **Ocultar visão geral do pagamento** para ocultar os pagamentos.</span><span class="sxs-lookup"><span data-stu-id="b4a41-131">Choose the **Hide payment overview** button to hide the payments.</span></span> 
17. <span data-ttu-id="b4a41-132">Clique em **Criar pagamentos**.</span><span class="sxs-lookup"><span data-stu-id="b4a41-132">Click **Create payments**.</span></span> <span data-ttu-id="b4a41-133">Antes de escolher **Criar pagamentos**, você pode clicar com o botão direito do mouse na grade e exportar a lista de faturas para o Excel.</span><span class="sxs-lookup"><span data-stu-id="b4a41-133">Before choosing **Create payments**, you can right click on the grid and export the list of invoices to Excel.</span></span> <span data-ttu-id="b4a41-134">O botão **Criar pagamentos** criará os pagamentos de fornecedor no diário de pagamentos.</span><span class="sxs-lookup"><span data-stu-id="b4a41-134">The **Create payments** button will create the vendor payments in the payment journal.</span></span>  
18. <span data-ttu-id="b4a41-135">Examine os pagamentos e verifique se o método de pagamento está definido para todos os pagamentos.</span><span class="sxs-lookup"><span data-stu-id="b4a41-135">Scan your payments and make sure the method of payment is defined for all payments.</span></span> <span data-ttu-id="b4a41-136">Se você gerar os pagamentos, como imprimir um cheque ou criar um pagamento eletrônico, o método de pagamento deve ser definido.</span><span class="sxs-lookup"><span data-stu-id="b4a41-136">If you generate the payments, such as printing a check or creating an electronic payment, the method of payment must be defined.</span></span> <span data-ttu-id="b4a41-137">O método de pagamento também padronizará a conta bancária a partir da qual o pagamento será feito.</span><span class="sxs-lookup"><span data-stu-id="b4a41-137">The method of payment will also default the bank account from the payment will be made.</span></span>  
19. <span data-ttu-id="b4a41-138">Clique em **Novo** para criar um pagamento único.</span><span class="sxs-lookup"><span data-stu-id="b4a41-138">Click **New** to create a one-off payment.</span></span> <span data-ttu-id="b4a41-139">Um pagamento único pode ser adicionado a um diário de pagamentos a qualquer momento antes do envio.</span><span class="sxs-lookup"><span data-stu-id="b4a41-139">A one-off payment can be added to a payment journal at any time prior to posting.</span></span> <span data-ttu-id="b4a41-140">Isso é feito clicando no botão **Novo** e adicionando as informações de pagamento manualmente, em vez de usar a **Proposta de pagamento**.</span><span class="sxs-lookup"><span data-stu-id="b4a41-140">This is done by clicking the **New** button and adding the payment information manually, rather than using the **Payment proposal**.</span></span>  
20. <span data-ttu-id="b4a41-141">Selecione o fornecedor para o qual o pagamento será feito.</span><span class="sxs-lookup"><span data-stu-id="b4a41-141">Select the vendor to whom the payment will be made.</span></span>
21. <span data-ttu-id="b4a41-142">Se existe uma fatura para pagar, selecione **Liquidar transações** para selecionar a fatura para pagamento.</span><span class="sxs-lookup"><span data-stu-id="b4a41-142">If an invoice exists to pay, select **Settle transactions** to select the invoice for payment.</span></span> <span data-ttu-id="b4a41-143">Se este for um pagamento antecipado, essa etapa é opcional.</span><span class="sxs-lookup"><span data-stu-id="b4a41-143">If this is a prepayment, this step is optional.</span></span> <span data-ttu-id="b4a41-144">Você pode criar o pagamento sem selecionar nenhuma fatura.</span><span class="sxs-lookup"><span data-stu-id="b4a41-144">You can create the payment without selecting any invoice.</span></span> 
22. <span data-ttu-id="b4a41-145">Marque todas as faturas que serão pagas.</span><span class="sxs-lookup"><span data-stu-id="b4a41-145">Mark any invoices that will be paid.</span></span> <span data-ttu-id="b4a41-146">Se você usar a opção **Liquidar transações** para selecionar as faturas para pagamento, o valor do pagamento será calculado automaticamente com base nas faturas que você marcou para o pagamento e em qual valor você inseriu em **Quantia a liquidar**.</span><span class="sxs-lookup"><span data-stu-id="b4a41-146">If you use the **Settle transactions** to select the invoices for payment, the payment amount will automatically be calculated based on what invoices you mark for payment, and what amount you enter in the **Amount to settle**.</span></span>
23. <span data-ttu-id="b4a41-147">Clique em **OK**.</span><span class="sxs-lookup"><span data-stu-id="b4a41-147">Click **OK**.</span></span>
24. <span data-ttu-id="b4a41-148">Se quiser excluir um pagamento, marque a linha.</span><span class="sxs-lookup"><span data-stu-id="b4a41-148">If you want to delete a payment, mark the row.</span></span>
25. <span data-ttu-id="b4a41-149">Clique em **Excluir**.</span><span class="sxs-lookup"><span data-stu-id="b4a41-149">Click **Delete**.</span></span> <span data-ttu-id="b4a41-150">Excluir um pagamento apenas irá excluir o pagamento.</span><span class="sxs-lookup"><span data-stu-id="b4a41-150">Deleting a payment will only delete the payment.</span></span> <span data-ttu-id="b4a41-151">Qualquer fatura marcada para pagamento ainda estará disponível para ser paga em outro pagamento.</span><span class="sxs-lookup"><span data-stu-id="b4a41-151">Any invoices marked for payment will still be available to be paid by another payment.</span></span>
26. <span data-ttu-id="b4a41-152">Clique em **Sim**.</span><span class="sxs-lookup"><span data-stu-id="b4a41-152">Click **Yes**.</span></span>
27. <span data-ttu-id="b4a41-153">Escolha **Gerar pagamento** para imprimir cheques ou criar o arquivo de pagamento eletrônico.</span><span class="sxs-lookup"><span data-stu-id="b4a41-153">Choose **Generate payment** to print checks or create the electronic payment file.</span></span>
28. <span data-ttu-id="b4a41-154">Selecione o método de pagamento que deseja gerar.</span><span class="sxs-lookup"><span data-stu-id="b4a41-154">Select the method of payment that you want to generate.</span></span> <span data-ttu-id="b4a41-155">O diário de pagamento pode conter pagamentos de cheques e pagamentos eletrônicos, mas você só pode gerar um tipo de pagamento por vez.</span><span class="sxs-lookup"><span data-stu-id="b4a41-155">The payment journal can contain payments for both Checks and electronic payments, but you can only generate one payment type at a time.</span></span>
29. <span data-ttu-id="b4a41-156">Selecione a conta bancária da qual irá gerar os pagamentos.</span><span class="sxs-lookup"><span data-stu-id="b4a41-156">Select the bank account from which to generate the payments.</span></span>
30. <span data-ttu-id="b4a41-157">Clique em **OK**.</span><span class="sxs-lookup"><span data-stu-id="b4a41-157">Click **OK**.</span></span> <span data-ttu-id="b4a41-158">Pagamentos só serão gerados para pagamentos que correspondam ao **Método de pagamento** e à **Conta bancária** que você selecionou.</span><span class="sxs-lookup"><span data-stu-id="b4a41-158">Payments will only be generated for payments that match the **Method of payment** and **Bank account** you selected.</span></span>
31. <span data-ttu-id="b4a41-159">Se você está gerando **Cheques**, escolha **Documento** para garantir o destino de impressão correto para os cheques.</span><span class="sxs-lookup"><span data-stu-id="b4a41-159">If you are generating **Checks**, choose **Document** to ensure the correct print destination for the checks.</span></span>
32. <span data-ttu-id="b4a41-160">Clique em **OK**.</span><span class="sxs-lookup"><span data-stu-id="b4a41-160">Click **OK**.</span></span>
33. <span data-ttu-id="b4a41-161">Clique em **OK** para gerar os pagamentos.</span><span class="sxs-lookup"><span data-stu-id="b4a41-161">Click **OK** to generate the payments.</span></span>
34. <span data-ttu-id="b4a41-162">Clique em **Lançar** se todos os pagamentos foram aprovados e gerados.</span><span class="sxs-lookup"><span data-stu-id="b4a41-162">Click **Post** if all the payments are approved and generated.</span></span>
