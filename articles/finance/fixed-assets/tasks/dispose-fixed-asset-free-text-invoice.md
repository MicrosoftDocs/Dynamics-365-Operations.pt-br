# <a name="dispose-of-a-fixed-asset-using-a-free-text-invoice"></a><span data-ttu-id="b8af7-103">Descartar um ativo fixo usando uma fatura de texto livre</span><span class="sxs-lookup"><span data-stu-id="b8af7-103">Dispose of a fixed asset using a free text invoice</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="b8af7-104">Este tópico explica como alienar um ativo fixo usando a fatura de texto livre.</span><span class="sxs-lookup"><span data-stu-id="b8af7-104">This topic explains how to dispose of a fixed asset using the free text invoice.</span></span>

1. <span data-ttu-id="b8af7-105">No Painel de Navegação, vá para **Módulos > Contas a receber > Faturas > Todas as faturas de texto livre**.</span><span class="sxs-lookup"><span data-stu-id="b8af7-105">In the navigation pane, go to **Modules > Accounts receivable > Invoices > All free text invoices**.</span></span>
2. <span data-ttu-id="b8af7-106">Selecione **Novo**.</span><span class="sxs-lookup"><span data-stu-id="b8af7-106">Select **New**.</span></span>
3. <span data-ttu-id="b8af7-107">No campo **Conta de cliente**, insira ou selecione um valor.</span><span class="sxs-lookup"><span data-stu-id="b8af7-107">In the **Customer account** field, enter or select a value.</span></span>
4. <span data-ttu-id="b8af7-108">Valide a data padrão da **fatura** e edite, se aplicável.</span><span class="sxs-lookup"><span data-stu-id="b8af7-108">Validate the default **Invoice** date and edit if applicable.</span></span>
5. <span data-ttu-id="b8af7-109">Valide os campos restantes de cabeçalho padrão, como **Moeda**, e edite, se aplicável.</span><span class="sxs-lookup"><span data-stu-id="b8af7-109">Validate remaining default header fields, such as **Currency** and edit if applicable.</span></span>
6. <span data-ttu-id="b8af7-110">Na seção **Linhas da fatura**, valide os campos **Descrição** e **Conta principal**.</span><span class="sxs-lookup"><span data-stu-id="b8af7-110">In the **Invoice lines** section, validate the **Description** and **Main account** fields.</span></span>
7. <span data-ttu-id="b8af7-111">Valide os campos padrão **Grupo de impostos sobre vendas** e **Grupo de impostos do item**.</span><span class="sxs-lookup"><span data-stu-id="b8af7-111">Validate the default **Sales tax group** and **Item sales tax group** fields.</span></span>
8. <span data-ttu-id="b8af7-112">Insira o **Preço unitário** ou o **Valor** de venda do ativo fixo.</span><span class="sxs-lookup"><span data-stu-id="b8af7-112">Enter the **Unit price** or the **Amount** of the sale of the fixed asset.</span></span>
9. <span data-ttu-id="b8af7-113">Selecione a seção **Detalhes da linha**.</span><span class="sxs-lookup"><span data-stu-id="b8af7-113">Select the **Line details** section.</span></span>  
10. <span data-ttu-id="b8af7-114">Digite ou selecione um valor no campo **Ativo fixo**.</span><span class="sxs-lookup"><span data-stu-id="b8af7-114">Enter or select a value in the **Fixed asset** field.</span></span>
11. <span data-ttu-id="b8af7-115">Selecione **Lançar**.</span><span class="sxs-lookup"><span data-stu-id="b8af7-115">Select **Post**.</span></span>
