# <a name="create-a-credit-note-on-project-invoices"></a><span data-ttu-id="65101-103">Criar uma nota de crédito em faturas de projeto</span><span class="sxs-lookup"><span data-stu-id="65101-103">Create a credit note on project invoices</span></span>

[!include [banner](../../includes/banner.md)]

1. <span data-ttu-id="65101-104">Vá para Gerenciamento e contabilidade de projeto > Projetos > Todos os projetos.</span><span class="sxs-lookup"><span data-stu-id="65101-104">Go to Project management and accounting > Projects > All projects.</span></span> 
2. <span data-ttu-id="65101-105">Na lista, localize e selecione o PDV desejado.</span><span class="sxs-lookup"><span data-stu-id="65101-105">In the list, find and select the desired record.</span></span> 
3. <span data-ttu-id="65101-106">Selecione a ID do projeto usada para a nota de crédito.</span><span class="sxs-lookup"><span data-stu-id="65101-106">Select the Project ID that is used for the Credit Note.</span></span> <span data-ttu-id="65101-107">Na lista, clique no link na linha selecionada.</span><span class="sxs-lookup"><span data-stu-id="65101-107">In the list, click the link in the selected row.</span></span> 
4. <span data-ttu-id="65101-108">No Painel de Ação, clique em Gerenciar.</span><span class="sxs-lookup"><span data-stu-id="65101-108">On the Action Pane, click Manage.</span></span> 
5. <span data-ttu-id="65101-109">Clique em Diário de faturas.</span><span class="sxs-lookup"><span data-stu-id="65101-109">Click Invoice journals.</span></span> 
6. <span data-ttu-id="65101-110">Selecione a transação para a nota de crédito.</span><span class="sxs-lookup"><span data-stu-id="65101-110">Select the transaction for credit note.</span></span> <span data-ttu-id="65101-111">Na lista, marque a linha selecionada.</span><span class="sxs-lookup"><span data-stu-id="65101-111">In the list, mark the selected row.</span></span> 
7. <span data-ttu-id="65101-112">Clique em Funções.</span><span class="sxs-lookup"><span data-stu-id="65101-112">Click Functions.</span></span> 
8. <span data-ttu-id="65101-113">Clique em Selecionar para a nota de crédito.</span><span class="sxs-lookup"><span data-stu-id="65101-113">Click Select for credit note.</span></span> 
9. <span data-ttu-id="65101-114">Selecione a transação para a nota de crédito marcando a caixa de seleção.</span><span class="sxs-lookup"><span data-stu-id="65101-114">Select the transaction for credit note by marking the check box.</span></span>
10. <span data-ttu-id="65101-115">Clique em OK.</span><span class="sxs-lookup"><span data-stu-id="65101-115">Click OK.</span></span> 

<span data-ttu-id="65101-116">Após esta etapa, a nota de crédito estará disponível para a Proposta de fatura.</span><span class="sxs-lookup"><span data-stu-id="65101-116">After this step, the Credit Note will be available for Invoice Proposal.</span></span>
