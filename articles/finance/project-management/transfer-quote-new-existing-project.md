# <a name="transfer-a-quotation-to-a-project"></a>Transferir uma cotação para um projeto

[!include [banner](../includes/banner.md)]

Quando uma cotação de projeto tiver o status **Confirmado**, você poderá transferir algumas ou todas as linhas de cotação para um projeto novo ou existente usando o assistente **Transferir para projeto**. 

Para usar o assistente, siga estas etapas.

1. Vá para **Gerenciamento e contabilidade do projeto** > **Cotações** > **Cotações do projeto**.
2. Selecione a cotação que você deseja transferir e, na guia **Acompanhamento**, no grupo **Modificar**, selecione **Transferir para projeto**.
3. Acompanhe o assistente **Transferir cotação para projeto** para associar a cotação a um projeto e transferi-la para ele.
