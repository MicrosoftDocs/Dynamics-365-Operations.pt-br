# <a name="confirm-update-and-send-a-project-quotation"></a>Confirmar, atualizar e enviar uma cotação de projeto

[!include [banner](../includes/banner.md)]

Depois de criar e enviar uma cotação de projeto a um cliente, você deverá obter confirmação do cliente antes de atualizar o status da cotação para **Enviado**. As modificações solicitadas pelo cliente podem ser atualizadas na cotação. Depois de atualizar o status da cotação para **Enviado**, você não poderá modificá-lo. O procedimento a seguir descreve as opções para enviar uma cotação para confirmação, fazer atualizações com base em comentários e enviar a cotação.

## <a name="send-a-project-quotation-confirmation"></a>Enviar uma confirmação de cotação de projeto  

Você pode enviar uma cotação de projeto existente para confirmação por email ou como cópia impressa. 

1. Vá para **Gerenciamento e contabilidade de projeto** > **Cotações** > **Cotação do projeto**. 
2. Na página **Cotação do projeto**, selecione a cotação que deseja enviar para confirmação. 
3. Na guia **Acompanhamento**, no grupo **Gerar**, selecione **Confirmar**. 
4. Na página **Confirmar cotação**, defina os parâmetros necessários. Por exemplo, para imprimir a confirmação, nas opções de **Impressão**, ative **Imprimir confirmação** e selecione **OK**.
5. Na página **Enviar cotação**, selecione **Opções** e na página **Configurações de destino de impressão**, selecione se deseja que a cotação seja enviada para **Tela**, **Email**, **Arquivo**, **Imprimir arquivo** ou **Impressora**. Em seguida, faça os ajustes que desejar na área de **Especificação** para encaminhar a cotação.
6. Na página **Configurações de destino de impressão**, selecione **OK**.  

## <a name="update-a-project-quotation"></a>Atualizar uma cotação de projeto

Para modificar uma cotação de projeto existente, o status da cotação deve ser **Criado**. Conclua as etapas a seguir para atualizar uma cotação existente. 

1. Vá para **Gerenciamento e contabilidade do projeto** > **Cotações** > **Cotações do projeto**.
2. Na página **Cotações do projeto**, selecione a cotação que deseja atualizar e, no Painel de Ações, selecione **Editar**.
3. Atualize as informações de campo necessárias e, no Painel de Ações, selecione **Salvar**.  

## <a name="send-a-project-quotation-to-a-customer"></a>Enviar uma cotação de projeto para um cliente 

1. Vá para **Gerenciamento e contabilidade do projeto** > **Cotações** > **Cotações do projeto** e selecione a cotação do projeto que você deseja enviar.
2. Na página **Cotações do projeto**, na guia **Cotação**, no grupo **Processo**, selecione **Enviar cotação**. O status da cotação é atualizado para **Enviado**.

> [!NOTE]
> Não é possível modificar uma cotação de projeto após o status ser alterado para **Enviado**.
