# <a name="create-a-fixed-asset"></a>Criar um ativo fixo

[!include[task guide banner](../../includes/task-guide-banner.md)]

Este guia de tarefa usa a empresa demo USMF.  Ele criará um novo ativo fixo usando a página de lista de Ativo fixo.

1. Vá para Ativos fixos > Ativos fixos > Ativos fixos.
2. Clique em Novo.
3. No campo Grupo de ativo fixo, insira ou selecione um valor.
    * O número será padrão se você tiver habilitado a funcionalidade de ativos fixos Autonumber nos parâmetros de ativos fixos e no grupo de ativos fixos.  Se não, você deve inserir um número exclusivo para identificar o ativo fixo.  
4. No campo Nome, digite um valor.
    * Insira informações adicionais que sua empresa precisa para este ativo.  
5. Clique em Registros.
6. No campo de Data de aquisição, insira uma data.
7. No campo Preço de aquisição, insira um número.
    * Insira informações adicionais que sua empresa precisa para este registro.  
    * Insira informações adicionais que sua empresa precisa para os registros restantes.  
8. Feche a página.
