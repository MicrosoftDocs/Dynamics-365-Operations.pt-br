----------------------

<span data-ttu-id="ad876-108">Para qualquer entidade legal que tenha um endereço no Brasil, o código do IBGE (Instituto Brasileiro de Geografia e Estatística) deve estar configurado no estado e/ou cidade.</span><span class="sxs-lookup"><span data-stu-id="ad876-108">For any legal entity that has an address in Brazil, the Instituto Brasileiro de Geografia e Estatística (IBGE) code must be set up on the state and/or city.</span></span> <span data-ttu-id="ad876-109">O código do IBGE é usado para identificar a região na qual o endereço está localizado, para que os impostos corretos sejam considerados.</span><span class="sxs-lookup"><span data-stu-id="ad876-109">The IBGE code is used to identify the region where the address is located, so that correct taxes are considered.</span></span> <span data-ttu-id="ad876-110">As informações do código IBGE vem do IBGE, que é uma organização oficial que mapeia a geografia brasileira.</span><span class="sxs-lookup"><span data-stu-id="ad876-110">The IBGE code information comes from the IBGE, which is an official organization that maps the Brazilian geography.</span></span>

1.  <span data-ttu-id="ad876-111">Na página **Configuração de endereço**, no grupo de campo **Estado/província**, no campo **País/região**, selecione **BRA**.</span><span class="sxs-lookup"><span data-stu-id="ad876-111">On the **Address setup** page, in the **State/province** field group, in the **Country/region** field, select **BRA**.</span></span> <span data-ttu-id="ad876-112">Em seguida, no campo **código IBGE**, insira o código IBGE do estado.</span><span class="sxs-lookup"><span data-stu-id="ad876-112">Then, in the **IBGE code** field, enter the IBGE code for the state.</span></span>
