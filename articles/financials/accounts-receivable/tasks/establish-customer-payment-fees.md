<span data-ttu-id="e298b-105">Esta tarefa usa a empresa de demonstração USMF.</span><span class="sxs-lookup"><span data-stu-id="e298b-105">This task uses the USMF demo company.</span></span>

1. <span data-ttu-id="e298b-106">No **Painel de Navegação**, vá para **Módulos > Contas a receber > Configuração de pagamentos > Taxa de pagamento**.</span><span class="sxs-lookup"><span data-stu-id="e298b-106">In the **Navigation pane**, go to **Modules > Accounts receivable > Payments setup > Payment fee**.</span></span>
2. <span data-ttu-id="e298b-107">Clique em **Novo**.</span><span class="sxs-lookup"><span data-stu-id="e298b-107">Click **New**.</span></span>
3. <span data-ttu-id="e298b-108">No campo **ID de taxa**, insira um ID de taxa.</span><span class="sxs-lookup"><span data-stu-id="e298b-108">In the **Fee ID** field, enter a Fee ID.</span></span> <span data-ttu-id="e298b-109">Exibe a ID de taxa em diários de pagamentos, o que o tornam descritivo para compreender a taxa que será avaliada.</span><span class="sxs-lookup"><span data-stu-id="e298b-109">The Fee ID displays on payment journals, so make it descriptive to understand what fee is being assessed.</span></span>  
4. <span data-ttu-id="e298b-110">No campo **Nome**, insira um nome de taxa.</span><span class="sxs-lookup"><span data-stu-id="e298b-110">In the **Name** field, enter a fee name.</span></span>
5. <span data-ttu-id="e298b-111">No campo **Descrição de taxa**, insira uma descrição para a taxa.</span><span class="sxs-lookup"><span data-stu-id="e298b-111">In the **Fee description** field, enter a description for the fee.</span></span>

7. <span data-ttu-id="e298b-116">No campo **Tipo de diário**, selecione o tipo de diário que pode usar essa taxa de pagamento.</span><span class="sxs-lookup"><span data-stu-id="e298b-116">In the **Journal type** field, select the type of journal that can use this payment fee.</span></span> <span data-ttu-id="e298b-117">Se essas taxas são usadas para pagamentos de cliente, o tipo de diário será provavelmente Pagamento de cliente.</span><span class="sxs-lookup"><span data-stu-id="e298b-117">If these fees are used for customer payments, the journal type will likely be Customer payment.</span></span>  
8. <span data-ttu-id="e298b-118">Clique em **Salvar**.</span><span class="sxs-lookup"><span data-stu-id="e298b-118">Click **Save**.</span></span>
9. <span data-ttu-id="e298b-119">Clique em **Configuração de taxa de pagamento**.</span><span class="sxs-lookup"><span data-stu-id="e298b-119">Click **Payment fee setup**.</span></span> <span data-ttu-id="e298b-120">A configuração da taxa de pagamento é usada para definir os critérios para a taxa de pagamento que será avaliada.</span><span class="sxs-lookup"><span data-stu-id="e298b-120">The Payment fee setup is used to define the criteria for when the payment fee will be assessed.</span></span>  <span data-ttu-id="e298b-121">Por exemplo, você pode definir que a taxa será calculada se a conta bancária for USMF OPER, e o método de pagamento é cheque.</span><span class="sxs-lookup"><span data-stu-id="e298b-121">For example, you can define that the fee will be calculated if the bank account is USMF OPER, and the method of payment is check.</span></span>

11. <span data-ttu-id="e298b-126">No campo **Relação do banco**, selecione um grupo bancário ou uma conta bancária.</span><span class="sxs-lookup"><span data-stu-id="e298b-126">In the **Bank relation** field, select either a bank group or a bank account.</span></span> <span data-ttu-id="e298b-127">Se você selecionou Tabela, a pesquisa exibirá contas bancárias.</span><span class="sxs-lookup"><span data-stu-id="e298b-127">If you selected Table, the lookup will display bank accounts.</span></span> <span data-ttu-id="e298b-128">Se você selecionou Grupo, a pesquisa exibirá grupos bancários.</span><span class="sxs-lookup"><span data-stu-id="e298b-128">If you selected Group, the lookup will display bank groups.</span></span>  
12. <span data-ttu-id="e298b-129">Na lista, clique no link na linha selecionada.</span><span class="sxs-lookup"><span data-stu-id="e298b-129">In the list, click the link in the selected row.</span></span>
13. <span data-ttu-id="e298b-130">No campo **Método de pagamento**, selecione o método de pagamento para o qual essa taxa será avaliada.</span><span class="sxs-lookup"><span data-stu-id="e298b-130">In the **Mehtod of payment** field, select the Method of payment for which this fee will be assessed.</span></span> <span data-ttu-id="e298b-131">Por exemplo, você pode classificar uma taxa para os clientes se enviar pagamentos como um cheque, em vez de como um pagamento eletrônico.</span><span class="sxs-lookup"><span data-stu-id="e298b-131">For example, you may assess a fee to your customers if they send payments as a check, rather than as an electronic payment.</span></span>  
14. <span data-ttu-id="e298b-132">Na lista, localize e selecione o registro desejado.</span><span class="sxs-lookup"><span data-stu-id="e298b-132">In the list, find and select the desired record.</span></span>
15. <span data-ttu-id="e298b-133">Se relevante, no campo **Moeda do pagamento**, insira uma moeda de pagamento.</span><span class="sxs-lookup"><span data-stu-id="e298b-133">If relevant, in the **Payment currency** field, enter a payment currency.</span></span> <span data-ttu-id="e298b-134">A moeda de pagamento é usada como critérios adicionais para se a taxa que será avaliada.</span><span class="sxs-lookup"><span data-stu-id="e298b-134">The payment currency is used as an additional criteria for whether the fee will be assessed.</span></span>  <span data-ttu-id="e298b-135">Por exemplo, seu banco pode cobrar uma taxa adicional para pagamentos recebidos na moeda USD, desde que transacionem normalmente somente na moeda de Euro.</span><span class="sxs-lookup"><span data-stu-id="e298b-135">For example, your bank may charge an extra fee for payments received in USD currency, since they normally only transact in EUR currency.</span></span>

18. <span data-ttu-id="e298b-141">No campo **Moeda da taxa**, selecione a moeda da taxa.</span><span class="sxs-lookup"><span data-stu-id="e298b-141">In the **Fee currency** field, select the currency of the fee.</span></span> <span data-ttu-id="e298b-142">Esta é a moeda na qual a taxa será criada.</span><span class="sxs-lookup"><span data-stu-id="e298b-142">This is the currency in which the fee will be created.</span></span>  
19. <span data-ttu-id="e298b-143">Clique em **Salvar**.</span><span class="sxs-lookup"><span data-stu-id="e298b-143">Click **Save**.</span></span>
