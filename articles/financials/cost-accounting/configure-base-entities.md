# <a name="configure-base-entities"></a>Configurar entidades base 

[!include[banner](../includes/banner.md)]


A contabilização de custo previsto é um modulo independente que exige que estas entidades base sejam configuradas corretamente no Dynamics 365 for Finance and Operations, Enterprise Edition.

-  Moeda
-  Taxa de câmbio
-  Calendário fiscal  

Antes de utilizar o módulo contabilização de custo previsto, você precisa de uma moeda, uma taxa de câmbio e um calendário fiscal definidos para sua empresa. 

Para saber mais sobre essas entidades, execute o curso de treinamento de eLearning [Módulo 2 Configuração de moeda e taxa de câmbio](https://mbspartner.microsoft.com/AX/CourseModules/1215) (requer conta do CustomerSource).
