# <a name="set-up-a-ledger-account-alias"></a><span data-ttu-id="1bf34-103">Configurar um alias de conta contábil</span><span class="sxs-lookup"><span data-stu-id="1bf34-103">Set up a ledger account alias</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="1bf34-104">Este procedimento mostra como criar um apelido de conta que fornece um atalho para inserir um número de conta.</span><span class="sxs-lookup"><span data-stu-id="1bf34-104">This procedure shows how to create an account alias that provides a shortcut for entering an account number.</span></span> <span data-ttu-id="1bf34-105">Este procedimento usa a empresa de dados de demonstração USMF.</span><span class="sxs-lookup"><span data-stu-id="1bf34-105">This procedure users demo data company USMF.</span></span>

1. <span data-ttu-id="1bf34-106">Vá para Contabilidade > Plano de contas > Contas > Apelido da conta contábil.</span><span class="sxs-lookup"><span data-stu-id="1bf34-106">Go to General ledger > Chart of accounts > Accounts > Ledger account alias.</span></span>
2. <span data-ttu-id="1bf34-107">Clique em Novo.</span><span class="sxs-lookup"><span data-stu-id="1bf34-107">Click New.</span></span>
3. <span data-ttu-id="1bf34-108">No campo Apelido da conta contábil, digite um valor.</span><span class="sxs-lookup"><span data-stu-id="1bf34-108">In the Ledger account alias field, type a value.</span></span>
4. <span data-ttu-id="1bf34-109">No campo Estrutura de conta, selecione a estrutura à qual a conta e as dimensões pertencem.</span><span class="sxs-lookup"><span data-stu-id="1bf34-109">In the Account structure field, select the structure the account and dimensions belong to.</span></span>
5. <span data-ttu-id="1bf34-110">No campo Empresa, clique no botão suspenso para abrir a pesquisa.</span><span class="sxs-lookup"><span data-stu-id="1bf34-110">In the Company field, click the drop-down button to open the lookup.</span></span>
6. <span data-ttu-id="1bf34-111">Na lista, localize e selecione a empresa à qual o apelido se aplica.</span><span class="sxs-lookup"><span data-stu-id="1bf34-111">In the list, find and select the company that the alias applies to.</span></span>
7. <span data-ttu-id="1bf34-112">Na lista, clique no link na linha selecionada.</span><span class="sxs-lookup"><span data-stu-id="1bf34-112">In the list, click the link in the selected row.</span></span>
8. <span data-ttu-id="1bf34-113">No campo Definição do apelido da conta contábil, especifique a conta e as dimensões.</span><span class="sxs-lookup"><span data-stu-id="1bf34-113">In the Ledger account alias definition field, specify the account and dimensions.</span></span>
    * <span data-ttu-id="1bf34-114">A conta e as dimensões serão preenchidas quando o atalho for usado.</span><span class="sxs-lookup"><span data-stu-id="1bf34-114">The account and dimensions will be populated when using the shortcut.</span></span>  
9. <span data-ttu-id="1bf34-115">No campo Foco inicial, selecione a dimensão que terá foco quando o apelido é usado.</span><span class="sxs-lookup"><span data-stu-id="1bf34-115">In the Initial focus field, select the dimension that will have focus when the alias is used.</span></span>
    * <span data-ttu-id="1bf34-116">Após digitar o atalho e a conta e as dimensões serem preenchidas, o campo Foco inicial é para onde o cursor ou foco será movido.</span><span class="sxs-lookup"><span data-stu-id="1bf34-116">After you type the shortcut, and the account and dimensions are populated, the Initial focus field is where the cursor or focus will move to.</span></span>
