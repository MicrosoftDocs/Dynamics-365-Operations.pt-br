# <a name="maintain-financial-dimension-default-templates"></a>Manter modelos padrão de dimensão financeira

[!include [task guide banner](../../includes/task-guide-banner.md)]

Este procedimento mostra como configurar modelos padrão da dimensão financeira. Os modelos podem ser usados para distribuir valores rapidamente em documentos de origem.

1. Vá para Contabilidade > Plano de contas > Dimensões > Modelos padrão da dimensão financeira.
2. Clique em Novo.
3. No campo ID modelo, insira um ID modelo.
4. Insira o percentual para os primeiros valores de dimensão.
5. Insira os valores de dimensões financeiras que correspondem a porcentagem inserida.
6. Clique em Adicionar.
7. Insira outra porcentagem para os próximos valores de dimensões.
    * Por exemplo, insira uma porcentagem para que ainda assim você tenha menos de 100% atribuído.  
8. Insira os valores de dimensões financeiras que correspondem a porcentagem inserida.
9. Clique em Adicionar.
10. Insira outra porcentagem, que faz a porcentagem total igual a 100%.
11. Insira os valores de dimensões financeiras que correspondem a porcentagem inserida.
12. Clique em Salvar.
