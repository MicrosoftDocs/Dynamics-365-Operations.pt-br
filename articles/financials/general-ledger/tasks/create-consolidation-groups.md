# <a name="create-consolidation-groups-and-additional-consolidation-accounts"></a>Criar grupos de consolidação e contas de consolidação adicionais

[!include [task guide banner](../../includes/task-guide-banner.md)]

Este procedimento mostra como criar um grupo de contas de consolidação e depois adicionar contas ao grupo. Este procedimento usa a empresa de dados de demonstração USMF.


## <a name="create-a-consolidation-account-group"></a>Criar um grupo de contas de consolidação
1. Vá para Contabilidade > Plano de contas > Contas > Grupos de contas de consolidação.
2. Clique em Novo.
3. No campo Grupo de contas de consolidação, insira um identificador exclusivo para o grupo de contas de consolidação.
4. No campo Nome, digite um valor.

## <a name="add-accounts-to-consolidation-account-group"></a>Adicionar contas ao grupo de contas de consolidação
1. Feche a página.
2. Vá para Contabilidade > Plano de contas > Contas > Contas de consolidação adicionais.
3. Clique em Novo.
4. No campo Conta principal, clique no botão suspenso para abrir a pesquisa.
5. Na lista, clique na conta principal que deseja mapear.
6. No campo Grupo de contas de consolidação, clique no botão suspenso para abrir a pesquisa.
7. Na lista, clique no grupo de contas de consolidação.
8. No campo Conta de consolidação, insira um valor.
9. No campo Nome da conta de consolidação, insira um valor.
