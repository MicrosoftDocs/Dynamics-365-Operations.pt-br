# <a name="view-journal-entries-or-transactions"></a><span data-ttu-id="84874-103">Exibir entradas de diário ou transações</span><span class="sxs-lookup"><span data-stu-id="84874-103">View journal entries or transactions</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="84874-104">Este procedimento mostra como usar a consulta das transações de comprovante para pesquisar por entradas de diário ou transações.</span><span class="sxs-lookup"><span data-stu-id="84874-104">This procedure shows how to use the Voucher transactions inquiry to search for journal entries or transactions.</span></span>

1. <span data-ttu-id="84874-105">Vá para Contabilidade > Consultas e relatórios > Transações de comprovante.</span><span class="sxs-lookup"><span data-stu-id="84874-105">Go to General ledger > Inquiries and reports > Voucher transactions.</span></span>
2. <span data-ttu-id="84874-106">Selecione o campo para o qual você deseja definir um critério de filtragem.</span><span class="sxs-lookup"><span data-stu-id="84874-106">Select the field for which you want to define a filter criteria.</span></span>
3. <span data-ttu-id="84874-107">Insira os critérios de filtragem para o campo selecionado.</span><span class="sxs-lookup"><span data-stu-id="84874-107">Enter your filter criteria for the selected field.</span></span>
    * <span data-ttu-id="84874-108">É possível filtrar um valor único ou um intervalo.</span><span class="sxs-lookup"><span data-stu-id="84874-108">You could filter on a single value or a range.</span></span> <span data-ttu-id="84874-109">Ao definir um intervalo, certifique-se de que a sintaxe correta seja usada.</span><span class="sxs-lookup"><span data-stu-id="84874-109">When defining a range, make sure the correct syntax is used.</span></span> <span data-ttu-id="84874-110">Os valores devem ser separados por dois pontos finais (..).</span><span class="sxs-lookup"><span data-stu-id="84874-110">The values should be separated by a double period (..).</span></span>  
4. <span data-ttu-id="84874-111">Clique na guia Junções para adicionar tabelas extras para a filtragem.</span><span class="sxs-lookup"><span data-stu-id="84874-111">Click the Joins tab to add additional tables from which to filter.</span></span>
5. <span data-ttu-id="84874-112">Na árvore, selecione 'Tabelas\Entrada de diário geral'.</span><span class="sxs-lookup"><span data-stu-id="84874-112">In the tree, select 'Tables\General journal entry'.</span></span>
6. <span data-ttu-id="84874-113">Clique em Adicionar junção de tabela.</span><span class="sxs-lookup"><span data-stu-id="84874-113">Click Add table join.</span></span>
7. <span data-ttu-id="84874-114">Clique em Cancelar se você decidir não adicionar uma tabela extra.</span><span class="sxs-lookup"><span data-stu-id="84874-114">Click Cancel if you decide not to add an additional table.</span></span>
8. <span data-ttu-id="84874-115">Clique na guia Variação.</span><span class="sxs-lookup"><span data-stu-id="84874-115">Click the Range tab.</span></span>
9. <span data-ttu-id="84874-116">Clique em OK para executar a consulta.</span><span class="sxs-lookup"><span data-stu-id="84874-116">Click OK to run the query.</span></span>
10. <span data-ttu-id="84874-117">Clique em Origem da transação.</span><span class="sxs-lookup"><span data-stu-id="84874-117">Click Transaction origin.</span></span>
    * <span data-ttu-id="84874-118">Vários botões em torno da grade podem ser usados para procurar por informações adicionais sobre o registro de comprovante selecionado.</span><span class="sxs-lookup"><span data-stu-id="84874-118">Various buttons about the grid can be used to research additional information about the selected record of the voucher.</span></span> <span data-ttu-id="84874-119">Alguns botões podem não estar disponíveis, dependendo do tipo e das características da transação.</span><span class="sxs-lookup"><span data-stu-id="84874-119">Some buttons may not be available, depending on the type of transaction and characteristics of the transaction.</span></span>  
11. <span data-ttu-id="84874-120">Feche a página.</span><span class="sxs-lookup"><span data-stu-id="84874-120">Close the page.</span></span>
12. <span data-ttu-id="84874-121">Clique em Documento original.</span><span class="sxs-lookup"><span data-stu-id="84874-121">Click Original document.</span></span>
13. <span data-ttu-id="84874-122">Feche a página.</span><span class="sxs-lookup"><span data-stu-id="84874-122">Close the page.</span></span>
