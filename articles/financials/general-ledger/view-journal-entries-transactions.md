<span data-ttu-id="1badf-108">**Transações de comprovante** é uma página de consulta na qual você pode selecionar várias tabelas e campos para especificar critérios para o saldo ou a transação que está sendo pesquisando.</span><span class="sxs-lookup"><span data-stu-id="1badf-108">**Voucher transactions** is an inquiry page where you can select from various tables and fields to specify criteria for the balance or transaction that you're searching for.</span></span> <span data-ttu-id="1badf-109">Por exemplo, você pode exibir todas as transações para uma data ou uma conta específicas, ou todas as transações do tipo **Operação** que estão em um nível de lançamento específico.</span><span class="sxs-lookup"><span data-stu-id="1badf-109">For example, you can view all transactions for a specific date or account, or all transactions of the **Operating** type that are in a specific posting layer.</span></span> <span data-ttu-id="1badf-110">Por padrão, a página mostra o número do diário, o comprovante, a data e a conta principal, mas você pode adicionar tabelas, campos e os critérios para restringir a pesquisa.</span><span class="sxs-lookup"><span data-stu-id="1badf-110">By default, the page shows the journal number, voucher, date, and main account, but you can add additional tables, fields, and criteria to narrow down your search.</span></span>

## <a name="audit-trail"></a><span data-ttu-id="1badf-111">Trilha de auditoria</span><span class="sxs-lookup"><span data-stu-id="1badf-111">Audit trail</span></span>

-   <span data-ttu-id="1badf-123">**Diário de impressão** – esse relatório mostra o resultado de um diário lançado.</span><span class="sxs-lookup"><span data-stu-id="1badf-123">**Print journal** – This report shows the result of a posted journal.</span></span> <span data-ttu-id="1badf-124">Você pode executar o relatório por número de lote de diário ou tipo de diário ou adicionar campos adicionais.</span><span class="sxs-lookup"><span data-stu-id="1badf-124">You can run the report by journal batch number or journal type, or add additional fields.</span></span>
