<span data-ttu-id="b4f62-104">As estruturas da conta usam a conta principal e as dimensões financeiras para criar um conjunto de regras que determina a ordem e os valores usados ao inserir o número de conta.</span><span class="sxs-lookup"><span data-stu-id="b4f62-104">Account structures use the main account and financial dimensions to create a set of rules that determine the order and values used when entering the account number.</span></span> <span data-ttu-id="b4f62-105">Você pode configurar quantas estruturas de contas você precisar para sua empresa.</span><span class="sxs-lookup"><span data-stu-id="b4f62-105">You can set up as many account structures as you need for your business.</span></span> <span data-ttu-id="b4f62-106">As estruturas de conta são atribuídas a uma configuração de contabilidade de uma empresa, de forma que possam ser compartilhadas.</span><span class="sxs-lookup"><span data-stu-id="b4f62-106">The account structures are assigned to a company’s ledger setup, so they can be shared.</span></span>
