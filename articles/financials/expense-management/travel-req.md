# <a name="travel-requisitions"></a><span data-ttu-id="be38b-104">Requisições de viagem</span><span class="sxs-lookup"><span data-stu-id="be38b-104">Travel requisitions</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="be38b-105">Uma *requisição de viagem* é um documento de origem que lista as despesas que serão incorridas para a viagem.</span><span class="sxs-lookup"><span data-stu-id="be38b-105">A *travel requisition* is a source document that lists the expenses that will be incurred for the purpose of travel.</span></span> <span data-ttu-id="be38b-106">Uma requisição de viagem é enviada para revisão e pode então ser usada para autorizar despesas.</span><span class="sxs-lookup"><span data-stu-id="be38b-106">A travel requisition is submitted for review and can then be used to authorize expenses.</span></span>

<span data-ttu-id="be38b-107">Sua organização pode exigir que uma requisição de viagem seja enviada antes que os funcionários incorram alguma despesa que seja cobrada da organização.</span><span class="sxs-lookup"><span data-stu-id="be38b-107">Your organization might require that a travel requisition be submitted before employees incur any expense that is charged to the organization.</span></span> <span data-ttu-id="be38b-108">Essa exigência se aplicará quando os funcionários cobrarem as despesas a um cartão de crédito corporativo, gastarem dinheiro recebido de um adiantamento ou incorrerem despesas pagas do próprio bolso que serão reembolsadas pela organização.</span><span class="sxs-lookup"><span data-stu-id="be38b-108">This requirement applies, regardless of whether employees charge expenses to a corporate credit card, spend cash that they received from a cash advance, or incur out-of-pocket expenses that will be reimbursed by the organization.</span></span>

<span data-ttu-id="be38b-109">As requisições de viagem e as diretivas podem ser usadas para ajudar com controle de orçamento.</span><span class="sxs-lookup"><span data-stu-id="be38b-109">Travel requisitions and policies can be used to help with budget control.</span></span> <span data-ttu-id="be38b-110">Por exemplo, se a sua organização estiver trabalhando em um projeto de preço fixo que exige viagens, as despesas de viagem dos membros da equipe do projeto devem se adequar ao seu orçamento.</span><span class="sxs-lookup"><span data-stu-id="be38b-110">For example, if your organization is working on a fixed-price project that requires travel, the travel expenses of the project's team members must fit within the budget for the project.</span></span> <span data-ttu-id="be38b-111">Ao exigir que as despesas de viagem sejam aprovadas antes de serem incorridas, a organização pode ajudar a garantir que o projeto permaneça dentro do orçamento.</span><span class="sxs-lookup"><span data-stu-id="be38b-111">By requiring that travel expenses be approved before they are incurred, the organization can help make sure that the project remains within budget.</span></span>

<span data-ttu-id="be38b-112">Você pode configurar políticas para requisições de viagem na página **Requisição de viagem**.</span><span class="sxs-lookup"><span data-stu-id="be38b-112">You can set up policies for travel requisitions from the **Travel requisition** page.</span></span>
