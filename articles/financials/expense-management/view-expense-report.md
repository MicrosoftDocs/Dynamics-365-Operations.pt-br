# <a name="view-an-expense-report"></a><span data-ttu-id="09772-103">Exibir um relatório de despesas</span><span class="sxs-lookup"><span data-stu-id="09772-103">View an expense report</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="09772-104">Quando você executa uma consulta para exibir transações na contabilidade ou em compra e fornecimento, a consulta pode retornar informações demais.</span><span class="sxs-lookup"><span data-stu-id="09772-104">When you run a query to view transactions in General ledger or Procurement and sourcing, the query might return too much information.</span></span> <span data-ttu-id="09772-105">Por isso, pode ser interessante refinar sua pesquisa.</span><span class="sxs-lookup"><span data-stu-id="09772-105">Therefore, you might want to narrow your search.</span></span> <span data-ttu-id="09772-106">Para obter as informações necessárias, você pode fazer uma busca detalhada em uma transação para ver o relatório de despesas no qual ela apareceu.</span><span class="sxs-lookup"><span data-stu-id="09772-106">To get the information that you require, you can drill down into a transaction to see the expense report that it appeared on.</span></span>

<span data-ttu-id="09772-107">Em relatórios de despesas, você pode fazer uma busca detalhada de uma transação até o relatório original armazenado no Gerenciamento de despesas.</span><span class="sxs-lookup"><span data-stu-id="09772-107">For expense reports, you can drill down from a transaction to the original report that is stored in Expense management.</span></span> <span data-ttu-id="09772-108">Além do valor total das despesas, você pode exibir as transações discriminadas, as transações qualificadas para a recuperação do imposto sobre valor agregado (VAT) e os recebimentos associados às despesas.</span><span class="sxs-lookup"><span data-stu-id="09772-108">In addition to the total amount of expenses, you can view the itemized transactions, the transactions that are eligible for recovery of value-added tax (VAT), and the receipts that are attached to the expenses.</span></span>

<span data-ttu-id="09772-109">Para fazer uma busca detalhada no relatório de despesas original em que uma transação aparece, selecione a transação e, em seguida, selecione **Documento original**.</span><span class="sxs-lookup"><span data-stu-id="09772-109">To drill down to the original expense report that a transaction appeared on, select the transaction, and then select **Original document**.</span></span>

<span data-ttu-id="09772-110">Ao exibir transações individuais, você também pode exibir os dados adicionais associados a uma transação.</span><span class="sxs-lookup"><span data-stu-id="09772-110">When you view individual transactions, you can also view any additional data that is attached to a transaction.</span></span> <span data-ttu-id="09772-111">Por exemplo, para uma despesa discriminada de hotel, você pode exibir uma lista detalhada de despesas, como acesso à Internet, chamadas telefônicas e serviço de quarto.</span><span class="sxs-lookup"><span data-stu-id="09772-111">For example, for an itemized hotel expense, you can view a detailed list of expenses such as internet access, telephone calls, and room service.</span></span>
