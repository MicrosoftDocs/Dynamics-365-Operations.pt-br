<span data-ttu-id="ce73b-104">Use este procedimento para definir os valores padrão e as políticas de segurança para o catálogo de endereços global.</span><span class="sxs-lookup"><span data-stu-id="ce73b-104">Use this procedure to set the default values and security policies for the global address book.</span></span> 

<span data-ttu-id="ce73b-105">A empresa de dados demo usada para criar esta tarefa é USMF.</span><span class="sxs-lookup"><span data-stu-id="ce73b-105">The demo data company used to create this task is USMF.</span></span> <span data-ttu-id="ce73b-106">Esta tarefa está direcionada para a equipe de Planejamento e configuração.</span><span class="sxs-lookup"><span data-stu-id="ce73b-106">This task is intended for the Planning and configuration team.</span></span>

1. <span data-ttu-id="ce73b-107">No Painel de navegação, vá para **Módulos > Administração da organização > Catálogo de endereços global > Parâmetros do catálogo de endereços global**.</span><span class="sxs-lookup"><span data-stu-id="ce73b-107">In the Navigation pane, go to **Modules > Organization administration > Global address book > Global address book parameters**.</span></span>

5. <span data-ttu-id="ce73b-111">Em **Exibir número DUNS no endereço**, selecione se deseja exibir o número DUNS em endereços.</span><span class="sxs-lookup"><span data-stu-id="ce73b-111">In **Display DUNS number on addresses**, select whether to display the DUNS number on addresses.</span></span>
6. <span data-ttu-id="ce73b-112">Em **Verificar número DUNS exclusivo**, selecione se é necessário verificar a existência de números DUNS exclusivos.</span><span class="sxs-lookup"><span data-stu-id="ce73b-112">In **Check for unique DUNS number**, select whether to check for unique DUNS numbers.</span></span>
7. <span data-ttu-id="ce73b-113">No campo **Participante**, selecione uma opção.</span><span class="sxs-lookup"><span data-stu-id="ce73b-113">In the **Party** field, select an option.</span></span>
8. <span data-ttu-id="ce73b-114">No campo **Cliente**, selecione uma opção.</span><span class="sxs-lookup"><span data-stu-id="ce73b-114">In the **Customer** field, select an option.</span></span>
9. <span data-ttu-id="ce73b-115">No campo **Fornecedor**, selecione uma opção.</span><span class="sxs-lookup"><span data-stu-id="ce73b-115">In the **Vendor** field, select an option.</span></span>
10. <span data-ttu-id="ce73b-116">No campo **Cliente potencial**, selecione uma opção.</span><span class="sxs-lookup"><span data-stu-id="ce73b-116">In the **Prospect** field, select an option.</span></span>
11. <span data-ttu-id="ce73b-117">No campo **Concorrente**, selecione uma opção.</span><span class="sxs-lookup"><span data-stu-id="ce73b-117">In the **Competitor** field, select an option.</span></span>
12. <span data-ttu-id="ce73b-118">Clique na guia **Segurança da localização privada**.</span><span class="sxs-lookup"><span data-stu-id="ce73b-118">Click the **Private location security** tab.</span></span>

14. <span data-ttu-id="ce73b-121">Clique em **Salvar**.</span><span class="sxs-lookup"><span data-stu-id="ce73b-121">Click **Save**.</span></span>
