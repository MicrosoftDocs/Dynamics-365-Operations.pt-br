# <a name="create-an-operating-unit"></a><span data-ttu-id="3b293-103">Criar uma unidade operacional</span><span class="sxs-lookup"><span data-stu-id="3b293-103">Create an operating unit</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="3b293-104">Uma unidade operacional é uma organização usada para dividir o controle de recursos econômicos e os processos operacionais em uma empresa.</span><span class="sxs-lookup"><span data-stu-id="3b293-104">An operating unit is an organization that is used to divide the control of economic resources and operational processes in a business.</span></span> <span data-ttu-id="3b293-105">As pessoas de uma unidade operacional têm um imposto para maximizar o uso de recursos escassos, melhorar os processos e a conta para o desempenho.</span><span class="sxs-lookup"><span data-stu-id="3b293-105">People in an operating unit have a duty to maximize the use of scarce resources, improve processes, and account for their performance.</span></span> <span data-ttu-id="3b293-106">Os tipos de unidades operacionais incluem centros de custos, unidades de negócios, departamentos e fluxos de valor.</span><span class="sxs-lookup"><span data-stu-id="3b293-106">The types of operating units include cost centers, business units, departments, and value streams.</span></span> <span data-ttu-id="3b293-107">Use o procedimento a seguir para criar uma unidade operacional.</span><span class="sxs-lookup"><span data-stu-id="3b293-107">Use the following procedure to create an operating unit.</span></span> <span data-ttu-id="3b293-108">A empresa de dados demo usada para criar este procedimento é USMF.</span><span class="sxs-lookup"><span data-stu-id="3b293-108">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="3b293-109">Vá para Administração da organização > Organizações > Unidades operacionais.</span><span class="sxs-lookup"><span data-stu-id="3b293-109">Go to Organization administration > Organizations > Operating units.</span></span>
2. <span data-ttu-id="3b293-110">Clique em Novo para abrir a caixa de diálogo suspensa.</span><span class="sxs-lookup"><span data-stu-id="3b293-110">Click New to open the drop dialog.</span></span>
3. <span data-ttu-id="3b293-111">Na lista, localize e selecione o PDV desejado.</span><span class="sxs-lookup"><span data-stu-id="3b293-111">In the list, find and select the desired record.</span></span>
    * <span data-ttu-id="3b293-112">Selecione o tipo de unidade operacional que você deseja criar.</span><span class="sxs-lookup"><span data-stu-id="3b293-112">Select the type of operating unit you want to create.</span></span>  
4. <span data-ttu-id="3b293-113">Na lista, clique no link na linha selecionada.</span><span class="sxs-lookup"><span data-stu-id="3b293-113">In the list, click the link in the selected row.</span></span>
5. <span data-ttu-id="3b293-114">No campo Nome, digite um valor.</span><span class="sxs-lookup"><span data-stu-id="3b293-114">In the Name field, type a value.</span></span>
    * <span data-ttu-id="3b293-115">Expanda a seção Geral, se necessário.</span><span class="sxs-lookup"><span data-stu-id="3b293-115">Expand the General section, if necessary.</span></span>  
    * <span data-ttu-id="3b293-116">Forneça informações gerais sobre o unidade operacional, como um número de identificação, um número DUNS, e um gerente.</span><span class="sxs-lookup"><span data-stu-id="3b293-116">Provide general information about the operating unit, such as an identification number, DUNS number, and manager.</span></span>    
    * <span data-ttu-id="3b293-117">Expanda a seção Endereços, se necessário.</span><span class="sxs-lookup"><span data-stu-id="3b293-117">Expand the Addresses section, if necessary.</span></span>  
    * <span data-ttu-id="3b293-118">Insira informações de endereço, como o nome da rua e o número, o CEP, e a cidade.</span><span class="sxs-lookup"><span data-stu-id="3b293-118">Enter address information, such as the street name and number, postal code, and city.</span></span> <span data-ttu-id="3b293-119">Clique em Adicionar para inserir um novo registro de endereço, ou clique em Editar para modificar um registro de endereço existente.</span><span class="sxs-lookup"><span data-stu-id="3b293-119">Click Add to enter a new address record, or click Edit to modify an existing address record.</span></span>   
    * <span data-ttu-id="3b293-120">Expanda a seção Informação de contato, se necessário.</span><span class="sxs-lookup"><span data-stu-id="3b293-120">Expand the Contact information section, if necessary.</span></span>  
    * <span data-ttu-id="3b293-121">Insira informação sobre métodos de comunicação, como endereços de e-mail, URLs, e números de telefone.</span><span class="sxs-lookup"><span data-stu-id="3b293-121">Enter information about methods of communication, such as email addresses, URLs, and telephone numbers.</span></span> <span data-ttu-id="3b293-122">Para inserir um novo registro de comunicação, clique em Novo.</span><span class="sxs-lookup"><span data-stu-id="3b293-122">To enter a new communication record, click New.</span></span> <span data-ttu-id="3b293-123">Para modificar um registro de comunicação existente, clique em Mais opções > Avançado.</span><span class="sxs-lookup"><span data-stu-id="3b293-123">To modify an existing communication record, click More options > Advanced.</span></span>   
6. <span data-ttu-id="3b293-124">Clique em Salvar.</span><span class="sxs-lookup"><span data-stu-id="3b293-124">Click Save.</span></span>
