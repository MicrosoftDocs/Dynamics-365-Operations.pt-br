<span data-ttu-id="b1b45-122">Antes de usar o recurso **Abrir em nova janela**, é necessário configurar o bloqueador de pop-ups do navegador para permitir pop-ups do URL do site do Finance and Operations.</span><span class="sxs-lookup"><span data-stu-id="b1b45-122">Before using the **Open in new window** feature, you should configure your browser's pop-up blocker to allow pop-ups from the URL of the Finance and Operations site.</span></span> <span data-ttu-id="b1b45-123">Como um exemplo, você poderia permitir pop-ups de "\*.dynamics.com".</span><span class="sxs-lookup"><span data-stu-id="b1b45-123">As an example, you could allow pop-ups from "\*.dynamics.com".</span></span>

<span data-ttu-id="b1b45-124">O recurso **Abrir em nova janela** estará disponível apenas quando houver mais de uma página aberta na janela.</span><span class="sxs-lookup"><span data-stu-id="b1b45-124">The **Open in new window** feature is only available when there is more than one page open in the window.</span></span> <span data-ttu-id="b1b45-125">Além disso, a janela pop-up será fechada automaticamente quando não houver mais páginas abertas (ou seja, quando a última página nessa janela estiver fechada.)</span><span class="sxs-lookup"><span data-stu-id="b1b45-125">Also, the pop-up window automatically closes when there are no more pages open (that is, when the last page in that window is closed).</span></span> <span data-ttu-id="b1b45-126">O Finance and Operations também fecha as páginas em aberto quando você navega para uma área diferente no aplicativo.</span><span class="sxs-lookup"><span data-stu-id="b1b45-126">Finance and Operations also closes open pages when you navigate to a different area in the application.</span></span> <span data-ttu-id="b1b45-127">Portanto, se você tem janelas pop-up abertas e navega para uma área diferente no aplicativo, as janelas pop-up são fechadas automaticamente porque as páginas nas janelas foram fechadas pelo sistema.</span><span class="sxs-lookup"><span data-stu-id="b1b45-127">Therefore, if you have pop-up windows open and navigate to a different area in the application, the pop-up windows are automatically closed because the pages in those windows were closed by the system.</span></span>
