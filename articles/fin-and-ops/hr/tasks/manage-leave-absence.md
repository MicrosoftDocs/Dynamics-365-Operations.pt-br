# <a name="manage-leave-of-absence"></a>Gerenciar licença

[!INCLUDE [task guide banner](../../includes/task-guide-banner.md)]

Este procedimento mostra a criação de registros de licença de funcionário. Você pode rastrear tempo de licença para os motivos pelos quais incluem atividades, médicas de formação educacional, ou parentais. A empresa de dados demo usada para criar este procedimento é USMF.

1. Vá para Recursos humanos > Trabalhadores > Funcionários.
2. Na lista, selecione um funcionário.
3. Exiba informações detalhadas sobre o funcionário selecionando o nome do funcionário.
4. Clique na guia Emprego.
5. Clique na licença.
6. Clique em Novo.
7. No campo Tipo de licença, clique no botão suspenso para abrir a pesquisa.
    * Você pode associar um tipo de licença para um código de salário no formulário Tipos de licença. Se um tipo de licença está associado a um código de trabalho, uma linha de salário será gerada com o código associado de trabalho durante o período de licença inseridos.  
8. Na lista, selecione um tipo de licença. 
    * Por exemplo: Adoção  
9. Insira a data em que a licença será iniciada. Exemplo '2015-10-26'
    * Por exemplo: 2015-10-26  
10. Insira a data em que a licença será iniciada. 
    * Por exemplo: 2015-11-20  
11. No campo Nota, insira uma descrição.
    * Por exemplo: Deixar para adoção  
12. Clique em Salvar.
