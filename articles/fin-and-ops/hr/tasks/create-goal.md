# <a name="create-a-goal"></a>Criar uma meta

[!include[task guide banner](../../includes/task-guide-banner.md)]

Você pode criar metas, adicionar atividades a meta, rastrear, medições e vincular metas a revisão. Este procedimento foi criado usando a empresa de dados de demonstração USMF. Este procedimento é para um recurso que foi adicionado à versão 1611 do Dynamics 365 for Operations.

1. Vá para Todos os espaços de trabalho > Autoatendimento para funcionários.
2. Clique em Exibir todas as metas.
3. Clique em Novo.
4. No campo Nome, digite "Concluir o projeto Contoso".
5. No campo de Visão geral, insira "terminarei o projeto da Contoso em 30 dias".
    * As metas podem se aplicar a uma pessoa, a uma equipe, ou até a uma empresa. A configuração padrão para metas de funcionários é individual.  
6. No campo Categoria de meta, insira ou selecione um valor.
7. No campo Data inicial, altere a data exibida.
8. No campo Data final, altere a data exibida.
    * Quando você concluir uma meta, insira a data que você concluiu a meta aqui.  
9. Clique em Salvar.
10. Expandir a seção Medidas. Se já está expandido, clique na seta e expanda duas vezes.
11. Clique em Adicionar medida para abrir a caixa de diálogo suspensa.
12. No campo Medida, insira ou selecione um valor.
13. No campo Valor alvo, insira um número.
14. Clique em OK.
15. Expandir a seção Atividades. Se já está expandido, clique na seta e expanda duas vezes.
16. Clique em Adicionar.
17. No campo Título, digite um valor.
18. No campo Descrição, digite um valor.
19. Clique em Salvar.
20. Clique em Exibir seções para abrir a caixa de diálogo suspensa.
21. Selecione Não no campo Exibir medidas.
22. Clique em Salvar.
23. Clique em Exibir seções para abrir a caixa de diálogo suspensa.
24. Selecione Sim no campo Exibir medidas.
25. Clique em Salvar.
    * Há que frequência as metas que são iguais ou semelhantes para membro de equipe ou todos os funcionários.     Seu administrador ou gerente de horas podem criar modelos de meta para as metas. Você pode adicionar um meta de um modelo, e em altera-o para corresponder as linhas de tempo.  
    * Há frequentemente várias metas que são as mesmas ou semelhantes para os membros da equipe ou todos funcionários.     Seu administrador ou gerente de horas podem criar grupos de metas que contêm um ou mais modelos de meta. Você pode adicionar um meta de um grupo de meta, e em altere-o para corresponder as linhas de tempo.  
    * Quando você cria uma revisão, você pode adicionar a meta a revisão. Consulte o guia de tarefa para revisões para verificar como você pode adicionar suas revisões a meta.
