# <a name="change-the-banner-or-logo"></a>Alterar a faixa ou o logotipo

[!include [banner](../../includes/banner.md)]

O procedimento a seguir lista as etapas que os administradores do sistema podem usar para atualizar a imagem da faixa ou do logotipo que é exibida para uma entidade jurídica. A empresa de dados demo usada para criar este procedimento é USMF.

1. Acesse **Painel de navegação > Módulos > Administração da organização > Organizações > Entidades legais**.
2. Na lista à esquerda, selecione a entidade legal para a qual você deseja atualizar a faixa ou o logotipo. Se já estiverem selecionadas, Acesse a próxima etapa.
3. Expanda a guia **Imagem no painel**.
4. Selecione **Alterar**.
    
    A resolução ideal para uma imagem da faixa é de 1920 x 281 pixels. A largura ideal para uma imagem do logotipo pixels é 350 pixels.
    
5. Selecione **Salvar**.
6. Acesse **Painel de navegação > Módulos > Comum > Comum > Painel padrão**. Você deverá ver a nova imagem da faixa ou do logotipo no painel.  
    
    Talvez seja necessário modificar sua empresa, usando o separador da empresa, para aquela carregada para a faixa.
