[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="51716-104">O procedimento a seguir lista as etapas que os administradores do sistema podem usar para atualizar a imagem da faixa ou do logotipo que é exibida para uma entidade jurídica.</span><span class="sxs-lookup"><span data-stu-id="51716-104">The following procedure lists the steps that system administrators can use to update the banner or logo image that is displayed for a legal entity.</span></span> <span data-ttu-id="51716-105">A empresa de dados demo usada para criar este procedimento é USMF.</span><span class="sxs-lookup"><span data-stu-id="51716-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="51716-106">Vá para **Painel de navegação > Módulos > Administração da organização > Organizações > Entidades legais**.</span><span class="sxs-lookup"><span data-stu-id="51716-106">Go to **Navigation pane > Modules > Organization administration > Organizations > Legal entities**.</span></span>
2. <span data-ttu-id="51716-107">Na lista à esquerda, selecione a entidade legal para a qual você deseja atualizar a faixa ou o logotipo.</span><span class="sxs-lookup"><span data-stu-id="51716-107">In the list on the left, select the legal entity for which you want to update the banner or logo.</span></span> <span data-ttu-id="51716-108">Se já estiverem selecionadas, vá para a próxima etapa.</span><span class="sxs-lookup"><span data-stu-id="51716-108">If it is already selected, go to the next step.</span></span>

4. <span data-ttu-id="51716-110">Selecione **Alterar**.</span><span class="sxs-lookup"><span data-stu-id="51716-110">Select **Change**.</span></span>
    
    <span data-ttu-id="51716-111">A resolução ideal para uma imagem da faixa é de 1920 x 281 pixels.</span><span class="sxs-lookup"><span data-stu-id="51716-111">The ideal resolution for a banner image is 1920 x 281 pixels.</span></span> <span data-ttu-id="51716-112">A largura ideal para uma imagem do logotipo pixels é 350 pixels.</span><span class="sxs-lookup"><span data-stu-id="51716-112">The ideal width for a logo image is 350 pixels.</span></span>
    
5. <span data-ttu-id="51716-113">Selecione **Salvar**.</span><span class="sxs-lookup"><span data-stu-id="51716-113">Select **Save**.</span></span>
6. <span data-ttu-id="51716-114">Vá para **Painel de navegação > Módulos > Comum > Comum > Painel padrão**.</span><span class="sxs-lookup"><span data-stu-id="51716-114">Go to **Navigation pane > Modules > Common > Common > Default dashboard**.</span></span> <span data-ttu-id="51716-115">Você deverá ver a nova imagem da faixa ou do logotipo no painel.</span><span class="sxs-lookup"><span data-stu-id="51716-115">You should see the new banner or logo image on the dashboard.</span></span>  
    
    <span data-ttu-id="51716-116">Talvez seja necessário modificar sua empresa, usando o separador da empresa, para aquela carregada para a faixa.</span><span class="sxs-lookup"><span data-stu-id="51716-116">You may need to change your company, using the company picker, to the one you uploaded the banner for.</span></span>
