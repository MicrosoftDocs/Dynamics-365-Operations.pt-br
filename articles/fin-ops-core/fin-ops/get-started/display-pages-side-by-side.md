<span data-ttu-id="61a2a-104">Este artigo explica como exibir páginas lado a lado.</span><span class="sxs-lookup"><span data-stu-id="61a2a-104">This article explains how to display pages side by side.</span></span>

<span data-ttu-id="61a2a-121">Antes de usar o recurso **Abrir em nova janela**, é necessário configurar o bloqueador de pop-ups do navegador para permitir pop-ups da URL do site.</span><span class="sxs-lookup"><span data-stu-id="61a2a-121">Before using the **Open in new window** feature, you should configure your browser's pop-up blocker to allow pop-ups from the URL of the site.</span></span> <span data-ttu-id="61a2a-122">Como um exemplo, você poderia permitir pop-ups de "\*.dynamics.com".</span><span class="sxs-lookup"><span data-stu-id="61a2a-122">As an example, you could allow pop-ups from "\*.dynamics.com".</span></span>

<span data-ttu-id="61a2a-123">O recurso **Abrir em nova janela** estará disponível apenas quando houver mais de uma página aberta na janela.</span><span class="sxs-lookup"><span data-stu-id="61a2a-123">The **Open in new window** feature is only available when there is more than one page open in the window.</span></span> <span data-ttu-id="61a2a-124">Além disso, a janela pop-up será fechada automaticamente quando não houver mais páginas abertas (ou seja, quando você fecha a última página nessa janela).</span><span class="sxs-lookup"><span data-stu-id="61a2a-124">Also, the pop-up window automatically closes when there are no more pages open (that is, when you close the last page in that window).</span></span> <span data-ttu-id="61a2a-125">O sistema também fecha as páginas abertas quando você navega para uma área diferente no aplicativo.</span><span class="sxs-lookup"><span data-stu-id="61a2a-125">The system also closes open pages when you navigate to a different area in the application.</span></span> <span data-ttu-id="61a2a-126">Portanto, se você tem janelas pop-up abertas e navega para uma área diferente no aplicativo, as janelas pop-up são fechadas automaticamente porque o sistema fechou as páginas nessas janelas.</span><span class="sxs-lookup"><span data-stu-id="61a2a-126">Therefore, if you have pop-up windows open and navigate to a different area in the application, the pop-up windows close automatically because the system closed the pages in those windows.</span></span>
