# <a name="change-the-date-for-a-session"></a>Alterar a data de uma sessão

[!include [task guide banner](../../includes/task-guide-banner.md)]

Este tópico explica como alterar a data de uma sessão. Por padrão, a data atual é usada ao inserir e lançar entradas do diário ou documentos de origem. Você pode alterar a data usada para sua sessão atual. Use esse recurso para pós-datar entradas do diário ou documentos de origem, conforme necessário.

1. No Painel de Navegação, vá para **Módulos > Comum > Comum > Data e hora da sessão**.
2. No campo **Data**, insira uma data.
3. Selecione **OK**.
