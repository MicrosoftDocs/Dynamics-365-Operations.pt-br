# <a name="create-an-operating-unit"></a><span data-ttu-id="a0bc9-103">Criar uma unidade operacional</span><span class="sxs-lookup"><span data-stu-id="a0bc9-103">Create an operating unit</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="a0bc9-104">Uma unidade operacional é uma organização usada para dividir o controle de recursos econômicos e os processos operacionais em uma empresa.</span><span class="sxs-lookup"><span data-stu-id="a0bc9-104">An operating unit is an organization that is used to divide the control of economic resources and operational processes in a business.</span></span> <span data-ttu-id="a0bc9-105">As pessoas de uma unidade operacional têm um imposto para maximizar o uso de recursos escassos, melhorar os processos e a conta para o desempenho.</span><span class="sxs-lookup"><span data-stu-id="a0bc9-105">People in an operating unit have a duty to maximize the use of scarce resources, improve processes, and account for their performance.</span></span> <span data-ttu-id="a0bc9-106">Os tipos de unidades operacionais incluem centros de custos, unidades de negócios, departamentos e fluxos de valor.</span><span class="sxs-lookup"><span data-stu-id="a0bc9-106">The types of operating units include cost centers, business units, departments, and value streams.</span></span> <span data-ttu-id="a0bc9-107">Use o procedimento a seguir para criar uma unidade operacional.</span><span class="sxs-lookup"><span data-stu-id="a0bc9-107">Use the following procedure to create an operating unit.</span></span> <span data-ttu-id="a0bc9-108">A empresa de dados demo usada para criar este procedimento é USMF.</span><span class="sxs-lookup"><span data-stu-id="a0bc9-108">The demo data company used to create this procedure is USMF.</span></span>

3. <span data-ttu-id="a0bc9-111">Na lista, localize e selecione o PDV desejado.</span><span class="sxs-lookup"><span data-stu-id="a0bc9-111">In the list, find and select the desired record.</span></span> <span data-ttu-id="a0bc9-112">Selecione o tipo de unidade operacional que você deseja criar.</span><span class="sxs-lookup"><span data-stu-id="a0bc9-112">Select the type of operating unit you want to create.</span></span>  
4. <span data-ttu-id="a0bc9-113">Na lista, clique no link na linha selecionada.</span><span class="sxs-lookup"><span data-stu-id="a0bc9-113">In the list, click the link in the selected row.</span></span>
5. <span data-ttu-id="a0bc9-114">No campo **Nome**, digite um valor.</span><span class="sxs-lookup"><span data-stu-id="a0bc9-114">In the **Name** field, type a value.</span></span>
    + <span data-ttu-id="a0bc9-115">Expanda a seção **Geral**, se necessário.</span><span class="sxs-lookup"><span data-stu-id="a0bc9-115">Expand the **General** section, if necessary.</span></span>

    + <span data-ttu-id="a0bc9-117">Expanda a seção **Endereços**, se necessário.</span><span class="sxs-lookup"><span data-stu-id="a0bc9-117">Expand the **Addresses** section, if necessary.</span></span>
