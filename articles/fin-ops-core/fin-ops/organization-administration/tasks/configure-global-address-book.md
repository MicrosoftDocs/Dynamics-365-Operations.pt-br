[!include [banner](../../includes/banner.md)]

<span data-ttu-id="e086c-104">Use este procedimento para definir os valores padrão e as políticas de segurança para o catálogo de endereços global.</span><span class="sxs-lookup"><span data-stu-id="e086c-104">Use this procedure to set the default values and security policies for the global address book.</span></span> 

<span data-ttu-id="e086c-105">A empresa de dados demo usada para criar esta tarefa é USMF.</span><span class="sxs-lookup"><span data-stu-id="e086c-105">The demo data company used to create this task is USMF.</span></span> <span data-ttu-id="e086c-106">Esta tarefa está direcionada para a equipe de Planejamento e configuração.</span><span class="sxs-lookup"><span data-stu-id="e086c-106">This task is intended for the Planning and configuration team.</span></span>

1. <span data-ttu-id="e086c-107">No Painel de navegação, vá para **Módulos > Administração da organização > Catálogo de endereços global > Parâmetros do catálogo de endereços global**.</span><span class="sxs-lookup"><span data-stu-id="e086c-107">In the Navigation pane, go to **Modules > Organization administration > Global address book > Global address book parameters**.</span></span>
2. <span data-ttu-id="e086c-108">No campo **Sequência do nome**, selecione como os nomes devem ser mostrados.</span><span class="sxs-lookup"><span data-stu-id="e086c-108">In the **Name sequence** field, select how names should be shown.</span></span>
3. <span data-ttu-id="e086c-109">Em **Excluir participantes sem funções**, selecione se irá excluir participantes aos quais não foram atribuídas funções.</span><span class="sxs-lookup"><span data-stu-id="e086c-109">In **Delete parties with no roles**, select whether to delete parties with that have not been assigned a role.</span></span>
4. <span data-ttu-id="e086c-110">Em **Usar verificação de duplicatas**, selecione se deve ou não verificar registros duplicados.</span><span class="sxs-lookup"><span data-stu-id="e086c-110">In **Use duplicate check**, select whether to check for duplicate records.</span></span>
5. <span data-ttu-id="e086c-111">Em **Exibir número DUNS no endereço**, selecione se deseja exibir o número DUNS em endereços.</span><span class="sxs-lookup"><span data-stu-id="e086c-111">In **Display DUNS number on addresses**, select whether to display the DUNS number on addresses.</span></span>
6. <span data-ttu-id="e086c-112">Em **Verificar número DUNS exclusivo**, selecione se é necessário verificar a existência de números DUNS exclusivos.</span><span class="sxs-lookup"><span data-stu-id="e086c-112">In **Check for unique DUNS number**, select whether to check for unique DUNS numbers.</span></span>
7. <span data-ttu-id="e086c-113">No campo **Participante**, selecione uma opção.</span><span class="sxs-lookup"><span data-stu-id="e086c-113">In the **Party** field, select an option.</span></span>
8. <span data-ttu-id="e086c-114">No campo **Cliente**, selecione uma opção.</span><span class="sxs-lookup"><span data-stu-id="e086c-114">In the **Customer** field, select an option.</span></span>
9. <span data-ttu-id="e086c-115">No campo **Fornecedor**, selecione uma opção.</span><span class="sxs-lookup"><span data-stu-id="e086c-115">In the **Vendor** field, select an option.</span></span>
10. <span data-ttu-id="e086c-116">No campo **Cliente potencial**, selecione uma opção.</span><span class="sxs-lookup"><span data-stu-id="e086c-116">In the **Prospect** field, select an option.</span></span>
11. <span data-ttu-id="e086c-117">No campo **Concorrente**, selecione uma opção.</span><span class="sxs-lookup"><span data-stu-id="e086c-117">In the **Competitor** field, select an option.</span></span>
12. <span data-ttu-id="e086c-118">Clique na guia **Segurança da localização privada**.</span><span class="sxs-lookup"><span data-stu-id="e086c-118">Click the **Private location security** tab.</span></span>
13. <span data-ttu-id="e086c-119">Na lista, localize e selecione o PDV desejado.</span><span class="sxs-lookup"><span data-stu-id="e086c-119">In the list, find and select the desired record.</span></span> <span data-ttu-id="e086c-120">Pressione a tecla Shift para selecionar várias funções para adicionar ao painel **Funções selecionadas** e clique na seta para adicioná-las.</span><span class="sxs-lookup"><span data-stu-id="e086c-120">Press the Shift key to select multiple roles to add to the **Selected roles** pane and then click the arrow to add the selected roles.</span></span>  
14. <span data-ttu-id="e086c-121">Clique em **Salvar**.</span><span class="sxs-lookup"><span data-stu-id="e086c-121">Click **Save**.</span></span>
