# <a name="hiring-candidate-through-recruiting"></a>Contratar candidato por meio de recrutamento

[!include [banner](../../includes/banner.md)]

> [!IMPORTANT]
> A funcionalidade de recrutamento neste artigo será referida como Projetos de recrutamento e se concentra em candidatos, solicitações de emprego e projetos de recrutamento.  

Este procedimento permite que um recruta contrate um candidato que envia uma solicitação de emprego em um projeto de recrutamento específico. Ao contratar um candidato em um projeto de recrutamento, um novo registro de trabalhador será criado e o registro do candidato terá um status **Empregado**. A empresa de dados de demonstração **USMF** foi usada para criar este procedimento.

1. Acesse **Recursos humanos \> Recrutamento \> Solicitações de emprego \> Solicitações de emprego**. 
2. Selecione uma solicitação de emprego para um candidato externo.
3. Na lista, selecione o link na linha selecionada.
4. Selecione **Status da solicitação de emprego**.
5. Selecione **Contratar novo trabalhador**.
6. No campo **Data de início do emprego**, insira uma data e hora.
7. No campo **Posição**, selecione a seta suspensa para abrir a pesquisa.
8. Na lista, selecione o link na linha selecionada.
9. No campo **Iniciar atribuição**, insira uma data e hora.
10. Selecione **Contratar novo trabalhador**.
11. Expanda a Caixa de Dados **Histórico de emprego**.
12. Expanda a Caixa de Dados **Posições atuais**.
13. Expanda a Caixa de Dados **Histórico de emprego**.
14. Expanda a Caixa de Dados **Posições atuais**.
15. Expanda ou recolha a seção **Endereços**.
16. Expanda ou recolha a seção **Informações de contato**.
17. Expanda ou recolha a seção **Informações pessoais**.
