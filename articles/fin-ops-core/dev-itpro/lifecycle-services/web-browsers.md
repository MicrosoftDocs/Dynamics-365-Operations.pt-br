# <a name="supported-web-browsers-for-lifecycle-services"></a>Navegadores da web com suporte para o Lifecycle Services 

[!include[banner](../includes/banner.md)]

Os usuários podem acessar o Microsoft Dynamics Lifecyle Services (LCs) usando as versões mais recentes destes navegadores populares: 

- Microsoft Edge (recomendado: [Edge baseado no Chromium](https://support.microsoft.com/microsoft-edge/download-the-new-microsoft-edge-based-on-chromium-0f4a3dd7-55df-60f5-739f-00010dba52cf))
- Google Chrome
- Apple Safari

Para localizar a versão mais recente de cada navegador, Acesse o site do fabricante do software. Para obter o melhor desempenho e uma experiência ideal, recomendamos que você use a versão mais recente de um navegador moderno, especialmente Microsoft Edge.
