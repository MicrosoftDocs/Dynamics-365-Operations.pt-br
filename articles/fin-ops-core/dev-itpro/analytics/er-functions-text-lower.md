# <a name="lower-er-function"></a>Função de ER LOWER

[!include [banner](../includes/banner.md)]

A função `LOWER` retorna a cadeia de caracteres de texto especificada como um valor de *Cadeia de caracteres* após ser convertida em letras minúsculas.

## <a name="syntax"></a>Sintaxe

```vb
LOWER (text)
```

## <a name="arguments"></a>Argumentos

`text`: *Cadeia de caracteres*

Um valor de *Cadeia de caracteres* que especifica o texto.

## <a name="return-values"></a>Valores de retorno

*Cadeia de caracteres*

O valor de texto resultante.

## <a name="example"></a>Exemplo

`LOWER ("Sample")` retorna **"exemplo"**.
