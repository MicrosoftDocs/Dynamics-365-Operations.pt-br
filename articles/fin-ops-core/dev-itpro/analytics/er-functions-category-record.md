# <a name="list-of-er-functions-in-the-record-category"></a>Lista de funções ER na categoria de registro

[!include [banner](../includes/banner.md)]

As funções de registro de relatório eletrônico (ER) podem ser usadas para extrair informações de e executar operações em, fontes de tipos de dados do *Contêiner (registro)*. Este tópico fornece um resumo dessas funções.

## <a name="list-of-supported-functions"></a>Lista de funções com suporte

| Função | Descrição |
|----------|-------------|
| [NullContainer](er-functions-record-nullcontainer.md) | Esta função retorna um valor de *Contêiner (registro)* nulo que tem a mesma estrutura do registro ou da lista de registros especificada. |
| [EmptyRecord](er-functions-record-emptyrecord.md) | Esta função retorna um valor de *Contêiner (registro)* nulo que tem a mesma estrutura do registro ou da lista de registros especificada. |
