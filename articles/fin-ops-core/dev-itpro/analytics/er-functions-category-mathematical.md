# <a name="list-of-er-functions-in-the-mathematical-category"></a>Lista de funções ER na categoria matemática

[!include [banner](../includes/banner.md)]

As funções matemáticas de relatório eletrônico (ER) podem ser usadas para fazer vários cálculos matemáticos comuns. Este artigo fornece um resumo dessas funções.

## <a name="list-of-supported-functions"></a>Lista de funções com suporte

| Função | Descrição |
|----------|-------------|
| [Abs](er-functions-mathematical-abs.md)             | Essa função retorna o valor absoluto (módulo) do número especificado como um valor *Real*. Em outras palavras, ela retorna o número sem seu sinal. |
| [Potência](er-functions-mathematical-power.md)         | Essa função retorna um valor *Real* que representa o resultado da elevação do número positivo especificado à potência especificada. |
| [Arredondar](er-functions-mathematical-round.md)         | Essa função retorna o número especificado como um valor *Real* após ele ser arredondado para o número especificado de casas decimais. |
| [RoundDown](er-functions-mathematical-rounddown.md) | Essa função retorna o número especificado como um valor *Real* após ele ser arredondado para baixo para o número especificado de casas decimais. |
| [RoundUp](er-functions-mathematical-roundup.md)     | Essa função retorna o número especificado como um valor *Real* após ele ser arredondado para cima para o número especificado de casas decimais. |
