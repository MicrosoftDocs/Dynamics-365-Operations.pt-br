# <a name="ScreenDestinationType">Destino da tela</a>

[!include [banner](../includes/banner.md)]

Você pode configurar um destino de tela para cada componente de pasta ou arquivo de um formato de ER (Relatório eletrônico) que está configurado para gerar documentos de saída. Com base na configuração do destino, um documento gerado é aberto para visualização em uma guia separada do navegador.

Se você definir **Ativado** como **Sim**, será criada uma visualização da saída. Você pode exibir alguns tipos de arquivo, como o XML, TXT, ou PDF diretamente em uma janela do navegador. Para outros tipos de arquivos, como Microsoft Excel ou Word, é usado o serviço do Microsoft Office Online.
