# <a name="len-er-function"></a>Função de ER LEN

[!include [banner](../includes/banner.md)]

A função `LEN` retorna o número de caracteres na cadeia de caracteres especificada como um valor *Inteiro*.

## <a name="syntax"></a>Sintaxe

```vb
LEN (text)
```

## <a name="arguments"></a>Argumentos

`text`: *Cadeia de caracteres*

Um valor de *Cadeia de caracteres* que especifica o texto.

## <a name="return-values"></a>Valores de retorno

*Inteiro*

O valor numérico resultante.

## <a name="example"></a>Exemplo

`LEN ("Sample")` retorna **6**.
