# <a name="file-destination"></a>Destino de arquivo

[!include [banner](../includes/banner.md)]

Você pode configurar um destino de arquivo para cada componente de PASTA ou ARQUIVO de um formato de ER (Relatório eletrônico) que está configurado para gerar documentos de saída. Com base na configuração do destino, um documento gerado é disponibilizado para download no navegador da Web.

Na página **Configurações de destino**, se você definir **Habilitado** como **Sim**, uma caixa de diálogo para abrir ou salvar será aberta quando a configuração tiver concluído a execução.
