# <a name="access-to-finance-and-tax-reference-data"></a>Acesso a dados de referência financeira e tributária

[!include [banner](../../includes/banner.md)]



Todas as empresas funcionam com um conjunto básico de dados financeiros, como o ano civil fiscal, a moeda na qual a empresa é transacionada, as contas nas quais o dinheiro dos negócios entram e saem, taxas de imposto e remessas. Esses dados residem nos aplicativos de finanças e operações. No entanto, ele é exposto ao Dataverse para que os aplicativos Customer Engagement possam ter uma única fonte para dados financeiros e tributários. Dessa forma, os dados são uniformes no ecossistema comercial.

Os dados financeiros e tributários são integrados com os seguintes mapeamentos:

+ [Razão integrado](ledger-mapping.md)
+ [Mestre de imposto integrado](tax-mapping.md)
