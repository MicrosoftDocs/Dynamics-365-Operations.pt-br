> [!IMPORTANT]
> O recurso Inteligência de IoT descrito neste artigo será atualizado em breve para a versão 2.0 e, ao mesmo tempo, passará a se chamar *Sensor Data Intelligence*. A nova versão está disponível no momento como uma versão preliminar do recurso do Supply Chain Management versão 10.0.30 ou posterior.
> 
> O Sensor Data Intelligence apresenta um novo modelo de implantação no qual os componentes necessários do Azure são implantados em sua própria assinatura do Azure, em vez de serem gerenciados pela Microsoft. Esse novo modelo aprimora sua capacidade de personalizar o recurso e integrá-lo a sistemas de terceiros. A nova versão também inclui suporte a dois novos cenários de IoT para o gerenciamento de ativos.
>
> Devido a essas alterações futuras, a Microsoft não recomenda que você inicie novos projetos com base no recurso Inteligência de IoT existente. Se você tem interesse em adicionar recursos de IoT ao sistema, recomendamos que comece a trabalhar com a versão preliminar do Sensor Data Intelligence. Para saber mais sobre o Sensor Data Intelligence e como trabalhar com ele, consulte a [página inicial do Sensor Data Intelligence](/dynamics365/supply-chain/sensor-data-intelligence/sdi-home-page).
