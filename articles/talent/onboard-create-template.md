# <a name="create-an-onboarding-template"></a>Criar um modelo de integração

[!include [banner](includes/banner.md)]

O Microsoft Dynamics 365 Talent: Onboard fornece vários modelos que podem ajudar você a criar um guia de integração o mais rápido possível. Você pode usar um ou mais desses modelos, ou criar seus próprios modelos. O Onboard fornece textos de exemplo que você pode usar ao criar seus próprios modelos. Portanto, o processo é fácil mesmo que você comece do zero.

## <a name="create-an-onboarding-template-from-an-existing-template"></a>Criar um modelo de integração por meio de um modelo existente

1. No menu esquerdo, selecione **Modelos**.
2. Em **Modelos populares da galeria** ou **Meus modelos**, selecione um modelo. Para exibir mais modelos, selecione **Mais na galeria de modelos**.
3. Siga uma destas etapas:

    - Se o modelo for da galeria, selecione **Salvar como meu modelo**, digite um novo nome para o modelo e selecione **Salvar**.
    - Se o modelo for dos **Meus modelos**, selecione **Salvar como modelo**, digite um novo nome para o modelo e selecione **Salvar**.

    [![Criando um modelo por meio de um modelo existente](./media/onboard-save-template.png)](./media/onboard-save-template.png)

## <a name="create-a-new-onboarding-template"></a>Criar um novo modelo de integração

1. No menu esquerdo, selecione **Modelos**.
2. Em **Meus modelos**, selecione o bloco **Adicionar** (sinal de adição \[**+**\]).

    [![Criando um novo modelo](./media/onboard-create-new-template.png)](./media/onboard-create-new-template.png)

3. Na caixa de diálogo **Criar um modelo de guia**, insira um nome para o modelo e depois selecione **Salvar**.
