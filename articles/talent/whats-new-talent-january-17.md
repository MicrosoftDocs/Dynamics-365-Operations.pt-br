# <a name="whats-new-or-changed-in-dynamics-365-for-talent-core-hr-january-17-2019"></a>Novidades ou alterações no Dynamics 365 for Talent Core HR (17 de janeiro de 2019)

[!include [banner](includes/banner.md)]

**Compilação 8.1.2107**

Este tópico descreve os recursos novos ou alterados no Core HR.

## <a name="minor-changes-and-bug-fixes-included-in-this-release"></a>Alterações menores e correção de bug incluídos nesta versão

### <a name="new-position-assignment-start-datetime-default-has-been-corrected"></a>O novo padrão de data/hora de início da atribuição da posição foi corrigido
Ao transferir funcionários, o sistema usará como padrão de data/hora a data selecionada e a hora no início do dia, para o fuso horário do usuário.

### <a name="all-employees-have-the-same-position-description-in-the-exited-worker-task-management-list"></a>Todos os funcionários têm a mesma descrição de posição na lista de gerenciamento de tarefa de trabalhadores que saíram da empresa
Uma atualização for feita no espaço de trabalho **Gerenciamento de tarefa** para exibir descrições de posição corretas para todos os funcionários que saíram recentemente da empresa.

### <a name="action-requested-by-field-is-now-populated-on-workers-action-page"></a>O campo "Ação solicitada por" está agora preenchido na página Ação de trabalhadores
Com esta alteração, o padrão do campo **Ação solicitada por** é agora o usuário que está solicitando a alteração.

### <a name="ideas-portal-updated"></a>Portal de ideais atualizado
Os links de aplicativos para enviar uma ideia para o Talent foram atualizados no novo portal de ideias. Todas as ideias inseridas no portal antigo foram migradas, portanto todas as ótimas ideias ainda estão disponíveis para serem votadas por outros.
