# <a name="close-a-job"></a>Fechar um trabalho

[!include [banner](includes/banner.md)]

No Microsoft Dynamics 365 for Talent: Attract, você pode fechar um trabalho como **Trabalho cancelado** ou **Cargo preenchido**. Você pode cancelar um trabalho a qualquer momento.

Se desejar indicar que o cargo foi preenchido, um candidato deve ter aceito uma oferta para o trabalho.

## <a name="cancel-a-job"></a>Cancelar um trabalho

É possível cancelar um trabalho no Attract a qualquer momento.

1. Na guia **Trabalhos** no Attract, selecione o trabalho que deseja cancelar.

2. Selecione **Fechar trabalho**.

   ![Fechar trabalhos no Attract](./media/attract-close-job.png)

3. Na caixa de diálogo **Fechar este trabalho** , selecione **Trabalho cancelado**, adicione comentários e selecione **Fechar trabalho**.

   ![Fechar o trabalho como Cargo preenchido](./media/attract-close-job-as-cancelled.png)

## <a name="close-a-job-as-position-filled"></a>Fechar um trabalho como Cargo preenchido

É possível fechar um trabalho como **Cargo preenchido** depois que pelo menos um candidato ao trabalho tenha concluído o estágio de oferta.

1. Na guia **Trabalhos** no Attract, selecione o trabalho que deseja cancelar.

2. Selecione **Fechar trabalho**.

   ![Fechar trabalhos no Attract](./media/attract-close-job.png)

3. Na caixa de diálogo **Fechar este trabalho** , selecione **Cargo preenchido**, selecione o candidato que aceitou a oferta, adicione comentários e selecione **Fechar trabalho**.

   ![Fechar o trabalho como Cargo preenchido](./media/attract-close-job-as-position-filled.png)

   > [!NOTE]
   > Só é possível fechar um trabalho como **Cargo preenchido** quando pelo menos um candidato tenha concluído o estágio de oferta. Se a solicitação de emprego não tiver a opção **Oferta aceita** selecionada, ou se a solicitação de emprego não exibir um status **Aceito** na oferta, a opção **Cargo preenchido** não será exibida no menu suspenso **Motivo** .
