[!include [banner](includes/banner.md)]

<span data-ttu-id="7978a-104">O Microsoft Dynamics 365 Talent: Onboard permite compartilhar guias e modelos de integração com outros colaboradores, para que eles possam analisar os documentos e fazer alterações.</span><span class="sxs-lookup"><span data-stu-id="7978a-104">Microsoft Dynamics 365 Talent: Onboard lets you share onboarding guides and templates with other contributors so that they can review the documents and make changes.</span></span>

## <a name="share-an-onboarding-guide-with-other-contributors"></a><span data-ttu-id="7978a-105">Compartilhar um guia de integração com outros trabalhadores</span><span class="sxs-lookup"><span data-stu-id="7978a-105">Share an onboarding guide with other contributors</span></span>

1. <span data-ttu-id="7978a-106">No menu esquerdo, selecione **Guias** e depois a guia que você deseja compartilhar.</span><span class="sxs-lookup"><span data-stu-id="7978a-106">On the left menu, select **Guides**, and then select the guide that you want to share.</span></span>
2. <span data-ttu-id="7978a-107">Selecione o botão **Mais** (os três pontos verticais) no canto superior direito da página e depois **Compartilhar**.</span><span class="sxs-lookup"><span data-stu-id="7978a-107">Select the **More** button (the three vertical dots) in the upper-right corner of the page, and then select **Share**.</span></span>

    ![[<span data-ttu-id="7978a-108">Compartilhando um guia de integração com outros trabalhadores</span><span class="sxs-lookup"><span data-stu-id="7978a-108">Sharing an onboarding guide with other contributors</span></span>](./media/onboard-share-guide.png)](./media/onboard-share-guide.png)

3. <span data-ttu-id="7978a-109">Em **Compartilhar com**, se os destinatários puderem revisar o guia, defina a opção **Essas pessoas podem editar?** como **Ativada**.</span><span class="sxs-lookup"><span data-stu-id="7978a-109">Under **Share with**, if recipients should be able to revise the guide, set the **Can these people make edits?** option to **On**.</span></span> <span data-ttu-id="7978a-110">Se os destinatários tiverem que ser notificados por email de que você compartilhou o guia com eles, defina a opção **Enviar notificação por email para os destinatários** como **Ativado**.</span><span class="sxs-lookup"><span data-stu-id="7978a-110">If recipients should be notified by email that you've shared the guide with them, set the **Send email notification to recipients** option to **On**.</span></span>

    ![[<span data-ttu-id="7978a-111">Configurando preferências de email e de acesso para um guia de integração</span><span class="sxs-lookup"><span data-stu-id="7978a-111">Setting access and email preferences for an onboarding guide</span></span>](./media/onboard-share-guide-with-contributors.png)](./media/onboard-share-guide-with-contributors.png)

4. <span data-ttu-id="7978a-112">Insira um nome ou um endereço de email e selecione o destinatário desejado na lista suspensa.</span><span class="sxs-lookup"><span data-stu-id="7978a-112">Enter a name or email address, and select the desired recipient in the drop-down list.</span></span>
5. <span data-ttu-id="7978a-113">Na guia **Gerenciar o acesso**, você pode revisar ou alterar o que os destinatários pode fazer com o guia.</span><span class="sxs-lookup"><span data-stu-id="7978a-113">On the **Manage access** tab, you can review or change what recipients can do with the guide.</span></span> <span data-ttu-id="7978a-114">Para alterar o acesso de um destinatário, na lista suspensa próxima ao nome, selecione **Pode Exibir**, **Pode editar** ou **Remover**, conforme apropriado.</span><span class="sxs-lookup"><span data-stu-id="7978a-114">To change a recipient's access, in the drop-down list next to the recipient's name, select **Can View**, **Can edit**, or **Remove**, as appropriate.</span></span>
6. <span data-ttu-id="7978a-115">Quando terminar de adicionar ou modificar os destinatários, selecione **Atualizar**.</span><span class="sxs-lookup"><span data-stu-id="7978a-115">When you've finished adding or modifying recipients, select **Update**.</span></span>

## <a name="share-an-onboarding-template-with-other-contributors"></a><span data-ttu-id="7978a-116">Compartilhar um modelo de integração com outros trabalhadores</span><span class="sxs-lookup"><span data-stu-id="7978a-116">Share an onboarding template with other contributors</span></span>

2. <span data-ttu-id="7978a-118">Em **Meus modelos**, selecione o botão **Mais** (as reticências \[**...**\]) no canto superior direito do modelo que você deseja compartilhar e selecione **Compartilhar**.</span><span class="sxs-lookup"><span data-stu-id="7978a-118">Under **My templates**, select the **More** button (the ellipsis \[**...**\]) in the upper-right corner of the template that you want to share, and then select **Share**.</span></span>

    <span data-ttu-id="7978a-119">[![Compartilhando um modelo com outros trabalhadores](./media/onboard-share-template.png)](./media/onboard-share-template.png)</span><span class="sxs-lookup"><span data-stu-id="7978a-119">[![Sharing a template with other contributors](./media/onboard-share-template.png)](./media/onboard-share-template.png)</span></span>

    [!Definindo preferências de acesso e de email para um modelo]<span data-ttu-id="7978a-122">(./media/onboard-share-template-access-and-email.png)](./media/onboard-share-template-access-and-email.png)</span><span class="sxs-lookup"><span data-stu-id="7978a-122">(./media/onboard-share-template-access-and-email.png)](./media/onboard-share-template-access-and-email.png)</span></span>

4. <span data-ttu-id="7978a-123">Insira um nome ou um endereço de email e selecione o destinatário desejado na lista suspensa.</span><span class="sxs-lookup"><span data-stu-id="7978a-123">Enter a name or email address, and select the desired recipient in the drop-down list.</span></span>

6. <span data-ttu-id="7978a-126">Quando terminar de adicionar ou modificar os destinatários, selecione **Atualizar**.</span><span class="sxs-lookup"><span data-stu-id="7978a-126">When you've finished adding or modifying recipients, select **Update**.</span></span>
