# <a name="education-credit-basis"></a>Base de crédito de educação


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Este tópico descreve a opção da base de crédito de educação definida para o Dynamics 365 Human Resources.

Nome físico: mshr_hcmeducationcreditbasis

Essa enumeração fornece o conjunto de opções de valores para a base de crédito de educação do registro de formação educacional do candidato.

| Valor | Etiqueta | Descrição |
| --- | --- | --- |
| 200000000 | Em Branco | Nenhum valor foi selecionado. |
| 200000001 | Semestre | Semestre. |
| 200000002 | Trimestre | Trimestre. |
| 200000003 | Quadrimestre | Quadrimestre. |
| 200000004 | Termo | Termo. |
| 200000005 | Outros | Outros. |
