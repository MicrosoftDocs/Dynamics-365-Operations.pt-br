# <a name="period-unit"></a>Unidade de período

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Este tópico descreve a opção Unidade de período definida para o Dynamics 365 Human Resources.

Nome físico: mshr_periodunit

Essa enumeração fornece a opção definida de valores de unidades de medida para períodos de tempo.

| Valor | Etiqueta | Descrição |
| --- | --- | --- |
| 200000000 | Dias | Dia. |
| 200000001 | Meses | Mês. |
| 200000002 | Anos | Ano. |
