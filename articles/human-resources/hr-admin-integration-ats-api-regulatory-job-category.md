# <a name="regulatory-job-category"></a>Categoria de trabalho regulatório


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Este artigo descreve o conjunto de opções Categoria de trabalho regulatório para o Dynamics 365 Human Resources.

Nome físico: mshr_hcmregulatoryjobcategory

Essa enumeração especifica o conjunto de opções para os valores do conjunto de opções de Categoria de trabalho regulatório. Isso é fornecido no conjunto de opções existente de mshr_hcmregulatoryjobcategory.

| Valor | Etiqueta | Descrição |
| --- | --- | --- |
| 200000000 | Nenhuma | Nenhum. |
| 200000001 | Executivo | Funcionários e gerentes de nível executivo/sênior. |
| 200000002 | Gerente | Funcionários e gerentes de níveis principal e intermediário. |
| 200000003 | Profissional | Profissionais. |
| 200000004 | Técnico | Técnicos. |
| 200000005 | SalesWorker | Vendedores. |
| 200000006 | Administrativo | Funcionários de suporte administrativo. |
| 200000007 | CraftWorker | Artesãos. |
| 200000008 | Operante | Operantes. |
| 200000009 | Operário | Operários/ajudantes. |
| 200000010 | ServiceWorker | Prestador de serviços. |
