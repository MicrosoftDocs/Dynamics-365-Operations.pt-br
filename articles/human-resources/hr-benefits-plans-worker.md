# <a name="create-worker-benefit-plans"></a>Criar planos de benefícios do trabalhador

É possível criar planos de benefícios do trabalhador no Microsoft Dynamics 365 Human Resources para selecionar planos de benefícios para funcionários e confirmar seleções de planos de benefícios. Em geral, os funcionários selecionam os próprios planos de benefícios usando o autoatendimento para funcionários e depois um administrador de benefícios confirma as seleções. 

1. No espaço de trabalho **Gerenciamento de benefícios** em **Planos** , selecione **Planos de benefícios do trabalhador**.

2. Selecione **Novo**.

3. Especifique valores para os seguintes campos:

   | Campo | Descrição |
   | --- | --- |
   | Período | Especifica um período de benefícios a ser usado para filtrar os planos na guia rápida Planos. Filtre os planos para ajudar a selecionar um subconjunto de todos os registros do plano para que você possa confirmar o subconjunto. Por exemplo, selecione um período que você criou chamado 2015 para confirmar todas as seleções de inscrição de benefícios para 2015. |
   | Trabalhador | Especifica um trabalhador a ser usado para filtrar os planos na guia rápida Planos. Filtre os planos para ajudar a selecionar um subconjunto de todos os registros do plano para que você possa confirmar o subconjunto. |
   | Pessoa jurídica em geral | Especifica uma entidade legal a ser usada para filtrar os planos na guia rápida Planos. Filtre os planos para ajudar a selecionar um subconjunto de todos os registros do plano para que você possa confirmar o subconjunto. |
   | Opção de cobertura | Especifica uma opção de cobertura a ser usada para filtrar os planos na guia rápida Planos. Filtre os planos para ajudar a selecionar um subconjunto de todos os registros do plano para que você possa confirmar o subconjunto. |
   | Padrão | Filtra os planos de benefícios levando em consideração se eles são um plano padrão ou não. Filtre os planos para ajudar a selecionar um subconjunto de todos os registros do plano para que você possa confirmar o subconjunto. |
   | Status | Os filtros beneficiam os planos com base em seu status. Filtre os planos para ajudar a selecionar um subconjunto de todos os registros do plano para que você possa confirmar o subconjunto. |
   | Confirmação | Especifica o status de confirmação a ser usado para filtrar os planos na guia rápida Planos. Filtre os planos para ajudar a selecionar um subconjunto de todos os registros do plano para que você possa confirmar o subconjunto. |
   | Cancelamento | Especifica o status de cancelamento a ser usado para filtrar os planos na guia rápida Planos. Filtre os planos para ajudar a selecionar um subconjunto de todos os registros do plano para que você possa confirmar o subconjunto. |
   | Filtro de data de efetivação | Filtra os planos levando em consideração se eles expiraram, estão ativos ou estarão ativos no futuro. Marque as caixas de seleção que correspondem aos planos que você deseja ver na guia rápida Planos. |
   | Planos | A guia rápida Planos contém os planos que atendem aos critérios de filtro especificados. As opções de configuração relevantes definidas pela equipe de RH e as seleções de inscrição escolhidas pelos funcionários estão incluídas em cada linha. O campo Qualificado especifica se há um conflito de validação com a seleção do plano. |

4. Selecione **Salvar**.
