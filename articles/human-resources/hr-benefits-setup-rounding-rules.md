# <a name="set-up-rounding-rules"></a>Configurar regras de arredondamento

[!include [banner](includes/preview-feature.md)]

Você pode configurar regras de arredondamento no Dynamics 365 Human Resources para atender às necessidades da sua organização.

1. No espaço de trabalho **Gerenciamento de benefícios**, em **Configuração**, selecione **Regras de arredondamento**.

2. Selecione **Novo**.

3. Especifique valores para **Arredondar para o mais próximo**, **Cenário** e **Formulário de arredondamento**.

4. Selecione **Salvar**.
