# <a name="award-type"></a>Tipo de prêmio

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Este tópico descreve o conjunto de opções de tipo de prêmio para o Dynamics 365 Human Resources.

## <a name="option-set-award-type"></a>Conjunto de opções: tipo de prêmio

Nome físico: mshr_HrmCompVarAwardEmplType

Esta enumeração fornece o conjunto de opções para o tipo de prêmio de remuneração variável em entidades virtuais do Human Resources.

| Alíquota | Etiqueta | descrição |
| --- | --- | --- |
| 200000000 | Prêmio geral | Prêmio geral. |
| 200000001 | Pagamento fixo de quantia total | Pagamento fixo de quantia total. |
