# <a name="removed-or-deprecated-features-in-dynamics-365-human-resources"></a>Recursos removidos ou preteridos no Dynamics 365 Human Resources

Este artigo descreve os recursos que foram removidos ou foram planejados para remoção do Microsoft Dynamics 365 Human Resources.

- Um recurso *removido* não estará mais disponível no produto.
- Um recurso *preterido* não está no desenvolvimento ativo e poderá ser removido em uma atualização futura.

Esta lista é destinada a ajudá-lo a considerar essas remoções e reprovações para seu próprio planejamento.

## <a name="features-removed-or-deprecated-in-human-resources"></a>Recursos removidos ou preteridos no Human Resources 

### <a name="new-lcs-project-creation-in-human-resources-stand-alone"></a>Nova criação de projeto LCS no Human Resources (autônomo) 

[Provisionar o Human Resources](/hr-admin-setup-provision)

| &nbsp; | &nbsp; |
|------------|--------------------|
| **Motivo para a reprovação/remoção** | Substituídos pela criação do projeto na infraestrutura de finanças e operações. Para saber mais sobre a mesclagem de infraestrutura, consulte [Perguntas frequentes sobre mesclagem de infraestrutura do Dynamics 365 Human Resource](/hr-infrastructure-merge-faq). |
| **Substituída por outro recurso?**   | Sim |
| **Áreas afetadas do produto**         | Aplicativo |
| **Opção de implantação**              | Todos |
| **Status**                         | Preterido: a partir de julho de 2022, não há mais suporte para a criação de novos projetos no Human Resources (autônomo). |
