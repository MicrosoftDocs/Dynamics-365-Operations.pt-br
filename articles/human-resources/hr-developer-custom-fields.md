# <a name="custom-fields"></a>Campos personalizados

É possível adicionar campos personalizados a páginas no Dynamics 365 Human Resources. Para obter mais informações sobre como criar campos personalizados em uma página, consulte [Criar e trabalhar com campos personalizados](https://docs.microsoft.com/dynamics365/unified-operations/fin-and-ops/get-started/user-defined-fields).

As seguintes entidades no Common Data Service são compatíveis com campos personalizados criados no Human Resources:

- Trabalhador
- Origem étnica
- Situação militar
- Código do idioma
- Cargo
- Tipo de trabalho
- Função de trabalho
- Cargo
- Tipo de posição

Certifique-se de habilitar campos personalizados criados para que possa usá-los no Common Data Service. Para obter mais informações, consulte [Expor campos personalizados em entidades de dados](https://docs.microsoft.com/dynamics365/unified-operations/fin-and-ops/get-started/user-defined-fields#exposing-custom-fields-on-data-entities).
