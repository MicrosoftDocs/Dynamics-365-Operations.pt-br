# <a name="change-employee-self-service-workspace-name"></a>Alterar o nome do espaço de trabalho Autoatendimento para funcionários


[!INCLUDE [PEAP](../includes/peap-2.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Se você tem voluntários ou outras pessoas que não são funcionários, convém alterar o nome do espaço de trabalho **Autoatendimento para funcionários**. Você pode alterar esse espaço de trabalho para **Autoatendimento**.

> [!NOTE]
> Alterar o nome do espaço de trabalho **Autoatendimento para funcionários** também altera o item de menu usado internamente pelo Dynamics 365 Human Resources. Se anteriormente você aplicou as personalizações de segurança ao item de menu **HcmEmployeeSelfServiceWorkspace**, recomendamos aplicar as mesmas alterações a **HcmSelfServiceWorkspace** para manter a paridade.

1. No Human Resources, selecione **Gerenciamento de pessoal**, **Links** e **Parâmetros do Human Resources**.

2. Selecione a guia **Autoatendimento para funcionários**.

3. Em **Nome de exibição**, selecione **Autoatendimento**.

   ![Altere o nome do espaço de trabalho Autoatendimento para funcionários para Autoatendimento.](./media/hr-employee-self-service-workspace-name.png)

4. Selecione **Salvar**.
