# <a name="change-employee-self-service-workspace-name"></a><span data-ttu-id="fc414-103">Alterar o nome do espaço de trabalho Autoatendimento para funcionários</span><span class="sxs-lookup"><span data-stu-id="fc414-103">Change Employee self service workspace name</span></span>

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

<span data-ttu-id="fc414-104">Se você tem voluntários ou outras pessoas que não são funcionários, convém alterar o nome do espaço de trabalho **Autoatendimento para funcionários**.</span><span class="sxs-lookup"><span data-stu-id="fc414-104">If you have volunteers or other non-employees, you might want to change the name of the **Employee self-service** workspace.</span></span> <span data-ttu-id="fc414-105">Você pode alterar esse espaço de trabalho para **Autoatendimento**.</span><span class="sxs-lookup"><span data-stu-id="fc414-105">You can change this workspace to **Self service** instead.</span></span>

> [!NOTE]
> <span data-ttu-id="fc414-106">Alterar o nome do espaço de trabalho **Autoatendimento para funcionários** também altera o item de menu usado internamente pelo Dynamics 365 Human Resources.</span><span class="sxs-lookup"><span data-stu-id="fc414-106">Changing the name of the **Employee self-service** workspace also changes the menu item that is used internally by Dynamics 365 Human Resources.</span></span> <span data-ttu-id="fc414-107">Se anteriormente você aplicou as personalizações de segurança ao item de menu **HcmEmployeeSelfServiceWorkspace**, recomendamos aplicar as mesmas alterações a **HcmSelfServiceWorkspace** para manter a paridade.</span><span class="sxs-lookup"><span data-stu-id="fc414-107">If you previously applied security customizations to the **HcmEmployeeSelfServiceWorkspace** menu item, we recommend applying the same changes to **HcmSelfServiceWorkspace** to maintain parity.</span></span>

1. <span data-ttu-id="fc414-108">No Human Resources, selecione **Gerenciamento de pessoal**, **Links** e **Parâmetros do Human Resources**.</span><span class="sxs-lookup"><span data-stu-id="fc414-108">In Human Resources, select **Personnel management**, select **Links**, and then select **Human resources parameters**.</span></span>

2. <span data-ttu-id="fc414-109">Selecione a guia **Autoatendimento para funcionários**.</span><span class="sxs-lookup"><span data-stu-id="fc414-109">Select the **Employee self-service** tab.</span></span>

3. <span data-ttu-id="fc414-110">Em **Nome de exibição**, selecione **Autoatendimento**.</span><span class="sxs-lookup"><span data-stu-id="fc414-110">Under **Display name**, select **Self service**.</span></span>

   ![Alterar o nome do espaço de trabalho Autoatendimento para funcionários para Autoatendimento](./media/hr-employee-self-service-workspace-name.png)

4. <span data-ttu-id="fc414-112">Selecione **Salvar**.</span><span class="sxs-lookup"><span data-stu-id="fc414-112">Select **Save**.</span></span>
