# <a name="job-detail-api"></a>API de detalhes do trabalho

Este tópico fornece detalhes e um exemplo de consulta da entidade Detalhes do trabalho no Dynamics 365 Human Resources.

## <a name="properties"></a>Propriedades

| Propriedade<br>**Nome físico**<br>**_Tipo_** | Usar | descrição |
| --- | --- | --- |
| **ID do trabalho**<br>mshr_jobid<br>*Sequência de caracteres* | Somente leitura<br>Obrigatório | ID exclusiva de um trabalho. |
| **Limite de preço de mercado baixo**<br>mshr_marketpricelowthreshold<br>*Decimal* | | Um valor GUID gerado pelo sistema para identificar exclusivamente a posição.  |
