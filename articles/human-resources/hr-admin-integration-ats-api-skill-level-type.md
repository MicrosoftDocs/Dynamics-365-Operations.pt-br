# <a name="skill-level-type"></a>Tipo de nível de habilidade


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Este artigo descreve o conjunto de opções Tipo de nível de habilidade para o Dynamics 365 Human Resources.

Nome físico: mshr_hrmskillleveltype

Essa enumeração fornece o conjunto de opções de valores que categoriza o nível atribuído à habilidade de um candidato.

| Valor | Etiqueta | Descrição |
| --- | --- | --- |
| 200000000 | Real | Nível de habilidade real. |
| 200000001 | Alvo | Valor direcionado para o nível de habilidade. |
