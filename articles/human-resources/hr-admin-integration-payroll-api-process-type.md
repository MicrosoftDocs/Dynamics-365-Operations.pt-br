# <a name="process-type"></a>Tipo de processo

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Este tópico descreve o conjunto de opções Não/Sim para o Dynamics 365 Human Resources.

## <a name="option-set-process-type"></a>Conjunto de opções: tipo de processo

Nome físico: mshr_hrmCompProcessType

Esta enumeração fornece o conjunto de opções para o tipo de processo em entidades virtuais do Human Resources.

| Alíquota | Etiqueta | descrição |
| --- | --- | --- |
| 200000000 | Nenhuma | Nenhum. |
| 200000001 | Anual | Anual. |
| 200000002 | Aniversário de tempo de serviço | Aniversário de tempo de serviço. |
| 200000003 | Não cíclico | Não cíclico. |
| 200000004 | Semestral | Semestral. |
| 200000005 | Trimestral | Trimestral. |
| 200000006 | Mensalmente | Mensalmente. |
