# <a name="contact-type"></a>Tipo de contato


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Este tópico descreve o conjunto de opções de tipo de contato para o Dynamics 365 Human Resources.

Nome físico: mshr_logisticselectronicaddressmethodtype

Essa enumeração fornece o conjunto de opções de valores tipos de contato de candidato. 

| Valor | Etiqueta | descrição |
| --- | --- | --- |
| 200000000 | Nenhuma | Nenhum tipo foi selecionado. |
| 200000001 | Telefone | Número de telefone. |
| 200000002 | Endereço de email | Endereço de email. |
| 200000003 | URL | URL do site. |
| 200000004 | Telex | Número de telex. |
| 200000005 | Fax | Número de fax. |
| 200000006 | Facebook | Conta do Facebook. Identificado pela ID do usuário. |
| 200000007 | Twitter | Conta do Twitter. Identificado pelo @username. |
| 200000008 | LinkedIn | Conta do LinkedIn. Identificado pelo nome do usuário. |
