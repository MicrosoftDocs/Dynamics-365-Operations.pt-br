# <a name="payroll---gender-option-set"></a>Folha de pagamento - Conjunto de opções Sexo


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Este tópico descreve a opção Sexo definida para o Dynamics 365 Human Resources.

## <a name="option-set-gender-option"></a>Conjunto de opções: sexo

Essa enumeração fornece a opção definida de gêneros para o funcionário. Isso está disponível na opção mshr_hcmpersongender definida.

| Valor | Etiqueta | descrição |
| --- | --- | --- |
| 200000000 | Nenhuma | Nenhum sexo foi especificado. |
| 200000001 | Masculino | Masculino. |
| 200000002 | Feminino | Feminino. |
| 200000003 | Não especificado | Seleção para um sexo não específico. |
