# <a name="get-support"></a>Obter suporte

Este artigo explica como obter ajuda com o Microsoft Dynamics 365 Human Resources.

Para obter suporte para o Human Resources, use os recursos na tabela a seguir. Esses recursos estão listados na ordem que recomendamos ser seguida quando você estiver solucionando problemas.

| Tarefa | Mais informações |
|------|------------------|
| Pergunte à comunidade. | Vá para a [Comunidade do Dynamics 365 Human Resources](https://community.dynamics.com/365/talent) para obter ajuda com suas dúvidas. Para encontrar respostas, insira uma pergunta no campo de pesquisa próximo a **Tenho uma pergunta**. Para fazer uma pergunta, selecione **Fazer uma pergunta**. |
| Usar a ferramenta de suporte habilitado para a nuvem. | No [Microsoft Dynamics Lifecycle Services (LCS)](https://lcs.dynamics.com/), vá para seu projeto e selecione o bloco **Suporte**. Ali, é possível enviar problemas de duas maneiras:<ul><li>Na guia **Problemas ativos**, selecione o problema que deseja enviar e depois selecione **Enviar para a Microsoft**.</li><li>Na guia **Enviado à Microsoft**, selecione **Enviar um novo incidente** para enviar um novo problema.</li></ul> |
| Obter suporte no aplicativo. | Selecione o botão **Ajuda** (**?**) no canto superior direito da página. Selecione **Suporte**. Problemas são reportados na guia **Problemas ativos** no LCS, quando um administrador pode determinar se deseja fornecer suporte em casa ou enviar os problemas para a Microsoft. |
| Obter ajuda com dúvidas sobre o licenciamento do Human Resources. | Contate seu parceiro ou representante de vendas da Microsoft. |
| Solicitar novos recursos e funcionalidades no Human Resources. | No Human Resources, selecione o botão **Ajuda** (**?**) e, depois, **Ideias**. Você também pode acessar [Ideias de aplicação do Dynamics 365](https://experience.dynamics.com/ideas/) para exibir, buscar ou votar em ideias existentes, ou para adicionar novas ideias. |
