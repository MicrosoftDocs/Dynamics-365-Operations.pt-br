# <a name="screening-frequency"></a>Frequência de triagem


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Este artigo descreve o conjunto de opções Frequência de triagem para o Dynamics 365 Human Resources.

Nome físico: mshr_hcmfrequencyunit

Essa enumeração fornece o conjunto de opções de valores da frequência de triagem. 

| Valor | Etiqueta | Descrição |
| --- | --- | --- |
| 200000000 Apenas ocasional | A triagem é necessária apenas uma vez. |
| 200000001 Diária | A frequência de triagem é calculada em dias. |
| 200000002 Semanal | A frequência de triagem é calculada em semanas. |
| 200000003 Mensal | A frequência de triagem é calculada em meses. |
| 200000004 Anual | A frequência de triagem é calculada em anos. |
