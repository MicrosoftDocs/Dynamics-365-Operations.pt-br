# <a name="map-skills"></a>Mapear habilidades

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Você pode criar uma pesquisa de mapeamento de habilidades para encontrar uma pessoa qualificada no Dynamics 365 Human Resources. As pesquisas de mapeamento de habilidades retornam resultados para os critérios inseridos consultando as seguintes informações:

- Habilidades
- Formação
- Certificados
- Posições
- Experiência em Projetos

Por exemplo, você pode encontrar pessoas em sua organização que receberam o CPA.

Os perfis de mapeamento de habilidades permitem que você encontre funcionários atuais ou candidatos com qualificações que correspondam diretamente às necessidades do negócio.

> [!NOTE]
> Somente os trabalhadores, os candidatos e as pessoas de contato selecionados para serem incluídos em pesquisas de mapeamento de habilidade serão exibidos em uma lista dos resultados de um mapeamento de habilidades ou serem incluídos em um perfil de habilidades. Para incluir uma pessoa nas pesquisas de mapeamento de habilidades, defina a seleção **Incluir no mapeamento de habilidades** como **Sim** nas seguintes páginas:<br>
> - Trabalhador<br>
> - Funcionário<br>
> - Candidato<br>
> - Contatos<br>

Para criar um mapeamento de habilidades, Acesse **Desenvolvimento do funcionário > Links > Mapeamento de habilidades**. Para criar um perfil de mapeamento de habilidades, Acesse **Desenvolvimento do funcionário > Links > Perfis de mapeamento de habilidades**.

## <a name="skill-gap-analysis-and-skill-profile-analysis"></a>Análise da lacuna de habilidades e análise do perfil de habilidades

Você pode criar uma análise de perfil de habilidade para exibir uma lista das competências de uma pessoa. Você pode criar uma análise de lacuna de habilidades para comparar as habilidades de uma pessoa e as habilidades necessárias para um trabalho.

Para criar uma análise de lacunas, Acesse **Desenvolvimento do funcionário > Links > Trabalho de análise de lacuna de habilidades - pessoa**. Para criar uma análise de perfil de habilidades, Acesse **Desenvolvimento do funcionários > Links > Análise do perfil de habilidades**.
