# <a name="blank-yes-no"></a>Em branco Sim/Não

Este tópico descreve o conjunto de opções Em Branco Sim/Não para o Dynamics 365 Human Resources.

Nome físico: mshr_hrmblankyesno

Esta enumeração fornece o conjunto de opções para propriedades Sim/Não que também permitem espaços em branco.

| Valor | Etiqueta | Descrição |
| --- | --- | --- |
| 200000000 | Em Branco | Nenhum valor foi selecionado. |
| 200000001 | Sim | Sim. |
| 200000002 | Não | Nº |
