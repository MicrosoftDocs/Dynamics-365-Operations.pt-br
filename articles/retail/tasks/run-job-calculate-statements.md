# <a name="configure-and-run-a-job-to-calculate-statements"></a><span data-ttu-id="904e9-103">Configurar e executar um trabalho para calcular demonstrativos</span><span class="sxs-lookup"><span data-stu-id="904e9-103">Configure and run a job to calculate statements</span></span>

[!include[task guide banner](../includes/task-guide-banner.md)]

<span data-ttu-id="904e9-104">Este procedimento orienta como configurar e executar trabalhos em lotes recorrentes para criar e calcular demonstrativos para uma loja ou grupo de lojas selecionado.</span><span class="sxs-lookup"><span data-stu-id="904e9-104">This procedure walks through configuring and running recurrent batch jobs to create and calculate statements for a selected store or group of stores.</span></span> <span data-ttu-id="904e9-105">Este procedimento usa a empresa USRT nos dados de demonstração.</span><span class="sxs-lookup"><span data-stu-id="904e9-105">This procedure uses the USRT company in demo data.</span></span>

1. <span data-ttu-id="904e9-106">Vá para Todos os espaços de trabalho > Finanças da loja de varejo.</span><span class="sxs-lookup"><span data-stu-id="904e9-106">Go to All workspaces > Retail store financials.</span></span>
2. <span data-ttu-id="904e9-107">Clique em Calcular demonstrativos.</span><span class="sxs-lookup"><span data-stu-id="904e9-107">Click Calculate statements.</span></span>
    * <span data-ttu-id="904e9-108">Selecione uma loja específica ou um nó se você deseja criar trabalho em lotes para um grupo de lojas.</span><span class="sxs-lookup"><span data-stu-id="904e9-108">Select either a specific store, or a node if you want to create the batch job for a group of stores.</span></span>  
    * <span data-ttu-id="904e9-109">Clique na seta para adicionar sua seleção.</span><span class="sxs-lookup"><span data-stu-id="904e9-109">Click the arrow to add your selection.</span></span>  
3. <span data-ttu-id="904e9-110">Clique na guia Executar em segundo plano.</span><span class="sxs-lookup"><span data-stu-id="904e9-110">Click the Run in the background tab.</span></span>
4. <span data-ttu-id="904e9-111">Em Processamento em lotes, selecione 'Sim'.</span><span class="sxs-lookup"><span data-stu-id="904e9-111">Under Batch processing, select 'Yes'.</span></span>
5. <span data-ttu-id="904e9-112">Clique em Recorrência.</span><span class="sxs-lookup"><span data-stu-id="904e9-112">Click Recurrence.</span></span>
6. <span data-ttu-id="904e9-113">No campo Data de início, insira uma data.</span><span class="sxs-lookup"><span data-stu-id="904e9-113">In the Start date field, enter a date.</span></span>
7. <span data-ttu-id="904e9-114">No campo Hora de início, insira um horário.</span><span class="sxs-lookup"><span data-stu-id="904e9-114">In the Start time field, enter a time.</span></span>
8. <span data-ttu-id="904e9-115">Selecione a opção Nenhuma data de término.</span><span class="sxs-lookup"><span data-stu-id="904e9-115">Select the No end date option.</span></span>
9. <span data-ttu-id="904e9-116">No campo Unidade padrão, insira 'Dias'.</span><span class="sxs-lookup"><span data-stu-id="904e9-116">In the PatternUnit field, enter 'Days'.</span></span>
10. <span data-ttu-id="904e9-117">No campo Por, insira um número.</span><span class="sxs-lookup"><span data-stu-id="904e9-117">In the Per field, enter a number.</span></span>
11. <span data-ttu-id="904e9-118">Clique em OK.</span><span class="sxs-lookup"><span data-stu-id="904e9-118">Click OK.</span></span>
12. <span data-ttu-id="904e9-119">Clique em OK.</span><span class="sxs-lookup"><span data-stu-id="904e9-119">Click OK.</span></span>
