## <a name="install-the-one-click-designer"></a><span data-ttu-id="348aa-109">Instalar o designer de um clique</span><span class="sxs-lookup"><span data-stu-id="348aa-109">Install the one-click designer</span></span>

1. <span data-ttu-id="348aa-110">No Microsoft Dynamics 365 for Retail, use o menu na parte superior esquerda para navegar para **Varejo** **e comércio** &gt; **Configuração de canal** &gt; **Configuração de PDV** &gt; **PDV** &gt; **Layouts de tela**.</span><span class="sxs-lookup"><span data-stu-id="348aa-110">In Microsoft Dynamics 365 for Retail, use the menu in the upper left to navigate to **Retail** **and commerce** &gt; **Channel setup** &gt; **POS setup** &gt; **POS** &gt; **Screen layouts**.</span></span>
2. <span data-ttu-id="348aa-111">Selecione qualquer layout que tenha um tipo de aplicativo **Modern POS para Windows** ou **PDV em nuvem**, em seguida clique em **Designer do layout**.</span><span class="sxs-lookup"><span data-stu-id="348aa-111">Select any layout that has an application type of **Modern POS for Windows** or **Cloud POS**, and then click **Layout designer**.</span></span>
3. <span data-ttu-id="348aa-112">Na barra de notificação exibida na parte inferior da janela do Internet Explorer, clique em **Abrir** para instalar o designer de um clique.</span><span class="sxs-lookup"><span data-stu-id="348aa-112">On the notification bar that appears at the bottom of the Internet Explorer window, click **Open** to install the one-click designer.</span></span> <span data-ttu-id="348aa-113">(A barra de notificação pode aparecer em um local diferente em outros navegadores).</span><span class="sxs-lookup"><span data-stu-id="348aa-113">(The notification bar might appear in a different place in other browsers.)</span></span>
4. <span data-ttu-id="348aa-114">Na caixa de mensagem que aparece **Execução de Aplicativo - Aviso de Segurança**, clique em **Executar** para instalar o host designer varejista.</span><span class="sxs-lookup"><span data-stu-id="348aa-114">In the **Application Run - Security Warning** message box that appears, click **Run** to install the Retail designer host.</span></span> <span data-ttu-id="348aa-115">O indicador de progresso mostra o progresso do processo de instalação.</span><span class="sxs-lookup"><span data-stu-id="348aa-115">A progress indicator shows the progress of the installation.</span></span>
5. <span data-ttu-id="348aa-116">Depois que a instalação for concluída, na página **Entrar**, insira seu nome de usuário e senha do Microsoft Dynamics 365 for Retail, e clique em **Entrar** para iniciar o designer.</span><span class="sxs-lookup"><span data-stu-id="348aa-116">After the installation is completed, on the **Sign in** page, enter your Microsoft Dynamics 365 for Retail user name and password, and then click **Sign in** to start the designer.</span></span>
6. <span data-ttu-id="348aa-117">Depois que suas credenciais forem validadas e o designer for iniciado, você poderá criar o seu próprio layout ou modificar o formato existente.</span><span class="sxs-lookup"><span data-stu-id="348aa-117">After your credentials are validated and the designer starts, you can design your own layout or modify the existing layout.</span></span>
