    <span data-ttu-id="9bea0-122">As notificações são mostradas somente para operações adicionadas na Guia Rápida **Notificações**, e você só poderá adicionar operações lá se a caixa de seleção **Habilitar notificações** para essas operações tiver sido selecionada na página **Operações de PDV**.</span><span class="sxs-lookup"><span data-stu-id="9bea0-122">Notifications are shown only for operations that are added on the **Notifications** FastTab, and you can add operations there only if the **Enable notifications** check box for those operations has been selected on the **POS operations** page.</span></span> <span data-ttu-id="9bea0-123">Além disso, as notificações para uma operação são mostradas aos trabalhadores somente se a operação for adicionada às permissões de PDV para esses trabalhadores.</span><span class="sxs-lookup"><span data-stu-id="9bea0-123">Additionally, notifications for an operation are shown to workers only if the operation is added to the POS permissions for those workers.</span></span>

    > [!NOTE]
    > <span data-ttu-id="9bea0-124">As notificações podem ser substituídas no nível do usuário.</span><span class="sxs-lookup"><span data-stu-id="9bea0-124">Notifications can be overridden at the user level.</span></span> <span data-ttu-id="9bea0-125">Abra o registro do trabalhador, selecione **Permissões do PDV** e, em seguida, edite a subscrição de notificação do usuário.</span><span class="sxs-lookup"><span data-stu-id="9bea0-125">Open the worker's record, select **POS permissions**, and then edit the user's notification subscription.</span></span>
