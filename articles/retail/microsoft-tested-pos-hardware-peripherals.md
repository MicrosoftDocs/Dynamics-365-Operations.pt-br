# <a name="pos-hardware-peripherals"></a>Periféricos de hardware do PDV

[!include[banner](includes/banner.md)]


O ponto de venda moderno manager (POS) e o retail do nuvem podem utilizar uma grande variedade de periféricos de hardware do pos, com várias opções de interfaces e implantação obter vários cenários comerciais de um fornecedor. 

Para dar suporte a maior seleção de dispositivos entre fabricantes e modelos, PDV utiliza interfaces padrão como OLE para Retail POS (OPOS), Unidades de dispositivo Windows e ponto Windows de interfaces de programa aplicativo de serviço (APIs). Geralmente, o PDV funcionará nesses dispositivos fornecidos onde a unidade apropriada e fornecida. No entanto, como cada fabricante e implementação de desenvolvedor de software desses padrões pode variar, há diferenças frequentes nas funcionalidades suportadas ou comportamentos.

A lista a seguir contém os modelos de dispositivo, em cada classe, que foram testados internamente pela Microsoft.

**Dispositivos OPOS**

-   Código de barras – Motorola DS9208
-   MSR – HP IDRA-334133, Magtek PN - 21073062
-   LineDisplay – Epson M58DC
-   Pinpad – Verifone 1000SE
-   Dispositivo de assinaturas – Scriptel ST1550
-   Impressora – EPSON TM-T88IV, TMT88V
-   Caixas registradoras – Star SMD2-1317BK44
-   Escala – Datalogic Magellan 8400

**LTM leitor de cartão de crédito do teclado**

-   USB de Magtek

**Terminal de pagamento**

-   Equinox L3500
-   Verifone MX925

**Dispositivos de rede**

-   Impressora – Star TSP650II
-   Caixas registradoras – APG Atwood
-   Terminal de pagamento – MX915, MX925

**MPOS direcionam CPI somente**

-   Código de barras – Honeywell 1900, HP LS2208
-   MSR – Magtek PN - 21073075
