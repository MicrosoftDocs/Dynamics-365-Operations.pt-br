# <a name="configure-linked-products"></a><span data-ttu-id="52076-103">Configurar produtos vinculados</span><span class="sxs-lookup"><span data-stu-id="52076-103">Configure linked products</span></span>

[!include[task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="52076-104">Este procedimento orienta na configuração de produtos vinculados.</span><span class="sxs-lookup"><span data-stu-id="52076-104">This procedure walks through configuring linked products.</span></span> <span data-ttu-id="52076-105">Este procedimento usa os dados de demonstração da USRT, e presume-se que os produtos já tenham sido criados e liberados.</span><span class="sxs-lookup"><span data-stu-id="52076-105">This procedure uses the USRT demo data, and assumes both products have already been created and released.</span></span>

1. <span data-ttu-id="52076-106">Vá para Gerenciamento de informações do produto > Produtos > Produtos liberados.</span><span class="sxs-lookup"><span data-stu-id="52076-106">Go to Product information management > Products > Released products.</span></span>
2. <span data-ttu-id="52076-107">Na lista, localize e selecione o PDV desejado.</span><span class="sxs-lookup"><span data-stu-id="52076-107">In the list, find and select the desired record.</span></span>
3. <span data-ttu-id="52076-108">No Painel de Ação, clique em Varejo.</span><span class="sxs-lookup"><span data-stu-id="52076-108">On the Action Pane, click Retail.</span></span>
4. <span data-ttu-id="52076-109">Clique em Produtos vinculados.</span><span class="sxs-lookup"><span data-stu-id="52076-109">Click Linked products.</span></span>
5. <span data-ttu-id="52076-110">Clique em Editar.</span><span class="sxs-lookup"><span data-stu-id="52076-110">Click Edit.</span></span>
6. <span data-ttu-id="52076-111">No campo Número do produto vinculado, clique no botão suspenso para abrir a pesquisa.</span><span class="sxs-lookup"><span data-stu-id="52076-111">In the Linked product number field, click the drop-down button to open the lookup.</span></span>
7. <span data-ttu-id="52076-112">Na lista, clique no link na linha selecionada.</span><span class="sxs-lookup"><span data-stu-id="52076-112">In the list, click the link in the selected row.</span></span>
8. <span data-ttu-id="52076-113">No campo Unidade, clique no botão suspenso para abrir a pesquisa.</span><span class="sxs-lookup"><span data-stu-id="52076-113">In the Unit field, click the drop-down button to open the lookup.</span></span>
9. <span data-ttu-id="52076-114">Na lista, clique no link na linha selecionada.</span><span class="sxs-lookup"><span data-stu-id="52076-114">In the list, click the link in the selected row.</span></span>
10. <span data-ttu-id="52076-115">No campo Quantidade, insira um número.</span><span class="sxs-lookup"><span data-stu-id="52076-115">In the Quantity field, enter a number.</span></span>
11. <span data-ttu-id="52076-116">Clique em Salvar.</span><span class="sxs-lookup"><span data-stu-id="52076-116">Click Save.</span></span>
