# <a name="replenishment"></a><span data-ttu-id="40876-103">Reabastecimento</span><span class="sxs-lookup"><span data-stu-id="40876-103">Replenishment</span></span>

[!include[banner](../includes/banner.md)]


<span data-ttu-id="40876-104">Este tópico descreve as estratégias de reabastecimento que estão disponíveis para os depósitos que usam a funcionalidade que está disponível no gerenciamento de depósito.</span><span class="sxs-lookup"><span data-stu-id="40876-104">This topic describes the replenishment strategies that are available for warehouses which use the functionality that is available in Warehouse management.</span></span> <span data-ttu-id="40876-105">As informações não se aplicam à solução de armazenamento disponível no Gerenciamento de estoque.</span><span class="sxs-lookup"><span data-stu-id="40876-105">The information doesn't apply to the warehousing solution that is available in Inventory management.</span></span> <span data-ttu-id="40876-106">As seguintes estratégias de reabastecimento estão disponíveis:</span><span class="sxs-lookup"><span data-stu-id="40876-106">The following replenishment strategies are available:</span></span>

-   <span data-ttu-id="40876-107">**Reabastecimento baseado em demanda** – essa estratégia cria o trabalho de reabastecimento para ordens de saída ou cargas, caso o estoque não esteja disponível quando o trabalho for criado pela onda.</span><span class="sxs-lookup"><span data-stu-id="40876-107">**Wave demand replenishment** – This strategy creates replenishment work for outbound orders or loads if inventory isn't available when the wave creates work.</span></span> <span data-ttu-id="40876-108">Por exemplo, se a quantidade necessária para uma ordem de venda não estiver disponível quando uma onda for processada, será possível criar o trabalho de reabastecimento.</span><span class="sxs-lookup"><span data-stu-id="40876-108">For example, replenishment work can be created if the quantity that is required for a sales order isn't available when a wave is processed.</span></span>
-   <span data-ttu-id="40876-109">**Reabastecimento mín./máx.** – essa estratégia usa limites mínimo e máximo de estoque para determinar quando as localizações deverão ser reabastecidas.</span><span class="sxs-lookup"><span data-stu-id="40876-109">**Min/Max replenishment** – This strategy uses minimum and maximum stocking limits to determine when locations should be replenished.</span></span> <span data-ttu-id="40876-110">Os critérios de item e localização definem o estoque avaliado para reabastecimento.</span><span class="sxs-lookup"><span data-stu-id="40876-110">The item and location criteria define the inventory that is evaluated for replenishment.</span></span> <span data-ttu-id="40876-111">Modelos de reabastecimento mín./máx. são o principal mecanismo para manter os níveis ideais em locais de separação.</span><span class="sxs-lookup"><span data-stu-id="40876-111">Min/Max replenishment templates are the primary mechanism for maintaining optimal levels in picking locations.</span></span> <span data-ttu-id="40876-112">Para garantir que o estoque nominal escolhido suficiente esteja disponível para ser compatível com a demanda da onda, você pode usar a demanda de reabastecimento como um suplemento entre ciclos de reabastecimento mín./máx.</span><span class="sxs-lookup"><span data-stu-id="40876-112">To ensure that enough pick face inventory is available to meet wave demand, you can use demand replenishment as a supplement between Min/Max replenishment cycles.</span></span>
-   <span data-ttu-id="40876-113">**Carregar o reabastecimento por demanda** – essa estratégia soma a demanda por várias cargas e cria o trabalho de reabastecimento necessário para os locais de separação relevante de ações.</span><span class="sxs-lookup"><span data-stu-id="40876-113">**Load demand replenishment** – This strategy sums the demand for several loads and creates the replenishment work that is required in order to stock the relevant picking locations.</span></span> <span data-ttu-id="40876-114">Essa estratégia ajuda a garantir que as cargas criadas possam ser separadas no depósito depois que elas são lançadas.</span><span class="sxs-lookup"><span data-stu-id="40876-114">This strategy helps to ensure that the loads that are created can be picked in the warehouse after they are released.</span></span>

<span data-ttu-id="40876-115">Todas as três estratégias criam trabalho de reabastecimento, com base em um modelo de reabastecimento.</span><span class="sxs-lookup"><span data-stu-id="40876-115">All three strategies create replenishment work, based on a replenishment template.</span></span>

## <a name="wave-demand-replenishment"></a><span data-ttu-id="40876-116">Reabastecimento de demanda de onda</span><span class="sxs-lookup"><span data-stu-id="40876-116">Wave demand replenishment</span></span>

<span data-ttu-id="40876-117">O reabastecimento de demanda de onda cria um trabalho de reabastecimento, baseado na demanda, se a quantidade que é necessária para ordens de produção, kanbans, ordens de saída ou carregamentos não está disponível quando uma onda cria trabalho.</span><span class="sxs-lookup"><span data-stu-id="40876-117">Wave demand replenishment creates replenishment work, based on demand, if the quantity that is required for production orders, kanbans, outbound orders or loads isn't available when a wave creates work.</span></span> <span data-ttu-id="40876-118">O modelo de reabastecimento contém informações sobre os critérios de item, a unidade de medida, o incremento de demanda e o local.</span><span class="sxs-lookup"><span data-stu-id="40876-118">The replenishment template contains information about the item criteria, the unit of measure, the demand increment, and the location.</span></span> 

<span data-ttu-id="40876-119">Diretivas locais são usadas para determinar qual local deve ser reabastecido.</span><span class="sxs-lookup"><span data-stu-id="40876-119">Location directives are used to determine which location should be replenished.</span></span> <span data-ttu-id="40876-120">Você vincula essas diretivas de local ao modelo de reabastecimento usando o campo **Código de diretiva**.</span><span class="sxs-lookup"><span data-stu-id="40876-120">You link these location directives to the replenishment template by using the **Directive code** field.</span></span> <span data-ttu-id="40876-121">Se o campo **Código de diretiva** não for definido, as consultas são usadas para determinar qual diretiva local deve ser usada.</span><span class="sxs-lookup"><span data-stu-id="40876-121">If the **Directive code** field isn't set, queries are used to determine which location directive should be used.</span></span> <span data-ttu-id="40876-122">Observe que se não for especificado um código de diretiva no modelo de reabastecimento e a diretiva local tiver um código de diretiva, a diretiva local será ignorada, mesmo se a consulta na diretiva local estiver correta.</span><span class="sxs-lookup"><span data-stu-id="40876-122">Note that if a directive code isn't specified in the replenishment template, and the location directive has a directive code, the location directive will be ignored, even if the query on the location directive is correct.</span></span> <span data-ttu-id="40876-123">Escolha diretivas locais que são usadas para determinar de onde obter o estoque para o reabastecimento.</span><span class="sxs-lookup"><span data-stu-id="40876-123">Pick location directives are used to determine where to get inventory for the replenishment.</span></span> 

<span data-ttu-id="40876-124">Além de criar um modelo, você deve especificar algumas configurações de reabastecimento no modelo de onda.</span><span class="sxs-lookup"><span data-stu-id="40876-124">In addition to creating a template, you must specify some replenishment settings in the wave template.</span></span> <span data-ttu-id="40876-125">O modelo de onda deve conter uma etapa de onda para reabastecimento executada somente se a alocação de um item não for bem-sucedida.</span><span class="sxs-lookup"><span data-stu-id="40876-125">The wave template should contain a wave step for replenishment that is run only if allocation of an item isn't successful.</span></span> <span data-ttu-id="40876-126">Esta etapa de onda de reabastecimento usa um código de etapa de onda para determinar qual modelo de reabastecimento deve ser usado.</span><span class="sxs-lookup"><span data-stu-id="40876-126">This replenishment wave step uses a wave step code to determine which replenishment template should be used.</span></span> <span data-ttu-id="40876-127">Além de ter uma etapa de onda para reabastecimento, você deve garantir que **Reabastecer** seja selecionado na seção **Métodos** do modelo de onda.</span><span class="sxs-lookup"><span data-stu-id="40876-127">In addition to having a wave step for replenishment, you need to make sure that **Replenish** is selected in the **Methods** section of the wave template.</span></span> 

<span data-ttu-id="40876-128">A página **Modelo de reabastecimento** inclui uma caixa de seleção **Permitir que a demanda de onda use quantidades não reservadas**.</span><span class="sxs-lookup"><span data-stu-id="40876-128">The **Replenishment template** page includes an **Allow wave demand to use unreserved quantities** check box.</span></span> <span data-ttu-id="40876-129">Você deve selecionar esta caixa de seleção se desejar permitir que o reabastecimento de demanda deduza quantidades de trabalho não reservadas geradas do modelo selecionado de reabastecimento.</span><span class="sxs-lookup"><span data-stu-id="40876-129">You should select this check box if you want to allow demand replenishment to deduct unreserved quantities from work that is generated from the selected replenishment template.</span></span> <span data-ttu-id="40876-130">Selecione esta caixa de seleção para cada modelo existente de reabastecimento para permitir que modelos de reabastecimento de demanda usem esta lógica.</span><span class="sxs-lookup"><span data-stu-id="40876-130">To enable demand replenishment templates to use this logic, select this check box for every existing replenishment template.</span></span> <span data-ttu-id="40876-131">Quando o reabastecimento de demanda é disparado no depósito, ele deduzirá a demanda de trabalho de reabastecimento existente com quantidades não reservadas, se o trabalho for gerado por modelos de reabastecimento com a caia de seleção **Permitir que a demanda da onda use quantidades não reservadas** selecionada.</span><span class="sxs-lookup"><span data-stu-id="40876-131">When demand replenishment is triggered in the warehouse, it will deduct the demand from existing replenishment work that has unreserved quantities, if the work originates from replenishment templates where the **Allow wave demand to use unreserved quantities** check box is selected.</span></span>


<span data-ttu-id="40876-132">O reabastecimento de demanda é suportado para pedidos de vendas, ordens de transferência, ordens de produção e kanbans.</span><span class="sxs-lookup"><span data-stu-id="40876-132">Demand replenishment is supported for sales orders, transfer orders, production orders, and kanbans.</span></span> 

## <a name="minmax-replenishment"></a><span data-ttu-id="40876-133">Reabastecimento mín./máx.</span><span class="sxs-lookup"><span data-stu-id="40876-133">Min/Max replenishment</span></span>
<span data-ttu-id="40876-134">No reabastecimento mín./máx., estoque é reabastecido para que esteja também entre os limites máximo e mínimo que foram definidos.</span><span class="sxs-lookup"><span data-stu-id="40876-134">In Min/Max replenishment, stock is replenished so that it's between the minimum and maximum limits that have been set.</span></span> <span data-ttu-id="40876-135">Normalmente, este processo ocorre uma vez por dia para garantir que todos os locais de separação são preenchidos para o nível máximo antes da separação ser iniciada.</span><span class="sxs-lookup"><span data-stu-id="40876-135">Typically, this process occurs one time every day to ensure that all picking locations are filled to the maximum level before picking starts.</span></span> 

<span data-ttu-id="40876-136">Os valores mínimo e máximo são definidos em um modelo de reabastecimento.</span><span class="sxs-lookup"><span data-stu-id="40876-136">The minimum and maximum amounts are set in a replenishment template.</span></span> <span data-ttu-id="40876-137">Muitas das configurações no modelo são semelhantes as configurações nos modelos que são usadas no reabastecimento de demanda da onda.</span><span class="sxs-lookup"><span data-stu-id="40876-137">Many of the other settings in the template resemble the settings in templates that are used in Wave demand replenishment.</span></span> <span data-ttu-id="40876-138">O modelo deve conter uma linha para cada item e localização.</span><span class="sxs-lookup"><span data-stu-id="40876-138">The template should contain one line for each item and location.</span></span> <span data-ttu-id="40876-139">Quando você executa o reabastecimento usando o trabalho em lotes, o Finanças e Operações avaliará se o reabastecimento é necessário na sequência na qual as linhas são organizadas.</span><span class="sxs-lookup"><span data-stu-id="40876-139">When you run replenishment by using the batch job, Finance and Operations evaluates whether replenishment is required, in the sequence that the lines are organized in.</span></span> 

<span data-ttu-id="40876-140">Observe que a estratégia de reabastecimento mínima/máxima não pode reabastecer um local vazio, a menos que o local esteja definido como o local fixo para o item.</span><span class="sxs-lookup"><span data-stu-id="40876-140">Note that the Min/Max replenishment strategy can't replenish an empty location unless the location is set as the fixed location for the item.</span></span> <span data-ttu-id="40876-141">Se a localização que deve ser reabastecida não estiver em um local fixo, não é possível determinar qual item para reabastecer.</span><span class="sxs-lookup"><span data-stu-id="40876-141">If the location that must be replenished isn't a fixed location, it's not possible to determine which item to replenish.</span></span> <span data-ttu-id="40876-142">Portanto, pelo menos alguma quantidade disponível é necessária antes de reabastecimento.</span><span class="sxs-lookup"><span data-stu-id="40876-142">Therefore, at least some on-hand quantity is required before replenishment occurs.</span></span>
