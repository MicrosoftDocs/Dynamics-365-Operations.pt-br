## <a name="white-papers"></a>White papers
* O <a href="https://mbs.microsoft.com/customersource/northamerica/AX/learning/documentation/white-papers/msd365optgtstcostacc/">Microsoft Dynamics 365 para operações - Introdução à contabilização de custos</a> white paper apresenta as etapas básicas para configurar o Microsoft Dynamics 365 para operações – custo contábeis para análise de custo real. Ele também fornece uma série de procedimentos que você pode concluir na ordem em que aparecem.
* O <a href="https://mbs.microsoft.com/customersource/northamerica/AX/learning/documentation/white-papers/leanmanufkanban365opt/">Dynamics 365 para operações - Lean Manufacturing: capaz de promessa e o planejamento de trabalho Kanban</a> white paper descreve os detalhes do plano de trabalho kanban e apresenta o conceito de fazer de ordem de venda que usa o método promessa (CTP) capaz para controle da data de entrega. Ela também explica como CTP pode ser combinada com kanbans de evento para modelar cenários de montagem sob encomenda ou ordem de compilação.
* O <a href="https://mbs.microsoft.com/customersource/northamerica/AX/learning/documentation/white-papers/365operationsbomcalsheet/">Microsoft Dynamics 365 para operações - cálculo de BOM usando uma folha de custos</a> white paper descreve como configurar uma folha de custos que inclui material e fabricação, e como a configuração afeta o cálculo de BOM resultados. Você pode fazer os exercícios no Apêndice para fins de aprendizado e usar o conhecimento que você adquiriu da leitura deste documento para fazer análise de cálculo de BOM.
