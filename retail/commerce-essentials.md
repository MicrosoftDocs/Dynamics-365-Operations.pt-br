# <a name="commerce-essentials-overview"></a>Visão geral de fundamentos de comércio

[!include[banner](includes/banner.md)]


Este artigo fornece informações sobre fundamentos de comércio, que é um ponto de venda de varejo e opção de configuração de comércio eletrônico para Microsoft Dynamics 365 for Operations. Os fundamentos de comércio fornecem uma carga de trabalho de varejo da saída de caixa que podem ajudar fornecedores a entrar no ar rapidamente com a solução de Retail com base no Microsoft Dynamics e permite que eles foquem nas funcionalidades específicas de Retail. 

<a name="overview"></a>Visão Geral
--------

Os ambientes de software empresarial costumam incluir vários sistemas, e cada qual executa funções muito específicas. Em geral, os varejistas desejam aproveitar os aspectos de gerenciamento de varejo do Microsoft Dynamics 365 for Operations, mas não querem substituir o sistema financeiro ou de RH existente. Para esses varejistas, é necessária uma carga de trabalho específica de varejo. Os fundamentos de comércio oferecem esses recursos específicos de varejo como uma opção de configuração do Microsoft Dynamics 365 for Operations. Quando a configuração é concluída, todos os aspectos da solução que não são de varejo são ocultos da exibição, destacando o que é importante: gerenciar uma organização de varejo. Os fundamentos de comércio foram apresentados primeiro como uma opção de configuração no Microsoft Dynamics AX 2012 R3 CU8. Com o tempo, houve muitas melhorias para torná-los uma solução para varejistas de omnicanal, e não apenas para organizações tradicionais.

## <a name="configuration"></a>Configuração
Para configurar os fundamentos de comércio, vá para **Administração de sistema** &gt; **Configuração** &gt; **Configuração de licença** e role para baixo até **Varejo**. Expanda o nó **Varejo** e marque a caixa de seleção de **Fundamentos de comércio**. Clique em **Salvar**, examine as alterações e atualize a página. Para alternar para a solução total, vá para **Fundamentos de comércio** &gt; **Configuração da sede** &gt; **Configuração de licença** e role para baixo até **Varejo**. Expanda o nó **Varejo** e desmarque a caixa de seleção de **Fundamentos de comércio**. Clique em **Salvar**, examine as alterações e atualize a página.

## <a name="what-is-included"></a>O que é incluído?
Os fundamentos de comércio incluem todos os recursos de varejo back office que são criados na solução de varejo principal do Dynamics 365 for Operations, exceto recursos do call center. Qualquer recurso de canal, trabalhador, produto, loja, registro e gerenciamento de dispositivo é incluído em um único nó de navegação. Além disso, todos os espaços de trabalho de varejo são incluídos para fornecer logo páginas de aterrissagem específicas de função. Os fundamentos de comércio se destinam a varejistas que têm um cenário tradicional ou móvel em que podem se beneficiar de Retail Modern POS (MPOS) e de canais de comércio eletrônico. Ao usarem os fundamentos de comércio, os varejistas podem gerenciar esses canais em um único local e, assim, obter verdadeiros recursos de omnicanal.

## <a name="how-is-commerce-essentials-different"></a>Qual é o diferencial dos fundamentos de comércio?
Além de oferecer páginas de navegação unificada e apenas de varejo, os fundamentos de comércio ocultam todos os elementos de página que não são de varejo, como controles e campos. O resultado é uma interface de usuário simplificada que ajuda fornecedores a se concentrarem na tarefa à mão. Os varejistas com uma licença completa do produto podem usar fundamentos de comércio como um ponto de partida para usuários mais orientados ao varejo. Outros elementos, como recursos avançados de RH, podem ser adicionados a fundamentos de comércio ao serem requisitados, de acordo com as condições do contrato de licença em vigor. Como parte da simplificação oferecida pelos fundamentos de comércio, o recurso de exportação de contabilidade permite que varejistas mapeiem os totais que são acumulados em fundamentos de comércio par aos números de conta no sistema contábil existente. À medida que o sistema de varejo gera dados financeiros, esses dados podem ser facilmente exportados de fundamentos de comércio para o sistema contábil do varejista. Se forem necessários recursos de integração mais avançados, os fundamentos de comércio também incluirão a estrutura completa de importação/exportação que será incluída no Dynamics 365 for Operations. As diversas entidades de dados que são incluídas por padrão dão aos fundamentos de comércio a flexibilidade para se ajustar a um ambiente de várias camadas onde diversos sistemas especializados são usados.
