# <a name="commerce-essentials-overview"></a>Visão geral de fundamentos comerciais

Este artigo fornece informações sobre o comércio fundamentos, um ponto de venda manager (POS) e opção de configuração comércio eletrônico para Microsoft Dynamics 365 para as operações. Os fundamentos de comércio fornecem uma carga de trabalho de varejo da saída de caixa que podem ajudar fornecedores a entrar no ar rapidamente com a solução de Retail com base no Microsoft Dynamics e permite que eles foquem nas funcionalidades específicas de Retail. 

<a name="overview"></a>Visão Geral
--------

Os ambientes de software empresarial costumam incluir vários sistemas, e cada qual executa funções muito específicas. Frequentemente, os varejistas desejam que tem as vantagens os aspectos do management do Microsoft Dynamics 365 para operações, mas não deseja substituir a existência sistema (HR) de recursos humanos. ou financeiros Para esses varejistas, é necessária uma carga de trabalho específica de varejo. Fundamentos os de comércio anotam esses recursos específicos varejo- como uma opção de configuração para dynamics 365 para as operações. Quando a configuração é concluída, todos os aspectos da solução que não são de varejo são ocultos da exibição, destacando o que é importante: gerenciar uma organização de varejo. Os fundamentos de comércio foram apresentados primeiro como uma opção de configuração no Microsoft Dynamics AX 2012 R3 CU8. Com o tempo, houve muitas melhorias para torná-los uma solução para varejistas de omnicanal, e não apenas para organizações tradicionais.

## <a name="what-is-included"></a>O que é incluído?
Fundamentos os de comércio incluem todos os recursos de varejo back office que são criados em dynamics 365 principal para a solução de varejo de operações, exceto de recursos. call center Qualquer recurso de canal, trabalhador, produto, loja, registro e gerenciamento de dispositivo é incluído em um único nó de navegação. Além disso, todos os espaços de trabalho de varejo são incluídos para fornecer logo páginas de aterrissagem específicas de função. Fundamentos os de comércio destinam fornecedores que têm um cenário físicas almofariz ou móveis que possa se beneficiar de canais modernos de POS (MPOS) e eletrônico de comércio. Ao usarem os fundamentos de comércio, os varejistas podem gerenciar esses canais em um único local e, assim, obter verdadeiros recursos de omnicanal.

## <a name="how-is-commerce-essentials-different"></a>Qual é o diferencial dos fundamentos de comércio?
Além de oferecer páginas de navegação unificada e apenas de varejo, os fundamentos de comércio ocultam todos os elementos de página que não são de varejo, como controles e campos. O resultado é uma interface de usuário simplificada que ajuda fornecedores a se concentrarem na tarefa à mão. Fornecedores com uma licença plena de produto podem usar fundamentos comerciais como ponto de partida para varejo- orientados mais usuários. Outros elementos, como recursos avançados, de hora podem ser adicionados a fundamentos comerciais como são necessários, conforme as condições do contrato de licença que está em vigor. Como parte da simplificação oferecida pelos fundamentos de comércio, o recurso de exportação de contabilidade permite que varejistas mapeiem os totais que são acumulados em fundamentos de comércio par aos números de conta no sistema contábil existente. À medida que o sistema de varejo gera dados financeiros, esses dados podem ser facilmente exportados de fundamentos de comércio para o sistema contábil do varejista. Se recursos mais avançados de integração são necessários, os fundamentos comerciais também incluem a estrutura total de exportação/importação que é incluída em dynamics 365 para as operações. As diversas entidades de dados que são incluídas por padrão dão aos fundamentos de comércio a flexibilidade para se ajustar a um ambiente de várias camadas onde diversos sistemas especializados são usados.
