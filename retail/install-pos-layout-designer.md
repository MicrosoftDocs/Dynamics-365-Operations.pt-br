# <a name="install-the-retail-pos-layout-designer"></a>Instalar o designer do layout de PDV

[!include[banner](includes/banner.md)]


Você pode usar o designer de um clique para criar moderno de POS (diferente MPOS) e para nublar-se layouts POS, no modo de retrato ou em modo de retrato, para lojas, registradoras, caixas, gerentes e.

O design gráfico da interface do MPOS ou Nuvem PDV é controlado pelo layout da gaveta. O layout controla a posição de vários objetos. Exemplos incluem o layout do total, o layout da grade de itens, o layout do cliente, o layout de pagamento e o layout de vários botões de menu. Os layouts também incluem a aparência geral da interface de vendas que é apresentada aos trabalhadores.

## <a name="install-the-one-click-designer"></a>Instalar o designer de um clique
1.  No Microsoft Dynamics 365 for Operations, use o menu na parte superior esquerda para navegar para **Retail** **e comércio** &gt; **Configuração de canal** &gt; **Configuração de PDV** &gt; **PDV** &gt; **Layouts de tela**.
2.  Selecione qualquer de layout que tenha um tipo de **POS para O Windows moderno** aplicativo **POS do nuvem** ou, clique em **Designer do layout**.
3.  Na barra de notificação exibida na parte inferior da janela do Internet Explorer, clique em **Abrir** para começar a instalação do designer de um clique. (A barra de notificação pode aparecer em um local diferente em outros navegadores).
4.  **Solicitação executada - aviso de segurança** Na caixa de mensagem que aparece, clique **Entrega**para instalar o host varejista designer. O indicador de progresso mostra o progresso do processo de instalação.
5.  Depois que a instalação for concluída, na página **Entrar**, insira seu nome de usuário e senha do Microsoft Dynamics 365 for Operations, e clique em **Entrar** para iniciar o designer.
6.  Depois que suas credenciais forem validadas e o designer for iniciado, você poderá começar a criar o formato de recibo ou modificar um formato existente. [![Designer de layout em um clique](./media/screenlayoutdesign_mposdownload-1024x664.png)](./media/screenlayoutdesign_mposdownload.png)

## <a name="troubleshoot-the-installation-of-the-layout-designer"></a>Solucionar problemas da instalação do designer do layout
-   Quando você clica **Designer**, o prompt para baixar ou execução (a) o installer não aparecem, nem as configurações de segurança atuais não permite baixe o arquivo. **Soluções:**
    -   Internet Explorer, verifique se o bloqueador de pop-ups será desabilitado para esse local. Clique em **Configurações** &gt; **Opções** &gt; **Privacidade** &gt; **Localizar bloqueador de pop-up** e alterar a configuração, se uma alteração for necessária.
    -   Internet Explorer, adicione o dynamics 365 para a URL de operações para seus sites confiáveis. Clique **Configurações** &gt; **Opções** &gt; **Segurança** &gt; **Sites de confiança** &gt; **Sites** &gt; **Adicionar**.
-   O programa não for iniciado, e você é orientado para contatar o fornecedor. **Solução:** Internet Explorer, adicione o dynamics 365 para a URL de operações para seus sites confiáveis. Clique **Configurações** &gt; **Opções** &gt; **Segurança** &gt; **Sites de confiança** &gt; **Sites** &gt; **Adicionar**.

**Problema):** O designer não funcionará corretamente em browsers de Google Chrome e Mozilla Firefox. Estamos trabalhando para solucionar o problema.
