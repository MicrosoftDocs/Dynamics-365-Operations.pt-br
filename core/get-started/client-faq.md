# <a name="dynamics-365-for-operations-client-faq"></a>Dynamics 365 para o FAQ de cliente de operações

Este artigo de perguntas frequentes sobre o Microsoft Dynamics 365 para o cliente de operações.

<a name="why-arent-symbols-loaded-when-i-use-dynamics-365-for-operations"></a>Por que os símbolos não é cobrado ao uso ao 365 para operações?
-----------------------------------------------------------------

As configurações de segurança em seu navegador podem impedir que os símbolos sejam carregados corretamente. Para resolver esse problema, siga as etapas a seguir:

-   Se você estiver passando esse problema Internet Explorer, clique ** ** ferramentas, clique em opções da Internet ** **.  Na caixa de diálogo opções da Internet, privacidade ** ** na guia, clique ** costume nível **, certifique-se ** e de download de fonte ** a opção selecionada.
-   Se não, pode ser necessário adicionar ao 365 para o site de operações a lista de sites de confiança.

## <a name="i-miss-the-ribbon-from-dynamics-ax-2012-can-i-keep-action-pane-tabs-open-all-the-time"></a>Eu a falto fita do dynamics AX 2012. Eu pode manter as guias do painel de ações abre qualquer tempo?
Planejando estamos para implementar logo este recurso. Os usuários poderão escolher para mantê-lo tempo dos guias painéis ações abertas. Caso contrário, as guias serão recolhidas quando não estiverem sendo usadas, para obter mais espaço de tela para a página.

## <a name="why-do-i-sometimes-see-different-shortcut-menus-when-i-rightclick"></a>Por que eu consulto ocasionalmente menus de atalho diferentes ao rightclick?
Se você clicar com o botão direito do mouse em um campo editável (ou se o texto for selecionada), o menu de atalho do navegador será exibido. Esse menu fornece acesso aos comandos **Recortar**, **Copiar**, e **Colar**. Nós poderemos não inserir esses comandos em dynamics 365 para menus de atalho operações como, por motivo de segurança, o browsers não permitem que os acessem forma programática a área de transferência do sistema.

Se você clicar com o botão direito em um rótulo de campos ou o valor de um controle somente leitura, você verá ao 365 para o menu de atalho de operações.

Para facilitar o acesso do teclado, nós planejamos para implementar futuramente um atalho do teclado que abre o dynamics 365 para o menu de atalho de operações.

## <a name="where-is-the-view-details-functionality-in-dynamics-365-for-operations"></a>Qual é a funcionalidade de exibir detalhes em dynamics 365 para operações?
A opção **Exibir detalhes** está disponível de algumas formas:

-   Se um controle tiver os recursos de **Exibir detalhes **, e se o controle tiver um valor, esse valor será exibido como um hiperlink. Você pode clicar no hiperlink para abrir uma página que contém detalhes adicionais.
-   ** Exiba detalhes ** também é uma opção em dynamics 365 para menus de atalho de operações. Para obter mais informações sobre quando o dynamics 365 para menus de atalho de operações é exibida quando você clicar com o botão direito do mouse, consulte a seção anterior.
