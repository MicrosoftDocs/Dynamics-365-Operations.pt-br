# <a name="create-record-templates"></a>Crie modelos de registro

Este artigo introduz o conceito de modelos de registro e explica como podem ser usados para criar registros que compartilham informações.

Os modelos de registro ajudam a criar rapidamente mais registros no Microsoft Dynamics 365 para as operações. Você pode criar modelos de registro para somente alguns de registro no Microsoft Dynamics 365 para as operações. Por exemplo, o **** imagina que você está inserindo informações alugado para uma empresa de contratação carros é de que está localizado em San Francisco. Como é provável que a maioria dos clientes sejam da área de San Francisco, seria interessante já preencher automaticamente os valores dos campos **Estado**, **País** e **Cidade** no formulário de aluguel. ** Observação: ** É possível aplicar modelos somente para as áreas do 365 para as operações às quais você tem acesso. No entanto, todos os títulos de modelo são visíveis para você ao criar um novo registro, e também para outros usuários, se você está criando modelos que serão disponibilizados para todos os usuários. Considere isso ao nomear modelos. Por exemplo, evite o uso de nomes que incluam palavras como "comissão" se for confidencial que alguns funcionários da empresa recebem salários com base em comissões. Quando um ou mais modelos aos quais você tem acesso existem para um formulário específico, e você tenta criar um novo registro no formulário, a página **Selecione um modelo para** é exibida. Quando você seleciona um modelo da lista, o novo registro é criado e contém as informações padrão baseadas no modelo que você selecionou. Se não deseja usar modelos ao criar novos recordes, não marque ** parte novamente ** a caixa de seleção ** selecione um modelo para ** na página. Para exibir novamente a caixa de diálogo de seleção de modelo, clique com o botão direito em qualquer registro, clique ** ** informações de registro, e clique em mostrar ** a seleção de modelo **.
