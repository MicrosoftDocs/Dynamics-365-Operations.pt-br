# <a name="configure-address-books"></a>Configurar catálogos de endereços

[!include[task guide banner](../../includes/task-guide-banner.md)]

Use esse procedimento, e as decisões feitas no tópico Planejando a configuração do catálogo de endereços global e de catálogos de endereços adicionais, para configurar catálogos de endereços adicionais para sua organização.

A empresa de dados demo usada para criar esta tarefa é USMF. Esse registro é destinado aos membros da equipe de Planejamento e configuração.


## <a name="configure-address-books"></a>Configurar catálogos de endereços
1. Vá para Administração da organização > Catálogo de endereços global > Catálogos de endereços.
2. Clique em Novo.
3. No campo Nome, digite um valor.
4. No campo Descrição, digite um valor.
5. Clique em Salvar.
6. Na lista, localize e selecione o PDV desejado.
7. Clique na seta para adicionar os times disponíveis selecionados ao catálogo de endereços.
8. Clique em Salvar.
