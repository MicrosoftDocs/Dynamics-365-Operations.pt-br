# <a name="deliver-an-employee-benefits-program"></a>Entregar um programa de benefícios do funcionário

[!include[task guide banner](../../includes/task-guide-banner.md)]

Essa tarefa mostrará como criar elementos de benefício que será usado ao criar um novo benefício. A empresa de dados demo usada para criar esta tarefa é USMF. Esta tarefa está direcionada para uma compensação e beneficia o gerente.


## <a name="create-benefit-elements"></a>Criar elementos do benefício
1. Essa tarefa parte da página de listagem de benefícios. Inicie a tarefa com essa página ou pesquisando a página de listagem de benefícios.
2. Clique em Novo.
3. No campo Tipo, digite o nome do tipo de benefício que está sendo criado.
4. No campo Descrição, digite um valor.
5. No campo Inscrição de concorrente, selecione uma opção.
    * Para restringir a capacidade dos funcionários de inscrever-se em vários planos médicos, selecione um registro por tipo.  
6. No campo Categoria da folha de pagamento, selecione uma opção.
7. Clique na guia Planos.
8. Clique em Novo.
9. No campo Plano, digite um valor.
10. No campo Descrição, digite um valor.
11. No campo Tipo, clique no botão suspenso para abrir a pesquisa.
12. Na lista, localize e selecione o registro desejado.
13. Na lista, clique no link na linha selecionada.
14. No campo Impacto da folha de pagamento, selecione uma opção.
15. Clique na guia Opções.
16. Clique em Novo.
17. No campo Opção, digite um valor.
18. No campo Descrição, digite um valor.

## <a name="create-a-benefit"></a>Criar um benefício
1. Feche a página.
2. Ir para Recursos humanos > Benefícios > Benefícios.
3. Clique em Novo para abrir a caixa de diálogo suspensa.
4. No campo Plano, clique no botão suspenso para abrir a pesquisa.
5. Na lista, localize e selecione o registro desejado.
6. Na lista, clique no link na linha selecionada.
7. No campo Opção, clique no botão suspenso para abrir a pesquisa.
8. Na lista, localize e selecione o registro desejado.
9. Na lista, clique no link na linha selecionada.
10. No campo Efetivo, insira uma data e hora.
11. Clique em Criar benefício.
12. Alternar a expansão da seção Detalhes da folha de pagamento.
13. No campo Frequência, clique no botão suspenso para abrir a pesquisa.
14. Na lista, localize e selecione o registro desejado.
15. Na lista, clique no link na linha selecionada.
16. No campo Base, selecione uma opção.
17. No campo Valor ou taxa, insira um número.
